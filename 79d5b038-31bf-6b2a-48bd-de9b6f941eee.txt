<br/>				一时间，她沉默着，静坐着，眉头紧拧。<br/>　　<br/>　　“凤公子？凤公子？”<br/>　　<br/>　　见凤九静坐着，城主连唤了两声，直到见他缓过神来，才道：“不知凤公子还有没其他想问的？若是有我定知无不言。”<br/>　　<br/>　　闻言，凤九看了他一眼，道：“既然这样，那，我想问一下城主，能否让我在城主府的藏书楼中查查资料呢？”<br/>　　<br/>　　听到这话，凌城主微顿：“这个……”<br/>　　<br/>　　他有些迟疑地看了看凤九，又看了看他儿子，最后才道：“只要凤公子同意解开契约，那么，我可以让凤公子在我们城主府的藏书楼中待三天，只不过，三天之后，无论凤公子有没找到你所想要的资料，都得离开。”<br/>　　<br/>　　一旁的凌天羽见他父亲为了他竟答应了这要求了，不由的动了动嘴，想说话却又不知说什么好。难道要开口拒绝？然后让他给这个凤九当奴隶吗？<br/>　　<br/>　　想到这，他咬了咬牙，目光不善的盯着那一袭红衣的凤九一眼，冷哼一声，别开了头。<br/>　　<br/>　　凤九也没去理会那凌天羽，而是在听到城主的话后，唇角微勾笑了起来：“好，可以。”<br/>　　<br/>　　于是，城主以及那凌天宇两人在凤九的面前许下了天地誓言，承诺只要凤九解开奴隶签约，他们不会追究，而且也不会派人暗杀他，找他麻烦。<br/>　　<br/>　　最后，凤九便将凌天羽眉心之处的奴隶印记抹去，解开了两人之间的契约。<br/>　　<br/>　　见此，城主轻松了一口气，露出笑容来：“凤公子，今晚就在这里休息吧！明天一早我再让人带你过去藏书阁。”<br/>　　<br/>　　“如此便打扰了。”凤九朝城主行了一礼，礼数也多了起来。<br/>　　<br/>　　“不打扰。”城主笑了笑，唤来了管家，让他带凤九去休息。<br/>　　<br/>　　待凤九离开后，那凌天羽来到他父亲身边，闷声问：“父亲，难道就这样算了吗？”<br/>　　<br/>　　负着手而立的城主睨了他一眼，黑沉着的脸色带着怒火：“若不是你成天在外胡作非为，惹下祸端，我堂堂一城之主何需向人低声下气相求？还允他在藏书楼中看书三天？这些都是你惹出来的麻烦，如今连不找他麻烦的话都说，怎么？你还想反悔不成？”<br/>　　<br/>　　他黑着脸盯着他，沉声说着：“我警告你，这段时间给我乖乖在家中修炼，若是让我知道你又跑出去胡作非为，看我不打断你的腿！”警告的话语落下后，凌城主衣袖一甩，迈步往外走去，留下凌天羽一人独自在厅中站着。<br/>　　<br/>　　凤九被安顿在城主府的一处院落中，如同客人般住了下来，只不过城主府里的人都知道，是他们少城主输给了他，才让城主将人请到家中来的。<br/>　　<br/>　　若非有城主在，估计这少城主就得给他当奴隶了。想到这红衣男子年纪轻轻实力居然这么强，城主府中明处暗处的那些人都在暗暗的打量着他。<br/>　　<br/>　　尤其城主府中的侍女，更是看他容颜俊美，气质出众，不由得猜测着，这位红衣公子究竟是什么来头？竟能让他们少城主栽了个大跟头，到最后还能被城主奉为座上宾，这手段当真是不得了。<br/>
