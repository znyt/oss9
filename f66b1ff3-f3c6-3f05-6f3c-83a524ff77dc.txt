　　消化掉了鸡冠蛇的全部记忆，涂小安全然没有了吞噬一条觉醒蛇应该有的开心。
<br/>　　如此忠心的一条宠物蛇，被自己吞掉，实在可惜了。
<br/>　　这条蛇身上是一段励志的故事，为了能给自己死去的主人报仇，它开始修炼，从一条无毒家蛇，变成了可怕的变异鸡冠王蛇，拥有可怕的手段。
<br/>　　一天杀一人，六天六人，其实这样的杀戮手段在看完鸡冠蛇的记忆后，涂小安觉得是相当的温柔了。
<br/>　　充斥着如此满腔仇恨，花了整整五十年修炼，就算一口气将赵家满门杀之，又算的了什么。
<br/>　　那惨死的主人，跟她腹中的孩子，何其之冤，赫然是人间悲剧。
<br/>　　无人伸冤，无人帮其昭雪，唯有这条蛇想为自己的主人做点什么。
<br/>　　而即使如此，鸡冠蛇还遵守着跟自己主人生前的约定。
<br/>　　它煞费苦心的逼着赵老爷子自杀，它本来可以成功的。
<br/>　　只是被自己给破坏了。
<br/>　　此时的涂小安万分的歉疚。
<br/>　　这是一条善良的蛇，对主人有情有义。
<br/>　　如果有那个可能，涂小安愿意将鸡冠蛇吐出来，他宁愿自己的系统任务失败，宁愿不要那些奖励，也让这条蛇做它所做，报仇雪恨。
<br/>　　可惜，木已成舟，一切枉然。
<br/>　　“叮...恭喜宿主成功吞噬觉醒类蛇种鸡冠王蛇，等级直接提升两级，第一阶段进化开始...”
<br/>　　霍..．
<br/>　　宅院之中的金银蛇全身骤然笼罩这一股炽热的白光，白光将他缓缓的从地面托起，悬浮在半空中。
<br/>　　白光包裹着金银蛇，将他变的神圣不可侵犯，圣洁无比。
<br/>　　此时的涂小安感觉这股白光疯狂的改造自己的身体，将鸡冠蛇的蛇族血脉跟他彻底融为一体。
<br/>　　将他的血统变的更加的高贵不凡。
<br/>　　他的实力再次得到质的提升，全身充满了用之不尽的力量，身体强度，意识力全面提升。
<br/>　　舒服，愉悦，身体此时的感觉棒极了，鸡冠蛇的血脉让涂小安有种第二次发育的机会，他能明显感受到自己身体的细微变化，身体喀嚓作响，仿佛有一种挣脱囚笼的快感。
<br/>　　但是这种感觉很难形容出来。
<br/>　　有种鲤鱼跃龙门的提升。
<br/>　　都说蛇化蛟，蛟化龙，仿佛化成龙，才是一条蛇最好的归宿。
<br/>　　但既然为蛇，为何偏要化为龙，任何生物，当自身的血脉不断的提升，实力暴涨，依然可以俯视一切，唯我独尊。
<br/>　　天上地下，不是只有龙才可以睥睨天下，他这条蛇，也可以让万物臣服。
<br/>　　龙能大能小，能升能隐，他同样可以做到。
<br/>　　如果问一条蛇，你最大的愿望是什么？
<br/>　　它回：“我要变成龙”
<br/>　　不过小志也，若此时问涂小安，你最大的愿望是什么。
<br/>　　“我要吞龙”
<br/>　　蛇吞万物，吞龙才爽，才是蛇王应该具备的霸气。
<br/>　　龙是高高在上的，蛇不过是一条爬虫，逆袭才是王道。
<br/>　　如果这个世界上真有龙的存在，那他便做吞噬龙的毒蛇，岂不快哉。
<br/>　　昂...
<br/>　　被炽热白光包裹着的金银蛇发出了一声长啸声，无形之中，一股俯视一切的王者之气激荡开来。
<br/>　　霎时间，方圆十公里内蛰伏的蛇类全部瑟瑟发抖，诚惶诚恐，匍匐在地，像是迎接自己的王降临。
<br/>　　“叮...恭喜宿主的等级成功提升到十七级”
<br/>　　“叮...恭喜宿主彻底融合鸡冠王蛇血脉，完成第一阶段的进化，现在赋予雷电属性能力”
<br/>　　劈...
<br/>　　一道闪电在涂小安的脑海中一闪而过，像一把利剑插入了他的心中，顿时他觉得自己多了一种神奇的力量。
<br/>　　“叮...雷电能力赋予成功”
<br/>　　倏然，涂小安掀开眼，一双蛇眸精光四射，眸中蕴含着一道雷电，像要挣脱而出。
<br/>　　他金银色条纹上此时电光流转，空气中滋滋的爆响，一条一条闪电如银蛇窜出，惊雷显化，在他的周身缭绕窜动，可怕的异象迸发。
<br/>　　空间仿佛要被撕裂，充斥的雷电白光，连夜晚那皎洁的月光都不敢媲美。
<br/>　　赫然，这使涂小安成为了一条真正掌控雷电的蛇王，周身一道道惊雷闪烁，将赵家宅院照的宛如白昼。
<br/>　　噼里啪啦的惊雷声不绝于耳。
<br/>　　这个夜晚，对于赵家人来说，各种离奇的事情已经让他们麻木，颠倒三观。
<br/>　　他们感觉现在任何事情的发生，都不会大惊小怪了。
<br/>　　可此时，一个个瞠目结舌，目瞪口呆，撑着难以置信的眼神望着悬浮在半空，四处惊雷闪烁的金银蛇身上。
<br/>　　一条蛇沐浴在雷电之中，一股威严俯视一切的气势让人硬生生地升起一种想要顶礼膜拜的冲动。
<br/>　　“蛇王，是蛇王显化，都给我跪下”
<br/>　　赵老爷子颤栗着佝偻的身子，从轮椅上滚了下来，朝着半空雷电异象缭绕的金银蛇膜拜。
<br/>　　此时哪里还用赵老爷子多说什么，白镇人民信仰的蛇王显化了，这阵仗让人不禁俯首称臣。
<br/>　　“蛇王万岁，蛇王万岁”
<br/>　　赵家人疯狂的在地上磕头，蛇王降临，除恶扬善，帮赵家人手刃邪蛇，拯救他们一家与死亡边缘。
<br/>　　鸡冠蛇何等厉害，简直就是蛇妖，最后被这条来历不明的蛇打败了，还吞了，原来它是蛇王。
<br/>　　这便是最好的证明了。
<br/>　　唯有蛇王才能如此神通，对蛇王的信仰与崇拜，白镇人民首当其冲。
<br/>　　别说的白镇人民，就算是普通人，看到上空那条周身缭绕惊雷异象的蛇，那睥睨的气势，威严不凡的样子，不经意的也要臣服。
<br/>　　什么是蛇王，为什么蛇王那么受白镇人民信仰跟崇拜，因为蛇王是庇佑他们的神灵。
<br/>　　赵家家蛇变邪蛇，不断死人，蛇王便降临了，惩戒邪蛇。
<br/>　　亲身经历呀。
<br/>　　白镇人民什么时候开始信奉崇拜蛇王，这点不可追溯了，太过遥远，但蛇王经常会显化，救黎民于水火，这点早有传颂。
<br/>　　传说，蛇王修炼得道在佘山，荫庇一方，期初是统治蛇族一脉，不让毒蛇肆意的伤人，古代有一年，白镇地区发生了可怕的大霍乱，死了很多乡民，后来人们祈求蛇王出世。
<br/>　　次日，突然一条庞大的蛇影显化在虚空之上，口吐焰火，驱除了瘟疫。
<br/>　　乡民得救，后立庙奉蛇王为菩萨，从此香火不断，并于每年农历7月7日举行蛇王节，以为纪念关。
<br/>　　至今，白镇依然在每年农历的七月七，举行蛇王节，蛇王雕像更是挨家挨户都有。
<br/>　　赵家宅院蛇王显化，风云搅动，惊雷不断，不单单是赵家，整个赵宅村的村民无一不是跪在地上，磕头许愿。
<br/>　　蛇王显化，百年难得一遇，这是大吉之兆呀。
