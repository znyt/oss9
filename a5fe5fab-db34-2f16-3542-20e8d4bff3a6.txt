<br/>				  第二天，陈默练完功夫从外边走进别墅的时候，就看到小姐姐正蹲在地上，逗弄两只小家伙。<br/>　　<br/>　　  昨天搬家，将求收养的一家三口都搬了过来，和大黄大灰一样，虽然对这里还有些不适应，不过见到陈默也一起过来后，也就无所谓了。只要饭票在，哪里都一样。这是小赤看着陈默的目光，对此，陈默也是无语的。<br/>　　<br/>　　  至于大黄和大灰两个家伙，陈默在哪里它们就在哪里，不存在环境是不是适应的问题。<br/>　　<br/>　　  小姐姐半蹲在地上，从后面望去，真的是成标准的葫芦形身材，黄金比例标准，看的陈默一阵眼热。<br/>　　<br/>　　  “这两只小狗狗是什么品种？”听到身后大门响，就知道陈默回来了，所以回头疑问道。<br/>　　<br/>　　  陈默除了汗以外，真的是无语了！这两个小家伙哪里看着像小狗了？这是狐狸好不好？<br/>　　<br/>　　  不过看上去，小一小二这两个小家伙，可能是来到陈默这里，吃的好睡的好，所以身材胖了不少。<br/>　　<br/>　　  本来是倒三角的脸型，竟然看上去有些呈圆形了，肥嘟嘟的小身材，相得益彰的粗尾巴，一眼看上去，还真的有些像是小狗狗的说。转头看小赤，也是一样，尤其是脸型，到三角的脸，都有些变形。<br/>　　<br/>　　  看到这样的一家三口，真的有些哭笑不得。自己是不是对它们太好了，整天的吃吃吃!结果一家三口都变胖了不说，也聪明了很多。<br/>　　<br/>　　  要知道，它们平时吃着乾坤珠内的鱼肉不说，还喝着溪水，真的能让这些小动物变聪明很多。<br/>　　<br/>　　  小一和小二，见陈默走了进来，立刻移动着圆~滚滚的身子，跑到陈默的腿边求抚摸。<br/>　　<br/>　　  小赤也从窝里走了出来，半蹲着看着陈默。刚才小姐姐在逗弄它的两个孩子的时候，它可是没有出来的。<br/>　　<br/>　　  虽然有些紧张自己的孩子，但是从小姐姐的身上没有感觉到什么危险，也就随她去了。<br/>　　<br/>　　  “它们是狐狸！”看着两个小家伙萌萌的，只能伸手摸了摸两个家伙，安慰了一下它们之后，对小姐姐说道。<br/>　　<br/>　　  “什么？它们是狐狸？”小姐姐有些半张嘴巴，很是好奇！<br/>　　<br/>　　  “有这么肥的小狐狸么？”小姐姐再次蹲下，盯着两只小狐狸。<br/>　　<br/>　　  “现在不就是了，看看它们！”陈默笑呵呵的也一起蹲下，然后将小一和小二都抱了起来，两只小狐狸在陈默的手里，非常的安静，一动不动。不过大大的眼睛却盯着陈默，似乎有些好奇被抱起。<br/>　　<br/>　　  “这个叫小一，就是下巴这里有些白的小家伙。这个是小二。”陈默一一将两只狐狸的名字给小姐姐介绍道。<br/>　　<br/>　　  “那它叫什么？”小姐姐又指着小赤问道。<br/>　　<br/>　　  “小赤！”<br/>　　<br/>　　  “哈哈哈哈!这是谁给它们起的名字啊，真是够衰的！不会是你吧！”小姐姐有些哈哈哈大笑的问道。<br/>　　<br/>　　  陈默汗颜：“还真是就是我起的！”对于叫小一小二，还有小赤的名字，他感觉挺好听的啊！怎么凡是听到这个名字的人，都要哈哈大笑呢？<br/>　　<br/>　　  “是你的风格！”小姐姐说道，并且使劲拍拍陈默的肩膀。当然并不是真的使劲，要不然凭借小姐姐武者的身份，她也担心自己将陈默给收拾了。至于说实际上能不能收拾掉陈默，可能就要另说了！<br/>　　<br/>　　  两个人一起吃了顿陈默做的早饭，很简单，就是点小米粥，两个凉菜和馍馍鸡蛋。不过虽然简单，但是沈婷婷却吃着很好吃，感觉有种小时候吃过饭的味道。<br/>　　<br/>　　  “你这里的伙食真好吃！”小姐姐吃完饭以后，夸奖的说道。<br/>　　<br/>　　  “你喜欢就好。中午给你做鱼和鸡，都是我这里养殖的，昨天晚上有些匆忙，没有准备。等会我让工人准备一下，给你做几个菜，让你尝尝。”陈默笑着说道。<br/>　　<br/>　　  “好！”小姐姐点头答应。从昨天晚上到现在，她舒服的都不想回西市了，住在这里真的非常舒心，并且还能吃到好吃的，真的是有些让人不想走。<br/>　　<br/>　　  吃完饭以后，由于陈默还要去工地转转，所以就邀请小姐姐一起。两个人也就并肩去转了转，并且今天陈默以前住的那个临时建筑，要被于经理带人过来改动一下。<br/>　　<br/>　　  陈默虽然不知道袁若珊那个时候将人送过来，但是却不能让人等房间不是。所以自己搬过去以后，对这个临时建筑就开始动手改变。施工要求也就几天的时间，必须加快，要不然病人到了，却没有住的地方，可不就搞笑了？<br/>　　<br/>　　  其实他可以打个电话与袁若珊商量一下的，但是他自己却不想这么做，主要是因为袁若珊按个母老虎，实在是太气人！<br/>　　<br/>　　  “这个地方你收拾出来要做什么？”小姐姐看到临时集装箱房子里，工人在施工隔墙，似乎是增加房间。<br/>　　<br/>　　  “当病房。”陈默说道。<br/>　　<br/>　　  “这种地方合适么？”在沈婷婷的概念中，病房不是那种无菌、干净卫生的房间么，怎么临时用房也能用？<br/>　　<br/>　　  “这种建筑用来做病房是没有任何问题的。不过也就是技术上的事情而已，尤其是细节上，要采用一些密闭空间，空气循环系统等等。在国际救援上，还有部队等等，都有这种建筑组成的医院。”<br/>　　<br/>　　  陈默倒是非常清楚，集装箱房式医院，在国际上都有通用的，不过不是他这里这种改建，而是专门按照标准尺寸制作的。<br/>　　<br/>　　  “我这里主要是疗养，不涉及其他治疗，所以就没有太高的卫生要求。”陈默说道。<br/>　　<br/>　　  “哦！原来如此！”小姐姐也听明白了。<br/>　　<br/>　　  途中，有碰到大海，就让他中午前，给他送点鱼和鸡子。鱼就要鳗鱼和鲈鱼，鸡子弄一只就成。<br/>　　<br/>　　  “没有问题，等一会我让工人厨房哪里将鸡子杀好以后，包括鱼也收拾好，就给你送过去。”大海挤眉弄眼的说道。<br/>　　<br/>　　  陈默知道他挤眉弄眼的是做什么，于是就白了一眼，撇撇嘴。对于大海的示意，他也解释不清，爱咋地就咋地吧！<br/>　　<br/>　　  等转了一圈回到别墅，就在这个院子的凉亭位置休息。<br/>　　<br/>　　  别墅这里的花园，可以说就大很多了。前院就有将近两亩大小，可以说陈默在别墅盖好以后，种植了不少的花花草草，还有个以前一样，这里挖了一个好大的鱼塘，呈不规则的长方形，凉亭就盖在鱼塘的上面，可以说说凉亭四周都是水，像是一个浮岛一样，在鱼塘的一角，通过连廊才能出去。<br/>　　<br/>　　  鱼塘很大很深，几乎有半米多深，里面不仅放了些锦鲤，陈默还弄了很多的荷花等水生植物，由于水里有陈默投入的溪水，无论是锦鲤和水生植物，都生长的非常好，可能在过几天，荷花就会开了。<br/>　　<br/>　　  人在凉亭坐着，看到四周不时有锦鲤游过，还别说真的有些情趣。<br/>　　<br/>　　  再者说了，这个院子还种植着很多果树和植物，可以说在凉亭中坐着，很是享受。<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
