　　而在赛后当叶帝他们和楚白衣几人握手时的对话传出去后，王哥、社会王再次登顶热搜，倒是引来了无数的乐子。
<br/>　　因为这周叶帝他们就一场比赛，所以在比赛结束之后，叶帝便是跑到了星娱公司那边，准备去看看他的歌曲准备的怎么样了。
<br/>　　所以在周日这天，叶帝在中午的时候便是来到了星娱公司，等他到这里的时候，林奇等人还在那里工作着。
<br/>　　“那两首曲子做的怎么样了？”看着几人，叶帝开口问道。
<br/>　　“第一首做完了，第二首也是到尾声了，你先听听，看看哪里有问题没。”林奇开口说道。
<br/>　　听着他的话，叶帝便是带上了耳机，在那里专心的听着这第一首曲子，还别说几人这专业的真不是盖的，居然一点瑕疵都没有。
<br/>　　“完美，一会请你们吃饭。”听过之后，叶帝笑着开口说道。
<br/>　　“曲子倒是不错，歌词也给我们看看啊，我现在都有些被你折磨的受不了了！”一旁的王沙跟着说道。
<br/>　　“等今天晚上吧，晚上我会在直播间里唱，到时候你们好好来听听。”叶帝笑着说道，坚决不把歌词给他们。
<br/>　　这么一首装逼的歌，自然是要唱出来才更有感觉，哥只是个传说，不知道今天晚上自己唱出这首歌，又要有多少人来骂自己了。
<br/>　　人生寂寞如雪啊，谁来赐我一败！叶帝仰头45度角，一脸的落寞。
<br/>　　看着叶帝的这个样子，一旁的林奇几人顿时一脸的无语，自己几人不过就是问了一个歌词而已，你至于这么装逼么，还来个45度角望天，你特么那忧郁的样子是给谁看的。
<br/>　　“不过你这第二首曲子我有点没听明白，完全听不懂什么意思。”一旁的孟建犹豫了一下后开口说道。
<br/>　　听着孟建的话，叶帝这边差点笑出声来，我的滑板鞋这么经典的曲子要是能让你们听明白，那你们岂不是也都是地球过来的了。
<br/>　　“嗯，这是我换风格的一首歌曲，还没有完善好，等过两天在直播的时候唱吧。”叶帝装模作样的说道。
<br/>　　等了大概半个小时，当几人将我的滑板鞋这首曲子最后的部分完成之后，叶帝听了一遍感觉没什么毛病之后，便是带着几人出去吃饭去了。
<br/>　　至于这几首曲子到时候他们用电脑直接传给自己就行了，反正自己只是用一个伴奏，至于歌曲相信到时候肯定会有无数人录制下来的。
<br/>　　叶帝完全有那个自信，他在直播的时候和在录音棚里唱的一样好，因为在这个世界他除了大魔王还有着另外一个绰号——行走中的cd！
<br/>　　吃过饭后，叶帝便是向着基地内返去，这一阵子基本上都是他一个人出行的，因为他相信就算是日本那边在想杀死自己，也肯定不敢在这个时候这样光明正大的动手。
<br/>　　甚至可能他们还会尽全力保护自己，以自己如今的影响力，只要自己被刺杀，那么那些粉丝还有很多人肯定会将这口锅放到日本的身上。
<br/>　　如若真的那样的话，日本的经济恐怕最少也要倒退数年，不要怀疑，这就是如今叶帝的影响力。
<br/>　　返回基地的时候已经是下午4点多了。
<br/>　　“咦？你们今天不是有比赛么，怎么回来的这么早。”看着在屋子内闲聊着的蒋念等人，叶帝有些诧异的问道。
<br/>　　“比完了啊。”蒋念理所当然的说道，不过那样子怎么看起来都有些装逼的味道。
<br/>　　“这速度也太慢了吧，1点开始比赛，现在才回来，要是我们，估计3点就到家了！”虽然心里确实感叹对方的速度快，不过这个时候叶帝怎么可能让蒋念继续把这个逼装下去。
<br/>　　听着叶帝那嘲讽的声音，屋子内的几人都是一脸的无语。
<br/>　　你特么刚刚还在那里诧异呢，结果瞬间就过来打人家的脸了，简直太不要脸了。
<br/>　　“无耻！”蒋念咬着牙有些气急败坏的说道。
<br/>　　刚刚他还真准备好了一大堆要装逼的话语呢，可是哪知道就这么被叶帝一句话直接给打回去了。
<br/>　　“你这装逼的功夫还太嫩了一点，怎么和我叶哥比，蒋念啊，你还是好好学着点吧。”香蕉拍了拍蒋念的肩膀，有些同情的说道。
<br/>　　“哎，本来还想听着蒋念装下逼呢，心疼我家小念念！”林萧笑着说道。
<br/>　　“滚滚滚，你才是小念念呢，你全家都是小念念。”听着林萧的话，蒋念心中别提多气了，连自家的队友都过来打击自己了，这真的没法活了。
<br/>　　看着蒋念那一脸郁闷的样子，基地内倒是一片欢声笑语，反正大家都是互相打击，今天要是不把蒋念好好打击一下，那等到蒋念下次嘲讽他们的时候，那他们岂不是会很亏。
<br/>　　“小叶子你今晚要直播么？”知道叶帝今天去弄歌曲去了，秦霄有些好奇的问道。
<br/>　　“嗯，今天晚上直播，两首歌曲的伴奏已经做出来了，不过今天就唱一首好了，细水长流啊，毕竟我的才华也是有尽头的。”叶帝开口说道。
<br/>　　“小念念看见没，叶哥这才叫真正的装逼，一个礼拜出创作出来两首歌，然后跟我们说江郎才尽，这个意境估计你是永远体会不到的。”听着叶帝的话，一旁的香蕉再次高山仰止，又拍了拍蒋念的肩膀开口说着。
<br/>　　“卧槽，你特么崇拜就崇拜，你拍我干什么啊！”蒋念感觉自己非常的受伤。
<br/>　　虽然他刚刚也看出来了叶帝确实是在装逼，但是自己刚刚被叶帝给打击了，这个时候怎么可能开口说话。
<br/>　　“我这不是在教你么，下次在叶哥面前不要装那么浅显的逼，要尽量装的有内涵，叶哥装逼从来都不是那种直接说出来的，他总是能够从侧面来烘托自己，你好好学着，下次别在被打击了。”香蕉不以为然，继续给蒋念上着课。
<br/>　　在众人闲聊声中，时间倒是过的很快，吃过晚饭后，各自又回到房间休息了一阵，等到时间到达7点后，众人都是准时的来到了训练室内。
