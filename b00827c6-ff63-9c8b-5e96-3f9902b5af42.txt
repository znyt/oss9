　　一天很快过去，黑幕降临，月光如水，当万籁具寂的时候。
<br/>　　一栋住所中一道光的人影射了出去，然后在古学院中游荡。
<br/>　　此人自然就是涂小安了，他心中惦记这那个诡异的湖泊，想去一探究竟。
<br/>　　昏暗的环境中，他朝着花圃后面的湖泊走去，此时接近凌晨了，学员们都会宿舍睡觉，校园内很安静。
<br/>　　走到湖泊面前的时候，一丝凄凉的笛声忽远忽近的从平静的湖面离奇的传荡上来，涂小安站在小湖面前，圣洁的月光散在他的身上，将他清秀的面容衬托的俊逸不凡。
<br/>　　这笛声还是这般的凄凉跟幽怨，仿佛像世人诉说这人世间最悲凉的苦楚。
<br/>　　涂小安认真的聆听这笛声想传递出来的意思，心中似明镜，他闭着双眼悄然的感觉到附近的一草一木。
<br/>　　他看到两只鸟儿在一棵树上交颈而眠，他看到一只老鼠在小湖阴暗处鬼鬼祟祟，天空有一只不知名的鸟拍着翅膀划过。
<br/>　　“意识力渗透”
<br/>　　涂小安强大的意识力投放在湖面，他要渗透进去，寻找笛声的出处。
<br/>　　可旋即，涂小安蹙眉起来，这湖面波光潋滟，微微荡漾，好像有一股神秘的力量笼罩，他的意识力既然无法渗透湖水之中。
<br/>　　莫非下面真有一个古阵法？
<br/>　　然后古阵法中镇压一头三阶妖灵吗。
<br/>　　涂小安胡思乱想，足足思忖了半晌，他释然了。
<br/>　　下水看看就行了，他这条蛇，还真的没下过水。
<br/>　　湖面底下，哪怕是龙潭虎穴，涂小安也有胆气走一遭，说不定有什么奇遇呢。
<br/>　　猫精海棠说湖底好像有一种似蛇似鱼的怪物，顺便看看是什么怪物。
<br/>　　注意已定，涂小安精光一闪，整个人纵深一跃，直接跳入了湖面之中。
<br/>　　旋即沉沦不见，也还好这一幕没被人看见，不然还以为他投湖自杀呢。
<br/>　　一入湖中，湖水冰凉无比，湖中光线很暗，涂小安目光如光如电，视力并没有受到什么太大的影响，不过也只能看到百来米范围内的景物。
<br/>　　在湖中，涂小安耳畔依稀还能听到若有似无凄凉的笛声，好像从四面八方传来，根本让人难以辨别那个方向。
<br/>　　涂小安只能不断的下潜，越是下潜，他暗暗的吃惊，这湖泊的底下好像深不见底，宛如一个深渊似的。
<br/>　　随着继续的下潜，这算是地下湖了，深水已经传来层层的压力，这些对普通人是致命的，但对于涂小安并无大碍。
<br/>　　足足下潜了几百米有余，地下湖中，涂小安眸光环视附近的景物，旋即他诧异了下，他看到了一个地下溶洞，很是神奇。
<br/>　　里面有钟乳石，石笋，石柱，石花，石幔，石瀑等岩溶堆积物，千姿百态，瑰丽神奇。
<br/>　　笛声，若有似无的笛声近了，近的就像眼前站着一个人在吹笛。
<br/>　　莫非是在这个地下溶洞之中？
<br/>　　看上去有点灵异，若不是涂小安现在对自己的实力太过自信，他还真不敢下水，这湖泊太过神秘了。
<br/>　　涂小安朝着地下溶洞游去，溶洞的旁边有一大堆的乱草在密密麻麻的摇曳着，跟一只只触手怪一样。
<br/>　　细看一眼，草，不对，这并不是乱草横生。
<br/>　　待涂小安看清楚，所谓的乱草既然是无数条水蛇在溶洞前触目惊心的舞动。
<br/>　　没错，蛇，当今世上，恐怕没有人比涂小安更认识蛇了，这在水中的自然是水蛇。
<br/>　　这些水蛇莫约一米，条纹灰褐色，并没什么过人之处，只是数量有点多。
<br/>　　地下溶洞旁边爬满了这些水蛇，里面甚至更多，好像这溶洞是一个水蛇的蛇窟。
<br/>　　也亏了是涂小安下水来到这里，要换做别的人，这密密麻麻如乱草舞动的水蛇，就足够让人心生绝望，绝不敢越雷池半步。
<br/>　　涂小安继续朝着溶洞游去，几乎不到十米的距离，这些水蛇仿佛一下子觉醒了，水中有嘶嘶作响的声音，一股脑，风卷残云的朝着他淹没过去。
<br/>　　其画面简直惊恐无比。
<br/>　　“哼，本蛇王在此，也敢放肆”
<br/>　　涂小安冷哼一声，一股蛇王气息自体内爆涌了出来，席卷出去。
<br/>　　倏然地，数量庞大的水蛇群一下子怔住了，发出惶恐不安的声音，根本无法承受涂小安的蛇王气息。
<br/>　　这些水蛇每条大概不过五六级的等级，如何受得了这登基坐殿的蛇王气息。
<br/>　　震慑住了水蛇群，涂小安顺利的潜游到了地下溶洞之中，一进其中，涂小安就看到不少造型各异的钟乳石，石柱或者一些石幔等。
<br/>　　两边岩壁有着一些美丽多变的岩画，是自然形成，没什么规律，但却上面爬满了蠕动的水蛇，有点恶心。
<br/>　　涂小安见溶洞也是深不见底，里面应该是凶险万分的。
<br/>　　这笛声，涂小安已经确定就从这个溶洞深处传出，就像是给涂小安的指路明灯，指引着他深入探险。
<br/>　　忽然，正在此时，一股强烈不安的感觉袭上了涂小安的心间，他感觉到了一股巨大的危机感，水流在急速的涌动起来。
<br/>　　他抬头望去，只见溶洞中一条似鱼非鱼，似蛇非蛇的水怪像箭一般的快速朝他袭来，水怪长约一丈，头生独角，蛇身鱼尾，正张着血盆大口向他咬来。
<br/>　　“卧靠”
<br/>　　涂小安惊了惊，急忙向旁闪去，水怪那锋利如剑的牙齿与他擦身而过，但它那如蛇一般的蛇尾却一下子缠上了他的腰腹。
<br/>　　在水中，涂小安做不到灵活自如，除非显出蛇形。
<br/>　　“这逼是什么怪物品种”
<br/>　　涂小安腰腹被水怪蛇尾缠住却并不多惊慌，一个意识洞察丢了过去。
<br/>　　“叮...鱼妖巨蛇，混合品种，等级十五级，吞噬可获得300进化能源”
<br/>　　鱼妖巨蛇？闻所未闻，恐怕只有在这地下湖深处可见吧。
<br/>　　区区十五级，既然长成这般巨大，足足三丈，等同十米了。
<br/>　　样子着实怪异吓人，难不成是鱼跟蛇的杂交？
<br/>　　管你是不是混合品种，但凡有蛇的基因在，它就是以下犯上，涂小安蛇王气息激荡出去。
<br/>　　“还不给我臣服”
<br/>　　“昂吼...”
<br/>　　鱼妖巨蛇的双眼幽然闪烁，看起来透着森森然的寒意，旋即，涂小安感觉缠在腰腹的蛇尾力量在加大，是准备将他绞杀死。
<br/>　　不受自己的蛇王气息影响？
<br/>　　这尼玛的到底是不是蛇呀。
