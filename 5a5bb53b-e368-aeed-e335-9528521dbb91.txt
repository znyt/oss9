　　苏莫离开了天浑第一斗场，换了一副面容之后，在城中闲逛了起来。
<br/>　　逛了一刻钟，他将太云神剑卖掉了，只卖了六万下品源石。
<br/>　　卖掉这柄神剑，主要是以后不能再使用了，不然的话，很容易被让你认出来。
<br/>　　当然，他又重新买了一柄下品神剑，品质和威力，比太云神剑更甚一分。
<br/>　　而后，苏莫没有引起任何人的注意，便离开了天浑虚界。
<br/>　　他走出了石碑，回到了八卫城。
<br/>　　石碑所在之地，人来人往，可谓是人山人海，根本没有人能注意到他。
<br/>　　而后，苏莫没有丝毫的停了，就准备离开八卫城。
<br/>　　不过，刚刚走到城门之前，他却是遇到了沁儿和褚彦等人。
<br/>　　“段惊天，你也在这里？”
<br/>　　沁儿眼尖，一眼便看到了苏莫，立刻轻呼了起来。
<br/>　　褚彦和骆妤、姬月松三人闻言，均是向苏莫望了过去。
<br/>　　“呵呵，路过此地，进城买件神器！”苏莫微微一笑，便大步的走到了沁儿身前。
<br/>　　“你有没有进天浑虚界，今日，天浑虚界出了一位厉害的妖孽？”沁儿问道。
<br/>　　“哦？是谁啊？”苏莫故作惊讶，疑惑的问道。
<br/>　　“此人自称冷无敌，也是一品低级修为，战力已然逆天！不过，不知道他的真实身份！”沁儿微微叹了口气说道。
<br/>　　“这么厉害？可惜无缘领教一番！”苏莫摇了摇头道。
<br/>　　“你省省吧！”
<br/>　　褚彦开口了，嗤笑一声道：“段惊天，就你还要领教一番，你虽然闯到了熔岩窟十八层，也是一品低级的修为，但是，你给人家提鞋都不配！”
<br/>　　褚彦心中郁闷，心情很不好，现在见到苏莫，顿时找到了发泄的地方。
<br/>　　他知道苏莫战力肯定也是超强，尤其是防御力强大，不然不可能闯到熔岩窟十八层，但是，即便如此，也不可能和冷无敌相提并论。
<br/>　　“是吗？那褚彦兄与此人切磋了？”苏莫仿佛不在意褚彦的讽刺，好奇的问道。
<br/>　　“我……！”
<br/>　　褚彦顿时语塞，一张脸阴沉了下来，随即他深深的吸了口气，道：“切磋了，我略逊一筹！”
<br/>　　苏莫心中暗笑，对方真是不害臊啊，明明是毫无反抗之力，居然被对方说成了略逊一筹。
<br/>　　“佩服！”
<br/>　　苏莫点了点头，随即看向沁儿，问道：“沁儿姑娘，你们这是要回人王宫吗？”
<br/>　　“是啊，你呢？”沁儿反问道。
<br/>　　“我再转转，过几日再回去！”苏莫道。
<br/>　　“沁儿，我们走吧！”
<br/>　　姬月松没有理会苏莫，催促了一句，便大步出城。
<br/>　　褚彦也没有再言，跟着姬月松，大步而去。
<br/>　　“段惊天，奉劝你一句，你以后少来找沁儿，不然在人王宫，你很难待的长久！”
<br/>　　骆妤脸色冷然，淡淡的对苏莫说了一句，便拉着沁儿远去。
<br/>　　“这……！”
<br/>　　沁儿有些无语，只能给苏莫一个抱歉的眼神，便跟着骆妤而去。
<br/>　　“骆妤姐，你干嘛这么不待见段惊天？”走到城外，追上了姬月松和褚彦，沁儿无奈的问向骆妤。
<br/>　　“沁儿，你身份特殊，不宜和一个外人果断亲近，而且，这段惊天还配不上你！”骆妤告诫道。
<br/>　　“大家只是同门而已！”沁儿苦笑着说道。
<br/>　　“你太单纯了，防人之心不可无！”骆妤道。
<br/>　　沁儿闻言，无语的摇了摇头，她也懒的和骆妤争论。
<br/>　　而后，四人身形冲天而起，向人王宫而去。
<br/>　　远远的，见到沁儿等人离开，苏莫也离开了八卫城。
<br/>　　他没有返回人王宫，而是向相反的方向而去。
<br/>　　“今日之事，估计会传到帝一魂的耳中！”飞行的天空之中，苏莫低声自喃，虽然他展露的不多，但是，传到帝一魂耳中的可能性，依旧很大。
<br/>　　苏莫不傻，他来到神域的消息，帝一魂定然已经知晓了。
<br/>　　那么，这几年的时间之中，帝一魂肯定在寻找他。
<br/>　　所以，今日的消息一旦传出，帝一魂肯定会有所猜测。
<br/>　　所以，他要暂时离开轮回神朝，出去转转。
<br/>　　当然，他并不是单纯的出去转转，而是要在其它地方，搞出一些动静。
<br/>　　以此，来分散帝一魂的注意力，这样的话，他藏身在人王宫就会安全很多。
<br/>　　苏莫向着霸天神朝飞了过去，一边修炼，一边赶路。
<br/>　　时间悠悠，苏莫耗费了一年多，才到达霸天神朝。
<br/>　　他在霸天神朝待了半个月，又去了天龙神朝，然后又去了天池神朝。
<br/>　　……
<br/>　　天王宫，一座金碧辉煌的宫殿，坐落在天王宫深处，清静而悠然。

<br/>　　蟒袍人没有停止修炼，声音从其口中传出，略显沙哑。

<br/>　　“还有呢？”蟒袍人继续问道，他知道，天尘宇既然来见他，必定有其它消息。
<br/>　　“不过几年，在神域各地貌似都出现了类似苏莫的人！”天尘宇道。
<br/>　　蟒袍人没有在开口，静静的听着。
<br/>　　少倾，天尘宇又道：“三年前，霸天神朝出现了一人，拥有吞噬之能，差点将两名霸天族人生生吞噬；两年前，天龙神朝出现一人，战力逆天，拥有混沌源力；一年前，天池神朝无双界域出现一人，拥有多种元神和至尊战体！”
<br/>　　说到此处，天尘宇凝视着蟒袍人，略一停顿，又道：“这些人，都有可能是苏莫，你怎么看？”
