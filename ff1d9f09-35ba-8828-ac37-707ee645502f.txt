　　第35章
<br/>　　农田内，辣椒幼苗长势很好。原先程大雷还担心天气太热，所以特意用树干和茅草搭起凉棚，为幼苗遮阳。如今看来，这番苦工没有白费，幼苗绿汪汪一片，再有一两个月就可以收成。
<br/>　　程大雷心中腾起莫名的成就感，他想起刚才与黄三元的话头，道：“打听出来什么消息？”
<br/>　　“护送寿礼的队伍，这十几天就会从青牛山下经过，从其他山寨得到的消息，这次薛半川应该会派陆哼护送。”
<br/>　　“陆哼？这个人你知道多少？”程大雷知道黄三元当初在黑石城，对于黑石城内的人员情况，他比所有人都了解。
<br/>　　“陆哼，今年差不多三十岁，善使一条大棒，他是个孤儿，是薛半川收留照顾他，此人无父无母，无亲无故，普天之下谁都不认，只认薛半川，是个厉害人物。”黄三元道。
<br/>　　“当然是厉害人物，不厉害薛半川不会让他出这趟差。”程大雷看着眼前的辣椒苗，问：“其他山寨打算怎么劫？”
<br/>　　“无非就是夜袭，下毒，在路上设伏。”黄三元道：“反正是猫有猫道，狗有狗道，大当家，咱们就不插手么？”
<br/>　　程大雷想了想，长叹一声道：“群狗抢肉，凭咱们蛤蟆寨这几块料，想要争，也得有和人争的资格呐。”
<br/>　　站在高处往下望，密林之中隐隐能嗅到火药味，当然，青牛山没有火药味，可那种不安分的味道，已经弥漫得到处都是。
<br/>　　……
<br/>　　“婉儿，婉儿，过来给你看个好东西！”李行哉大步踏进屋子，冲李婉儿招招手。
<br/>　　“哥，你能不能不要一副色眯眯的表情，让我感觉你又把丫鬟的小衣偷了。”李婉儿。
<br/>　　“这……过去的事能不能不要老提，走，出去我给你看点东西。”
<br/>　　“究竟是什么呐？”李婉儿极其不情愿的随李行哉走出土屋。
<br/>　　顺着蛤蟆寨的地势往上爬，最后李婉儿眼前出现一座池子，上面盖着茅草。
<br/>　　“哥，这是什么，洗澡用的，露天？”李婉儿道。
<br/>　　“不，关键是池子里面的东西。”李行哉显得很兴奋：“泉水从山上下来，被引到池子中，上面盖着茅草，飞虫落尘进不去。池子中还有碎石，木炭……”
<br/>　　“木炭，有了木炭水还能喝嘛？”李婉儿。
<br/>　　“能喝，他们说木炭能搁住水里面的小虫。”
<br/>　　“水里面会有小虫？”李婉儿道。
<br/>　　“这个……我也不是很懂。不过大费周章挖这么个池子，终归是有些用吧。”李行哉道：“走，我再领你去看个东西。”
<br/>　　李行哉拽着李婉儿往下走，李婉儿拧着眉头道：“哥，你带我到茅房中干什么，不嫌臭嘛？”
<br/>　　“去，进去看看。”李行哉。
<br/>　　“我不！”
<br/>　　“好啦，不去就不去，我给你聊聊。”李行哉道：“这茅房的下面和马圈、猪圈通着，地底下有一个大坑。他们说茅房也有小虫，飞出来进入人身体，人就会得病。以后可以挖出来肥田。”
<br/>　　“哥，这……有什么用？”李婉儿一头雾水。
<br/>　　“我……也不知道有什么用，可总觉得这里面有大用处。”李行哉兴致勃勃，拍着手掌道：“行千里路，知千里事，婉儿，你现在还觉得这座山寨简单嘛。”
<br/>　　“一共六个人，再不简单又能怎样？”
<br/>　　“是啊，一共六个人，何必如此大费周章……”李行哉眼中闪烁着异样的光彩：“这个山寨的主人，图谋不小呐。”
<br/>　　“哥，你能不能别盯着茅房一副色眯眯的样子。”李婉儿捂着鼻子：“再说你也是……”
<br/>　　“嘘，噤声！”李行哉神色一凛，声音已低下来。
<br/>　　到了晚间，土屋内点着蜡烛，烛光透过窗户纸洒出来。
<br/>　　“写得怎么样了？”程大雷这个时候走进土屋。
<br/>　　“一直在写。”李行哉道：“这是我在京州的见闻。”
<br/>　　“继续写吧。”
<br/>　　程大雷抓起一张已写完的，如饥似渴的获取着这个世界的信息。帝国十三州一百单八城，各州各城在什么位置，程大雷一无所知。而且关于帝国的历史，也是程大雷要知道的。他从苏樱那里了解到一些，可苏樱的见识明显比不上李行哉。
<br/>　　“大当家，你了解这些作什么啊？”
<br/>　　“写书。”程大雷顺嘴胡说。
<br/>　　“写书！”李行哉和李婉儿都睁大眼睛。
<br/>　　“你是个诗人，我是个作家，我的梦想就是写出一本传世巨作。”程大雷抬起头来：“难道我长得不像。”
<br/>　　李行哉和李婉儿目瞪口呆，想说像实在违心，想说不像，可如今又是寄人篱下。
<br/>　　“你不是个诗人么？”程大雷道：“写得诗呢，让我看看。”
<br/>　　“哈，倒是写了几首，也让大当家鉴赏鉴赏，与小生评点一二。”
<br/>　　李行哉从纸堆中翻出几页纸，递到程大雷面前。
<br/>　　程大雷一看大嘴巴就咧起来了，口中发出『啧啧』怪叫。
<br/>　　“怎么，我这诗难道不好？”关于这一点，李行哉还是有些自信的。在京州，他被人称为诗才无双，有高人评点过，说他诗中有兵戈之音。
<br/>　　“这也能叫诗……”程大雷嘴都快咧到嘴巴：“帝国的诗才，啧啧，真的是堪忧呐。”
<br/>　　“喂，你到底懂不懂啊，我哥哥的诗在整个京州都是有名的。”李婉儿不忿道。
<br/>　　“好啦，好啦，也让你们开开眼，知道什么叫写诗。”程大雷拿起毛笔，饱蘸浓墨，抬起头，看到李行哉和李婉儿都盯着自己：“哎，有我这一句，若是传出去，怕是整个帝国的才子都不敢拿笔了。”
<br/>　　“哼，口气倒是不小。”
<br/>　　李婉儿冷哼一声，视线还是落在程大雷笔尖上，微微有些出神，却是在想：此人如此口气，不知道肚子里的东西怎样。
<br/>　　程大雷紧紧攥住笔杆，笔尖走在纸上，出现歪歪扭扭鬼画符般的几行字。看到他的字迹，李婉儿就忍不住笑出声，刚想嘲讽几句，可等她看清纸上的内容，眼睛突然睁大。
<br/>　　程大雷停下笔，看了目瞪口呆的二人一眼，施施然道：“好了，天不早了，早点歇着。”
<br/>　　“一句，为什么只有一句？”李婉儿忍不住道。
<br/>　　“一句就够了，怕都拿出来吓死你们。”走到门口的程大雷补充一句。
<br/>　　李行哉早已失神，手捧着那歪七扭八的字迹，口中呐呐道：“不说天下才子，有这一句，至少我这一生是提不动笔了。”
<br/>
