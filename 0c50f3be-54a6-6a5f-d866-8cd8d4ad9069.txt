　　望着远处的村庄，慕清霄身影一闪就出现在山道上，顺着崎岖的山路，向木叶忍者村走去。
<br/>　　短短几十秒后，让的身影就出现在了木叶忍者村的大门口，而门口执勤的忍者，显然也注意到了他。
<br/>　　“站住，你是那个村子的忍者，有通行证吗？”
<br/>　　望着站在门口的两名忍者，年纪不大，二十岁左右的样子，估计也不知道，他身上的红云风衣代表着什么。
<br/>　　“通行证，我回木叶，还需要通行证吗？”说着，慕清霄抬起头来，斗笠下面，星眸中浮现三颗妖异的勾玉。
<br/>　　两名执勤忍者，身躯皆是一怔，眼中也倒映出三颗勾玉，随后目光就变得呆泄起来。
<br/>　　“叮铃铃......”
<br/>　　伴随着清脆悦耳的风铃声，慕清霄就与执勤忍者擦肩而过，向村内走去。
<br/>　　走在偏向现代化的街道上，孩童嬉戏打闹声绕耳不绝，两侧建设着玲琅满目的商店，比起当年，房屋都显得崭新，似乎重建过。
<br/>　　“妈妈，你快看，那里有一个奇怪的人。”
<br/>　　“......别乱指，快跟妈妈回家。”
<br/>　　奇装异服，显得诡异，想不引起路上行人的注意都难，甚至有的路人指指点点，议论纷纷。
<br/>　　慕清霄并不在意行人的目光，顺着街道，很快经过一个小木屋时，其中传来一阵香味，令得他停住了脚步。
<br/>　　掀开帘幕，进入其中，慕清霄随意坐在了前面的板凳上。
<br/>　　“先生，要来一碗拉面吗？”
<br/>　　店老板见这个时间段有客人来此，转过身来，见到慕清霄的装扮后，瞳孔微微一缩，但很快就恢复正常。
<br/>　　慕清霄只是瞥了他一眼，微微点头，道：“那就来一碗拉面吧。”
<br/>　　“好嘞，请稍等片刻。”
<br/>　　几分钟后，老板将香喷喷的拉面端了上来，同时店外也迎来了一群不速之客。
<br/>　　察觉到他们的气息后，慕清霄淡然自若，悠哉悠哉的吃起了拉面。
<br/>　　十分钟后，放下筷子，起身向外走去。
<br/>　　来到店外，此时周围已经被封锁了起来，街道上看不到一个人影，而四周的建筑物上，则站着一群气息强悍的忍者。
<br/>　　“你是谁，为何穿着晓的服饰，来木叶有何目的？”最先说话的是，站在最前面，手中拿着一本书，满头银发的男子。
<br/>　　“旗木卡卡西，真是有许多年没见了。”
<br/>　　说着，慕清霄伸手取下斗笠，在周围忍者扫视一圈，其中还站着几位熟人。
<br/>　　“宇智波......清霄，真的不敢置信，十多年过去，你的容貌居然也没有留下一丝岁月的痕迹。”
<br/>　　映入眼帘的俊逸面孔，令得旗木卡卡西目光凝重无比，接着道：“十几年前，你就已经叛出木叶，如今又回来有何目的？”
<br/>　　“真的是你吗？宇智波清霄，没想到我们会以这样的方式见面。”
<br/>　　人群中响起一道颇为响亮的声音，只见一名身穿绿色紧身衣，梳着西瓜头，盯着一对浓眉的男子走了出来。
<br/>　　并不在意他们的称呼，慕清霄古井无波，道：“迈特凯，真是好久不见，你的肉身，原来如此，易筋经修炼至圆满了吗？”
<br/>　　“没错，早在几年前，我便将这门博大精深的修炼方式，修炼至圆满，在这里，我也要代替小李谢谢你。”
<br/>　　说着，迈特凯上前一步，恐怖的压力自他身躯中爆发而出，道：“但是，如果你是来木叶捣乱的，我绝不会手下留情。”
<br/>　　虽然话是这么说，但迈特凯自己心里都没底。
<br/>　　毕竟，在十多年前，他曾经开启八门遁甲，与慕清霄战斗过，使出全力的他，却未曾伤到后者一丝一毫。
<br/>　　他不是白痴，可不会认为，十多年过去，慕清霄依旧在原地踏步。
<br/>　　而且慕清霄现在给他的感觉就是，完全看不透，比起当年，更加的深不可测。
<br/>　　“我对木叶忍者村，没多大的兴趣。”慕清霄看了看四周的忍者，道：“回来，只是来找人，顺便看一看。”
<br/>　　“咻咻——”
<br/>　　就在这时，空中一道苦无，携着破风声，插在了慕清霄两米外的地面。
<br/>　　旋即，风沙弥漫，只见一道穿着红白相间长袍的青年，已经出现在前面，青年满头金发，胳膊上绑满了绷带，身上有着一股上位者的气质。
<br/>　　“我们，是不是也很久没见了。”
<br/>　　望着面前的青年，慕清霄微微一笑，道：“是啊，当年满口梦想的毛头小子，都已经成长到这一步了吗？”
<br/>　　在他的眼中，漩涡鸣人可不就是当年的毛头小子么。
<br/>　　对于慕清霄的称呼，鸣人并没有大呼小叫，若是当年，恐怕早就冲上来，而此时，他的眼中只有慎重。
<br/>　　作为火影，若是这点忍耐力都没有，谈何守护村子。
<br/>　　“你来村子，有什么事情吗？”
<br/>　　“嗯，回来找人。”
<br/>　　说着，慕清霄便戴上斗笠，无视周围的忍者，自漩涡鸣人身边走过，朝着西边方向走去。
<br/>　　刚走没几步，脚步一顿，手中就出现一个容器，其中放着四颗眼珠，随手一抛，道：“还给它们原来的主人吧。”
<br/>　　将容器攥在手中，漩涡鸣人脸色微变，将其收了起来，盯着慕清霄的背影。
<br/>　　“对了，你结婚了吗？”
<br/>　　对于慕清霄的问题，鸣人一愣，微微点头道：“已经结婚多年了。”
<br/>　　“是吗？你妻子叫什么？”
<br/>　　“日向花火。”
<br/>　　鸣人眉头皱了皱，语气依旧平淡的说道。
<br/>　　“恭喜。”
<br/>　　慕清霄微微点头，丢下一句莫名其妙的话语，便在众忍者的目光中，向着村子西边走去。
<br/>　　见慕清霄离去，最后消失不见，旗木卡卡西上前一步，道：“要跟上去吗？毕竟他依旧是叛忍。”
<br/>　　鸣人抬起手示意，道：“不必，他估计是去日向家，当年的事情已经过去，不要在提起，多生事端。”
<br/>　　说着，鸣人就转身离去，其余忍者也皆是散去。
<br/>　　只有他清楚，站在慕清霄如此近的距离，压力是多么的沉重，仿佛站在他面前的不是一个人，而是一头不可战胜的巨兽。
<br/>　　比起当年的十尾，带给他的压迫感更大。
<br/>　　坐在火影的位置上，他需要为村子考虑，叛忍是弱者的标志，给强者打上叛忍的标签，无异于自取灭亡。
