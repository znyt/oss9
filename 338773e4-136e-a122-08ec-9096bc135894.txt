　　“易霄副殿主先别急。”
<br/>　　许鹤颜连忙出言。
<br/>　　“我们确实有理由怀疑你身后的那群武者，乃是邪修。”
<br/>　　“根据我们的情报，这群人，来历不明，而且一直行事低调。”
<br/>　　“若非我们各主殿联手，甚至连他们的些许情报都寻不到。”
<br/>　　“种种迹象，皆表明，他们很可能是邪君府中人…”
<br/>　　“许鹤颜，你也要教我分辨邪修？”萧逸双眼一眯。
<br/>　　“是否觉得，易某连邪修都不会分辨？”
<br/>　　“又是否认为，这些天，易某斩杀的无数邪修，血洗的无数邪修分部，都被易某枉杀了不少，错认了不少？”
<br/>　　“不敢。”许鹤颜连忙拱手。
<br/>　　今时今日，若说谁更了解邪修，绝不是上古八殿。
<br/>　　而是有着‘邪修克星’之称的紫炎易霄。
<br/>　　眼前这位短短时间被斩杀无数邪修，血洗无数邪修分部的强者，放眼中域，谁敢说教他如何去分辨邪修？
<br/>　　邪修，向来诡秘，行事低调，又残忍至极。
<br/>　　但，但凡被面前这位强者狙击上的邪修，从无幸免者。
<br/>　　哪怕是那位声名赫赫的毒袍，也仅仅只能以诈死这般低劣的手段去逃命。
<br/>　　说教他如何分辨邪修？
<br/>　　此话若传出，怕是江流火许鹤颜二人，瞬间就会成为整个中域的笑柄。
<br/>　　萧逸手持卷宗，冷声道，“通缉理由，两个。”
<br/>　　“一为黑云学教给出的掠夺药尊殿天材地宝。”
<br/>　　“二为你们5殿给出邪修身份。”
<br/>　　“两个理由，无一成立，你们还有何话可说？”
<br/>　　“这些不过是易霄副殿主的推测罢了，并无凭据。”白月、荆无等人，还想说些什么。
<br/>　　“我提醒易霄副殿主一句，这般污蔑我们各大主殿主…”
<br/>　　萧逸淡漠打断道，“若你们认为易某连这些事情都查不出真伪，那便继续狡辩。”
<br/>　　“若你们认为你们的手段比邪修更高明，更隐秘的话，也大可继续。”
<br/>　　“易某也提醒你们一句，至今为止，易某出手追杀之人，能活命者，仅毒袍一人，你们可以试试。”
<br/>　　“紫炎易霄，你好大的胆子，胆敢威胁我们…”白月冷喝一声。
<br/>　　但明显可以看到，他脸上尽是慌惧之色。
<br/>　　即便是冷喝，也显得外强中干。
<br/>　　嘭…
<br/>　　滔天火焰，瞬间弥漫长空。
<br/>　　惊人的紫晶灵炎，化作火海，封锁了四周。
<br/>　　“威胁你们？”萧逸冷漠一笑，“你们够格吗？”
<br/>　　“我没兴趣与你们浪费时间。”
<br/>　　“事实如何，我压你们回各自总殿，自有定论。”
<br/>　　“你…”周遭主殿主，脸色顷刻大变。
<br/>　　以紫炎易霄的实力，若他要用强，在场谁能阻止反抗？
<br/>　　嗖…
<br/>　　恰在此时。
<br/>　　远方，一道身影疾速而至。
<br/>　　“破。”
<br/>　　人未至，一声如若天地妙音的轻喝，已然落下。
<br/>　　轰…
<br/>　　漫天紫炎，瞬间消散。
<br/>　　即便是承风殿主二人之前布下的修罗锁天阵亦一并溃散。
<br/>　　“嗯？”萧逸双眼一眯，直视远方。
<br/>　　承风殿主二人，同样皱了皱眉。
<br/>　　三人几乎瞬间就能确定，来人极强，不，准确来说，是强得可怕。
<br/>　　嗖…
<br/>　　数个呼吸后。
<br/>　　一个老者，稳稳的落在了高空中。
<br/>　　“妙殿主。”周遭主殿主中，白月、荆无二人明显脸色大喜。
<br/>　　“妙生花。”承风殿主，直接道出了来人姓名。
<br/>　　“承风殿主，横天殿主，许久不见。”老者，淡淡一笑，对着二人拱了拱手。
<br/>　　“妙殿主。”横天殿主二人，同样回以一礼。
<br/>　　老者的目光，最后定格在萧逸身上。
<br/>　　“这位，便是鼎鼎大名的紫炎易霄，易霄副殿主了吧。”
<br/>　　“妙殿主。”萧逸同样拱拱手。
<br/>　　老者，他第一次见，但姓名，他却听说过。
<br/>　　而且，单单这个名字，已足以让他目光凝重到极点。
<br/>　　妙生花，天机殿总殿主殿主。
<br/>　　论职位，他大概等同承风殿主二人在风刹殿、修罗殿中的地位。
<br/>　　可此人，乃是千年前便已成名的传奇阵法师。
<br/>　　一双妙手，弹指成阵。
<br/>　　阵之精妙，法之绝伦，此人掌控得淋漓尽致。
<br/>　　哪怕是上古大阵，在此人手中也撑不过一时三刻。
<br/>　　此人，乃是个当之无愧的传奇阵法师。
<br/>　　“妙殿主来此，是…？”萧逸皱眉问道。
<br/>　　妙生花淡淡一笑，道，“自然是解决此事。”
<br/>　　“区区主殿通缉令，也值得妙殿主亲至？”萧逸皱了皱眉，心头闪过一丝不妙的念头。
<br/>　　妙生花摆摆手，“易霄副殿主不必误会。”
<br/>　　“区区主殿通缉令，还有一群不过普通绝世实力的通缉者，自然不值得老夫亲至。”
<br/>　　“不过，也正是这区区主殿通缉令，还有一群区区通缉者，却生生闹得风刹、修罗两殿主殿主，还有那位声名赫赫的萧逸副殿主都亲自来此。”
<br/>　　“现在，连你这位三殿接班人都亲至，老夫若还不来，可就说不过去了。”
<br/>　　妙生花笑笑，道，“江流火、许鹤颜二人，乃是你炎殿和药尊殿之人，老夫管不着。”
<br/>　　“老夫来，只是处理白月、荆无二人的事。”
<br/>　　“白月，荆无。”妙生花霎时脸色一肃，威严中，带着冰冷。
<br/>　　“你真当你们能瞒得过总殿？”
<br/>　　“八殿自有铁规，若犯之，也必严惩。”
<br/>　　“即日起，你二人革去主殿主与副殿主头衔，降为分殿主，百年内不再录用。”
<br/>　　“参与此通缉令追捕的执法队、殿内武者，亦一律连降三级。”
<br/>　　“百年之内，若还有差池，便数罪并罚，压入黑魔牢狱，重罚百年。”
<br/>　　“这…”白月、荆无二人，脸色一变。
<br/>　　当他们听到降为分殿主时，本想说些什么。
<br/>　　中域，可不是寻常地方，八殿头衔和职位要累计功绩，是一件极其困难的事。
<br/>　　能当上主殿殿主，甚至是进入总殿成为副殿主，哪个不是耗费百年时间，一直兢兢业业，方有机会成就。
<br/>　　现在即刻革去身份，甚至降为分殿主，这对他二人来说，乃是极大的打击，甚至是重罚。
<br/>　　但听到‘黑魔牢狱’四个字时，二人又身躯一颤，再不敢多说什么。
<br/>　　“谨遵妙殿主之命。”二人连忙躬身。
<br/>　　妙生花点了点头，目光，看回萧逸，“易霄副殿主，这般处理，可满意？”
