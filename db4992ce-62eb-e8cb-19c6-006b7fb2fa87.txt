　　楚风眠拿起了这一瓶精血后，便是将那白玉棺材盖上，随即一步离开了这密室。
<br/>　　这狮霸武祖的最终宝藏，就是这一瓶精血，一瓶武祖精血。
<br/>　　虽然论起数量，这一瓶武祖精血，还不如外面光幕之中的宝藏有多。
<br/>　　但是论起价值的话，只怕是外面所有光幕之中的宝藏加起来的价值，都不如这一瓶武祖精血要高。
<br/>　　天人精血，遗留下来的寥寥无几，每一滴都是无比珍贵之物。
<br/>　　一滴都无比珍贵，一滴天人精血，如果是被一尊半步天人境的强者的得到。
<br/>　　吞噬其中得到的力量是小，但是天人精血之中，蕴含的还有天人境的力量，熟悉领悟，未来在冲击天人境的时候，会有着极大的帮助。
<br/>　　一滴天人精血，都无比尊贵，武祖精血更是比起寻常的天人精血还要珍贵不知道多少倍。
<br/>　　现在楚风眠得到的武祖精血，何止一滴，这一个玉瓶之中的武祖精血，足足有着数百滴之多。
<br/>　　数百滴精血的失去，对狮霸武祖来说，可都是伤筋动骨，甚至是可能损失实力，掉落境界的事情。
<br/>　　如果不是最终狮霸武祖寿元将尽的话，只怕他也不会留下这样一瓶精血。
<br/>　　参悟天人境，对楚风眠来说还太早了，但是单单是吞噬了这一瓶精血，楚风眠就有把握，足有提升他三成以上的实力。
<br/>　　楚风眠看了一眼这一瓶武祖精血，手心一点，玉瓶上的禁制纷纷崩开。
<br/>　　武祖精血哪怕是经历数十万年的岁月，其中也蕴含着无比恐怖的力量。
<br/>　　禁制崩开，单单是凭借那一个玉屏，根本困不住这其中的武祖精血。
<br/>　　咔嚓一声。
<br/>　　玉瓶之上便是打开无数裂痕，随即其中的武祖精血，当即是飞了出去，就在空中化为了一头雄狮。
<br/>　　这一头雄狮，不去管楚风眠，而是直接向着外面飞去。
<br/>　　血脉有灵，像是狮霸武祖这个级别的强者，哪怕是身死，留下的精血都有着足以凝聚生灵的力量。
<br/>　　看着那武祖精血要逃，楚风眠冷笑一声，突然出手，一把抓向这武祖精血化为了雄狮。
<br/>　　一股灵力笼罩在这武祖精血化为的雄狮之上，只见这一头雄狮，在疯狂的咆哮，但是却无法挣脱楚风眠的手心。
<br/>　　以楚风眠的实力比起寻常的九劫古帝还要强横，一团精血想要从楚风眠的手中逃脱升天，绝无可能。
<br/>　　尤其是这一团武祖精血，已经是经历了数十万年的时间，哪怕是在玉瓶之中尘封。
<br/>　　但是其中的力量，依然十不足三，楚风眠想要收取起来，自然轻松无比。
<br/>　　随着楚风眠手心用力一拍，这一头雄狮便是在楚风眠的手心之中化为了一团血雾，恢复了一团精血的模样。
<br/>　　也不在挣扎。
<br/>　　楚风眠顺势直接运转起吞天秘术来，他的身躯都是化为了一道巨大无比的漩涡，将这精血全部吞噬。
<br/>　　一经融入到楚风眠的身躯之中，他身躯之中的力量都在节节攀升起来，尤其是战龙血脉。
<br/>　　这战龙血脉如今变得疯狂无比，大量的吞噬这武祖精血之中蕴含的力量，大量的灵力吞噬其中。
<br/>　　突然之间，在楚风眠身躯之中的十二万九千六百滴远古战龙精血之中，一滴精血突然飞了出来。
<br/>　　这一滴精血吞噬的最快，在这一滴金色的战龙精血之上，突然间浮现出了无数纹路。
<br/>　　这纹路，就在这一滴战龙精血之上开始凝聚起来，随着一点点的完善，便是化为了一道法阵的模样。
<br/>　　轰隆！
<br/>　　当这法阵彻底凝聚完成的一刻，这一滴战龙精血的力量，疯狂攀升，足足是攀升了上百倍。
<br/>　　龙皇精血！
<br/>　　楚风眠看到这一滴战龙精血的转变，他也清楚，这便是战龙决向着第二层开始修炼的景象。
<br/>　　血中凝阵，这正是龙皇精血的模样。
<br/>　　一尊远古龙皇，比起寻常的远古战龙要强横百倍，这一滴龙皇精血蕴含的力量也是战龙精血的百倍。
<br/>　　虽然一滴的转化，对楚风眠实力的不值一提，但是如果楚风眠身上所有的战龙精血都转化为龙皇精血的话，那楚风眠的实力将会提升数百倍。
<br/>　　这一滴龙皇精血只是第一滴开始转化的。
<br/>　　一瓶武祖精血蕴含的力量，远远不止如此。
<br/>　　第二滴，第三滴。
<br/>　　不断的有着战龙精血，在汲取了足够的力量之后便是就开始发生这转变，凝聚成了龙皇精血。
<br/>　　一千滴！
<br/>　　等到最后一滴武祖精血被吞噬殆尽之后，楚风眠身躯之中战龙精血的转变，也逐渐的结束。
<br/>　　足足是一千滴战龙精血，在吞噬了这一瓶武祖精血之后晋升，成为了龙皇精血。
<br/>　　一千滴龙皇精血，蕴含的力量几乎是跟十万滴战龙精血一样。
<br/>　　虽然十二万九千六百滴战龙精血之中，仅仅只是其中的一千滴转化成了龙皇精血。
<br/>　　但是楚风眠身上的气息，他的力量足足是提升了七成左右。
<br/>　　带来的好处比起楚风眠估计的提升，还要更大。
<br/>　　楚风眠身上的境界，也随着力量的提升，一举踏入到了八劫古帝巅峰的层次。
<br/>　　距离九劫古帝，只有一步之遥。
<br/>　　这提升之快，简直是无法想象，几日前楚风眠才刚刚度过了帝尊大劫，从准帝境界一举踏入到了八劫古帝的地步。
<br/>　　如今随着这一瓶武祖精血的吞噬，楚风眠的境界更是直接达到了八劫古帝巅峰。
<br/>　　“怪不得九域的出世让如此之多的人疯狂，一个大型宝藏就足以带来如此之大的好处。”
<br/>　　楚风眠感叹了一声。
<br/>　　如果不是得到这一瓶武祖精血，楚风眠想要让战龙血脉再度转变，只怕耗费数年的时间都未必做得到。
<br/>　　但是有了这一瓶武祖精血的相助，楚风眠轻而易举的就是将身躯之中的一千滴远古战龙精血，已经是转化成为了龙皇精血。
<br/>　　这就是机缘，大型宝藏的机缘。
<br/>　　得到一个这样的机缘，都足以让一个小人物，一步登天。
<br/>　　五行刀祖，碧螺老祖！
<br/>　　楚风眠眼神之中露出了几分杀意。
<br/>　　那建木树枝，楚风眠可还记得，现在是时候从碧螺老祖的手中，讨回这建木树枝了。
