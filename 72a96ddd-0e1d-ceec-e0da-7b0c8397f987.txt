　　药王谷谷主的话音落下。
<br/>　　全场尽皆陷入了一片沉默。
<br/>　　萧逸冰冷的双眸，直视着面前老者。
<br/>　　而老者那苍老却不失浑浊的眼眸中，显然不是在开玩笑。
<br/>　　“苏谷主这是何意？”萧逸沉声问道。
<br/>　　药王谷谷主，全名苏震玄，实力深不可测。
<br/>　　就单单萧逸的感知中，此人单单身上溢出的些许气息，已让自己无力抵抗。
<br/>　　那是一种真正的无力感。
<br/>　　在他面前，自己甚至连战斗的欲望都生不起来。
<br/>　　药王谷谷主笑笑，道，“只是单纯想让小友当我药王谷长老罢了。”
<br/>　　“客卿长老，享长老待遇，却无需履行任何职责。”
<br/>　　“每月可到我药王谷宝库领取一定修炼资源，却无需时刻待在谷中。”
<br/>　　“简单来说，只是给予你一份头衔，以及待遇，你什么都不需付出。”
<br/>　　“这般好事，小友仍旧不愿意？”
<br/>　　萧逸摇了摇头，沉声道，“我不明白苏谷主的意思。”
<br/>　　药王谷谷主，可是成名已久的传奇人物之一。
<br/>　　若他亲自出手擒下自己。
<br/>　　哪怕自己有猎妖殿分殿主的身份都无用。
<br/>　　他何须如此礼遇自己？
<br/>　　药王谷谷主笑道，“看来不解释清楚，易霄小友这般谨慎，是不肯愿意的了。”
<br/>　　“客卿长老，确实无需你付出什么。”
<br/>　　“但，若我药王谷需要你帮忙时，你有义务前来相助。”
<br/>　　“当然，那只是义务，来不来，取决于你。”
<br/>　　萧逸皱起了眉头，问道，“易某并不认为能帮药王谷什么。”
<br/>　　“能。”药王谷谷主脸色一正，“比如，药楼里的星辰之火。”
<br/>　　“如今，已被你吸收殆尽，虽禁制仍在，但若有你帮忙留下火焰种子。”
<br/>　　“禁制的运行速度更快。”
<br/>　　“另外，易霄小友如今的炼药之术，已相当了不起。”
<br/>　　“我药王谷有时也需寻你帮忙炼药。”
<br/>　　“就这么简单？”萧逸谨慎问道。
<br/>　　“就这么简单。”药王谷谷主重重地点了点头。
<br/>　　“小友接受这份头衔后，在我药楼凝聚出火焰种子，即可离去。”
<br/>　　“我药王谷绝不阻挠半分。”
<br/>　　“日后，我药王谷，你亦可出入自如。”
<br/>　　药王谷谷主的话，让得在场所有人，脸色突变。
<br/>　　木青云更是脸色发黑，却不敢多说什么。
<br/>　　萧逸思索了半分，半晌，点了点头，“既然苏谷主盛意拳拳，易某自当答应。”
<br/>　　“如此甚好。”药王谷谷主满意一笑。
<br/>　　萧逸，则心头仍旧凝重。
<br/>　　他愿意答应下，只是不想多生变故。
<br/>　　苏震玄这样的强者，没必要骗他。
<br/>　　但若忽然翻脸，自己今日恐怕真的走不出药王谷。
<br/>　　所幸，苏震玄似乎并非那种霸道之人。
<br/>　　药王谷，向来有两脉。
<br/>　　一脉是谷主苏家一脉。
<br/>　　一脉是各大长老的木家一脉。
<br/>　　虽分两脉，但药王谷传承千百年以来，从未出现过内讧之事。
<br/>　　两脉以苏家为首，异常团结。
<br/>　　言归正传。
<br/>　　萧逸去药楼之巅，凝聚了星辰之火的种子后，转身便走。
<br/>　　木青云等一众长老，果真不再阻难。
<br/>　　萧逸心头微微松了口气，连忙带着叶铭离去。
<br/>　　……
<br/>　　药王谷内，苏震玄看向白墨寒与血焰，道，“两位小友，可以离开药王谷了。”
<br/>　　“是。”白墨寒和血焰，拱了拱手，行了一礼，而后离去。
<br/>　　原地，只剩下苏震玄以及一众长老。
<br/>　　木青云当即说道，“谷主，就这么便宜了易霄，放他离去？”
<br/>　　“那小子身上，可是蕴含了我药王谷千百年的无数珍稀天材地宝药力。”
<br/>　　苏震玄笑笑，道，“不放他走，你能如何？”
<br/>　　“丹药既已被吞下，药力怕也已经被他吸收了。”
<br/>　　“难道真的将他击杀，重新炼化出药力？”
<br/>　　“此等邪恶之事，若传出去，药王谷必定千夫所指。”
<br/>　　“更何况，他可是猎妖殿分殿主，功绩累累。”
<br/>　　“那也不能就这样放他走。”木青云强硬地说道。
<br/>　　药王谷储存了千百年的无数天材地宝，何等惊人与珍贵。
<br/>　　足以让任何人疯狂，更不可能让外人所得。
<br/>　　木青云之前之所以直接对易霄出手，便也是因此。
<br/>　　苏震玄闻言，沉默了一下。
<br/>　　半晌，才说道，“青云，你可知我刚才去哪了？”
<br/>　　木青云皱眉道，“谷主不是在闭关，给少谷主寻求医治之法吗？”
<br/>　　苏震玄摇了摇头，“我始终想不出办法，所以去了一趟猎妖殿。”
<br/>　　“嗯？”木青云疑惑道，“连殿主那等人物，都没有办法吗？”
<br/>　　“没有。”苏震玄摇了摇头，“浅沫是武魂出了问题。”
<br/>　　“武魂，是天地规则力量所化，谁也奈何不了。”
<br/>　　“此事先不提。”
<br/>　　苏震玄摆摆手，道，“我去一趟猎妖殿，你可知，殿主与我说了什么？”
<br/>　　“什么？”木青云问道。
<br/>　　苏震玄沉声道，“他给了易霄三个任务。”
<br/>　　“哦？”木青云脸色更加疑惑。
<br/>　　“殿主作为炎武王国所有猎妖殿的主殿主，可不会特地给出任务。”
<br/>　　“除非…”
<br/>　　木青云，忽然脸色大变。
<br/>　　“不错。”苏震玄点了点头，“他在找接班人。”
<br/>　　“那三个任务，便是考核。”
<br/>　　“易霄今日已展现出足够的实力和天资。”
<br/>　　“日后，只要他在其它方面，也满足殿主的考核。”
<br/>　　“那么，他便是猎妖殿主殿下一任殿主。”
<br/>　　“这下遭了。”木青云脸色变得更为难看。
<br/>　　“怎么？”苏震玄皱了皱眉头。
<br/>　　木青云沉声道，“谷主怕是不知道，易霄此子，虽声名在外。”
<br/>　　“但，他的来历极为神秘。”
<br/>　　“除了知晓他叫易霄，以及来自于北山郡外，再无其它信息。”
<br/>　　“谷主可还记得十数年前那场灾难。”
<br/>　　“那个疯子，同样姓易，同样来自北山郡，同样大闹我药王谷。”
<br/>　　说着，木青云脸上，杀意凛然。
<br/>　　“原来这就是你铁了心要对付他的原因。”苏震玄脸色一松。
<br/>　　“此事你不必多想，这二人没有任何关系。”
<br/>　　不知为何，苏震玄的语气，极其肯定。
<br/>　　木青云咬牙道，“谷主，事情不可能如此巧合，我们…”
<br/>　　苏震玄打断道，“那只是你的猜测，而且，就算你的猜测是对得。”
<br/>　　“那又如何？”
<br/>　　苏震玄的目光，极其深邃，脸色，也变得凝重。
<br/>　　“青云，眼光放远些。”
<br/>　　“这个世界，是会变的，命运的齿轮，无人能阻碍分毫。”
<br/>　　“哪怕我这个境界的强者，甚至是殿主，终有一天也会死去的。”
<br/>　　木青云脸色一阵萧瑟，道，“谷主，生死关到底是什么？”
<br/>　　苏震玄摇了摇头，“你现在无需知道，日后自己会明白的。”
<br/>　　“易霄，他还年轻。”
<br/>　　“今日我药王谷与他交好，日后会有收获的。”
<br/>　　“总之，你只需记住，以后与他交好即可。”
<br/>　　说罢，苏震玄转过身，刚欲离去。
<br/>　　恰在此时，木长老忽然惊呼一声。
<br/>　　“哎呀，倒是忘了。少谷主的事，或许该找那小子帮忙。”
<br/>　　“现在，那小子估计走没影了。”
<br/>　　木青云撇撇嘴，道，“连谷主都解决不了的事，找他何用？”
<br/>　　“毕竟他现在还未成长起来。”
<br/>　　“可不能这样说。”木长老认真说道，“那小子，是一个总能让人吃惊的家伙。”
<br/>　　“谷主没办法，不代表他每办法。”
<br/>　　苏震玄闻言，叹了口气，道，“算了，下次他来药王谷，再看他是否有那个能力吧。”
<br/>　　显然，苏震玄也不太相信易霄有办法。
<br/>　　“浅沫的命，我还能再保两年。”
<br/>　　“若两年内再无他法，也只能…”
<br/>　　木青云以及众长老齐声道，“谷主请放宽心，少谷主吉人天相，必能无恙。”
<br/>　　“两年内，我等必能想到办法。”
<br/>　　“但愿吧。”苏震玄叹了口气。
