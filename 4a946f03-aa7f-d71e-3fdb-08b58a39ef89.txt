　　古桐城，在巨大的太阴星上，都是一座非常有名的雄城。
<br/>　　此城属于啸天狼族的管辖范围，人口十数亿，占地方圆万里，辽阔无比。
<br/>　　古桐城中相当繁华，可谓是车水马龙，人流如织。
<br/>　　苏莫和吕罡两人，大步的行走在古桐城的街道上，能亲身感受到古桐城的繁华。
<br/>　　“苏兄，你的那些宝物，若是想卖的高价，还是交由拍卖场拍卖最好！”吕罡对苏莫说道。
<br/>　　“嗯，不过拍卖场比较耗时间吧？”苏莫问道。
<br/>　　“也用不了多少时间，古桐城大商会多的是，基本上每天都有拍卖会！”吕罡说道。
<br/>　　“也好！那我们就先询问一番，哪里明日举行拍卖会！”苏莫道。
<br/>　　“我知道几家大商会，我现在就带你去！”
<br/>　　吕罡说道，言罢，他便带着苏莫，在偌大的古桐城转悠了起来。
<br/>　　连续询问了四家大型的商会，他们终于找到了一家商会，在明日举行拍卖会。
<br/>　　六壬商会，一间装饰奢华的偏殿之中。
<br/>　　苏莫和吕罡端坐于于下首，上首主坐上端坐着一名年过古稀的藏袍老者。
<br/>　　“老夫龚忠，两位是有物品在明日拍卖？”老者打量了吕罡和苏莫一番，先是自我介绍了一句，然后微笑着问道。
<br/>　　他的目光看到苏莫之时，眸中闪过一道微不可查的精光，因为他一眼便能看出，苏莫改变了容貌。
<br/>　　“不错，我有不少还算不错的宝物，准备明日拍卖！”苏莫点头说道。
<br/>　　“呵呵，两位小友来我六壬商会，算是来对了地方，明日的拍卖会，是一次大型的拍卖会，武者众多，你们的物品，绝对能拍卖出高价！”老者龚忠微笑着说道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即手掌一番，掌中便出现了一枚储物戒，递给了龚忠，道：“这里是在下需要拍卖的物品，前辈可以清点一番。”
<br/>　　龚忠点了点头，便接过了储物戒，将储物戒内的物品，仔细的清点了一番。
<br/>　　“你这些物品共有，一百八十三件，其中伪圣器三件，尊级上品宝器一件，尊级中品宝器……！”清点完所有物品之后，龚忠将数量详细的报了出来。
<br/>　　三件伪圣器，自然便是三支庚金破虚箭，一件尊级上品宝器，便是武仁的那张金色大弓。
<br/>　　至于其它的宝物，基本上都是尊级中品和下品，有兵器也有功法武技，而且以下品居多。
<br/>　　“嗯，没错！”苏莫点了点头，他之前已经清点过了。
<br/>　　“关于佣金，老夫要事先严明，我们会抽取一成！”龚忠沉声说道。
<br/>　　“明白！”苏莫道，拍卖会自然会抽取佣金，这在何地都是如此。
<br/>　　“好，明日辰时，还望两位来参加拍卖会！”
<br/>　　龚忠面带微笑，手中出现了一张金灿灿的请柬，递向苏莫。
<br/>　　苏莫接过请柬，稍一查看，便收了起来。
<br/>　　请柬非常简单，上面只记录了一个贵宾包间的房号。
<br/>　　“前辈，既然如此，那我们便告辞了！”苏莫长身而起，向龚忠抱拳。
<br/>　　“不送！”龚忠同样抱了抱拳。
<br/>　　随后，苏莫和吕罡便离开了六壬商会。
<br/>　　“苏莫，我们现在去哪里？”走在古桐城的街道上，吕罡问道。
<br/>　　“你不是要购买高品阶的拳吗？”苏莫反问道，他的那些拍卖的宝器之中，并没有高品阶的拳套，不然他就直接送于对方了。
<br/>　　“明天就是拍卖会了，当然是在拍卖会中购买！”吕罡说道，虽然拍卖会中价格比外面要稍高，但选择的余地也更多。
<br/>　　“嗯，那我们找个客栈休息一晚吧！”苏莫点头，随即便和吕罡两人，随便找了一家客栈住了下来，准备明日参加拍卖会。
<br/>　　住下以后，苏莫便再次进入了空间玉镯之中，继续尝试属性融合。
<br/>　　……
<br/>　　在古桐城，另一家客栈的大堂之中，三名年轻的武者，一边饮酒一边交谈。
<br/>　　这三名武者是两男一女，其中一名男子正是扬天成。
<br/>　　而另外两人之中，那名男子身穿华服，面容和扬天成有三分相像，那名女子则是身穿绿衣，面容娇美，如同大家闺秀。
<br/>　　“天威族兄，这次六壬商会的拍卖会，听说规模颇大，我们多淘些好东西！”扬天成将一杯美酒一饮而尽，然后对华服青年说道。
<br/>　　“嗯，还有不足三个月，古魔墓地就要开启了，我们尽可能多买些能增加战力和防御的宝物！”华服青年说道。
<br/>　　华服青年名为杨天威，乃是扬天成的族兄，同样也是天虚宗真传弟子，而且是仅次于厉恨天的存在。
<br/>　　“是啊，到时候强者太多，想要获得天魔花，难度太大了！”扬天成点了点头。
<br/>　　古魔墓地，是特殊的地方，其中蕴含着大量的古魔精气，百万年不散，形成奇特的天魔花，此花对于武者有极大的好处。
<br/>　　每十年开启一次的古魔墓地，也是七大势力的真传弟子，和太阴族子弟的磨炼，其中的争锋凶险异常，陨落都是常有的事情。
<br/>　　“天成师兄，那个苏莫已经成了道宗的真传弟子，到时候是不是也会去古魔墓地？”这时，那名娇美的女子绿衣女子说道。
<br/>　　听闻娇美女子之言，扬天成的脸色顿时一沉，想到可恨的苏莫，他就气不打一处来。
<br/>　　“高师妹，他若是敢去，我绝对会让他死无葬身之地！”扬天成咬牙切齿的说道，声音中充满了无尽的杀机。
<br/>　　娇美女子见此，心中不禁暗叹，她从未见过扬天成如此的憎恨一个人。
<br/>　　以前的扬天成，温文尔雅，气度不凡，让她痴迷。
<br/>　　但现在的扬天成，脾气变得相当暴躁，感觉性格都变得怪张了起来。
<br/>　　“天成，他应该不会去！”杨天威摇了摇头，在他看来苏莫不傻，明知道他们天虚宗要对付他，怎么可能敢去？
<br/>　　“也是！”扬天成闻言叹了口气，最近一段时间，根据宗内派出的人查探，苏莫回到道宗之后，便一直没有再出来，想必短时间内是不敢出来了。
<br/>　　随后，三人便不再谈论苏莫，又开始谈论此次的拍卖会。
<br/>　　闲谈了良久之后，三人便也在客栈中住了下来，准备参加明日的拍卖会。
<br/>(本章完)
