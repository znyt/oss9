　　听着顾凉晨的话，叶永阳也深觉有理，渐渐放下了警惕心道：“那你们狼牙？”
<br/>　　听到这一带有怀疑的语气，顾凉晨不禁笑出声：“这年头还有谁敢明目张胆的打狼牙主意？更不可能有人能够破开狼牙的防护。”
<br/>　　充满自信的话响起在这一机舱内，让叶永阳愣了半响，之后他只能摇头苦笑：“我花了大把的时间远走高飞，一路躲到西部，甚至家里人还死了大半，结果却一无所获，到头来还是只能依靠某一势力的援手。”
<br/>　　叶永阳的经历无疑很凄惨，身上背负的命运更让整个叶家都陷入水深火热，他只是想保护家族，却被各方人马逼的走投无路。
<br/>　　顾凉晨顿了顿，皱了下眉问道：“虽然很突兀但我还是想问，白家到底想从你身上得到什么？”
<br/>　　叶永阳的双目炸闪，盯着前方的顾凉晨身影道：“你们狼牙基地，还有那个楚涵上将，难道真的不怕神秘家族，不怕白家么？你们这样把我接走，也瞒不了多久吧？”
<br/>　　“压根没打算瞒。”顾凉晨的嘴角勾起，笑容带着一丝邪气：“至于你说的‘怕’，那更是个笑话，或许你这几年都在奔波中不知道很多事，等你到了狼牙就会知道，当下唯一敢跟神秘家族正面对抗的，还有谁。”
<br/>　　霸道无比的话从顾凉晨口中说出，一改他平日里冷静低调的作风，显得耀眼无比。
<br/>　　叶永阳愣住，他的确信息获知的很少，但从顾凉晨的话里也能轻而易举的听出某个惊天事实。
<br/>　　“你们连白家都敢杠上？”叶永阳的声音无比震惊，简直不敢相信自己的耳朵。
<br/>　　“楚涵长官率领黑芒战队仅仅二十人，在上京基地杀的血流成河，在最后一秒救下了牧司令和洛老，撤退时就放了狠话，要跟神秘家族的白家和罗家不死不休。”顾凉晨淡淡的说着，思绪也渐渐回忆第一次听到这一信息时的心情。
<br/>　　叶永阳则是早已惊呆，呐呐出声：“现在的年轻人，真是狠啊……”
<br/>　　顾凉晨轻轻一笑：“您老也不差，一把年纪了不也还在与白家对抗？”
<br/>　　叶永阳沉默了一会儿，而后忽然道：“不仅是白家啊，我是在和四大神秘家族对抗。”
<br/>　　眼看话匣子已经打开，顾凉晨立即发问：“其他三家也参与其中？但他们并没有派人来截你啊！”
<br/>　　“那是因为叶子博跟随了白忧。”叶永阳不假思索的回答道：“其他三家没有露头，一来是没有找到叶家人接线，二来是想看白家先行动，他们随后抢好处吧。”
<br/>　　顾凉晨双眼眯了眯：“你不知道当下其余三家的状况？”
<br/>　　“恩？”叶永阳一声惊咦，问道：“什么状况？”
<br/>　　顾凉晨道：“肖家已经灭族，当下只剩下了一位家族成员，还是个有着一半普通人血脉的混血，不过因为是唯一，她也就代表了罗家，并且是返祖者。”
<br/>　　“罗家在此之前都定居海外，末世爆发后才迁移到华夏，家族人员数量庞大，高战力者却稀少，与白家关系匪浅，但也相互隐瞒颇多，不过他们家族还未出现返祖者。”
<br/>　　“高家就最有意思了，除了有一个返祖者，还有一个神女，家族成员不算多，但似乎人人都战力不俗，只不过这个家族向来不爱管事，一直处于隐居状态。”
<br/>　　“白家的情况你应当有所猜测，当下为四大神秘家族之首，家主不知所踪但也还在世，少族长白忧战力深不可测，还有个同样混血的返祖者，名叫白允儿，是家族嫡系与一个普通人生的女儿。”
<br/>　　听到这里，叶永阳的眼神充满了复杂情绪，显然是第一次听到这些：“你怎么知道这么多？难道现在的四大神秘家族，已经在世上不算秘密，家族内部情况都人尽皆知了？”
<br/>　　“当然不可能。”顾凉晨笑道：“这都是我们狼牙情报部掌控的资料，知道的人不超过十个。”
<br/>　　叶永阳震惊无比：“你们的情报系统这么厉害？怎么打探到的？”
<br/>　　“光靠狼牙情报部当然不可能知道这些，虽说狼牙的这一情报部门已经为华夏顶尖，但有关神秘家族的一切，都是我们长官亲自打探得来的。”顾凉晨解释着，紧接着道：“顺便一提，我们楚涵长官杀过罗家的一名正统血脉者，狼牙战团也战胜过罗家的一支半血统成员部队，我们并不是没有跟神秘家族交过手。”
<br/>　　叶永阳此时已经震撼的说不出话，与神秘家族交手获胜，甚至杀死正统成员的这种事，这一千年来整个叶家都不敢想，可在这个兵荒马乱的时代，竟然有人能做到？
<br/>　　那个传闻很不好听，黑历史一抓一大把，甚至风评极差的那个楚涵？
<br/>　　感受到后方叶永阳的震惊，顾凉晨又及时添了一把火，道：“对了，阴阳谷我们狼牙的人也去过很多次，并且第一批进入阴阳谷的人，其中两位就是我们狼牙战团的成员，两个人还都是战队队长。”
<br/>　　听到阴阳谷三个字，叶永阳差点惊得跳起来，一下子牵扯到身上的内伤钻心的疼。
<br/>　　看到叶永阳激动的样子，顾凉晨再次皱眉：“你又不知道，你到底在路上耽搁了多少时间？”
<br/>　　“两年……”叶永阳下意识的回答，而后连忙发问：“你们真的去过阴阳谷？你们……你们去了阴阳谷的那块地方？”
<br/>　　顾凉晨的笑容浮起的微不可查，故作思索的回答道：“不止我们去过，神秘家族的那三个返祖者也过去，还有一些进去探索的部队，不过他们大部分都只能踏足于阴谷，出来后并没找到阳谷的入口，极少一部分从阴谷的出口直接进入了阳谷，有人死在了里面，也有人从内安全走出。”
<br/>　　“至于我们长官嘛……”说着顾凉晨一顿，笑道：“楚涵长官出入阴谷阳谷很多次，他探索过的很多地方我们听都没听说过，什么白家旧址，水族沙滩，制裁之地，制裁牢笼之类的，哦对了，还有水族墓地……”
