　　整个帝玄山脉，虽然聚集了亿万武者，但却寂静无声，非常的安静。
<br/>　　所有的人，俱是抬头凝望着高空上的青年，这位已经站在巅峰，敢向武帝邀战的青年。。
<br/>　　很多认识苏莫的人，或者和苏莫有过交集的人，都有些恍惚。
<br/>　　仿佛只是转瞬之间，苏莫便从一个后辈天才，成长为了巅峰强者，速度快到众人难以适应。
<br/>　　苏莫静静的盘坐在虚空中，他闭上了双眼，凝神调息。
<br/>　　炼制十五柄本命灵剑，消耗了很多精血，让他略有疲惫，精神有些许萎靡，所以，他要在最短的时间内，恢复到巅峰状态。
<br/>　　时间缓慢流逝，整个帝玄山脉又逐渐的喧嚣了起来，嘈杂的声浪直冲云霄。
<br/>　　“我观苏莫好像底气十足啊！”
<br/>　　“底气有什么用，他根本不清楚武帝强者的恐怖！”
<br/>　　“以老夫的眼力，苏莫的实力虽然，但与武帝强者相比，差距还是很大，这一战没有什么悬念了！”
<br/>　　“……”
<br/>　　无数的武者在谈论，很多人甚至已经认定了结果。
<br/>　　在众人的心中，武帝是苍穹世界的神话，是至高无上的武道大能，绝对是所向武帝，横扫一切。
<br/>　　苏莫听着众人的议论，虽然自己不被看好，但他的心境平静如水，不受众人的影响。
<br/>　　他控制着自己的心境，只有冷静的头脑，才能发挥出巅峰的战力，才有希望打败巫森。
<br/>　　蓦地，苏莫好似想到了什么事情，突然睁开了眼眸。
<br/>　　眼眸转动，他找到了战乱，目光落在了远处山巅之上战乱的身上。
<br/>　　“战兄，待我斩杀巫森之后，你我去苍穹神宫一叙！”苏莫朗声开口，声音洪亮传遍八方。
<br/>　　战乱闻言一怔，疑惑的望着苏莫，不知道对方为何邀请他去苍穹神宫！
<br/>　　他和苏莫可不熟，只是有一战的交集而已。
<br/>　　而此刻，帝玄山脉的无数武者，听闻苏莫的话，均是面带愕然之色。
<br/>　　苏莫此言意味深长啊！
<br/>　　斩杀巫森？
<br/>　　难道苏莫如此自信？自认为他的实力比武帝强者还要强大！
<br/>　　这怎么可能？
<br/>　　而且，就算苏莫真有武帝的实力，也不可能斩杀巫森吧！
<br/>　　毕竟，打败和斩杀是两回事！
<br/>　　这到底是自信还是自大？
<br/>　　众人心中惊疑不定，别说苏莫有斩杀巫森的实力，就只是说有打败巫森的实力，众人都不太相信！
<br/>　　战乱闻言沉默了片刻，点头道：“此战你若不死，我会去苍穹神宫！”
<br/>　　虽然不知道苏莫找自己何事，战乱还是答应了苏莫，但却有一个前提，那就是苏莫能在此战中不死。
<br/>　　若是苏莫此战不死，就算败了，也说明有了抗衡武帝的实力。
<br/>　　到那时候，苍穹世界将会是苍穹神宫和黄泉魔宗分庭抗礼的局面，他们自然要选择站队。
<br/>　　相比于黄泉魔宗，战乱和战神殿，更倾向于苍穹神宫。
<br/>　　其一是苏莫天赋强大，潜力无可估量。
<br/>　　其二是黄泉魔宗乃是魔道势力，有何试炼者有千丝万缕的关系。
<br/>　　“嗯！”苏莫闻言点了点头，随即一句话也不愿多说，再次闭上了眼眸，静心调息。
<br/>　　苏莫抛却了一切杂念，心静如水，脑中还在暗自琢磨着二十四星宿剑阵。
<br/>　　时间缓缓流逝。
<br/>　　一个时辰又一个时辰，约莫三个时辰之后，天地的尽头突然出现了一片黑色。
<br/>　　唰！
<br/>　　黑色刚刚涌现，苏莫便第一时间睁开了双眼，凝目望了过去。
<br/>　　这一片黑色铺天盖地，是一道道黑袍身影，密密麻麻仿若蝗虫一般。
<br/>　　这是黄泉魔宗的人来了！
<br/>　　这些黄泉魔宗之人，人数不下五千人，而最让人震惊的是，这些人全部都是武皇境强者。
<br/>　　帝玄山脉聚集的无数武者，自然也看到了这些黄泉魔宗之人，一看之下纷纷心神巨震，倒吸凉气。
<br/>　　黄泉魔宗，居然有这么多武皇强者？这怎么可能？
<br/>　　如此多的武皇强者，怕是足有其它所有势力之和了！
<br/>　　黄泉魔宗居然强大到了这等地步！
<br/>　　而在五千名黄泉魔宗强者的前方，是四名准帝强者，分别是二魔老、阳鼎天、殷棕和北堂平华。
<br/>　　至于大魔老，却并在此，可能留守九幽魔渊坐镇。
<br/>　　苏莫也是心中震惊，黄泉魔宗果然强大，若不是准帝强者数量不够，以前就能横扫整个苍穹世界了。
<br/>　　苏莫目光转动，并没有发现黄泉魔宗宗主巫森，因为这些人之中，并无武帝强者。
<br/>　　不过，没有看到巫森，苏莫却是看到了两个熟人，巫山和皇甫擎。
<br/>　　此刻的巫山，修为也有增长，赫然是和苏莫同样的境界，武皇境四重的修为。
<br/>　　对此，苏莫没有什么意外，但对方的实力在他如今的眼中，已经不值一提。
<br/>　　让苏莫稍稍有些意外的是皇甫擎，他当初已经废除了对方的修为，对方居然还有实力！
<br/>　　“气力？”
<br/>　　苏莫的目光落在了皇甫擎的身上，仔细的打量了一番，立刻发现对方的血肉中蕴含大量的气力，在穿梭游走。
<br/>　　这就说明，对方是炼体强者！
<br/>　　“太古龙象诀！”
<br/>　　苏莫在皇甫擎的身上感受了熟悉的气息，对方的气力居然和他一模一样，正是修炼了太古龙象诀练就的气力。
<br/>　　这让苏莫不仅心中一沉，对方居然也练就了太古龙象诀。
<br/>　　转念一想，苏莫便明白了，对方必然是从李峰身上得到的此诀。
<br/>　　毕竟，李峰曾是阶下囚，又被巫山占据了肉身。
<br/>　　苏莫眸中陡然杀意闪动，凌厉冰寒的眸光，如同刀锋一般的扫视巫山和皇甫擎。
<br/>　　不过，巫山和皇甫擎却是丝毫无惧，和一众黄泉魔宗强者来到帝玄山脉外围伫立，目光和苏莫对视，眼眸中还有冷笑之意。
<br/>　　苏莫心中杀机大起，就欲先行击杀这两人，但就在此刻，他身前数千丈外的空间，突兀的波动了起来。
<br/>　　嗡~
<br/>　　空间一阵轻微的波动，随即，一名高大的身影突兀的出现了。
<br/>　　这具身影身穿紫黑色长袍，是一名四十岁许的中年人，此人身形挺拔如松，脸盘消瘦，鹳骨高凸，给人一种阴森狠辣之感。
<br/>　　此人的眼眸，漆黑如墨，又如同一个无尽的深渊，让人不敢对视。
<br/>　　最主要的是，此人的额头之上，有一个黑色的三叉戟标志，和苏莫曾在二魔老头上见过的一般无二。
<br/>　　此人，正是黄泉魔宗宗主，巫森。
