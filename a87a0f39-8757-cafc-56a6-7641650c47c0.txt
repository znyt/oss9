　　过去，子午还是GHOST身份的时候有着很大的烟瘾，一天两包烟都算是少的。子午曾经一度认为，自己如果不是因为行踪败露，被逮捕最后死在暗无天日的牢房中，就是死在香烟上；而子午个人更倾向于死在香烟上这种可能，因为在赵天宇找到他之前，子午并不认为有人能抓到他。
<br/>　　加入龙隐部队，脱去传奇黑客GHOST的身份，成为“子午”后，他逐渐把烟戒掉了。子午已经不记得上次抽烟是什么时候了，当辛辣的尼古丁从自己的肺部循环而过，子午忍不住居然忍不住咳嗽了起来。子午看着夹在自己手指间的香烟，嗤笑一声，没想到自己还会有抽烟呛到的一天。
<br/>　　烟头一明一暗，闪烁着微弱的光亮，没多久，就结了一段烟灰，手一抖，吧嗒，烟灰从香烟上掉落下来，摔得粉碎。子午把烟掐灭在桌子上，活动了一下手指，看着液晶屏幕上的摄像头，说道：“来吧，SHADOW。不，影天使！”
<br/>　　电脑屏幕上，再次出现一个十秒钟的倒计时，而在十秒钟倒计时出现的时候，放置在房间角落里的塑胶炸药上的指示灯从红色变成了绿色，并且随着倒计时开始闪烁，进入到启动状态。
<br/>　　当电脑屏幕上十秒钟倒计时归零，键盘背光灯再次亮起，键盘重新解锁的时候，塑胶炸药上的计时器也同时启动。
<br/>　　影天使立刻在键盘上敲打起来，开始攻击自己这边门锁中，子午设置的防御程序；而反观子午那边，原本应该是分秒必争的时刻，子午却是又从烟盒中抽出一支烟，塞到嘴里，用打火机点燃，慢慢的抽着。有之前那支烟习惯后，子午抽这支烟，没有再被呛得咳嗽。
<br/>　　“这个时候，你还有心思抽烟，似乎是对自己很有信心？”影天使抬头扫了眼墙壁上的液晶屏幕，问道。
<br/>　　“是的，非常有信心。”子午吸了一口烟，缓缓的吐出烟圈，说道：“你一定会输！”
<br/>　　子午抽完烟，把烟头掐灭，开始在键盘上敲打起来。
<br/>　　“GHOST，最后十分钟！”影天使继续着自己的垃圾话，说道：“你放弃了自己最后的机会，实在是太可惜了，本来你可以有比现在更好的前程。”
<br/>　　子午双手在键盘上飞快的敲打着，根本不在意影天使对自己说得那些垃圾话，很随意的说道：“你有自己的国家吗？”
<br/>　　“这种无聊的问题有什么意义吗？”影天使双手不停，在键盘上飞快的敲打着，反问道。
<br/>　　“有意义。”子午回答道：“曾经我也认为有没有自己的国家根本无所谓，对于我们这样的人就像是一朵蒲公英，随风而飘，随风而落，落在什么地方，就在什么地方生活，不管在哪儿其实都一样。可是直到他们找到我，把我从危险中救出去，带着我回国，我才知道回到自己的国家，回到自己出生的地方，能用自己所掌握的东西帮助自己国家做一些事情是多么美好。”
<br/>　　“愚蠢！”
<br/>　　影天使不屑地嗤笑，可是话音刚落，他就察觉到了不妙。这次跟子午之间的对决，不仅是进攻而且还要防御，影天使自然是在自己电脑上准备了监控程序，来监控子午那边门锁被攻击程度；现在过去了五分钟，但是自己设置的防御程序却没有遭到攻击，这让影天使感到很不对劲，以子午的水平，不可能过了五分钟，自己设置的防御程序还没有遭到攻击。
<br/>　　“GHOST，你到底在干什么！？”
<br/>　　影天使对着子午大声质问，从子午现在的动作，影天使已经意识到了情况的变化，察觉到了不妙。
<br/>　　“影天使，不，还是叫SHADOW比较顺口一点，毕竟这个名字叫了这么长的时间，一时间改过来还不习惯。”子午双手在键盘继续飞快的敲打着，以惊人的APM完成输入，对影天使问道：“你有学生吗？”
<br/>　　“学生？”影天使嗤笑一声，说道：“GHOST你是不是当兵当傻了？你居然会去收学生？我们之所以无法被超越，是因为别人会的东西我们都会，而我们会的东西别人不会！独一无二！”
<br/>　　“SHADOW，我们都会老都会死，等到我们老去的那天，就会有人取代我们。”子午摇头说道。
<br/>　　“但是在被人成功取代之前，我还是最强的那一个。”影天使嗤笑道：“而且就算是被人取代，我所留下的影响也无法被取代，传奇终究是传奇！”
<br/>　　“因为你没有学生，所以你这次输定了！”子午异常肯定的说道。
<br/>　　“不知所谓。”影天使轻轻摇头：“现在说输赢，是不是太早了一点？谁输谁赢，不到最后一刻，谁说得准呢？”
<br/>　　影天使加快自己的进攻，可是子午的防御却像是铜墙铁壁一样，不断化解着他的进攻。影天使觉得自己的进攻就像是丢入水塘中的石块，仅仅只是出现一片水花，紧接着就消失不见，根本看不到任何的成效。不管自己的进攻如何紧密，就算是撕开了一个口子，也很快就被子午给防御下来，而自己设置的防御程序却还没有遭到进攻的痕迹。
<br/>　　“GHOST，你到底想要干什么！？”影天使的情绪出现了很大的波动，对着子午大声质问。
<br/>　　“赢你！！”子午微笑着回答，说道：“我们两个人其实各有所长，你更擅长红域的进攻，而我更擅长蓝域的防守，如果我分心既要进攻同时还要防守，说不准到底谁会赢谁会输，可是如果我放弃进攻，只注重于防守，你觉得在十五分钟内你有把握从你那间屋子里逃出去吗？我们交手过这么多次，应该知道‘绝对防御’……”
<br/>　　“绝对防御？这是绝对防御！？”
<br/>　　影天使手速进一步提升，盯着电脑，摇头道：“不对，这不是‘绝对防御’，‘绝对防御’虽然厉害，但不是无法破解，这么长时间我一直在研究你的‘绝对防御’，这根本不是‘绝对防御’。”
