夏振国可从来没有想过去幽兰苑坐坐，再见见自己所谓的这个女婿。
<br/>
<br/>        幽兰苑处在靠山间的位置，根本就不在繁华市区，他总觉得那里是一个鬼地，不吉利。
<br/>
<br/>        夏振国迅速为难道，“妍妍，我就不去了吧，我那个女婿有什么好见的，都说他已经病入膏肓了，说不定过两天人都不在了，我不想去。”
<br/>
<br/>        这就是夏妍妍想要的，她必须将幽兰苑那个鬼夫曝光出来，这样舆论的雪球才会越滚越大，将夏夕绾给推到风口浪尖。
<br/>
<br/>        “爸，我是不知道绾绾是如何认识陆总，又如何骗到陆总的，像陆总这样的男人，就算绾绾离婚了，你不会真的以为陆总会娶绾绾吧？”
<br/>
<br/>        “当然不会，夕绾那是什么条件，陆总绝对看不上她的，现在顶多是尝尝鲜，妍妍，陆总最终要娶得肯定是你。”夏振国为表自己的忠心迅速慷慨激昂道。
<br/>
<br/>        而且他说的是真话，他从不认为陆寒霆是真的看上了夏夕绾。
<br/>
<br/>        “爸，你放心吧，我手里还有王牌，我对陆太太的位置志在必得，所以现在我需要你的帮助，这两天你就去幽兰苑见一见你那个女婿，然后将绾绾和陆总的事情说给他听，他虽然无权无势，但他能闹出一点动静最好，闹不出动静，如果他被气死了，那就更好了。”
<br/>
<br/>        夏振国点头，他后半辈子的希望都落在夏妍妍身上了，“妍妍，好，爸爸都听你的，只要你能嫁入陆家当少奶奶！”
<br/>
<br/>        夏妍妍心里冷笑，夏夕绾，如果你没有嫁入幽兰苑，或许你还能跟我抢陆寒霆，但是你已经嫁人了，还拿什么跟我抢？
<br/>
<br/>        ……
<br/>
<br/>        外面风起云涌，总裁办公室里一片昏黄宁静。
<br/>
<br/>        夏夕绾进了休息室，这里的休息室很大，而且低调奢华，一应俱全，一看就能猜到陆寒霆时常会在这里休息。
<br/>
<br/>        陆寒霆冲了一个冷水澡出来，一眼就看到了躺在他床上的那抹纤柔身躯，夏夕绾先洗澡的，现在她整个人钻在蚕丝被里，就露了一颗小脑袋出来，一双黑漉漉的澄眸像小奶猫般盯着他骨碌碌的转。
<br/>
<br/>        见他出来了，她还乖巧的往床里面挪了挪，给他腾床位。
<br/>
<br/>        陆寒霆掀开被子上了床，将她裹在身上的蚕丝被往下拉了拉，勾唇笑道，“这么害羞干什么，你会让我有一种你被子下没有穿衣服的错觉。”
<br/>
<br/>        “…”
<br/>
<br/>        夏夕绾一巴掌想招呼在他的俊脸上。
<br/>
<br/>        蚕丝被拉了下去，原来她穿了一件他的黑色衬衫，面纱已经摘了，清纯如丝绸般的秀发几缕缠在她的粉颈里，少女唇红齿白，顾盼流转，躺在他的床上，端的一个活色生香。
<br/>
<br/>        陆寒霆低头，“想亲你，可以么？”
<br/>
<br/>        夏夕绾察觉到了他不动声色的试探，换成平时，他都是行动派，直接亲上来的。
<br/>
<br/>        夏夕绾拎过被子藏住了自己的红唇，对着他摇了摇头。
<br/>
<br/>        不可以。
<br/>
<br/>        陆寒霆狠狠的盯了一眼她的红唇，“哦，那算了。”
