　　苏莫只感觉眼前场景一变，而后自己便出现在了一片广阔的荒原之上。
<br/>　　他转头打量四方，这荒原浩瀚无垠，无比无际，非常的辽阔。
<br/>　　荒原上长满了茂盛的植被，杂草丛生，还有一些低矮的古树，远处还坐落着一座座低矮的山峰。
<br/>　　“这是大祭司的空间宝器之中？”苏莫眸光精光一闪，那青璇会不会在此地？
<br/>　　苏莫立刻微微激动了起来，马上就想去寻找青璇。
<br/>　　但很快，他便压下了这个不切实际的念头，别说青璇不一定在此地，就算在此地的话，他也救不出去。
<br/>　　一旦暴露身份，必死无疑！
<br/>　　而且这里既然是大祭司的空间宝器之中，肯定都在大祭司的窥视之下。
<br/>　　现在他的目标已经快要完成了，不能因为急躁，而功亏一篑！
<br/>　　见四周没有半个人影，苏莫猜测他们参加这第三关考验的人，估计都被分开了。
<br/>　　而后，他努力静下心神，盘膝而坐，开始学习武技。
<br/>　　少司之位近在眼前，他要全力以赴，一举拿下。
<br/>　　苏莫仔细感受脑海中的武技，大祭司所传的武技是一种剑法，名为《天剑诀》。
<br/>　　不过，施展此剑法并不需要剑，因为此剑法讲究的是天地万物皆可为剑。
<br/>　　一草一木，一花一石，所有的一切，都能成为手中之剑。
<br/>　　“好玄妙的剑法！”苏莫心中惊叹，这巫族大祭司，居然能创出如此玄妙的剑法，不愧是武圣境的超级强者。
<br/>　　随即，苏莫仔细的参悟这门剑法，他发现这门剑法并没有等级之分，而且修炼难度并不是很大。
<br/>　　而这门剑法的威力大小，是根据施展之人的修为、控制力，以及对于剑法的感悟来决定的。
<br/>　　换言之，这门剑法修炼到大成之时，可以很强，也可以很弱，完全看施展之人的本事。
<br/>　　少倾之后，苏莫将剑法研读了几遍，并没有急着修炼，而是在脑中仔细的揣摩，揣摩其中的玄妙。
<br/>　　说实话，这门剑法很简单，知晓其中的原理之后，他便能立刻施展出来。
<br/>　　但是，想发挥出强大的威力，却是有些难度。
<br/>　　“起！”
<br/>　　静静的揣摩了一个时辰，苏莫轻喝一声，手掌一扬，他周身百丈之内的石子，枯叶，杂草等等都飘飞了起来。
<br/>　　“去！”苏莫一指前方，无数的石子和枯叶顿时如同疾驰的闪电，****了出去。
<br/>　　噗嗤！噗嗤！！
<br/>　　轰隆隆！
<br/>　　石子和枯叶等物，将数里之外一颗较大的古树射成了筛子，而后那颗古树轰然倒塌，溅起一片尘土飞扬。
<br/>　　“威力太低了！”苏莫摇了摇头，这种万物为剑的方式，他这个级别的武者基本上都能施展出来，但根本没有什么威力。
<br/>　　这种剑法，是以特殊的方法让得精神力调动万物，而后，同样是以精神力为引，将力量导入万物之中，从而形成强大的威力。
<br/>　　前者较为容易，后者则稍难不少。
<br/>　　随即，苏莫继续练习，一次次的施展天剑诀，周身千丈之内的各种碎石、枯叶等物，都被他毁灭的一干二净。
<br/>　　随着不断的练习，他对于这天剑诀的感悟越来越深，每一次施展此诀，威力亦是越来越大。
<br/>　　嗖！
<br/>　　只见苏莫飞升了高空之中，双手一扬，方圆百里之内无数乱七八糟的杂物通通漂浮了起来。
<br/>　　而后，在苏莫精神意念的控制之下，这无数的杂物仿佛化为了一柄柄娟秀小剑，洞穿长空，向远处一座小山爆射而去。
<br/>　　无数的小剑，密密麻麻，数之不尽，仿佛是一阵剑雨，铺天盖地的涌向小山。
<br/>　　下一刻。
<br/>　　轰！
<br/>　　一声爆响震天动地，只见千里之外那座两百余丈高的小山顿时被刺爆，化为了漫天碎石，洒落八方。
<br/>　　“不错！应该有小成的地步了吧！”苏莫微微一笑低声自喃，目前才过去了一天的时间而已，还剩下两天时间呢。
<br/>　　两天之内，修炼到大成境界，问题不大。
<br/>　　而且，这天剑决也很有讲究，他适才那种控制无数‘利剑’的方式，对于武者的要求和控制力非常的高。
<br/>　　也是他精神力强大，才能做到这一步。
<br/>　　他若是只控制少量的物体，这天剑决的威力还能更上一层楼。
<br/>　　随即，苏莫换了一处地方，继续练习天剑诀。
<br/>　　时间流逝，他的天剑决施展的越发纯属，威力越来越大，到得最后他全力施展的情况下，都快接近他本命灵剑阵的威力了。
<br/>　　虽然这个威力不算太强，但也非常的可观了！
<br/>　　苏莫之所以能修炼这么快，一是因为他本来就是剑修，对于剑的领悟自然多一些，二是他精神力强大无比、悟性超高，三则是此天剑诀修炼难度不高。
<br/>　　不过，他也不清楚目前算不算达到了大成的境界，但他感觉施展起来已经浑圆如意，一切都行云流水。
<br/>　　三天时间转瞬即过，某一刻，正在练习天剑决的苏莫，突然感觉到一股浩大的意念突然将他笼罩。
<br/>　　下一刻，他的身形突兀的消失，便被强行拉出了这处空间。
<br/>　　唰唰唰！！
<br/>　　圣巫广场上，人影闪烁，六十一名参加第三关考核的天才，纷纷出现在了广场上。
<br/>　　广场上与三天前没有任何变化，所有的人都在等待，等待第三关的结束。
<br/>　　见到苏莫等人出来，所有人都激动了起来，谁能通过考验，马上就要见分晓了。
<br/>　　谁能成为新的少司，说不定这一关就能定夺了！
<br/>　　大祭司的深邃的目光，在六十一名天才身上一一扫过，当扫到苏莫之时，微微停顿了一瞬，眸中闪过一丝微不可查的赞赏。
<br/>　　“好了，三日时间已过，接下来便是检验你们成绩的时候了！”大祭司朗声宣布道。
<br/>　　言罢，大祭司略一沉吟，继续道：“自认为没有将《天剑决》修炼到大成的人，自行离场吧！免得丢人现眼！”
<br/>　　大祭司此言一出，六十一名天才之中，立刻有不少人脸色变换，面现迟疑之色。
<br/>　　少倾，有四十五人缓缓离开，脸色黯然的走到了一旁。
<br/>　　这些离开的人，都有自知之明，清楚自己的修炼进度，别说修炼到大成境界了，连小成境界都不到。
<br/>(本章完)
