　　常言道，人不为已天诛地灭，那些海岛上的土著虽然并不知道这句话，但却知道其中的道理。
<br/>　　毕竟谁也不想自己头上多出一个‘太上皇’管着自己，由其是像少族长这样的更不想。
<br/>　　因为如此投降了大唐，那么大唐一定会派人管着他们，这样的话，他们的老爹就‘降’了一级，而他们这些‘土二代’们，当然也要跟着降，这是他们绝对无法接受的。
<br/>　　可是，这些土著并不知道，其实外面那些人只是‘路过’的，就算是他们想要投降，他们都不一定会接受。
<br/>　　因为在大唐看来，他们这些岛上的土著就跟秦岭里面随处可见的猴子没有什么区别，有谁会在乎自己领地里有一群猴子么？当然不会！
<br/>　　没人会费尽心思去教一群猴子说话，然后再教他们如何使用工具，大唐的官员都忙的很，没这个外国时间来陪他们玩。
<br/>　　所以他们并不知道，其实他们现在所在的岛已经是大唐的领土，他们这些人已经是大唐的……化外之民，投降是根本不存在的，当他们丢出手中第一根长矛的时候，他们就已经上了李二的必杀名单。
<br/>　　……
<br/>　　树木的砍伐工作还在继续，该死的热带雨林实在太适合植物的生长了，砍掉了一颗树，第二天这里就会被一大堆的藤蔓覆盖，再次变成一片绿色的海洋。
<br/>　　来自北方的禁军哪里见过这样的奇景，一个个都看的有些傻眼。
<br/>　　他们从来就没见过植物以肉眼可见的速度生长是什么样子，可是在这片热带雨林，他们终于开眼了，可是开眼界的同时，却让他们头大如斗。
<br/>　　生长如此快的植被，这特么要怎样才能开出一条路来？
<br/>　　最后，还是独孤青云派出自己的手下才解决了这个问题。
<br/>　　那些生于南海的家伙们早就已经习惯了这样的环境，登岛之后没用多少时间便已经清理出了很大一片空地，大量的植物被连根拔起，这样它们就不会再长出来。
<br/>　　同时，在清理空地的时候，这些海盗出身的家伙们还清理出近二十处简陋的陷阱，这让尉迟恭和秦琼两个看的毛骨悚然。
<br/>　　这还只是一小片区域，那些土著就布置了如此多的陷阱，如果回头进入丛林深处会怎么样，不问可知。
<br/>　　所以在第二天登岛之后，两人一起找到了李世民：“太上皇，这岛上太危险了，为了安全起见，您还是回到船上去吧。”
<br/>　　“朕不回去，数十年来什么样的危险朕没有见到过，区区土著又能把朕如何？”老李倔强的摇了摇头，叉着腰深深吸了一口带着雨林味道的空气，用半真半假的口吻说道：“这里的环境不错，朕以为完全可以在这里建一座行宫！”
<br/>　　建行宫？如果是程咬金在这里一定会说：您可拉倒吧，咱又不是猴子，在这个满是蚂蝗、虫蚁的破地方根本就呆不了好吧，到时候只怕用不了三天就能因为水土不服踏上西天路。
<br/>　　可是秦琼和尉迟恭都不是什么嘴皮子利索之人，心中尽管有些不赞同李二的主义，但却也没有再说什么，只是抱了抱拳领命离开，然后去找独孤青云。
<br/>　　“什么？两位将军是说陛下要在这里建行宫？”独孤青云听了秦琼的话之后也有些发懵，搞不清楚这位前皇帝陛下到底要闹哪样。
<br/>　　“怎么样，你觉得可行么？”秦琼也很无奈，不过李二说了他就得去办，必竟太上皇那也是皇，说出来的话那就是圣旨。
<br/>　　“可行道是可行，只是在这里建行宫没有三五年时间根本连地基都打不完，昨夜两位将军也看到那些植物的生长速度了，这个工程到底有多麻烦想必也清楚，更何况这里不产粮食，食物的问题也很难解决……”独孤青云到底是在南海混了十多年的人，经验比较丰富，一眨眼就提出了无数根本无法解决的困难。
<br/>　　秦琼和尉迟恭无奈的对视着，最后齐齐叹了口气，决定回去找李二把这件事情说一下。
<br/>　　……
<br/>　　而另一边，独孤青云在老秦两人离开之后，也叫来了百十个手下，将他们分成十队，然后又安排他们与禁军配合，向着丛林深处摸了过去。
<br/>　　太上皇有令，将昨日袭击船队的那些土著全部抓来，这才是独孤青云的首要任务。
<br/>　　于是乎，只带着短枪，也就是左轮手枪的禁军跟在那些‘民团’的后面摸进了丛林，小心的向着未知的方向摸了过去。
<br/>　　雨林里面树木和藤蔓要比海边茂盛的多，拿着那种长长的步枪根本就转不开身，所以那东西根本用不上。
<br/>　　铁菠萝同样不能用，原因同样是树木太密集，你把那东西丢出去，一个不好就有可能砸到什么东西再反弹回来，到时候敌人没炸死，自己就能先上西天。
<br/>　　所以他们能依靠的只有刀和手里的左轮手枪，再有就是在战场上留下的对危险的感知能力。
<br/>　　不过这已经足够了，对付一些土著已经有了完胜的把握，只要小心周围的环境，不要被那些莫名其妙的东西咬到，不要掉进那些隐蔽性极好的陷阱里面，他们这一次的任务便可以称为一次旅行。
<br/>　　可怜的土著们本以为自己占着地利，就算是唐军想要攻击他们也要等到数日之后，到时候他们已经可以做出是战是和的决定。
<br/>　　可是他们万万没想到，这次来的并不只有大唐的军队，还有一批与他们一样熟悉从林的‘民团’，在那些由南海海盗转正而来的‘民团’带领下，十支小队没费什么力气便已经接近了他们的部落的中心，一点一点将他们全都围了起来。
<br/>　　而此时的土著却依旧在争论着到底要不要投降的问题，突然间一声惊叫响起：“&amp;*）”接着是就‘嘭’的一声巨响。
<br/>　　唰的一下，不管是先知还是部落的族长，脸色瞬间就变了，因为刚刚那个声音分明就是在叫着：“唐军来了！”
