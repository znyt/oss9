　　叶少阳脑海中隐约闪过了一些奇怪的画面。海水，巨峰，四处纷飞的鬼魂和元神……
<br/>　　橙子元神和灵骨的力量，已经逐渐消融，灵力网离他的头顶只有两三尺高，而钟馗的帅旗，离突破最后一层灵力网，仍然还有一丝距离。
<br/>　　钟馗已经喝了好几口酒，骂了好些句三字经，仍然努力挺近帅旗……
<br/>　　如同火堆烧成灰烬之前，最后闪现出的一丝火光——橙子的元神之力爆发出最后一丝灵光，然后一切暗淡下去。
<br/>　　灵力网瞬间落下，缠绕上叶少阳的头顶……
<br/>　　恰在这时，一道紫气，从叶少阳背包里喷射而出，是一道人影，出来之后，立刻高举双手，生生托住了下落的灵力网。
<br/>　　是一个须发皆白的老道士，撑住灵力网之后，自己灵身也立刻颤抖起来，感到了巨大的压力，转头环视了一周，立刻辨认了处境，发现了冲击灵力网的帅旗所在，犹豫了一下，猛地冲过去，双手各擒着一波紫气，用尽全力，将最后一层灵力网撕开了缺口。
<br/>　　因为失去了抵抗，灵力网瞬间落下来，几乎将叶少阳裹住，好在帅旗也在一瞬间捅了进来，旗面展开，将叶少阳裹了进去。老道士也趁机从裂缝中飞了出去。
<br/>　　灵力网对帅旗这种非生灵的死物，并没有什么杀伤，更不要说这帅旗是钟馗的贴身法器，不可能损毁，钟馗稍一作法，将帅旗抽了出来，也是长出了一口气。
<br/>　　“他娘的，总算是对得起老崔了。”钟馗抽身急退，回身看去，这才注意到后方的战斗情况：萧逸云等人以多敌少，居然被逼的险象环生，如果不是人多，可以轮流抵挡，几乎就顶不住了。
<br/>　　这一切，都是因为那个黄脸男子，修为深不可测。
<br/>　　钟馗凝神观察了一会，摇头谈道：“一群废物！”
<br/>　　眼看萧逸云几乎要吃个大亏，钟馗飘身上前，用旗杆挡住了黄脸男子的重重一击，萧逸云等人狼狈退开，转头朝北斗七星阵中望去，一看没人了，忙问道：“钟天师——”
<br/>　　“救出来了。指望你们这群废物。”转头朝恶黄脸男子望去，道：“我认得你是谁。”
<br/>　　黄脸男子也定睛看着他，道：“几百年不见，别来无恙。钟天师要来一战？”
<br/>　　钟馗哼了一声道：“今日之劫，却与我无关，我只是帮老崔走这一趟而已，顷刻便走。你们等会接着折腾。”
<br/>　　黄脸男子听他这么说，给张云和苏沫使了个眼色，一起停手，站在一边等着。毕竟谁也不想跟钟馗这种人做对手。
<br/>　　“萧郎君，接着你媳妇。”钟馗说着将帅旗打开，迎风一抖，一道人影落在地上，是叶少阳，怀里抱着橙子。
<br/>　　橙子面如死灰，一动不动。
<br/>　　妖跟人一样，有肉身、魂魄、元神，她的元神已碎，连魂魄也没有留下，像人类死去一样，只留下一副躯壳。
<br/>　　萧逸云本来满怀期待地想要看到橙子，结果看到这一幕，浑身一颤，上前推开叶少阳，一把抱住橙子，手搭在她脉门上，释放罡气检验她的身体……
<br/>　　足足愣了有十秒钟。
<br/>　　“橙子！”萧逸云发出一声撕心裂肺的惨叫，上前一把抓住叶少阳的领子：“怎么会这样，怎么会这样啊，你告诉我！”
<br/>　　叶少阳面如死人，任凭他要摇晃身体，像是完全没有听见。
<br/>　　橙子的死，他心中的悲伤，绝对不亚于萧逸云。
<br/>　　钟馗在一旁也是懵了，之前破开灵力网的瞬间，情况紧急，帅旗只顾得上往里面一裹，根本没有注意到叶少阳二人是死是活。
<br/>　　“缘身俱灭，都已经散去了，没得救了。”广宗天师望着橙子的肉身，苍然说道。
<br/>　　众人这才注意到他的存在。
<br/>　　他身后站着林三生，望着橙子，也是一脸的茫然和哀伤。虽然救下了叶少阳，然而……
<br/>　　之前，在被阵法困住之后，林三生就立刻认清了现状：依靠自己三人的力量，根本不足以自救他。他并不知道钟馗和天子殿的人回来，他以为会是道风或者是谁来搭救，不过结果也都是一样，因此才让叶少阳打开山河社稷图，自己钻进去，当时不是为了避祸，而是去找他的师父广宗天师。
<br/>　　广宗天师在曾经血菩提留下的妖宫基础上建了道观，开宗立派，位置离山河社稷图的出入口比较远，他一进图中，立刻去寻找师父，结果还是耽误了一些时间，好在最后关头及时赶到，帮叶少阳争取到了一线生机。
<br/>　　林三生只是万万没想到，橙子会死……
<br/>　　“呵！”
<br/>　　一声爆喝，从山顶方向传来。
<br/>　　灵力碰撞产生的余波，将整个山顶笼罩起来，向着周围扩散出去。
<br/>　　很多修为略差的星宿海弟子，甚至被余波冲下了山谷。几乎所有人都停止了打斗，朝着同一个方向望了过去——
<br/>　　漫天梨花，如同寒风中雪花，铺天盖地旋转着，中间黑光缭绕，爆发出了一次又一次的撞击。最后一次可怕的撞击结束后，一切逐渐尘埃落定。
<br/>　　梨花迅速散落，最终出现了两道人影，道风头发散落，衣衫褴褛，身上多出无数道小的伤口，不过神情看上去仍然是无比坚毅，右手提着打神鞭，正对着黎山老母的眉心处。
<br/>　　黎山老母瘫坐在地上，情况看上去更加糟糕，微微抬起头望着道风，脸上尽是震惊和不可思议。她没想到，自己居然会输……
<br/>　　“我，大意了。”黎山老母眼神暗淡地长叹了一口气，喃喃说道，“你为什么不动手灭了我？”
<br/>　　“因为你是青牛转世，杀了你，我所做一切都没意义了。”道风说完，猛然栖身而上，一只手按住黎山老母的肩膀，目光在人群中寻找到摇光仙子，说道：“关闭阵法。”
<br/>　　摇光仙子眼中寒光隐现，她也是完全没有想到，自己的师姐，黎山老母会败在道风的手上。道风一只手按着黎山老母的肩膀、拿她来做人质这件事，更是让她感到强烈的羞辱。
