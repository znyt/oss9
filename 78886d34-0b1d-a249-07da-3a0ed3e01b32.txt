　　当天照3000飞速行驶的过程中，南署基地的九万人队伍，则是已经进入了最为紧张的状态里。
<br/>　　一百架战斗直升机稳稳的停在基地内部的各处楼顶之上，每一名军法处的驾驶员身旁，都坐着来自射击队的射击手，他们在以最快的速度熟悉直升机，这与平时手握枪支的感触完全不同，还需要更强的集中力。
<br/>　　这些人就这么吃喝拉撒都在房屋楼顶，随时随地可以进入直升机内起飞，因为谁也不知道大尸潮到底什么时候来，或许明天或许后天，也或许就是下一刻。
<br/>　　除却这些房屋顶部的人，基地内圈已经呈现为迷宫状的大·片房屋内，正是那五万新兵的所在地，后勤部将大批大批的食物和水放置在各处房屋中，恰好也是秦渊分配这五万人队伍的地点，整个基地的内圈每一处的最佳战斗地带，都被好几个小队占领。
<br/>　　游击战相对安全也简单，尤其是早已准备好的迷宫战区，和足足五万人的人数优势，让这里成为这座基地的最后一处防线。
<br/>　　其余射击队的成员也早已站在了四百个高台之上，与那些空战的成员一样，他们将在接下来的日子里，每时每刻都守着这里。
<br/>　　基地的外墙和内墙之间，已经有一万五千人准备就绪，他们排着队站在两墙各个通道的之间，守着这唯一可通过的道路。
<br/>　　防守队的全体成员，当下更是早已进入了外墙内部，他们每一人都待在了新武器的旁边，紧张无比的例行检查设备。
<br/>　　三战队的一千人也在随后一个个跳上外墙的最高点，站成一个圈成为了这个基地全方位的一个个点，他们瞭望那看起来什么都没有的远方，心里很清楚这是暴风雨来临前的平静。
<br/>　　其余的一万五新兵，则是全体待在内墙之后准备，和他们一起的还有分散在各处的侦查队成员。
<br/>　　面向南方的城墙后方，有一座极高的建筑，这同样也是经由苏行加以改造的地方，这里的瞭望台和射击队成员的高台一样高，站在这里往前能看到城外尸潮铺面而来的方向，往后则是将整个南署基地收入眼底。
<br/>　　这里作战指挥台。
<br/>　　何峰与几名眼中不满血丝的参谋部成员待在这里，屋外最靠近城墙的地方，则是站着沉默无声的蒋天庆。
<br/>　　“丧尸引诱剂已经安放妥当，两次检查完毕。”一名参谋部成员开口，将所有准备工作的最后一环汇报。
<br/>　　何峰点头，擦了擦额间的汗水：“接下来就是等待尸潮来临。”
<br/>　　“长官还是没到。”忽的一人开口，带着一种令人难以察觉的落寞。
<br/>　　何峰拍了拍他的肩膀：“他的精神与我们并肩作战，我们都已经准备好了不是么？”
<br/>　　众人相互对望了一眼，而后就都将目光看向了外面的蒋天庆，而蒋天庆的目光则是紧紧盯着瞭望台前方的高耸外墙。
<br/>　　对于大尸潮正面进攻的方向，那一面高墙上站着的自然是团战冲击最强的杀羽战队，只不过在这几百名杀羽战队成员的中间，还站着神隐战队的队长路冰泽。
<br/>　　蒋天庆正是在路冰泽，而此时的路冰泽却是满脸的汗水，紧闭着双目面色惨白如纸，好似正在受着什么极大折磨。
<br/>　　徐锋一声不吭的守在路冰泽旁边，长达好几个小时的安静，彰显着他的心情并不好。
<br/>　　就在这时路冰泽睁开了眼睛，有些痛苦的开口：“什么都没有，什么都探索不到，我所能联系到的极限距离还没有丧尸出现，那些动物都还在回应我，但我派出往前探索的动物们全部有去无回，一点消息也没传回来。”
<br/>　　徐锋瞳孔一缩，道：“说明尸潮已经在往这里过来，怕是数量恐怖才会有这种情况，而距离……”
<br/>　　路冰泽摇头：“无法确定。”
<br/>　　徐锋点点头，而后转身对着身后远处高台比了个手势，紧接着就沉默不语的继续守在原地。
<br/>　　路冰泽在休息了一会儿之后，就再次闭上了眼睛，开始又一次的努力尝试探查敌情。
<br/>　　看到徐锋手势的蒋天庆表情越发严肃，继而转身进入后方指挥室，将消息汇报给何峰。
<br/>　　——————
<br/>　　天照3000停在高空，纯黑色的外表看起来充满了金属质感，此时控制着这个飞行物稳停在半空不动的高少辉，正满脸震撼的望着内部的大屏幕。
<br/>　　“这，这得多少丧尸？！”他结结巴巴的出声，冷汗都快冒了出来。
<br/>　　此时在大屏幕上，正是在天照3000下方的场面，那是成片成片的黑色，仿佛黑色的海洋无边无际，它们在朝着前方淹没，吞噬着所过之地的一切生灵。
<br/>　　从南向北，直面南署基地！
<br/>　　看到旁边的楚涵面无表情，高少辉急道：“你还在等什么，还不赶快过去通知，最多半小时，这尸潮就要到了啊！”
<br/>　　楚涵点头：“比我预计的还要早两天。”
<br/>　　高少辉近乎崩溃：“都这时候了你还在想这个？”
<br/>　　楚涵随手拨·弄了下屏幕方向，声音冷静：“再等等。”
<br/>　　“你还等什么？！”高少辉觉得楚涵不可理喻，但很快他的眼神就一变，惊愕的看着屏幕上的场景：“这，这又是什么？！”
<br/>　　楚涵的神情终于发生了一丝变化，吐出了四个字：“飞行丧尸。”
<br/>　　比天照3000低了一些的高度，正有大·片大·片的飞行丧尸铺面而来，它们有些是鸟兽，有些是昆虫，有些则是变异的奇怪物种。
<br/>　　但无一例外的是，它们都皮肤苍白，没有瞳孔，尖锐的牙齿阴森恐怖。
<br/>　　高少辉哪里见过这种场面，吓的当场全身发软，而后不管楚涵直接启动天照3000往前飞行：“快快快，我们快跑！”
<br/>　　楚涵却在这时候镇定的可怕，他似笑非笑的上下了看了高少辉一眼：“你们神秘家族的人，不能杀人，异种和丧尸也算是人形物种，所以没尝试过算不算禁忌吧？”
<br/>　　高少辉一边将天照3000急速往前行驶，一边随口答道：“谁有那个胆子拿自己的性命冒险？而且不怕告诉你，肖家的灭门就跟丧尸有关，罗家设下的陷阱就是尸潮，但是真实情况谁也不知道，谁叫肖家都已经死·光了。”
