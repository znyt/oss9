　　而剩下的人还有十六人，这十六人之中，除了苏莫之外，还有巫人王、巫玛、巫幽兰、巫穹、巫墨羽、巫罡天、巫天御，以及其他几名天才人物。
<br/>　　当然，没有离场的人，并不是说就将天剑决练到了大成，结果如何还需要检验一番。
<br/>　　苏莫微微有些意外，巫天御居然没有离场，难道对方自认为修炼到了大成的境界不成？
<br/>　　仿佛感受到了苏莫的目光，巫天御向苏莫望了过来，而后冷冷一笑。
<br/>　　不过，苏莫却在对方的眼中，看出了一丝忐忑之色。
<br/>　　这说明巫天御并不自信，天剑诀估计并没有修炼到大成的地步，或者说对方也不确定是否达到了大成的地步。
<br/>　　剩下的十六人，个个目光环视其他人，均是眸中露出凝重之色。
<br/>　　因为这最后一关，实在太重要了，很可能一举决定少司的人选。
<br/>　　不少人的目光，都是落在了巫人王、巫玛和巫幽兰三人的身上。
<br/>　　但三人的目光却是都投到了苏莫的身上，三人均是将苏莫当成了劲敌，没有人再敢小觑了。
<br/>　　“施展你们修炼的天剑诀，攻击老夫，逐一进行！”大祭司开口了，朗声说道。
<br/>　　他虽然依旧知道了每个人的修炼情况，但别人都不知道，肯定要展露一番。
<br/>　　“这……！”巫人王等十几名天才闻言有些迟疑，他们哪里敢攻击大祭司啊？
<br/>　　“尽管攻击，无须顾忌，就从你开始吧！”大祭司沉声说道，言语间手指一点，指向巫天御。
<br/>　　“是，大祭司！”巫天御见此立刻点了点头，随即他深吸一口气，准备施展天剑诀。
<br/>　　“起！”
<br/>　　少倾之后，巫天御轻喝一声，霎时之间，他周身百丈之内，广场上的一些细碎的石子，都漂浮了起来。
<br/>　　咻咻咻！！
<br/>　　众多的石子仿佛一柄柄小剑，向大祭司疾驰而去，发出刺耳的音爆声。
<br/>　　瞬息之间，众多石子便临近了看台，激.射到了大祭司的身前。
<br/>　　“不合格！”
<br/>　　就在此刻，淡漠的声音从大祭司的口中响起，随即只见其袖袍一甩，众多的石子便飞回广场，回落到了原处。
<br/>　　看台上的众多巫族高层，都不禁暗暗摇头，太弱了，也就小成的境界。
<br/>　　巫天御闻言脸色一白，难看不已，对于这个结果，他虽然已经预料到了，但还是非常的不甘。
<br/>　　但是，既然大祭司都说了不合格了，他还有什么办法！
<br/>　　叹了口气，巫天御转身离开了广场中央，走到了边缘处。
<br/>　　对于没能成为少司，他虽然有些不甘，但也没有太过失望，毕竟他知晓自己的天赋，根本不可能和巫人王等人竞争。
<br/>　　“下一个！”大祭司朗声道。
<br/>　　而后，另一名巫族天才立刻飞升而起，同样施展了天剑诀，向大祭司展开了攻击。
<br/>　　不过，此人的攻击虽然被巫天御稍强半分，但同样没有达到大成的境界。
<br/>　　“不合格！”
<br/>　　第二个天才同样被淘汰了。
<br/>　　“不合格！”
<br/>　　“不合格！”
<br/>　　“不合格！”
<br/>　　记下来的时间里，大祭司的声音不断的响起，全部都是不合格，没有一人能将天剑诀修炼到大成。
<br/>　　就连少司候选榜排名前列的巫墨羽、巫罡天，都纷纷失败。
<br/>　　这让广场上所有观看之人都暗暗心惊，这一关，才是最难的一关。
<br/>　　很快，便只剩下了苏莫、巫人王、巫玛、巫幽兰和巫穹五人了，其他人全部淘汰了。
<br/>　　“下一个！”大祭司的声音再次响起。
<br/>　　苏莫没有动，他喜欢等到最后，先看清别人的情况。
<br/>　　巫人王也是无动于衷，巫玛和巫幽兰同样没有动。
<br/>　　巫穹看了看四人，见四人没有出手的意思，他冷笑一声，脚下一踏身形冲天而起，瞬间来到了百丈高空。
<br/>　　而后，巫穹瞬间便施展了天剑诀，他周身方圆十里之内的一切杂物，地上的细碎的石子、空气中漂浮的柳絮，全部化为了道道微小的利剑。
<br/>　　无数的利剑爆射长空，裹挟磅礴的威势，向大祭司爆.射而去。
<br/>　　“不错！”
<br/>　　只见大祭司微微颌首，口中发出一声轻赞。
<br/>　　巫穹见此顿时心中一喜，但是大祭司接下来的话，立刻将他打入了万丈深渊。
<br/>　　“不合格！”淡漠的声音从大祭司口气传出，然后便见其一挥袖袍，所有利剑便原路返回，回落大地之上。
<br/>　　巫穹顿时脸色一沉，居然不合格？这怎么可能呢？
<br/>　　他感觉自己的天剑诀，应该已经大成了，没想到大祭司直接就说不合格！
<br/>　　这门天剑诀，因为没有级别之分，火候的深浅，也是不好定夺，所以他也不清楚有没有达到大成境界。
<br/>　　“退下吧！下一个！”大祭司挥了挥手，朗声说道。
<br/>　　巫穹心有不甘，本还想询问一番，但见大祭司一脸的淡漠，他不敢触及眉头，只好黯然的退到了一边。
<br/>　　他倒要看看，其他人是否能超越他，大成境界的天剑诀达到了什么地步！
<br/>　　“巫人王、巫尘，在下先一步了！”
<br/>　　这时，巫玛瞥了巫人王和苏莫一眼，言罢，便身形拔空而起，飞到了数百丈的高空之上。
<br/>　　“起！”
<br/>　　一声轻喝，从巫玛的口中响起，随即，方圆数十里内，无数细小的碎石、泥土、微尘，全部飞了起来。
<br/>　　无数细小的碎石、泥土和微尘，化为了一阵浩大的剑雨，爆发出冲天的锋芒，向大祭司铺天盖地的狂涌而去。
<br/>　　众人见此，顿时心中微惊，这巫玛的天剑诀，果然不同凡响，比别人强大的太多了。
<br/>　　承然，巫玛武帝境六重的修为，对于此剑诀的威力肯定有很强的加持，但即便如此，其火候也是相当的高了。
<br/>　　眼见着铺天盖地的剑雨，将要击中大祭司之时，大祭司一挥袖袍，所有利剑全部原路返回。
<br/>　　这一次，大祭司却是没有开口，面现沉吟之色。
<br/>　　巫玛见此，顿时心中有些紧张了起来，目光紧紧的凝视着大祭司。
<br/>　　整个圣巫广场上，所有的人都凝视着大祭司，等待大祭司宣布结果。
<br/>　　“哎！稍差一分！”
<br/>　　少倾，大祭司叹了口气说道。
<br/>　　虽然他很看重巫玛，但事实如此，巫玛的天剑诀，虽然无限接近大成的地步，但终究还是稍差一分。
<br/>　　巫玛闻言，顿时身形微颤，身前有些恍惚。
<br/>　　失败了！他居然在最后一关失败了！
<br/>　　努力了这么久，他就等着这一次机会呢！没想到最后还是失败了！
<br/>　　“巫玛大哥，你没事吗？”巫幽兰飞到了巫玛的身边，面现担忧的问道。
<br/>　　“我没事！”巫玛强自笑了笑。
<br/>　　“那就好！”巫幽兰松了口气，随即望向大祭司，道：“大祭司，我的天剑诀火候境界，还不如巫玛大哥，所以，我主动退出！”
<br/>　　“幽兰……！”巫玛闻言脸色一变，他可是心中清楚，对方也是很想成为少司。
<br/>　　“巫玛大哥，我有自知之明，算了吧！”巫幽兰摇了摇头道。
<br/>　　她说的是实话，她的天剑诀火候还不如巫玛，后者都不合格，她根本不可能合格。
<br/>　　“嗯！退下吧！”大祭司颌首，微微摆了摆手。
<br/>　　这时，所有人的目光，都落在了苏莫和巫人王的身上。
<br/>　　众人心中均是期待了起来，其他人全部都被淘汰了，现在只剩下了巫尘和巫人王两人了。
<br/>　　他们两人之中，到底谁能通过这一关？还是全部失败？亦或是全部通过？
<br/>　　众人对于巫人王有很大的信心，对于巫尘也是不敢小觑，毕竟巫尘前两关表现的极为抢眼，堪称一匹黑马。
<br/>　　苏莫转头看向巫人王，发现对方同样在望着他，两人目光对视在一起，顿时如同两柄利剑相击，火星四溅。
<br/>　　“我必胜！”巫人王眼眸微眯，声音冷然，但其中却充斥着绝强的自信。
