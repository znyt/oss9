　　“嗡嗡嗡……”
<br/>　　圣剑被公孙轩辕抓在手中，一阵翁响后，便像是认命般的停了下来，不再颤动，而是安安静静地躺在他手心。
<br/>　　显然，公孙轩辕已经通过了它的考验，得到了它的认可。
<br/>　　圣剑有灵，择主而事！
<br/>　　如今，公孙轩辕便是它的主人。
<br/>　　“好剑！”
<br/>　　轩辕仔细打量一番手中的金黄色圣剑，只觉得与自己乃是天造地设的一对，他顿时大喜，赞叹道。
<br/>　　“师祖，剑名为何？”
<br/>　　接着，他手持圣剑，向元始问道，这剑他十分喜欢，果然，师祖所赠，没一样是简单的。
<br/>　　“圣剑本无名，它既然选择了主人，自然是随了主人之名。汝名轩辕，那这剑便叫‘轩辕’，又有何妨？”元始淡淡一笑，开口说道。
<br/>　　“轩辕？轩辕剑么，果然是好名字！伏羲有伏羲琴，神农有神农鼎，吾之剑，便叫轩辕剑！”公孙轩辕闻言，顿时眼前一亮，开口豪气冲云道。他的话，完全暴露了他的内心，他要与天皇、地皇比高。
<br/>　　“锵！”
<br/>　　圣剑好似感受到主人心中所想，顿时，随着响应起来，一道巨大的金色剑柱，冲天而起，撕裂这片天地！
<br/>　　元始一看，大袖一甩，瞬间便将这道巨大的金色剑柱化为虚无，圣剑立刻变得安稳下来，不敢再颤动。
<br/>　　“呃……”公孙轩辕瞬间便熄火了，脸色悻悻然的样子，看向元始，尴尬不已。
<br/>　　良久，他才恢复淡然模样，开口问道：“师祖，这剑为何没有剑鞘？”
<br/>　　“它为何要有剑鞘？”元始淡淡反问道。
<br/>　　“呃？”
<br/>　　公孙轩辕顿时无话可说了。是啊，剑为何不能没有剑鞘呢？谁规定剑一定要有剑鞘啊。可是，还没等他再说什么，便听元始那淡淡的声音传来：
<br/>　　“这剑实际上是有剑鞘的，本座适才已经演示了一遍，只是你没有发现而已。”
<br/>　　有剑鞘？这怎么可能，我怎么没看到啊。师祖，你确定没有在骗你的徒孙？公孙轩辕心中无语。等等，他脑海中好似一道闪电划过，忽然想到了什么。刚刚他好像是看见了自己的师祖从虚空之中将剑给拔了出来，难道……
<br/>　　果然，只听元始那淡淡的声音再次传了过来，证明了他的猜想：
<br/>　　“这剑何须要剑鞘来束缚它？若是真有剑鞘，那么，这片虚空便是它的剑鞘！”
<br/>　　公孙轩辕心中恍然，是啊，剑鞘只会束缚住圣剑之灵，以虚空为剑鞘，这是何等大气。他转而向元始拜谢道：
<br/>　　“多谢师祖赠剑！”
<br/>　　“好了，你下山去吧。”元始吩咐道，“人间此刻正是风云汇聚之际，你自己务必小心，祸福都是你自取，怪不得别人，你且去吧！”
<br/>　　“是，师祖。”公孙轩辕恭敬拜道。
<br/>　　接着，他转身毫不犹豫地向着山下走去，百年之前上山，他还只是一个小孩，什么都没有，百年之后，他从这里下山，却是已经学得一身强大本事，只等着回去便可大展鹏图。
<br/>　　……
<br/>　　当公孙轩辕走后，山头出现另一道身影，正是公孙轩辕的老师，广成子！
<br/>　　“老师。”
<br/>　　广成子遥遥看着公孙轩辕离去的背影，感慨道，“我这弟子一生坎坷，喜好杀伐，这一去，真不知道是好是坏？”
<br/>　　“那是他的使命，身为三皇之中的人皇，他从生下来便早已注定，一生征战，统一人族部落。不过，他的确是一生坎坷，他这一生，大战不断，也定会遇到强大的对手，到时候，便需要你们师兄弟帮忙了。人皇之功德，可不是那么容易拿的。”
<br/>　　“哦？”
<br/>　　广成子诧异，紧接着点点头，仿佛明白了什么。只是看着远去的公孙轩辕，他眼神闪烁，不知道在想些什么。
<br/>　　元始淡淡地瞟了他一眼，便消失不见了。原地只留下广成子一人默默无语。
<br/>　　……
<br/>　　神农继承共主之位，一百五十年后，他大力发展五谷生产，农业迅速崛起，人族基本解决温饱问题，变得强盛起来。
<br/>　　但是，也正是因为人族以凡体之躯，吃各种五谷杂粮，野兽之肉，上山采摘野果充饥，所以他们身上常有病痛。其中，更是不知有多少人死在这病痛之中，为此，神农苦恼不已。
<br/>　　神农本身就拜师玄都，玄都乃太上圣人的传人，太上更是洪荒之中最精通丹药之人，而神农自然也是随着学会了一些丹药之术。丹药虽然可以治疗众人的病，但毕竟丹药是需要许多珍贵的药材，还需要精通炼丹之术，才能炼制出来，而且数量还很少，只能小范围的使用，不能广泛使用。
<br/>　　为此，神农一直在改良各种丹方，使之简化，能够运用大量普通药材，解决人类疾病问题。经过大量实验，他的鼎内沉淀的药力也越来越多，炼丹速度与成功率一时之间大增。
<br/>　　但就算如此，依旧不能解决人族的病痛问题，因为炼丹之术是需要极高的资质才能学习，而人族之中，能够学会炼丹之术的人，依旧少之又少。
<br/>　　此路不行，有很大的限制性，神农不得不另想他路。
<br/>　　他经过思考，想到既然药草能够炼丹，那么单单只用药草，想必也能起到作用，解决疾病呢？
<br/>　　想到这种可能，他顿时眼神一亮，觉得这非常有可能。若是他能够找到一种直接食用药草，便能解决疾病的方法，便能解决很大一部分人族病痛问题了。
<br/>　　想到就做，他直接请示了自己的老师玄都，而玄都也没有让他失望，给了他一种丹药，那种丹药能够让食用过它的人，检验出草木的药性或者毒性。
<br/>　　“神农，你可一定要想好了，你可能因此死亡。”玄都劝谏道。
<br/>　　“老师，我决定了，我既然身为人族共主，便要为人族排忧解难。如今人族正在被无尽病魔折磨着，正是需要我的时候，我不得不站出来！”神农一脸坚定道。
<br/>　　“好，既然如此，我也便不再劝你了，你好自为之！”玄都深深看了一眼神农，说道。
<br/>　　翌日，神农召集族人，宣布了自己的决定，之后他便独自一人上路，离开了部落，踏上尝百草之路，他要以自己的身体，去尝试各种草药，以身试毒！
<br/>　　这一去，便是数十个春秋，数十年时间转瞬即逝，神农的足迹，遍布东方大地，在那大地之上，深山老林之内，时常能看到他的身影。
<br/>　　各种药草的毒性，药性他都会亲自去尝试，有些是大补，而有些却是身具剧毒，他甚至有好几次，都差点被剧毒给毒死了。
<br/>　　经过各种尝试，他也总结出了许多经验，知道哪些药草能食用，能够解决什么病，能够解什么毒，而哪些东西是剧毒之物，不能食，否则便会死亡。
<br/>　　这种以身试药的方法，试药承担巨大的后果的。随着时间久了，那各种各样的草木之毒与各种药草的药力，便会积累在他身体之内。
<br/>　　越来越久后，神农便会承受着这越来越强的痛苦与疾病，这痛苦如跗骨之蛆，仅仅贴在他身体之内，无法驱逐。他每天都会在这无边痛苦之中度过，可他依旧没有放弃。
<br/>　　他甚至根据身体之内的各种药力与毒力，创造出属于自己独特的功法，它能够吸收药力与毒力来修炼，以药毒为法的神功，它名曰《药神典》。
