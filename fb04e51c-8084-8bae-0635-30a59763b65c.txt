        “可以跟我说说吗？”童筱颖接着问。
<br/>
<br/>        马智杰的父亲点点头，将事情的经过娓娓道来：“我记得几个月前，小杰跟我说，他想找我借三十五万块钱。他说，他想拿这钱去把他喜欢的女孩子娶回家。我当时还臭骂了他一顿，但是他的性子倔，非得要那三十五万块钱。小杰他妈妈心软，于是就把银行卡和密码都给了小杰。后来，没过多久，小杰又将银行卡还给了他妈妈。小杰说，他没动银行卡里的钱。还说，他喜欢的女孩子，已经找到了一个好男人给嫁了，不需要他了。小杰的妈妈确实查过，里面的钱一分不少，而且银行流水账单里也没有现金进出账单。所以，小杰没撒谎。他没有动里面的钱。只是后来，小杰惹上了官司，在学校里跟他室友打架，把他室友打成重伤住进了医院。他那个室友病好了以后，就把小杰给告了。我和小杰的妈妈，为了小杰四处奔波，又是请律师，又是赔偿的。还是把我们夫妻两的积蓄全都花光了……”
<br/>
<br/>        “马智杰问你们要三十五万，是在他打伤他室友之前，还是之后？”童筱颖又问道。
<br/>
<br/>        马智杰的父亲想了想，才笃定地回答道：“他把他室友打伤后，他室友出院了，他才管我们去要的那三十五万。然后，他说他要拿着钱去云城西镇找那个女孩。后来，他从云城回来后把钱还给了我老婆。再后来，他就被他的室友给告了，还被学校给开除了……”
<br/>
<br/>        “……”听到这里，童筱颖沉默了，她的贝齿咬住了下唇，嘴唇在颤颤地发抖。
<br/>
<br/>        “姑娘啊！你还有什么要问我的吗？”马智杰的父亲见童筱颖不说话了，用暗哑的嗓音问道。
<br/>
<br/>        童筱颖回过神来，深吸了口气，调整了心态，接着问道：“那马智杰死前的那一晚，是谁把他约出去的？”
<br/>
<br/>        “这个我不知道，而且，警方拿走了他的手机，至今未归还给我。”马智杰的父亲微微摇了摇头。
<br/>
<br/>        童筱颖抿了抿唇，关掉了录音笔，将资料的最后一张抽了出来，让马智杰的父亲签字。
<br/>
<br/>        马智杰的父亲看到资料上写着“叁拾万”，顿时被吓到了。
<br/>
<br/>        “保险公司，真的能补偿我们这么多钱吗？”马智杰的父亲问道。
<br/>
<br/>        童筱颖强颜欢笑地回答道：“当然能！”
<br/>
<br/>        马智杰的父亲犹豫不决地拿起笔来，颤抖着手，在上面签了字。
<br/>
<br/>        为了装得更像一点，童筱颖还要马智杰的父亲在上面按了拇指印。
<br/>
<br/>        一切假装的手续程序都走完后，童筱颖直接从自己的手提包里，将那三十万整整齐齐地堆到了马智杰的父亲面前。
<br/>
<br/>        这个三十万，是她当初在“夜惑”里驻唱时的工资以及客人给她的打赏钱。
<br/>
<br/>        她本来是想留着给昼景还债的，但是最近这段时间，她发现，昼景似乎是不需要她的这点钱了。
<br/>
<br/>        所以，童筱颖只给自己留了五万块钱，用于自己的不备之需，其余的钱，她全取了出来交给了马智杰的父母。
<br/>
<br/>        童筱颖知道，马智杰的父母就只有马智杰这么一个儿子。
<br/>
<br/>        马智杰的父亲看到钱后，忍不住痛哭了起来。
<br/>
<br/>        其实，他作为父亲，最想看到的应该不是钱，而是让他的儿子活过来。
<br/>
<br/>        童筱颖一直强忍着自己，不让自己哭出来，既然钱已经交到位了，她收拾了茶几上的资料，起身微微颔首道：“马叔叔，请节哀。我先告辞了！”
<br/>
<br/>        “姑娘，你等等。”马智杰的父亲连忙起身。
<br/>
<br/>        童筱颖这才伫足，只见马智杰的父亲去了一趟卧室，再出来的时候，他手里多了一本小相册。
<br/>
<br/>        “姑娘，我其实觉得你跟我儿子收藏的这本相册里的女生长得很像。我知道，这本相册里的女生，就是我儿子生前喜欢的说是要娶回家的那个女孩。我没别的意思，我只是希望，你能否帮我到云城西镇上去找这个女孩，让这个女孩到时候能来参加我儿子的葬礼，见我儿子最后一面……”马智杰的父亲声音哽咽道。
<br/>
<br/>        童筱颖接过相册，翻开来看了看。
<br/>
<br/>        一张张、一页页，里面全是她被偷拍时的照片……
<br/>
<br/>        她洗衣服的样子，她洗被单的样子，她打扫卫生的样子，她刷鞋子的样子，还有她晒衣服晾被子的样子，甚至她站在他们宿舍门口走廊上撩发吹风的样子……
<br/>
<br/>        她真的从来没有意识到，那时候的马智杰竟然在偷拍她。
<br/>
<br/>        一点征兆也没有，甚至找不到一丝蛛丝马迹。
