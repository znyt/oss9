　　第633章
<br/>　　百里胜并没有宋伯康想象中那么强，至少没有表现出来那么强。他在长安时养尊处优，很少有与人动手的机会。虽然后来被派到凉州边关，但也主要负责统率之职，并不是经常舞枪弄棒。
<br/>　　然而，宋伯康说得不错，今日是百里胜的绝地，他没办法从容不迫，于是只能拼命。哪怕他胜一百次，其实也伤不到宋伯康的根本，但只要输一次就无回天之力。
<br/>　　与徐童的决斗已经是他的极限，连胜两将已经耗尽他的体力。况且徐童本就力大无穷，一柄铜锤抡开杀机赫赫，百里胜根本无法硬接，只能一味避让。
<br/>　　招招凶险，稍有疏忽脑袋怕就要被砸成稀巴烂。难道自己当真要死在此地？脑海里冒出这个想法，百里胜的心情倒是平静下来，死则死矣，拼了性命不要又如何。
<br/>　　体内仿佛有股莫名的力量涌起，强撑着攻进徐童，两杆鞭一起挥，砸在徐童脑门上。徐童大锤抡了出去，此刻收不回来，在百里胜面前露出破绽。
<br/>　　狼山关的人马发出阵阵狂呼，宋伯康一激灵：又败了，这百里胜当真如此了得。
<br/>　　赵子龙微微点头，点评刚才战斗的得失：“徐童太过急躁，他力气强过百里胜，就该守好门户，太过贪进了。”
<br/>　　宋伯康看着赵子龙，当真是吃了苍蝇一般恶心。你如此本事，为何不肯出手，现在却在此地说着风凉话。
<br/>　　此时此刻，百里胜已经领着兵马杀了过来，想要借着连战连胜的士气突围。
<br/>　　“杀过来啦，杀过来啦，快退，快退。”
<br/>　　赵子龙拨转马头，立刻往后撤，他可不想卷入这场乱战之中。宋伯康也只好跟着赵子龙一块儿撤，他是军中主帅，不能冒险。
<br/>　　况且，他也不想和百里胜决战，完全没有那个必要。
<br/>　　双方的战士交锋，宋伯康的军队却是避而不战。百里胜领着兵丁冲杀，但宋伯康布下重兵，他虽然战得上风，却根本冲不过对方的包围圈。
<br/>　　最后，只能无可奈何的选择退兵，收回到狼山关中。
<br/>　　此战过后，宋伯康再不与百里胜交锋，任他在阵前叫阵，只是高挂免战牌，避不出战。
<br/>　　虽然百里胜在战斗中占了上风，但他的粮草却已不能支持他继续战斗下去，时间每过去一日，他便向地狱迈进了一步。
<br/>　　而此战的消息也很快由赵子龙率领的龙字军传到程大雷耳中。
<br/>　　最近，程大雷一直留在朔方城中，一面关注着百里胜与宋伯康的战斗，但主要精力还是放在朔方城的经营上。
<br/>　　“百里胜已经是行将就木，撑不了多久了。”程大雷合上赵子龙差人送来的情报，发出一声长叹。
<br/>　　心中也想起与百里胜交往的事情，程大雷不得不承认，百里胜并不招人讨厌。宋伯康还一心只想着从乱世中谋取利益，相比之下，百里胜更像个军人。
<br/>　　“消灭掉百里胜后，怕宋伯康的目标就要针对我们了。”和珅最近也常在朔方城，蛤蟆城一切已迈入正规，有他没他都差不太多，但朔方城需要他做的事情就太多了。
<br/>　　“他有这个心，可未必有这个力，他没有取蛤蟆城的本事。”程大雷说这话还是有些自信的，他道：“朔方城现在情况怎样？”
<br/>　　“最近不断有流民涌入，据喜字队得到的情报，这暗中还是宋伯康推动的，也不知他打得什么主意。”
<br/>　　程大雷皱起眉头，隐隐觉得宋伯康怕没安什么好心。最近他已在安排训练人手，展开针对自己的洗白计划。不过一切需要时间，正式行动尚未展开。
<br/>　　“有人来到终究是好事，这些人的情况怎样？”
<br/>　　“流民么，其中多有不义的恶徒，但对我们来说问题不大。现在正是春耕，我已安排他们在城外开荒，等过了今年秋天，便可以保证自给自足，多余的粮食也能补充我们。然而，撑到秋天需要的粮食还是太多了，消耗太大。”
<br/>　　宋伯康还是低估了程大雷的组织能力，对付流民他已有经验，本身就是山贼出身，程大雷还收拾不了几个恶徒与懒汉。不过，这些人每天都在消耗粮食，却也是真事。再任这件事发展下去，虽然不至于耗尽蛤蟆城的粮草，但终究对程大雷现在的情况不利。
<br/>　　“宋伯康那边还得盯着，看局势如何发展，说不得会有一些变数。”程大雷道：“另外让喜字队打听一下，方圆有没有能动手的势力，蛤蟆城的存粮不能再消耗下去了，得想办法开源。”
<br/>　　程大雷逛在朔方城中，城中的房屋正在修葺，这件工程完成后，还要休整城墙，城外开荒种田，以及开掘河道。有建筑精通技能的加成，工程进度还是蛮快的。
<br/>　　原先一座死气沉沉的空城，如今已经渐渐有了人气，但究竟是欣欣向荣，还是浮沫一现，如今还说不好。
<br/>　　程大雷逛街的时候，刘发财正在与城内居民训话。程大雷想错了，刘发财也并非一点作用没有，他妖言惑众的属性正好适应现在的场合，狠狠的忽悠这些人。
<br/>　　乱时当用重典，处理起那些作奸犯科之徒时，程大雷也是绝对不手软。如此双管齐下，涌入城中的流民都被妥善安置，没有人故意捣乱。
<br/>　　嘟，朔方城人口突破一万，恭喜完成任务。
<br/>　　这时候，程大雷收到一条系统提示，接连有流民涌入，城内人口不断增长，不过几日光景，程大雷已经完成任务。
<br/>　　程大雷又回到城主府，将门仔细关上，清点自己的收获。
<br/>　　寻找华氏一族的任务完成后，程大雷还获得一次抽奖机会。只是一直没有使用，也是担心自己的隐藏属性发作，再召出了不得的敌人。
<br/>　　这次完成扩充人口的任务后，程大雷又收获一次抽奖的机会。现在程大雷手中握着两次抽奖机会，握在手中不用总是有些不甘心。
<br/>　　今日晴空万里，是个好天气，程大雷狠狠心决定抽奖。
<br/>　　那扭蛋机一样的事物在脑海中出现，随之程大雷便收到一条系统提示。
<br/>　　嘟，恭喜获得作物生长加速技能，使用次数1 。
