
<br/>　　<br/>　　    王惊蛰没有先去破庙，因为庙就在那里，也不会生了腿跑掉，他去的是停尸房，要看看五具尸体，前面三个工人的，后面凌虚子师徒的。<br/>　　<br/>　　    陪同他的是刑警队的潘利明队长，来的是两个人，除了他以外还有个毕业于最高警察学府的，人民公安学院的一个女警，嗯，说是警花比较合适，她叫郭茜茜。<br/>　　<br/>　　    一辆警车开往区医院，王惊蛰抱着胳膊坐在后面闭目养神，前边是潘利明开车，郭茜茜坐在副驾驶，两人对视一眼，用余光瞥着后面的青年，他俩都很奇怪，为什么这个年轻人会参与进来，甚至之前范成梁跟潘利明提起有外人也要调查这个案子的时候，他还不太同意，觉得这不合规矩，但后来范成梁有一句话打动了潘利明。<br/>　　<br/>　　    “让他参与进来，可能会对你以后的断案产生深远的影响，有些经验和知识书本里是没有的，现实中你也可能第一次遇到……”<br/>　　<br/>　　    潘利明当时也没同意，但范二哥很认真的问他，我给你一月的时间，限期破案，你敢立军令状吗？<br/>　　<br/>　　    潘队长没吭声。<br/>　　<br/>　　    范二哥手点着桌子，说道：“你破不了，但我叫来的人可以，事成以后，功劳全记你身上”<br/>　　<br/>　　    范成梁随后也承诺过潘利明，王惊蛰的掺和都会循规蹈矩的，并且对警方也会很透明的。<br/>　　<br/>　　    车子到了医院，停到停尸房门口，下车后两人领着王惊蛰到了一个冷冻室，这里面放着在破庙四周死了的这五具尸体。<br/>　　<br/>　　    “我能自己进去看看么？”王惊蛰看着他俩问道。<br/>　　<br/>　　    潘利明皱眉说道：“这不合规矩，而且，王先生你的到来，本来也不合规矩”<br/>　　<br/>　　    “那好吧，一起进去”王惊蛰也没强求，知道警方不可能让他单独进去的。<br/>　　<br/>　　    “哗啦……”三个冷柜被打开，里面装着三个工人的尸体，王惊蛰背着手低头望去，潘利明在他身后解释道：“当时翻车了，他们三个正好被压在车底下，两个死因死颅内出血，还有一个肋骨碎了，骨头插进了肺叶里”<br/>　　<br/>　　    “嗯”王惊蛰淡淡的点了下头，说道：“可以了，关上吧”<br/>　　<br/>　　    “可，可以了？”潘利明当即一愣，和郭茜茜对视一眼，他们不知道对方看这三具尸体都没用上几秒钟是啥意思，就为了看看？<br/>　　<br/>　　    王惊蛰要看的是死者的面相，自然扫两眼就足够了，他是想确定一下他们到底是属于横祸而死，还是意外之死。<br/>　　<br/>　　    “麻烦一下，另外两人的尸体在哪？”王惊蛰回头问道。<br/>　　<br/>　　    潘利明很无奈，但还是把凌虚子师徒的冷柜给打开了，他们两人的尸体已经被缝合上了，毕竟警方不可能把一堆碎尸收拾起来。<br/>　　<br/>　　    这一次，王惊蛰没有看一眼就收了，而是略微弯着腰很仔细的盯着看了半天。<br/>　　<br/>　　    两具尸体的头盖骨全被掀开了，四肢和躯干分离，肠子等内脏也被塞了回去，尸体上没有穿着衣服是裸着的，几处伤口上露着明显的针线缝合痕迹。<br/>　　<br/>　　    “他们两人的死很奇怪，头盖骨的伤口非常平等，根本不可能是被刀子这种工具割开的，到像是激光切割一样，还有其他部分也是如此，但说激光切割就太离谱了，所以我初步断定也许是高压水枪切开的，但现场没有任何痕迹……”<br/>　　<br/>　　    他俩的死是警方完全没办法解释的，破庙里只有凌虚子师徒的脚印和指纹，其次就是许主任和他司机的，除此以外再没有第五个人了。<br/>　　<br/>　　    王惊蛰没有吭声仍旧低着脑袋，潘利明朝着郭茜茜努了努嘴，示意她往下接着说。<br/>　　<br/>　　    “既然是杀人，就得有杀人动机，我们调查了下他们两人的社会背景还有身份，发现了……”郭茜茜整理了下思路，小声说道：“发现他们的社会背景很复杂，来历也很有问题，年龄稍大点的叫崔浩，今年四十五岁，是裕丰市的农民，小学文化，另外一个叫张堂，是他崔浩的远房亲戚，最近一年多的时间他们两个一直活动在省城，对外称他们是龙虎山的道士，会堪舆和算命，关于这一点我们也和龙虎山的道观确认过了，并没有他们二人，而在这一年多的时间里，崔浩和张堂结交了不少人，其中也有官员和富商，其目的就是骗取他们的钱财，涉案金额大概有两百多万左右，不过调查到这就暂时遇到了点困难，警方曾经询问过和他们有过接触的一个官员和商人，但对方都不承认跟他们熟悉，只说是点头之交或者通过朋友介绍吃过顿饭，其他的就一概不认了，警方初步认为有可能是崔浩和张堂骗取钱财以后，有人也许起了报复的心思”<br/>　　<br/>　　    潘利明叹了口气，这时说道：“真要是报复的话，也不至于用这种复杂的手段了，完全可以暗中找人收拾他们一顿，或者直接报警把他俩抓起来判刑，根本用不着费这么大的力气要这两条人命，因为崔浩这个人很聪明，他每次骗取被害人的钱时都不太多，少了几万，多了也不过十万左右，他可能知道这么点钱对于那些人来说不算什么，哪怕事后露馅了，人家最多骂两句，而不可能大张旗鼓的找他们什么麻烦，毕竟人都是要脸的，这种丢人的事谁会闹大了啊？所以，调查到这里就完全解释不通了”<br/>　　<br/>　　    “嗯，关上吧”两人滔滔的说了一通，王惊蛰完全没啥反应。<br/>　　<br/>　　    “这就完了？看看就完了？”郭茜茜和潘利明都不可置信的问道。<br/>　　<br/>　　    “啊，完了”<br/>　　<br/>　　    “不是，你这也没……这个，看出什么了嘛？”<br/>　　<br/>　　    王惊蛰两手一摊，说道：“我也不是透视眼，也不是X光，还能看出啥啊？这就行了”<br/>　　<br/>　　    郭茜茜和潘利明在心里都卧槽了一声，完全无言以对了。
