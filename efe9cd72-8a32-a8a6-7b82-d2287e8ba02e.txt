        &lt;h3&gt;第三卷第三百一十章  不同的较量&lt;/h3&gt;
<br/>
<br/>        寒伯安带着安男回了寒家老宅。
<br/>
<br/>        苓希还没有回家，苓希这一次是带着任务来的，当然是不会轻易离开的，寒伯安想起在美洲的遭遇便已经对苓希多了一份厌恶，但是寒伯安并没有表现出来，他要看看苓希堕落到何种程度。
<br/>
<br/>        和寒伯安一起的还有李长卿。
<br/>
<br/>        在路上，李长卿对安男说：“嫂子，我和寒总在一起的表现，不是你看到的那样。”
<br/>
<br/>        安男被李长卿说得莫名其妙：“那是那样？”
<br/>
<br/>        “反正不是你看到的那样。”李长卿再一次说。
<br/>
<br/>        “李长卿，你想说什么？你要记得你的身份，你是我的助理，助理扮演的角色很多，工作上、生活上的，那个角色需要你你就得扮演那个角色。”寒伯安对李长卿唧唧歪歪的话有些着急。
<br/>
<br/>        李长卿害怕安男误会。
<br/>
<br/>        “那我是不是就要一直扮演我很喜欢你？”李长卿期期艾艾的看了一眼安男，惹得安男笑起来了。
<br/>
<br/>        “有必要的情况下还是要扮演的，比喻见了沧岚雪的时候。”寒伯安捏了捏安男的手。
<br/>
<br/>        “你们在美洲是不是很凶险？”安男突然想到寒伯安那几天在美洲一定是很危险，要不然怎么会不给她回信息。
<br/>
<br/>        “我们倒不是很危险，危险的是你。”李长卿替寒伯安回答了。
<br/>
<br/>        寒伯安瞪了一眼李长卿，李长卿赶快将头转回去，坐在前排不吭声了。
<br/>
<br/>        “其实，也不算是很凶险，只是知道你在这里多有不便，怕我给你回信息太多了，反而让你更难堪。”寒伯安替安男将额头前的碎发轻轻的拂到脑后，露出安男光洁的额头，轻轻的印上去。
<br/>
<br/>        他们谈恋爱的时候，他曾经嫌弃安男黏着他，后来俩个人毕业了，回到的各地地方，开始各自的工作和生活，反而让他特别怀念安男像小猫咪一样依赖他的那种感觉。
<br/>
<br/>        如今的安男已经成为他的妻子，随着年龄的增长，安男已经褪去了青涩和娇柔，成长为一个有主见的人，再也不像之前那样的依赖他，就拿这一次来说，要是还在恋爱的时候，安男肯定会撒娇，耍小脾气，可是现在的安男没有，她考虑的是寒伯安的安危，考虑的是寒伯安的处境，她已经开始沉稳的处理各种事情了。
<br/>
<br/>        安男心里一暖，寒伯安的身份不一样，他考虑到事情也不一样。
<br/>
<br/>        一行人到了寒家老宅，李长卿挤开了安男和寒伯安，坐在了俩个人之间。
<br/>
<br/>        老爷子看见寒伯安回来特别高兴。
<br/>
<br/>        “你们可总算回来了，我这颗心啊也就放下来了。”寒老爷子笑着说。
<br/>
<br/>        “年轻的时候也是闯荡江湖的人，怎么，现在真的是老了？”寒伯安挤兑寒老爷子。
<br/>
<br/>        “我不老，要你何用？”寒老爷子也毫不示弱。
<br/>
<br/>        “你这话说的，还真是养儿防老，话说，你的儿子也不只是我一个人吗？没有我，你们不是会更好吗？”寒伯安嘴角噙着冷笑。
<br/>
<br/>        “阿琪，你说的什么话？父母亲谁会嫌弃自己的孩子多？你在美洲的事情也不是我们所为。”苓如兰开口了。
<br/>
<br/>        “你怎么知道我在美洲有事情？”
<br/>
<br/>        “你怎么知道他在美洲有事情？”寒伯安和寒老爷子同时问苓如兰。
<br/>
<br/>        “我，我猜的，那几天伯安都没有给安男打过电话，肯定是有事情，要不然伯安如此的爱安男怎么会连一个电话都不给安男呢？”苓如兰有一丝丝的慌乱。
<br/>
<br/>        “是啊，琪哥哥一定是很忙，琪哥哥，包括你身边的助理，你的十二星座凑齐了吗？”苓希满眼含春的看着寒伯安岔开了话题。
<br/>
<br/>        “你放心，就算是没有凑齐，也轮不到你来凑数。长卿，你说是吗？”寒伯安看着李长卿笑着问。
<br/>
<br/>        “十二星座是个什么东西？有我香吗？我陪着他不好吗？要你们来凑什么数？”李长卿冷冰冰且酸味十足的对苓希开火了，并且朝寒伯安的身边挪了挪。
<br/>
<br/>        苓希睁大了眼睛：“你们果然，你们........”
<br/>
<br/>        “安男，你看他们，他们......”苓希又指着李长卿对安男说。
<br/>
<br/>        “他们不是一直都这样吗?苓希你不知道？你不是说你很了解伯安吗？你不是说我们的婚姻就是一张纸吗？你说对了，就是你看到的样子。”安男很配合苓希，有些伤心的说。
<br/>
<br/>        苓如兰也睁大了眼睛：“老爷子，你看他们.......成何体统？”
<br/>
<br/>        “年轻人的事情你要管那么多干什么？管好你自己就行了。”寒老爷子拂袖而去。
<br/>
<br/>        苓如兰见老爷子离开了，她觉得无趣，便也离开了。
<br/>
<br/>        俩个长辈一走，就留下四个年轻人。
<br/>
<br/>        李长卿便无所顾忌了：“你说，这个苓希小姐是不是你以前的女朋友？”
<br/>
<br/>        “长卿，你想到哪里去了，她就是苓家的人，和我有什么关系呢，无非仗着是阿姨的侄女，在我们家住了一些时日，便口口声声说爱我，你觉得我是那么容易接受女人爱的人吗？”寒伯安掏出一支烟，看了一眼安男。
<br/>
<br/>        安男拿起茶座上的打火机“啪”的一声打着了火递到寒伯安的面前：“总是拿着我做幌子，今天怎么就不装了？”
<br/>
<br/>        “都是自己人有什么好装的，再怎么说，苓希也是我们家亲戚，曾经也深深的爱过我，没有必要隐瞒她什么。苓希你说是不是？”寒伯安很满意安男的配合，他老婆怎么就这么可爱，这么善解人意？
<br/>
<br/>        苓希微微的张大嘴：“寒伯安，你真的是够够的了，我有爱过你这样的人吗？不好意思，我还有事情，我要走了。”
<br/>
<br/>        苓希走了，寒伯安和李长卿哈哈大笑，寒伯安将安男拉过来，深深的拥在怀里：“宝贝，你好可爱啊，你说你怎么这么可爱，这么善解人意？”
<br/>
<br/>        李长卿妥妥的被塞了一脸狗粮，他实在是看不过去了，在旁边幽幽的提醒：“我们还要去北京参加陈子昂公司的发布会，两位是继续秀恩爱呢，还是准备行程？”
<br/>
<br/>        苓希气呼呼的跑去找苓如兰：“姑姑，原来寒伯安是那样的人，难怪对我不理不睬，我当初真的是瞎眼了，耗费了我那么长时间的青春。”
<br/>
<br/>        “你现在知道也不迟啊，别将心思浪费在他身上了。”
<br/>
<br/>        “难怪沧岚雪再三叮咛我们要注意李长卿，姑姑，怎么能将李长卿拿下以后为我们所用？”苓希问苓如兰。
<br/>
<br/>        “我对李长卿也不是很熟悉，你多想想办法啊，毕竟你们是年轻人。不过，苓希啊，谷家到底给了我们苓家多少资源？”苓如兰问
<br/>
<br/>        “你傻啊，谷家能给我们多少资源？能给我们资源的人不是谷家，而是将来的陆家，谷家也是给陆家打工的，这几年爸爸和陆家走得比较近，你没有发现我们苓家一天比一天强了吗？”
<br/>
<br/>        “那你知道谷家找我们寒家运输什么吗？”苓如兰想知道谷家到底做的是哪一种生意，寒家是她的夫家，儿子都有了，她可不想寒家出什么事情。
<br/>
<br/>        “我不知道，反正都是正规生意，要不是正规生意也不会找上寒家是不是？你让老爷子多给寒伯安说说，我到时候也找找李长卿，有生意不做是傻瓜。再说他们都已经将合约都签了，以后我多给陆家说说，寒家也会有好处的。”
