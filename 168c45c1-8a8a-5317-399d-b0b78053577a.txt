　　追着叶子博过去的几个叶家年轻人一开始还能看到叶子博的踪迹，但很快就发现自己等人迷失在了这片到处都是居民平房的地方，这个高原基地说大不大说小不小，可真要藏个人也不容易找出来。
<br/>　　叶家的后代纷纷焦躁起来，开始无头苍蝇的乱窜。
<br/>　　“找不到了！到底去了哪？”
<br/>　　“是不是故意的调虎离山？”
<br/>　　“留爷爷一个人在家，我总感觉不对。”
<br/>　　“走！回去！”
<br/>　　为首的叶家年轻人当机立断，立即带着人想要返回。
<br/>　　此时在旁边不远处的一间小屋内，看到这一幕的叶子博心中焦急起来，他一直在计算时间，按理说三分钟前叶永阳就应该被张玄呆了过来，而那些叶家年轻人在找不到他人之后返回正好与张玄回归的道路错开。
<br/>　　只是眼下这些年轻人的确是依照了他的计划折了回去，但张玄却没有回来，怎么回事？
<br/>　　想到这里叶子博不仅焦虑又紧张，快速的换了一身装备后，便再次走出平房，循着叶永阳所在的地方摸索过去。
<br/>　　——————
<br/>　　高原基地的最后方基地总部，杨飞杰正在照例处理公务，他留守在这里的时间已经很长，管理这个基地也得心应手，除了每当有新人入驻时需要注意，平日里大多数都只需要常规巡视基地周围即可。
<br/>　　高原上的狂化动物比较少，偶尔出现有威胁的都会被第一时间发现，然后出动猎杀队伍将其解决。
<br/>　　而丧尸和异种更不会没事来这里，相对于东部那些动不动好几万人扎堆的基地，他们这里还不足以引起丧尸和异种的注意。
<br/>　　同时高原地带民风淳朴，人少矛盾也少，相对于其他东部基地来说，杨飞杰还没真正感受到过从内部而来的威胁。
<br/>　　就在他今天照例依照流程处理公务的时候，一个慌张的声音从走廊大声传来，汇报人员门都没敲就直接破门而入。
<br/>　　房门被‘嘭’的一声撞开后，杨飞杰就听到了他管理基地这一年来最震惊的一则消息：“首领！基地大门被破，有人杀了守卫逃了！”
<br/>　　刷！
<br/>　　杨飞杰霍然而起，拧着眉头大惊问道：“具体说明情况，谁干的？”
<br/>　　“还在调查，现场除了被轰碎的大门就只剩下两具尸体，没有目击者，是周边劳作的人听到声音赶去之后才发现的。”汇报人员快速回答。
<br/>　　“怎么会好端端的杀人逃走？”杨飞杰有些不能理解：“想要出基地只需要依照流程走，我们这里又没有禁止外出。”
<br/>　　“首领！”此时汇报的人面色苍白，道：“今天看守大门的两名守卫，都是四阶新人类，依照死亡时间和死后状态推断，是被人一招秒杀的！甚至我还怀疑，凶手破开大门和杀人，都只用了一招！”
<br/>　　“什么！？”杨飞杰大惊，瞳孔剧烈收缩：“此人战力超绝，怎么进的我们基地？混进来的？不对，我们这里地势特别，只有大门可以入，除非他进来时也用了同样暴力的方式，否则只能入基地的流程一步步来。”
<br/>　　汇报的人员点头：“而且事情发生的太快，我们所有人都措手不及，我已经第一时间前来汇报，此时才刚过去五分钟。”
<br/>　　“还有机会，立即派出一批小队出基地搜寻，切记不要硬碰硬，对方战力太强根本打不过，我们需要的是调查情况。”杨飞杰迅速下达命令，接着说道：“另外将基地内的每一户人家都查一遍，哪些人不在基地，一查便知。”
<br/>　　“是！”
<br/>　　——————
<br/>　　在高原基地的内部展开调查行动的时候，叶家的几人也用最快的速度回到了住处，只是等待他们的却是一间空无，叶永阳的身影早已消失不见。
<br/>　　叶家的几人当场急红了眼，与过去一路走到这里的心态不同，此时他们已经知道了爷爷的苦心，更明白家族的重任和压力，哪个与神秘家族挂钩的秘密竟然还和他们叶家有关，无论如何都不能让爷爷出事。
<br/>　　“走！去找杨飞杰，我们眼下只能求助于他！”一人提议，其余人也二话不说的点头同意。
<br/>　　叶子博正好摸索到这里，听到了屋内的声音后立即明白了过来，叶永阳已经失踪了，但张玄却没有带着人来见他。
<br/>　　愤怒之下，叶子博只能小心的折回住处，想要碰碰运气，只是无论他怎么找，张玄和叶永阳都依旧消失。
<br/>　　而在事发不久之后，一批基地的内部队伍就来到了这片区域，挨家挨户的开始了人员清点。
<br/>　　“大哥，请问发生了什么事？”叶子博小心翼翼的凑上前问道。
<br/>　　被询问的士兵脸一横，喝道：“基地机密是你能随口问的？！”
<br/>　　叶子博心头不爽，但却不敢表现出来，只是挂着畏惧的表情哆嗦开口：“我刚来才几天，不知道规矩，就是看今天大家都人心惶惶的，我有点……有点怕。”
<br/>　　旁边的一人看到他脸都吓白了，拍了拍同伴的肩膀，朝着叶子博笑道：“可以理解，别害怕，我这兄弟今天也是被气到了，基地大门被人从内部毁了，两名守卫身亡，眼下已经确定是基地内的人行凶，所以我们在检查离开的人是谁。”
<br/>　　“哦……”叶子博应着，内心却波涛汹涌，怒火更是差点压制不住。
<br/>　　好一个张玄！

<br/>　　在这批检查队伍离开后，叶子博快速的收拾起来，利用了这一基地此时有些混乱的状况，小心翼翼的朝着大门摸索过去。

<br/>　　在叶子博找着机会离开基地的时候，失踪的人也在这时候调查了出来，正是昨天刚进基地的张玄，和叶家的家主叶永阳。
