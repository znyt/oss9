<br/>    沙海之外，眼中所见，皆是赤潮的营地，一处挨着一处，占据了一座座山峰，众多新招收到的血衣剑卫看到如此多的赤潮，也是惊叹不已，他们现在可是知道自己的实力的，三百人就足以端了一处妖王的老巢。
<br/>    
<br/>        而现在，他们满目所见，几乎都是赤潮。
<br/>    
<br/>        本来有些倨傲的他们，在此刻低下了头，对于燕山只剩下忠心，生不起任何其他的心思，原来他们，在燕山，只是沧海一粟而已。
<br/>    
<br/>        沙海之外，几乎处处都是红色的营地，这些是最开始入门的那六百记名弟子的兵马，他们毕竟先入门，拥有着更多的资源。
<br/>    
<br/>        第一次尝到了抄家的快感的燕山领主，只想十万大山再多一些妖王，这才过多久啊，就被师兄弟们全灭了，而一个个燕山领主，腰包也是越来越鼓。
<br/>    
<br/>        甚至于张鸣此行已经是集结了二十多位燕山领主，灭了一位神府境的妖王，赚了一波大的。
<br/>    
<br/>        张鸣骑马走在前方，旁边有另外一个燕山领主道:“张鸣师兄，听说了吗，师父已经回来了，听说这一次开拓令的战争经费又拨了百万灵石！”
<br/>    
<br/>        张鸣听到这个消息，也是心中一惊，“师父又拨了百万灵石，每一个师兄弟们都有吗？”
<br/>    
<br/>        那个燕山领主道:“是的，我这次回来就是准备将我麾下的血衣剑卫扩充到一千，很多师姐师妹没有出去作战，全部的灵石都投入了炼器房，现在她们可是囤积了大量的杀王剑和灵甲！”
<br/>    
<br/>        张鸣笑道:“那好啊，我们可以省一番功夫了，直接从她们那里购买杀王剑和灵甲，现在这个时代，虽说战乱不休，但是也是最好的机会，莫说数十年，就是数千年之间，也是一场规模最大的战争，在十万大山之外，可是有着更多的肥羊待宰啊！既然师父又拨了百万灵石，我决定了，将我麾下的血衣剑卫拉到一万去，再请一个神府境，我一个人的兵马，就足以横行神府境的势力！”
<br/>    
<br/>        那个燕山领主道:“我可没有师兄这么大的胃口，我最多也就拉到一千了，只能慢慢打过去积攒灵石扩充更多的兵马了！”
<br/>    
<br/>        张鸣道:“现在可是以战养战的最好机会，不要爱惜灵石，师父说过，灵石就是用来花的，放在钱庄里，那就一文不值，只有花出去，才有作用！现在十万大山已经打完了，只能想办法在十万大山之外动手了，可是十万大山之外可是有着法相境的存在的，必须要请大师姐出手了！”
<br/>    
<br/>        燕山领主战斗的倒是很起劲，毕竟对于他们来说，不就是去抢灵石，抢灵药，以及各种资源吗？
<br/>    
<br/>        张鸣心想既然师父又拨了百万灵石，那自己麾下的那些大妖俘虏，是不是可以卖给亲爱的师弟们啊，好像又可以赚一波啊！
<br/>    
<br/>        张鸣安排完了自己的兵马，进入沙海之中，如今的燕山已经又不一样了，里面的其他记名弟子，似乎都是驾云而行，而且速度飞快，而且云上还有着一句话:天高地也阔，安全第一条，腾云不规范，师徒两行泪。
<br/>    
<br/>        只要看上一眼，就轻松地认了出来，绝对是师父的手笔。
<br/>    
<br/>        待得张鸣来到燕山本部，才知道，所有的记名弟子，都可以在燕山领取一朵祥云作为坐骑。
<br/>    
<br/>        张鸣领取了自己的祥云，直接找到了银翼，对于银翼，他还是有些尊敬的，虽说当年银翼干过某些丢脸的事，但是银翼毕竟是直属师父的麾下，不过现在的话，大师姐也有权力调动银翼，“银翼叔，请帮我禀报大师姐，我有事请求！”
<br/>    
<br/>        银翼扫了一眼张鸣，这小家伙他有影响，没有想到啊，这才过多久，这个小家伙也是一方领主了，“我这就去禀报大宗主！”
<br/>    
<br/>        不过一会儿，银翼过来了，“大宗主在山上等你。”
<br/>    
<br/>        张鸣整理了一番行装，心怀忐忑地上山，实在是大师姐给的压力太大了，手中是仙器巨阙，并且实力已经是悟道境巅峰的存在。
<br/>    
<br/>        大师姐威严如山。
<br/>    
<br/>        嗯，冰山。
<br/>    
<br/>        上山之后，张鸣可以看的到，卓青瑶正在练剑，看到他来到了此处，卓青瑶收剑，问道:“说吧，有什么事。”
<br/>    
<br/>        张鸣道:“如今十万大山已经平定，我们众多师兄弟，都想要进军十万大山之外，但是十万大山之外，古仙门的腹地之中，有着众多的法相境，我们可能很难处理这些法相境，还要请大师姐出手。”
<br/>    
<br/>        卓青瑶愣了愣，问道:“很多法相境，你这个很多，到底是多少？”
<br/>    
<br/>        张鸣道:“几十位应该是有的吧。”
<br/>    
<br/>        卓青瑶思考了片刻，露出一抹寒光，吓得张鸣赶紧低头拱手道:“要是大师姐醉心于修行，我们也可以自己想办法！”
<br/>    
<br/>        只见卓青瑶收起巨阙，然后道:“这一次的战争限于大君之下，只要没有大君入场，应该没什么问题，正好我最近好久没有找人切磋了，就点起兵马，出征吧！”
<br/>    
<br/>        开玩笑，卓青瑶这个以战悟道的战狂会放弃这种好机会！
<br/>    
<br/>        卓青瑶小声道:“正好师父回来了又去找紫霞仙子他们汇合去了，师父不在，现在燕山我说了算，就带齐八万大军，出征！”
<br/>    
<br/>        卓青瑶看向山下，传音道:“银翼，速速叫三位主战大妖王前来见我！”
<br/>    
<br/>        卓青瑶看向张鸣，“你们可要跟紧一点，不然，可没有你们喝汤机会了，二师弟已经去了仙魔对抗的前线，我就将古仙门腹地内的所有附属鬼仙的王级势力，全部斩除吧，三师弟能够一统魔宗王级，我自然也能轻松征服整个古仙门腹地！”
<br/>    
<br/>        输，卓青瑶从来没有想到过这个问题，甚至于燕山的记名弟子都没有想到过这个问题。
<br/>    
<br/>        血衣剑卫的战斗力，他们可是清楚无比，以一当十不在话下，其他入道境难得有一件的灵甲，血衣剑卫可是全军灵甲，而且，血衣剑卫可是修行的兵书！
<br/>    
<br/>        这一日，燕山真正出兵出征十万大山之外，有修行者看到燕山那苍茫的无尽大军，只见一片片赤色的红云遍布天际。
<br/>    
<br/>        这个在仙门正道不过刚刚露面的神秘宗门，最开始只是有传言这个宗门有着数万炼丹师，富可敌国，而现在，燕山出兵，赤潮遍布。
<br/>    
<br/>        不少修行者抬头望天看到这一片片的红云，震撼地心神狂震，“这就是燕山吗？”
<br/>    
<br/>        “此等势力，就是古仙门也不过如此吧。”
<br/>    
<br/>        “原来这才是燕山的真正实力吗？”
<br/>    
<br/>        他们并不知道，这些不过只是最开始的六百名燕山领主的大军，以及卓青瑶的大军，还有三千名燕山领主正在前往整个仙门领地各处，召集散修，组建属于自己的血衣剑卫。
<br/>    
<br/>        仙鬼大战的战场中，燕山正式入局参战。
