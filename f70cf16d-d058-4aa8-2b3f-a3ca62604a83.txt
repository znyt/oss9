　　如今的局势，对于蛇博士并不是很有利，这吸血蛇成长的有点出人意料，被人肉喂养，实力比他预期的还要强大。
<br/>　　他本想利用银针封住吸血蛇的穴位，让它束手就擒，当初他也正是这样做。
<br/>　　可如今的吸血蛇整整二十米的庞大蛇躯，浑身鳞片坚硬如铁，他找不到任何的机会，找到了，也刺不进去。
<br/>　　一条粗大的蛇尾带着滔天之力席卷而来，砸到地面就是无法无天的震动，好在这座牢笼是专门为了关押吸血蛇特殊制作的，坚固无比。
<br/>　　否则早就要轰塌了。
<br/>　　“哈哈，师父，你不行了，还是逃吧，徒弟我也不想背负一个弑杀师父的罪名”江行泽看着陈默只能招架吸血蛇的攻击，在一边放肆的落井下石。
<br/>　　“师父，你一生控蛇，要是最后被蛇给吃了，岂不是晚年不保，有辱你的威名”
<br/>　　陈默不断闪躲的蛇尾，尤其不能被蛇缠住，否则就绝对脱不开身，听到江行泽的话，脸色有点铁青起来，额上的一条青筋涨了出来，连着太阳窝的几条筋，尽在那里抽动。
<br/>　　“我怎么教出你这样一个孽徒”
<br/>　　一身怒喝，一条蛇尾呼啸而来，陈默骤然脸色大变，来不及闪躲，双手一挡，整个人猛然的被砸到了墙壁上，五脏六腑翻江倒海，跪地就是一口鲜血喷出。
<br/>　　当一条蛇你控制不了，那么就要最好被反伤的准备。
<br/>　　尤其是这种大蟒蛇，型如巨，身如铁，跟滚刀肉一样，让你难以下手。
<br/>　　嚓了嚓嘴角的血，抬头便是铁门散发着森森的光，如果陈默想走，还是走的了的。
<br/>　　但是他不能走，更不能让江行泽带着吸血蛇离开。
<br/>　　否则就是放虎归山，出去害人害己。
<br/>　　在山野集团，他的眼皮底下，江行泽都将让吸血蛇吃人，那么如果出去，更是肆无忌惮，无法无天。
<br/>　　要是这样，他陈默就是千古罪人，不知道多少人要死在吸血蛇的手里。
<br/>　　铁门外响起了一阵脚步声，有人来了，陈默闻声，不但不高兴，反而着急起来。
<br/>　　这条成长到二十米的吸血蛇，不是普通人能对付的了，来多少人，都是送死。
<br/>　　“不要过来，不要过来”
<br/>　　情急之下，陈默疯狂大吼起来，朝着铁门冲了过去，他想将铁门关闭，进不来人，也出不去。
<br/>　　呼..．
<br/>　　空间震荡，滔天之力席卷，陈默直接被抽飞了出去，砸落在地，数口鲜血喷出。
<br/>　　这一刻，他知道，他真的对付不了这条变异吸血蛇了。
<br/>　　他拥有的控蛇手段，对于吸血蛇形同虚构了。
<br/>　　“师父，有人进来，就让他们进来呗，还能跟你一起并肩作战”
<br/>　　江行泽戏谑的说着，笑着，根本不在意有人进来，但与此同时，他也一直跟陈默保持距离，躲在吸血蛇的身后，毕竟瘦死的骆驼比马大。
<br/>　　“你们不要进来，不要进来”
<br/>　　陈默哪里还有时间跟江行泽置气，可转眼冲进来了五六名保安，看到陈默倒在地上，满口是血，大惊：“博士，你怎么了”
<br/>　　“快出去，有危险”
<br/>　　“吼...”
<br/>　　一道阴森恐怖的嘶吼声惊天而起，吸血蛇铜铃般的蛇眸满是贪婪，蛇尾席卷，顿时间五六名保安直接圈住。
<br/>　　突如其来的变化，一条二十米的巨蟒俯视，将五六名保安直接吓傻了，心沉坠得像灌满了冷铅。
<br/>　　一时之间都忘记了喊救命。
<br/>　　“吸血蛇，你的对手是我，不要滥杀无辜”
<br/>　　陈默挣扎的起身，直接甩出了三四根银针，朝着吸血蛇的蛇眸射去。
<br/>　　这吸血蛇的弱点恐怕就在于眼睛上了，别的地方防御太强。
<br/>　　吸血蛇凶煞狡黠，蛇头摇曳，就避开了自己的弱点，让银针扑了一个空。
<br/>　　“你们人类都该死，都是吾的食物”
<br/>　　阴森沙哑的声音从吸血蛇的血口下发出，充满了冷血肃杀之意。
<br/>　　“妈呀，博士救命...”
<br/>　　这一声喊救命，也是这一世最后的一次呐喊了，血口之下，焉有完卵。
<br/>　　陈默双目充血，蛇吃人残忍的一幕让他整个人都充满的愤怒，他的气息扑面涌出，衣服涔涔作响，隐晦间，他后悔了，他不该培养什么绝世蛇种出来。
<br/>　　“哈哈...吸血蛇，吃吧，吃饱了还有我师父等着让你吃，你也算你报仇了”
<br/>　　江行泽面目可憎，蛇吃人的残忍画面却让他兴奋不已。
<br/>　　他已经不将自己当成是一个人类了，被吃的这些人，更与他何干。
<br/>　　“孽徒”
<br/>　　陈默咬牙切齿，杀意暴涌而来，若狂风飚射，此时好机会，吸血蛇在吃人，反而将江行泽给遗忘了。
<br/>　　“孽徒，我先取了你的命在说”
<br/>　　咻..．
<br/>　　昏暗的牢笼，一道微星划过，一根银针骤然插在了江行泽的一只左眼上。
<br/>　　快如闪电，根本无法察觉。
<br/>　　“啊...我的眼”
<br/>　　江行泽脸上的笑容顿时凝固，旋即，痛苦的嚎叫起来。
<br/>　　陈默身形如若幻影，直接出现在江行泽的面前，痛心疾首道：“我这只手只救人，从不杀人，但没想到第一个要杀的却是自己的徒弟，但你该杀，该死”
<br/>　　一掌高高扬起，掌中蕴含着无与伦比的力量，朝着疯狂嚎叫的江行泽头顶拍去。
<br/>　　吸血蛇恐怖如斯了，陈默自认拿它不下了，只能先手刃了这个禽兽。
<br/>　　“吸血蛇救我”
<br/>　　江行泽直接瞎了一只左眼，血流不止，可看到怒气滔滔的陈默一掌拍来，更是吓得三魂不见了七魄。
<br/>　　电光火石间，一条长长粗大的蛇尾悄无声息的席卷而来，顷刻间将陈默的双脚缠住，直接拖起，抛到空中。
<br/>　　这一掌险险的与江行泽插肩而过。
<br/>　　“吸血蛇，杀了他，一口吞了他”
<br/>　　江行泽狰狞无比的尖叫，满脸是血，乖张暴戾。
<br/>　　被抛到空中的陈默犹如待宰羔羊，一张血口带着扑面而来血腥味吞噬过来。
<br/>　　陈默极力的扭动身形，可在空中，他无力可借，厄运难逃。
<br/>　　唯有拼死一搏。
<br/>　　陈默手中再次出现三根银针，用尽浑身力气，朝着那张血口甩了出去。
<br/>　　瞬息间，三根银针直接没入血口喉咙深处。
<br/>　　昂..．
<br/>　　吸血蛇发出一声作痛的嘶吼声，本想将陈默一口吞下，可距离还未够，直接吃痛的合上了血口。

<br/>　　这一合，血口下，青白獠牙咬住了陈默的双腿，蛇头摇曳，陈默双腿尽断，血肉淋淋的摔飞出去。
<br/>　　与此同时，铁门外很多全副武装的人员涌了进来，牢笼中的异常，已经引发了注意。
<br/>　　“吸血蛇，不玩了，我们走”
<br/>　　瞎了一只眼的江行泽，血流不止，在无心情玩下去，他要找个地方治疗。
<br/>　　另外看到师父陈默被吸血蛇咬断了双腿，恐怕也活不了了。

<br/>　　吸血蛇的存在没几个人知道，如今冲进来的武装人员看到这条二十米的巨蟒，凶煞无比，吓得纷纷散开，根本不敢阻拦。
<br/>　　“不能让蛇逃出山野集团”
<br/>　　一个角落中，陈默一张脸苍白之极，下身血肉模糊，大腿之下皆无，鲜血疯狂冒出。
<br/>　　他连忙用几根银针勉强封住了穴位，让血不在流出，用尽力气大吼了一声，直接不省人事。
<br/>　　不知是死是活。
<br/>　　堂堂蛇博士，人类觉醒者，都不是吸血蛇的对手，更别说这些冲进来的人，不被吓得屁滚尿流就不错了。
<br/>　　除非现在出现真正的武装人员，军事力量，用火力才能压制这条蛇。
<br/>　　可枉然，现如今，无人能阻止这条蛇冠冕堂皇的离开。
