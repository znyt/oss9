        季少，全名季运恒，季氏集团董事长的长子，花钱如流水，换女朋友比换衣服还快。
<br/>
<br/>        这种男人，开口说“喜欢”，可信度几乎为负值。
<br/>
<br/>        “小谢，当我女朋友，我会给你，你想要的一切。今晚，好好伺候我！”季运恒咧着嘴，连哄带骗地再次靠近童筱颖。
<br/>
<br/>        “对不起，季少，我已婚，不适合给您当女朋友。”童筱颖淡定地撂下话，转身既走。
<br/>
<br/>        “行了，小谢，你就不要再找什么破借口拒绝我了！咱们都是成年人，说话就直接点。本少今晚就是要你给我当女朋友！”季运恒不再跟童筱颖墨迹，语气终于不耐烦了起来。
<br/>
<br/>        他刚刚还是一副温柔的面孔，此刻骤然一变，露出了原本就好色的暴躁本性。
<br/>
<br/>        季运恒跨步上前，伸手便捉住了童筱颖的手腕，将她往墙壁上压去，并抬起另一只手，拉下她的衣领，按住她的脖颈就狂吻了下来。
<br/>
<br/>        “小谢，我要你！本少今晚就要你！你必须成为我的女人！”季运恒一边迫不及待地乱吻，一边胡言乱语起来。
<br/>
<br/>        “小谢，你真是个磨人的小妖精！”
<br/>
<br/>        童筱颖连吃奶的劲都使出来了，但她的力度在这男人面，就像蚍蜉撼树，自不量力。
<br/>
<br/>        此刻……
<br/>
<br/>        谁来救救她？
<br/>
<br/>        “夜惑”的公共卫生间里，黑色系列风格的洗手台前，一个俊逸脱俗，气质与周遭环境显得格格不入的男人，正站在洗手盆前，慢条斯理地洗着自己那双修长而好看大手。
<br/>
<br/>        其实，他的双手因为常年摸枪而生了老茧，只是最近退役从商后，每天坐办公室，倒是把他给养白了。
<br/>
<br/>        傅夜沉将身上那件高定版银灰色西装脱了下来挂在了手臂上，现在他一袭白色天蚕丝手工衬衣，完美地将他健硕的胸肌给贴衬了出来。
<br/>
<br/>        他今晚是怎么呢？
<br/>
<br/>        傅夜沉剑眉微蹙，缓缓地抬起头，看着镜子里的自己。
<br/>
<br/>        脑海里，又一次浮现出台上那个驻唱的“小谢”的倩影。
<br/>
<br/>        傅夜沉微微眯了眯眸子，恍若在镜子上看到了“漫雪”的微笑。
<br/>
<br/>        刚刚有很多次，让他产生一种台上的“小谢”，就是他心爱的女人“漫雪”的错觉。
<br/>
<br/>        或许，他已经病入膏肓了吧！
<br/>
<br/>        他是不是应该去找一个心理医生给自己看看？
<br/>
<br/>        傅夜沉收回思绪，抽了张纸巾，一边擦拭着双手，一边走出去洗手间。
<br/>
<br/>        救命……
<br/>
<br/>        “漫雪”的声音，不知从何处传入了他的脑海。
<br/>
<br/>        傅夜沉身心一怔。
<br/>
<br/>        当季运恒的大手摸到童筱颖底裤的边缘时，童筱颖冷不丁地打了个寒颤，忽然间害怕地颤抖了起来。
<br/>
<br/>        救命……
<br/>
<br/>        谁来救救我？
<br/>
<br/>        不要……
<br/>
<br/>        童筱颖神经紧绷，继续抗拒着季运恒，双手不停地推搡，身心一刻都不敢松懈。
<br/>
<br/>        突然，一阵不急不缓的脚步声，从那边楼道口传来。
<br/>
<br/>        童筱颖眼前一亮，满怀希冀地循声望去。
<br/>
<br/>        这里是安全通道楼梯口，所以很少有人会路过这里。
<br/>
<br/>        以至于季运恒敢在这里对她肆意妄为。
<br/>
<br/>        而这样的脚步声，对童筱颖来说，无疑是一根救命稻草。
<br/>
<br/>        只见，一道修长笔直的身影出现在了楼梯上。
<br/>
<br/>        楼道里的灯光，从上打了下来。
<br/>
<br/>        那道身影就站在逆光里，随着他从容不迫的脚步，一张棱角分明的俊脸，在一片阴影中缓缓地挪了出来。
<br/>
<br/>        居然是……他！
<br/>
<br/>        童筱颖也没想到，会在这种地方，再次邂逅那个被她救过的男人。
<br/>
<br/>        明明她在这之前，一点都不想再遇到他。
<br/>
<br/>        可此时此刻，她却多么希望，他能朝她走过来。
<br/>
<br/>        那男人居高临上地看着他们，英俊的脸还有一大片都淹没在暗影里，完全看不清他是何种表情。
<br/>
<br/>        季运恒看到这男人时，先是一愣，停下了手中所有的动作，忧心忡忡地揣测着这个男人的心思。
<br/>
<br/>        他似乎是碰巧路过此地，好像并不打算插手他们的事情。
<br/>
<br/>        季运恒心里笃定这个男人不会多管闲事后，紧紧地抓住童筱颖的手，一把拽着她，就想带她离开这里，然后去开间房，接着做这没做完的事情。
<br/>
<br/>        “季少，你放开我！”童筱颖挣扎着，手腕都被这男人给抓红了，她依旧不放弃任何求生的机会，趁机对刚刚那个男人大喊了一声，“先生，求你，救我！”
<br/>
<br/>        “你给本少闭嘴！”季运恒一边拽着童筱颖往那边的电梯口走去，一边恶狠狠地谩骂着童筱颖，“本少花了钱，打赏了你，你就得陪本少过夜！都来这里卖了，当了婊子还要立牌坊？贱女人，真作践！要叫，待会到了本少的床上再叫！”
<br/>
<br/>        季运恒说这话时，故意拉高了嗓音，像是说给那个男人听的。
<br/>
<br/>        他只不过是想警告那个男人，他手里拽走的这个女人就是个出来卖的，不值得“英雄救美”。
<br/>
<br/>        傅夜沉淡定地从楼梯上下来后，也就默默地站在走道上驻足，静静地看着他俩渐行渐远的身影，似乎确实没有要插手管这事的意思。
<br/>
<br/>        童筱颖见那男人对她的呼救无动于衷，顿时想死的心都有。
<br/>
<br/>        趁着季运恒注意力分散之际，童筱颖低下头去，在季运恒的手背上咬了一口。
<br/>
<br/>        季运恒“啊”地一声，吃痛地松了手，童筱颖见状拔腿就跑。
<br/>
<br/>        因为跑得太急，童筱颖一个踉跄，跌倒在傅夜沉的面前。
<br/>
<br/>        季运恒趁机追上她，再次将她从地上拽了起来，一边破口大骂“你这个贱女人，敢咬我！找死，是不是？”一边扬起另一只手，欲要给童筱颖一巴掌。
