　　吞天秘术，在楚风眠杀了陈少主，吞噬了属于陈少主身上的那一部分吞天兽血脉之后，威力足足增强了数倍。
<br/>　　这剧毒虽然难缠，但是在吞天秘术之中，还是被源源不断的抽取了出来，化为了精粹的灵力，打入到了楚风眠的身躯之中。
<br/>　　看到这一幕，楚风眠便是不在犹豫，心神一动，一股灵力打入到了吞天秘术之中。
<br/>　　当即吞天秘术运转的速度变的更快，源源不断的剧毒，都从这清水圣丹之中，被剥夺了出来。
<br/>　　三分钟的时间过去。
<br/>　　大量的剧毒都被剥夺了出来，这清水圣丹上面的一层绿色的纹路也逐渐的消失，呈现出了一种晶莹的颜色。
<br/>　　这才是属于清水圣丹真正的颜色，一股药香飘散而出。
<br/>　　旋即楚风眠停下了吞天秘术，将这清水圣丹拿了出来，捏在了手心之中，打量了一眼。
<br/>　　现在这清水圣丹上，已经是没有一点点的剧毒了，都被彻底的清理了出去。
<br/>　　虽然是在清理剧毒的时候，这清水圣丹也消耗了一部分的药力，不过依然还留有九成的效果。
<br/>　　九成药效的清水圣丹，也要比起如今这个时代同样类型的圣丹，效果要好得多。
<br/>　　而且这一批古圣丹的品级都极高，应该都是天元商会之中很厉害的炼药大师炼制而出的。
<br/>　　天元商会起家的根本，就是丹药，所以在炼药一途上，天元商会几乎是历史之上数一数二的势力了。
<br/>　　像是如今七大宗门之一的药王谷，也是以炼药之道闻名于世的，但是如果比起炼药一途的手段，远远不如天元商会。
<br/>　　“将这些古圣丹的剧毒，都给炼化再说！”
<br/>　　这一枚清水圣丹，被楚风眠放入到空戒之中，随即楚风眠看向了那一箱玉瓶，手心一会，每一个玉瓶都是当即打开。
<br/>　　各色圣丹，足足有着一百多枚，全部飞了出来，都来到了楚风眠的面前，被吞天秘术一口气的吞噬了进去。
<br/>　　随即大量的剧毒，都被源源不断的吞噬起来。
<br/>　　熟能生巧，在将那一枚清水圣丹的剧毒剥夺出去之后，楚风眠也明白了这剧毒的难缠之处，吞天秘术不断的转动的。
<br/>　　几分钟后，十几枚古圣丹便是飞了出来，这些古圣丹的剧毒都被全部的化解，恢复了曾经的药效。
<br/>　　防止是被毒气再次侵袭，都被楚风眠装入到了玉瓶之后，便是纷纷放入到了空戒之中。
<br/>　　这一批古圣丹在剥夺出剧毒后，剩下的药力，至少剩下了九成五。
<br/>　　随着越来越古圣丹被剥夺出药力，恢复过来之后，楚风眠的手法也越来越熟练，最后的一批，最为珍贵上古级别的古圣丹，几乎都保存了九成九的药力。
<br/>　　几乎是没有一点药力，流逝了出去，这样品级的中古圣丹，拿出去足以让无数强者疯狂。
<br/>　　让楚风眠更加欣喜的是，这一批古圣丹之中，有着十几枚，都是突破境界补充灵力之用的圣丹。
<br/>　　楚风眠要冲击帝尊境，这些古圣丹都是一大助力。
<br/>　　“横扫了这天元商会之后，我冲击帝尊境的把握，至少可以达到九成！”
<br/>　　楚风眠目光看向宫殿之外。
<br/>　　这样的宫殿，在整个遗迹之中，足足有着十几个，既然这一个宫殿里有宝藏，其他的几座宫殿也不会例外。
<br/>　　冲击境界失败会影响未来的实力，甚至是会影响未来楚风眠冲击更加境界，所有冲击帝尊境，楚风眠必须要一次成功才行。
<br/>　　楚风眠必须要是将把握达到最高程度的时候，才会选择冲击，纵然是现在冲击帝尊境，楚风眠都有着六成的把握，但是楚风眠还没有冲击的想法。
<br/>　　他要做好万全准备才行。
<br/>　　将所有的古圣丹都给装入到了玉瓶之中，随即便是全部都放入到了空戒之中。
<br/>　　这宫殿内，已经是没有了任何的宝藏，楚风眠便是身躯一闪，向着旁边的一座宫殿走去。
<br/>　　这一座宫殿，比起刚刚所在的那一座宫殿要更加靠近主殿，所以其中的毒气，要更加浓郁。
<br/>　　不过却是对楚风眠造不成什么影响，楚风眠步入其中，还是可以看见不少的骸骨。
<br/>　　显然是跟之前那宫殿一样，这宫殿里的天元商会弟子，也都是被毒雾直接毒死在了宫殿里面。
<br/>　　就连逃走都做不到。
<br/>　　楚风眠没有去管这些骸骨，而是目光一扫，看见了宫殿的一个石台。
<br/>　　这石台之上，有着一座鼎，看样子，应该是一座药鼎。
<br/>　　药鼎，是炼药师的必备之物，当然如果是炼制少量丹药的话，要不要药鼎倒是还好。
<br/>　　但是一旦是要炼制大批的丹药，就必须要药鼎了，有了药鼎，足以让炼药师在炼药是消耗的灵力消耗减少数倍。
<br/>　　而且利用药鼎炼药的成功率，炼成的品级也会高上不少。
<br/>　　楚风眠之前炼药，也是因为没有找到合适的药鼎，毕竟楚风眠的纯阳天火，实在是威力太强了。
<br/>　　天下间能够承受纯阳天火的材质，寥寥无几，寻常的药鼎，就连楚风眠的火焰都承受不住，何谈炼丹？
<br/>　　但是眼前这药鼎。
<br/>　　“九龙神苍鼎？”
<br/>　　楚风眠看向这药鼎之上，一共九座龙头的模样，眼神之中露出几分喜色。
<br/>　　九龙神苍鼎，这件药鼎，楚风眠可是曾经听说过的，在古籍之上，天下百鼎之中，位列第九。
<br/>　　天下百鼎，记载的便是九域历史之中出现过的最强的百座药鼎。
<br/>　　在上面记载的药鼎，几乎都是已经从历史之中消失了，不知道是落到了哪里去，现在药王谷成名的药鼎，是药王无极鼎，在天下百鼎的记载之中，位列第十一。
<br/>　　眼前这九龙神苍鼎，要比起药王谷成名的药鼎，还要更强。
<br/>　　根据古籍记载这九龙神苍鼎最后的一任主人，就是天元商会的末代会长，后来天元商会摔落，这九龙神苍鼎也就遗失不见了。
<br/>　　没想到居然是还在这天元商会的遗迹之中。
<br/>　　这一座宫殿，名为炼药殿，看来是天元商会内部用来炼药的宫殿，在这九龙神苍鼎的旁边，还有着不少木架。
<br/>　　这些木架，应该是摆放灵药，圣药所用，可现在却是空无一物，应该这些灵药，圣丹，都是在毒气的侵蚀下，全部消逝了。
