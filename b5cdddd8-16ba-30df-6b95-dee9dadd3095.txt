　　第248章
<br/>　　西北草原，远远行来一支商队。
<br/>　　骆驼连成行，车上拉着丝绸，茶叶，瓷器。他们从帝国出发，穿过丘陵，草原，戈壁到达遥远的极西之国。途中还有可能遭遇马匪，风暴，以及官兵的吃拿卡要。
<br/>　　这是一条九死一生的商路，当然，回报也是相当丰厚，值得人铤而走险。
<br/>　　这支商队是以蓝氏商行为首，以及其他一些客商加入。虽然这条路危险重重，但跟着蓝氏商行大家还是有些底气的。
<br/>　　蓝氏商行经营这条商路诸多年，家里养着许多好手，这次更动用三百武者护送，更何况，这次领队的还是蓝氏商行的大小姐——蓝青婷。
<br/>　　蓝青婷的母亲是一名西域女子，蓝氏商行的大老板在一次风流快活中有了她，之后因为她天生绿瞳，被家族当作妖孽，五岁时差点被丢进水塘。
<br/>　　但随着她的长大，渐露狰狞，可以陪着家里的武夫汉子大口喝酒大口喝肉，几次随队，从一个杂役做起，事情办得可以说是滴水不漏。有她随行的商队，竟然次次顺风顺水，于是妖女变成圣女，到她二十岁时，其亲父已经承认她是蓝家的大小姐。
<br/>　　此刻她骑马走在队伍前方，大家都看着她，某种程度上，她是大家的精神支柱。一身黑衣，脸上也蒙着黑布，据说，她小时候被火烫伤，面目奇丑无比，因此才一直用黑纱罩面。
<br/>　　“大小姐，天快要黑下来，咱们是不是找个地方歇脚。”一个汉子道。
<br/>　　蓝青婷摇摇头：“连夜前进，这一带有土匪出没，一刻也不能留，越快通过越好。”
<br/>　　“不扎营了？”汉子一惊：“可咱已经走了一天了。”
<br/>　　“嗯，三天内要穿过这片草原，什么时候穿过，什么时候歇息。”
<br/>　　“三天？”
<br/>　　毕吉睁大眼睛，最后无奈的摇摇头。他毕竟是第一次随行，还不知道蓝大小姐的风格。蓝青婷的风格就一个字：快！
<br/>　　以最快的速度前行，不眠不休，穿过土匪出没的地界，土匪还没想明白呢，蓝大小姐已经带着人不见了。
<br/>　　“大小姐，大小姐。”
<br/>　　前面的探子驱马回来，脸上露出恐怖的表情。
<br/>　　“您快来看，前面！”
<br/>　　“前面怎么了？”蓝青婷皱起眉头，这人是家里的老伙计，出生入死多少次，从没有像今日一样恐惧过。
<br/>　　“您……来看……”
<br/>　　其实越过前面一道山坡就能看见了，便是蓝青婷，绿色眸子里也露出寒冷的恐惧。
<br/>　　山坡下散落着几十具尸体，他们肢体被砍断，肺腑流在外面，如今尸体已经腐烂，苍蝇围着尸体嗡嗡直叫，几只豺狗正在吞咽尸体。
<br/>　　“他们和我们一样是商队，这是遇到马匪了。”有人惊恐道，似乎山坡下的尸体变成了自己。
<br/>　　“快走！
<br/>　　蓝青婷一惊，此地必然有马匪出没，甚至就在附近，必须趁早离开，只希望能摆脱。
<br/>　　在当天夜里就出事了。
<br/>　　商队后面远远缀着几个人，他们点着火把，似乎不惧被发现，然而也没有发动攻击，只是不紧不慢跟着。
<br/>　　据说草原上的狼发现黄羊群时，并不会第一时间攻击，只是在后面跟着。等狼群越聚越多，直到狼王出现，才一举扑来。而这个时候，羊群已因为恐惧，而只会跪在地上发抖。
<br/>　　商队后的火把越来越多，白天看不见他们的踪影，但一到晚上，火把就会如约出现。恐惧在队伍中蔓延，人人惶惶不可终日。
<br/>　　直到那一天夜里。
<br/>　　“大小姐，我带一队人去看看吧。”毕吉道：“看他们是哪一路的。”
<br/>　　“嗯，小心些，能不动手，就不要动手。”
<br/>　　毕吉带着一队人过去了，都是队伍中的好手。蓝青婷在原地等待，目光看着远处火把燃起的方向。
<br/>　　她不知道毕吉会如何处理这件事，如果对方可以谈的话，留下一笔财物，或者说每次从这里经过，向他们缴纳一笔保护费也可以。
<br/>　　商人逐利，一切以利益为尚。
<br/>　　只有这群马匪不是那拨人就可以。
<br/>　　时间一点一滴过去，毕吉还未回来，远处的火把摇曳，不知发生了什么事情。
<br/>　　正这时候，急促的马蹄声响起，只见毕吉骑在马上奔来。
<br/>　　蓝青婷一惊：“毕吉，什么情况，怎么只有你一个回来了。”
<br/>　　“大小姐，是他们啊……”
<br/>　　毕吉发出一声吼，那声音宛若在地狱中遭遇了魔鬼。蓝青婷惊愕的发现，他的肢体开始散开，四肢和躯干分离。此刻蓝青婷才发现，毕吉今日是被绑在马上，而他的四肢早已经被斩断，又被绳子缠在一起。
<br/>　　一箭射来，从毕吉的喉咙透出，随着他的尸身倒地，在他身后响起沉重马蹄声，有数百骑从黑暗中出现。
<br/>　　远处的火把还在摇曳，然而马匪已经近在眼前，清晰得能看到他们扬起的刀。
<br/>　　惨叫声响起，人们四散逃离，口中一声声喊着：是他们啊，是他们啊……
<br/>　　他们？边城鬼见愁。
<br/>　　鬼见愁盘踞在草原上，抢劫过往的客商，男人被当场杀死，女人被劫回去，据说在寒冷的时候，他们会将男人制成肉干。
<br/>　　没有人愿意碰到鬼见愁，因为他们就像地震，风暴，海啸一样，属于不可抗力。
<br/>　　杀戮在夜里开始，黎明结束，惨白的天光下，一具具尸体倒在地上，血发出刺鼻的腥味。
<br/>　　蓝青婷在手下的护送下，骑着一匹快马逃出，身后的马匪追赶，像是戏耍一样从身后放箭，随着破空声，蓝青婷身边一个个人倒下。
<br/>　　最后只剩下她自己，身后的马匪越来越近。
<br/>　　忽然，身下马一跌，倒在地上口吐白沫，蓝青婷也被摔倒在地上。
<br/>　　她抬起头，发现面前站着一个人，眼睛漆黑，一席白袍。
<br/>　　“救我，救我！”蓝青婷立刻哀求，忽然想起对方只有一个人，不过是马匪刀下亡魂而已：“你快走，你快走……”
<br/>　　男人没有回应，伸出手摘掉她脸上的黑纱，那一刻蓝青婷竟然忘了阻止。
<br/>　　黑纱剥落，露出一张堪称完美的脸颊，男人愣了愣，一时竟有些呆意。
<br/>　　“你真好看。”
<br/>　　
<br/>(本章完)
