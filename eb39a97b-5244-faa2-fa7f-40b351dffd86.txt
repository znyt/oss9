　　“这炼尸缸是我本命魂器，能祭炼一切生灵，为我所吸收……先天灵体的血，我很想要。”
<br/>　　靠！她居然打的是这个主意！
<br/>　　叶少阳怔怔到看着她，这个美女，跟苏沫一模一样的美女，果然跟她一样疯狂。

<br/>　　“先别问了，你先试试能不能冲开封印！”

<br/>　　“老大，不信啊！”美华用尽全力，结果还是打不开结界，自己也被尸气烧得够呛。

<br/>　　叶少阳立刻拿出朱砂笔，在阴阳镜上画下一道符印。
<br/>　　碧清看在眼里，说道：“你这是做什么。”
<br/>　　“封印住镜面，让她出不来，免得她跟我一起被你炼了。”
<br/>　　如果真的要死的话，自己决不让美华陪葬。阴阳镜是道门法器，自己的封印，一个普通人都可以擦掉，但是邪物不行，就算碧清再强，都没有办法毁去阴阳镜和上面的封印。
<br/>　　美华是安全的。
<br/>　　碧清眼神一动，道：“你倒是很仗义。”
<br/>　　叶少阳冷哼一声，道：“反正不像你这么冷血，简直不是人。”
<br/>　　“你接着骂。我不生气。”碧清呵呵呵地笑了几声。
<br/>　　“你……”
<br/>　　叶少阳也是无语了，体内的罡气，就快被消耗殆尽……得想个什么办法。
<br/>　　叶少阳上下望着碧清，目光落在她的脚上。

<br/>　　“好看啊，又白又嫩，要是能摸一把，嘿嘿……感觉肯定不错。”
<br/>　　“你！”碧清居然有点红了脸。
<br/>　　“原来你也会害羞啊，是不是你很喜欢我这么说，我告诉你，我有恋/足情结，你知道什么意思吗……”
<br/>　　叶少阳满口胡说了一堆，碧清从来没听过这么污的话，出于女性的本能，被他说的满脸绯红，心惊肉跳的，连心态都不稳了，对尸气的操控力也越来越弱。

<br/>　　彻底没戏了。
<br/>　　叶少阳也只好收敛心神，在炼尸缸底部坐下，体内的罡气，越来越少，五脏六腑也越来越热，他感到自己的血在沸腾……

<br/>　　“大周天吐纳心法？”一个声音，在耳边响起来。
<br/>　　叶少阳一个激灵，清醒过来，左右看去，并没有人，随即想到也许是在自己神识里问的，于是也在神识反问：“是谁？”
