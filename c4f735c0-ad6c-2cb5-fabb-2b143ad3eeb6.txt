        严文斌愣在原地，眼巴巴地看着连星儿的背影，心里顿时很不是滋味。
<br/>
<br/>        他喜欢的女孩子，就这样从他的眼前跑了。
<br/>
<br/>        不过待他缓过神来时，他才发现另一个眼中的问题。
<br/>
<br/>        “喂！星儿！你还没记下我的手机号码……”严文斌大声喊道。
<br/>
<br/>        但此刻，连星儿已经跑远了，身影很快就湮没在了人群之中。
<br/>
<br/>        此时，明司寒从餐馆那边走了过来，走到严文斌的身旁，一手叉腰，一手的手肘搭着严文斌的肩头，意味深长地说道：“喂，老兄。你知道连星儿是我什么人吗？”
<br/>
<br/>        严文斌回过神来后，偏头与明司寒四目相对时，立即本能地跳开一段距离。
<br/>
<br/>        明司寒站稳了身子后，看着严文斌咧嘴一笑：“我点的烤全乳猪、烧鹅和清蒸鲥鱼，是给我未婚妻的。”
<br/>
<br/>        顿时，严文斌的脸色一片惨白。
<br/>
<br/>        随后，明司寒又从自己的皮夹里掏出一百块钱，递给了严文斌，接着说道：“这是我未婚妻借你的钱，我替她还了。”
<br/>
<br/>        “……”严文斌看了明司寒一眼，尴尬地接过钱后，默不吭声。
<br/>
<br/>        更确切地来说，他是自惭形秽。
<br/>
<br/>        他总觉得，只有连星儿这样的女孩子才配得上自己，却没想过他自己这样的男孩子是否配得上人家连星儿。
<br/>
<br/>        明司寒拍了拍严文斌的肩膀，会心一笑，然后只身离开了。
<br/>
<br/>        另一边，连星儿搭出租车回到酒店后，在前台重新拿了一张房卡，准备收拾行李回临海城。
<br/>
<br/>        而这个时候，房门上的门铃，不逢其时地“叮铃”响起。
<br/>
<br/>        连星儿放下手中的活，走到房门口，通过门上的猫眼，看了一眼来访的人后，才将房门打开。
<br/>
<br/>        对方是个与她年龄相仿的女孩子，而且还是上午在医院里，和明司寒很亲热的那个女孩子。
<br/>
<br/>        “你好，我叫贺思思。”贺思思落落大方地向连星儿伸出手来。
<br/>
<br/>        连星儿微微一笑，礼貌性地握了握：“你好，我叫连星儿。”
<br/>
<br/>        贺思思也不打算跟连星儿绕弯子，粉色的小嘴一撅，甩了甩搭在肩头的斜马尾，直接开门见山地说道：“我能查到你在北城的具体住址，就已经能够说明我的来历不简单了。”
<br/>
<br/>        “嗯。”连星儿应了声。
<br/>
<br/>        贺思思双手抱臂，一副大小姐的脾气，问道：“我要进去坐着跟你谈。”
<br/>
<br/>        “好。”连星儿让了路。
<br/>
<br/>        贺思思进屋后，她才将房门合上。
<br/>
<br/>        “我和司寒哥哥是青梅竹马，我很爱他，所以，请你退出！”贺思思一边说着，一边在客厅的沙发上坐下后，翘起了二郎腿。
<br/>
<br/>        连星儿听她这么一说，立即凑到她的身边，挤着她坐下后，迫不及待地说道：“你若是真的很爱他，那你去跟他爷爷说，要他爷爷取消我跟他的婚约！”
<br/>
<br/>        面对连星儿这样的反应，贺思思诧异不已。
<br/>
<br/>        她来之前，可是把她当成情敌的，而且还已经做好了要跟她斗嘴开撕的准备。
<br/>
<br/>        怎么着，现在，这种情况，完全不是她所想象中的那个样子呢？
<br/>
<br/>        “你、你不爱司寒哥哥吗？”贺思思有些难以置信。
<br/>
<br/>        她家司寒哥哥长得那么帅，从小到大，被她“打跑”的情敌不计其数。
<br/>
<br/>        连星儿倒是一脸不解地看着贺思思：“我为什么要爱你的司寒哥哥？”
<br/>
<br/>        “难道，你是被迫联姻的？”贺思思顿时恍然大悟。
<br/>
<br/>        连星儿微微耸了耸肩，解释道：“也不算是被迫，但是自己也没得选。”
<br/>
<br/>        “只要你不爱我的司寒哥哥，我就有办法解除你俩的婚约。”贺思思打了个响指，随即向连星儿勾了勾手指，示意她凑过来侧耳倾听。
<br/>
<br/>        连星儿凑过去后，听完贺思思的计划后，连连摇头，否决道：“这样不好。”
<br/>
<br/>        “有什么不好的，只需要你配合，又不需要你做别的事情，一举两得的事情，我觉得最完美不过了。”贺思思洋洋得意地说。
<br/>
<br/>        连星儿却不以为然：“那是对于你来说是一举两得很完美。但是，对于明司寒来说，很不公平。”
<br/>
<br/>        “我看你就是‘口是心非’，表面上说什么想要解除婚约，实际就是不肯！”贺思思见连星儿不认可自己的计划，心底莫名其妙地恼火起来。
<br/>
<br/>        “你要我配合你一起算计明司寒，对不起，我做不到。现在，你可以走了！”连星儿也没好气地下逐客令。
<br/>
<br/>        贺思思从沙发上站了起来，白了连星儿一眼后，扬起下巴，高傲地甩手离开。
<br/>
<br/>        连星儿不屑地睨了贺思思一眼后，待贺思思一出门，她自己便是一脚把门给踢关了。
