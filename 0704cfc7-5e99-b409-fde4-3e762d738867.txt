        待到连华生离开后，童筱颖坐在医生的办公桌前，语重心长地问道：“医生，我为什么不能怀孕？”
<br/>
<br/>        “你的所有检查一切正常，你的子宫也没问题，各种激素都是正常水平，可是你的子宫卵泡就是不排卵，让我也觉得很疑惑。”女医生拿着童筱颖的各项化验检查结果，拧着眉头一筹莫展，甚至匪夷所思起来，“如果是遗传问题的话，应该也不存在。存在的话，这世上就没你了。难道是基因问题？要不，你去基因科挂个号，找那里的医生替你检查一下。”
<br/>
<br/>        “好的，谢谢。”童筱颖拿回了自己所有的检查资料，开门走了出去。
<br/>
<br/>        连华生见童筱颖忧心忡忡地从诊室里走出来，立即迎上前去，好心问道：“医生怎么说？”
<br/>
<br/>        “让我去基因科找医生看看。”童筱颖苦笑。
<br/>
<br/>        连华生一手托着手肘，另一只手摸了摸下巴，凝眉思忖了一会儿，纳闷道：“你这看妇科的，医生让你去基因科？”
<br/>
<br/>        “有什么问题吗？”童筱颖微微歪了歪脑袋看着连华生。
<br/>
<br/>        “意思就是，你没怀孕，是你的问题喽？”连华生讷讷地问。
<br/>
<br/>        童筱颖点了点头：“医生说我各项检查都没问题，只是子宫不排卵，让我去基因科检查。”
<br/>
<br/>        “那医生拐着弯说你基因变异啊！”连华生打趣。
<br/>
<br/>        童筱颖忍不住踹了连华生一脚：“你正经点，行吗？”
<br/>
<br/>        连华生打个响指，咧嘴一笑：“你要是基因上出了问题，那还不如去我们的实验室。”
<br/>
<br/>        “你们的实验室？”童筱颖十分诧异。
<br/>
<br/>        连华生得意洋洋地回答道：“基因方面的问题，我们实验室里的医生，比这里的更权威。”
<br/>
<br/>        “那你带我去见识见识。”童筱颖应了声。
<br/>
<br/>        连华生点了点头：“行！”
<br/>
<br/>        童筱颖跟着连华生上了车，坐在了副驾驶座上。
<br/>
<br/>        连华生一边开车，一边感慨道：“医生既然说你不能排卵的话，那就是说明了，你没法造孩子出来啊！”
<br/>
<br/>        “大哥需要继承人，对不对？”童筱颖眸光暗淡地望向正在开车的连华生。
<br/>
<br/>        连华生咳了一声，清了清嗓子，才回答道：“当然！傅氏集团不能只到阿沉这儿给断了。”
<br/>
<br/>        话题到这里，童筱颖的神色一敛，突然间就没下文了。
<br/>
<br/>        连华生意识到自己的心直口快，戳到了童筱颖的痛点，连忙安慰道：“如今这医学技术这么发达，你放心，你将来肯定能怀上阿沉的孩子。”
<br/>
<br/>        “谢谢你安慰我。”童筱颖黯然神伤地应了声，思绪惆怅地将目光落向了车窗外。
<br/>
<br/>        到了目的地后，连华生领着童筱颖，又是刷卡又是按指纹又是读取面貌的，终于进了他们实验室基地的内部中心室。
<br/>
<br/>        经连华生一番介绍后，一个穿着白马褂的女医生领着童筱颖去了血液科抽血。
<br/>
<br/>        连华生正好去中心室里，询问一下工作进展。
<br/>
<br/>        “连少，既然你来了，就麻烦你把这个药带给裴金澈。”一个上了年纪的男人，手里拿着一个盒子走了过来。
<br/>
<br/>        连华生接过盒子，点了下头，应了声：“好。”
<br/>
<br/>        这个盒子里装着抑制裴金澈体内快速生长因子的药。
<br/>
<br/>        裴金澈是这个实验室里的“试管婴儿”长成的孩子，拥有快速自愈的能力。
<br/>
<br/>        但是，连华生对裴金澈身上的“基因秘密”了解得并不多。
<br/>
<br/>        这个实验基地是他的曾爷爷，也就是他父亲的爷爷，留传下来的。
<br/>
<br/>        那个年代的世界大战，入侵国研究生化武器，他的曾爷爷是第一批海外留学生化科的学生，被上级带到这里秘密研究解毒武器，说白了就是研究一些病毒、细菌后面又往基因之类的方向发展。
<br/>
<br/>        后来，他的曾爷爷成了这个实验基地的团队领导人，就把这里作为了自己的遗产传承了下来。
<br/>
<br/>        现在，就到了他连华生这代，不过，这个实验基地的领导人物不是他，而是他的父亲。
<br/>
<br/>        他连华生有自己的制药厂要管理，只是平日里闲来无聊的时候，就会没事来他父亲这里闲逛。
<br/>
<br/>        童筱颖进去了很久还未出来，连华生不得不主动找过去。
<br/>
<br/>        他刚走到门口，就被一名工作人员告知：“童小姐被上头的人给带走了！”
<br/>
<br/>        听到这句话时，连华生立马感到事情的不妙，连忙质问道：“走了多久了？”
<br/>
<br/>        “半个小时。”工作人员回答道。
<br/>
<br/>        这个夏阳枭，为了把傅夜沉抓回来，连这种“卑鄙”的手段都使。
<br/>
<br/>        说了不带走童筱颖的，结果这只老狐狸不讲信用！
