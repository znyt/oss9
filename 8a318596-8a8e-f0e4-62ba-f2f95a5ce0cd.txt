<br/>    太初古林。
<br/>    
<br/>        众多王级修行者看向前方，树木生长成一片，交织成一片，几乎密不透风，只有在十丈高的地方，才有着能够过人的地方。森林之中寂静的可怕，只有风吹树叶的声音，没有飞鸟，同样也没有野兽，只有簌簌的风吹树叶的声音。
<br/>    
<br/>        “这里便是太初古林了，既然我等已经来到了这里，边想办法进入其中，寻找机缘造化吧！”
<br/>    
<br/>        “这里的木灵气极为浓郁，想必其中是有着木灵脉的！”
<br/>    
<br/>        “木灵脉还有着很多的灵药伴生，想必有着不少灵药，甚至是药王！”
<br/>    
<br/>        有不少修行者直接从天空中飞过，希望可以寻找到灵药，或是其他的宝物，过了三天之后，忽然有别的消息从太古古林之中传出，有人发现了一片药田，其中最为珍贵的，还有着五株千年药王！
<br/>    
<br/>        众多的修行者按照消息赶到那一片药田，只见灵药足足有着数百株，甚至在最中心的地方，有着五株人参般的千年药王！
<br/>    
<br/>        “那是千年雪参！”
<br/>    
<br/>        “只需要一株，就足以让一位道种境晋升到神府境！”
<br/>    
<br/>        “此等造化，我们为何还在此地止步不前？”
<br/>    
<br/>        有修行者道：“你们看看这周围的白骨，还有尸体，在药田之外，有着一具具人类的骨骸，同样还有着最近刚留下来的尸体！”
<br/>    
<br/>        修行者们看向那些尸体，不禁触目惊心，其中还有着不少王级的尸体。
<br/>    
<br/>        这药田的周围，绝对有着诡异莫测的危险。
<br/>    
<br/>        众多的修行者迟疑了一会儿，不过还是有人不愿意离去，“如今仙魔大战，战乱不休，不成大君，终究只是土鸡瓦狗，机缘在前，危险便是天道赐予我等的考验，怎么可以停止不前？”
<br/>    
<br/>        有一个王级直接飞向药田之中，他飞行的很慢，以免惊动了周围的潜藏危险，他缓缓落地在药田之中，并没有什么诡异的事情发生，他大笑道：“看来，这机缘是我的了！”
<br/>    
<br/>        众多王级看得两眼发红，“机缘在前，怎可让他一人独得？”
<br/>    
<br/>        “我们也去分一杯羹！”
<br/>    
<br/>        “果然，机缘是留给有大意志，道心坚定的人的！我来也！”
<br/>    
<br/>        最开始到的王级伸手，将一株灵药摘下，就在他摘下这一株灵药，根须离开地面的瞬间，一株大树忽然如同人一般站了起来，“该死的人类！”
<br/>    
<br/>        一株株大树人立而起，无数的枝丫将天空遮蔽，朝着众多的王级拍去，无数王级喋血，从空中落下，这些大树如同钢铁般的躯体，让他们根本没有机会反抗，王级们摘了灵药，想要逃走，飞身到了天空之中，却是看到周围密密麻麻都是树根飞天，将他的脚缠住，不让他们逃走。
<br/>    
<br/>        “我命休矣！”
<br/>    
<br/>        “这到底是什么怪物，恐怕都有着王级的力量！”
<br/>    
<br/>        “这些大树，肉身绝对堪比王级，而且树木众多，我命休矣！”
<br/>    
<br/>        就在这时，一柄柄灵剑忽然出现在四周，一柄灵剑施展一种剑法，不过片刻，数百株大树的根须被斩断，天空中的王级们也得以脱逃。
<br/>    
<br/>        修行者们看向这些飞剑，一一回到一个青袍年轻道士的拂尘之中，“这种实力，是大君！”
<br/>    
<br/>        “不知道是哪一位大君驾临了，不是大君来到此处，我命休矣！”
<br/>    
<br/>        “是曙光大君，这是曙光大君！”
<br/>    
<br/>        大树们抬头看向陈明，喝道：“人族大君，我劝你莫要多管闲事！”
<br/>    
<br/>        陈明笑了笑，道：“如果我一定要多管闲事呢，你能把我怎么样？让你们的头领来见我！”
<br/>    
<br/>        大树们嘶吼道：“我们的头领也是你能够见到的！”
<br/>    
<br/>        “灵药莫非不是生命，天生就应该被你等吃掉不成？”
<br/>    
<br/>        “诸位，我们不要去和这些残酷的人类说道理，杀了他们！”
<br/>    
<br/>        “直接杀了他们！”
<br/>    
<br/>        陈明看向赶到此处的其他四位大君，然后道：“看来，跟它们谈是没用了，先把这些家伙解决了，只有把他们打怕了，他们才知道畏惧！”
<br/>    
<br/>        大树们身上的树叶开始摇晃，一首用树叶唱出的歌谣在风中传递，歌谣清幽曼妙，传递到一处，就有着一棵大树站了起来，其中不仅仅是有这种百丈高的大树，同样还有着十丈高的大树，数十丈高的大树。
<br/>    
<br/>        整个太初古林，如同在这一刻醒来，大地微微开始摇晃，一棵棵大树从地面站了起来，根须形成一条条腿，它们正在向着这里赶来。
<br/>    
<br/>        陈明扫了一眼，这些大树，只怕是不下数百万，还真是有些难对付啊。
<br/>    
<br/>        而且，陈明还并没有看到堪比大君存在的大树，数千王级，数百万入道境还是有的。
<br/>    
<br/>        它们是想要拿这些大树堆死他们吗？
<br/>    
<br/>        不过说实话，就是刘玄刹此刻眼中也是露出了一抹惊色，“这太初古林之中，只是这些古树，就已经快要比得上一个古仙门了，这么多的古树，就是我们有五位大君，也无法全部斩杀殆尽啊！”
<br/>    
<br/>        雪寻梅也是道：“而且这些大树因为经历过流星雨洗礼都没死，火烧是没有用的。”
<br/>    
<br/>        苏青罡道：“如此一来，我们低估了这太古古林之中古树的力量，堪比大君的古树还没有露面，我等就束手无策了，恐怕要集结一个古仙门的力量，才能彻底降服它们！我们还是，撤吧！”
<br/>    
<br/>        刘玄刹思考了片刻道：“这其中一定有着不少灵药，足以众多古仙门用上很长一段时间，但是这些古树的力量，已经超过了我们能够掌控的局面了！”
<br/>    
<br/>        紫霞仙子道：“我观察过了，这些大树，是把这些灵药当成了自己人，亲自培育这些灵药，他们都是植物，绝对不会放任我们肆意采摘灵药。”
<br/>    
<br/>        陈明看了一眼，这太古古林，他是一定要进去的，而且还要找到太初宗的遗迹。此时，就剩下陈明还没有说话，四人也是将目光投向陈明，陈明却是道：“其实并不需要全部杀掉，只要能够吓唬住它们，我们就有机会和他们的头领谈条件了。”
<br/>    
<br/>        刘玄刹此刻也是惊异不已，“我知道紫霞仙子说过你在群战之中确实很强，但是你面对的，可是整整堪比一个古仙门的力量啊。”
<br/>    
<br/>        陈明微微一笑，道：“只是吓唬的话，还是有机会的，试试看吧。”
