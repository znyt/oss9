　　青萝小姐将古传送阵的具体位置，告诉了叶尘之后，并留下了一份地图，便直接离开了。
<br/>　　她毕竟是无垢圣殿的弟子，还要回宗门报告，而且一旦让外人知道，她和叶尘在一起，说不定会引来杀身之祸。
<br/>　　叶尘和幽冥炎蛇，并没有着急赶路，他们都受了伤，而且这妖兽山脉中，有不少实力不弱的妖兽，至少也要恢复到一定实力之后，才能在这里横行无忌。
<br/>　　叶尘从乾坤镯里取出之前，在万宝阁炼制的丹药，先找到几瓶适合妖兽的丹药，丢给了幽冥炎蛇，然后剩下的丹药，一股脑的全部倒入口中，顿时把幽冥炎蛇都给看呆了，
<br/>　　“主人，这些可都是四阶丹药！我隔着瓶子都能感受到里面的药劲有多强悍，您居然全都吃了？您不会是想不开，要自杀吧？”
<br/>　　叶尘随手将丹瓶丢到一旁，瞥了一眼幽冥炎蛇，没好气道：
<br/>　　“胡说八道什么呢！在一旁帮我护法，我要尽快恢复功力，不然我们俩都得完蛋！”
<br/>　　说完这话，叶尘不再理会幽冥炎蛇，随手在周围布置下一道禁制，便开始运起吞天神功，炼化体内的丹药之力。
<br/>　　呼呼呼！
<br/>　　一转眼，大半天的时间过去了，
<br/>　　叶尘猛然睁开双眼，眼眸中透着一抹喜色，
<br/>　　“原本估摸着，这次的伤势，至少也得十天半月才能缓过劲，没想到这才短短几个小时，就已经恢复了三四成，连修为也提升到了金丹七重！”
<br/>　　“是了！这应该是万古长青珠的功劳！有了这个宝贝，我以后就算是想死，恐怕都没有那么容易！哈哈！”
<br/>　　欣喜过后，叶尘又猛然想起一事，
<br/>　　“对了！若是将万古长青珠，置于吞天鼎的内部空间之中，应该对修复曦月的神魂也有帮助！”
<br/>　　一念及此，
<br/>　　叶尘左手一翻，取出吞天鼎。
<br/>　　同时右手跟着一翻，将万古长青珠取出，然后两者缓缓合在了一处。
<br/>　　轰~
<br/>　　将万古长青珠放入鼎内空间后，叶尘的意念也跟着进入其中。
<br/>　　滋滋滋~~
<br/>　　在那些生命元力的滋养之下，曦月的残魂，果然开始飞速的变强，魂魄也渐渐的变得凝实起来，似乎随时都要清醒过来！
<br/>　　然而，
<br/>　　还没等叶尘高兴起来，
<br/>　　轰隆隆！
<br/>　　鼎内空间一阵乱晃，竟然隐隐出现了要崩塌的趋势。
<br/>　　“奥维斯！什么情况？”
<br/>　　叶尘连忙沟通吞天鼎的器灵。
<br/>　　哗！
<br/>　　奥维斯幻化成一个白发老者，出现在鼎内空间之中，满脸的惶恐之色，
<br/>　　“主人！不知道为何，我正在被这个世界的法则之力攻击，我的鼎内空间快要维持不住了！！”
<br/>　　叶尘两眼一瞪，也有点懵，
<br/>　　“怎么会这样？难道是因为曦月的神魂变强了，所以就引来了法则之力？”
<br/>　　曦月之前曾经说过这个问题，
<br/>　　她的神魂通过月光宝鉴，跟随着叶尘的神魂，回到八百年前的地球。
<br/>　　因为被地球上的规则之力所限制，所以她的神魂，没能回到自己的身体上。
<br/>　　如今，曦月的神魂，被叶尘带回了修真界，这个世界就同时出现了两个曦月的意识，必然有违世界法则，遭到攻击也就不奇怪了！
<br/>　　想到这里，叶尘意念一动，当即将万古长青珠，从鼎内空间中移出。
<br/>　　轰隆隆~~
<br/>　　鼎内空间又晃动了一阵，随着曦月的神魂，再次陷入沉睡状态，空间也慢慢稳定了下来。
<br/>　　“果然是这样！”
<br/>　　找到了原因之后，叶尘的眉头顿时皱的更紧了。
<br/>　　他现在还有一点搞不清楚，也是最为重要的一点，
<br/>　　倘若曦月的神魂清醒过来，又或者离开吞天鼎内的鼎内空间，这个世界的规则之力，将会如何对待这道神魂？
<br/>　　是会将曦月的神魂直接摧毁抹去？
<br/>　　还是会将这道神魂，送回曦月的肉身之中？
<br/>　　这个才是最为关键的问题所在！
<br/>　　“不行！除非有绝对的把握，我一定不能冒这个险！”
<br/>　　“为今之计，我必须要让曦月的神魂继续沉睡！”
<br/>　　“而我现在的当务之急，是要先找到神器月光宝鉴，然后再去落霞宗找曦月！”
<br/>　　“只有借助月光宝鉴的力量，才能对抗法则之力，才有可能将曦月八百年后的意念，送回她的肉身之中……”
<br/>　　想明白了其中缘由之中，叶尘先是微微松了一口气，不过很快有遇到了新的难题，
<br/>　　“记得上一世，这月光宝鉴，是我在修真界十大绝地之一的‘迷罗星海’中意外所得！”
<br/>　　“这迷罗星海，位于中央大星域，离此地间隔亿万星辰，除非是那种超远距离的大型传送阵，否则光是进入中央大星域，恐怕都得好几年，甚至是十几年！”
<br/>　　“曦月的神魂，也不知道能不能撑到那个时候？”
<br/>　　叶尘越想心里越是烦躁，可是思来想去，却也想不出更好的办法。
<br/>　　“不管了，先去青萝小姐说的那个上古传送阵看看！”
<br/>　　打定主意之后，叶尘随手撤掉周围的禁制，从地上一跃而起。
<br/>　　一旁的幽冥炎蛇，也慌忙站起身来。
<br/>　　叶尘也不废话，直接手掌一翻，之前从赤羽天君手中得到的那艘赤羽飞舟，便已经从乾坤镯中飞了出来。
<br/>　　叶尘神念一扫，便轻松抹去了赤羽天君的精神烙印，很快便将飞舟完全炼化，
<br/>　　“走吧！”
<br/>　　说话这话，叶尘纵身一跃，便跳到飞舟之上，一旁的幽冥炎蛇也慌忙跳上飞舟。
<br/>　　嗖！
<br/>　　这赤羽飞舟的速度果然惊人，几乎一眨眼的功夫，便化作一道金光，消失在天际尽头。
<br/>　　……
<br/>　　只用了两三个小时，叶尘便已经抵达目的地，
<br/>　　这里是一座古老的城池废墟，位于一处隐秘的丛林之中，旁边还有一片湖泊，在茫茫无尽的妖兽山脉之中，如果不是事先得知，根本很难发现，这里居然有着一座古城。
<br/>　　“主人快看！传送阵在那里！”
<br/>　　幽冥炎蛇忽然指着一个方向，兴奋大喊了一声，然后便直接奔了过去。
<br/>　　叶尘正要跟着过去，却猛然眉头一皱，
<br/>　　“幽冥！小心！！”
<br/>　　就在叶尘声音落下的同时，
<br/>　　“吼！！”
<br/>　　一道犹如天雷般的嘶吼声，陡然在整个妖兽山脉响起！
<br/>　　紧接着，
<br/>　　一道庞大的身影，从一旁的湖泊中爆射而出！
<br/>　　等叶尘定睛一看，顿时为之一愣，
<br/>　　“吞天蟒？”
