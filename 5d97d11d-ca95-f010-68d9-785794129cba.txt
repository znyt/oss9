　　他昂起头，望着树杈上的一块鸡蛋大小的玉石，保持这个姿势也不知道多久了，突然，那玉石外层似乎融化了一般，变成一层明亮的流波，在树杈上原地滚动起来。
<br/>　　巨大的菩提神木一阵轻微地摇晃，有菩提叶不断落下，落在珠子上面，立刻就被融化了，同时，林三生感到身边开始有气息流动，附近的灵气开始向着珠子聚拢，不知道被吸进去了多少。
<br/>　　林三生隐约察觉到什么，转身看去，一团精魄从远处飞来，心中一动，喃喃道：“你果然来了！”
<br/>　　精魄飞到菩提树上，绕着那珠子转了几圈，最后贴附上去，进入了珠子里。
<br/>　　羊脂玉一般洁白的珠子上立刻渗出血来，流到菩提树上，树身再度摇晃起来，一股股的青色的气息，不断从枝叶上流出，灌入珠子里面去。
<br/>　　林三生知道，这是木之灵气，就像人体内的阳气一样，是植物存活的根本，树木存活的时间越长，积累的灵气就越多，到了一定地步，植物就能生出灵智，然后成精，之后不管修炼多久，本质上都是积累这种灵气。
<br/>　　菩提神木，在鬼域不知道生长了多少年，不知道为什么一直没成精，但是树木中间蕴含的灵气，差不多算是鬼域之最了，就算是三界之内，能与之相比的也没几棵植物了。
<br/>　　想到这，林三生有些明白，为什么道风要把珠子弄到这里来了。
<br/>　　珠子不断滚动，吸收着菩提神木的罡气，直到珠子完全蒙上了一层青绿色的印记，逐渐才停止了转动。林三生早把山河社稷图展开，一把抓过这珠子，丢进了图中，自己看看左右，知道此地不宜久留，赶紧开溜了，离开死寂迷林之后，立刻开辟虚空，先到了人间，见到了自己精心挑选的几个最得力的手下，护送着他回道了空界……
<br/>　　芮冷玉坐在石碑旁边，惶惶不安地等着结果——他其实不知道结果是什么，林三生只让她进来等着。
<br/>　　等了不知道多久，突然一颗红绿相见的珠子从天而降，立刻飞身借住，入手的感觉，明明就是之前交给林三生的龙珠，但颜色已经从洁白变成了眼前这样。
<br/>　　珠子落在她手中不久，突然变得冰凉刺骨，芮冷玉正不知道该拿他怎么办，那珠子突然融化，细看之下，竟然变成了一个小小的人，晶莹剔透的，盘膝坐在自己手掌心。
<br/>　　这人只有一根烟那么高，比平常人带在胸口的佛像大不了多少，但是前后饱满，线条十分细致，最令她难以置信的是，这透明小人身体内居然有一根根的红线，仔细看才知道是经脉，麻雀虽小五脏俱全，这小人的经脉竟然一根不少，
<br/>　　芮冷玉心中惊奇，仔细看这小人的打扮，身穿长衫，长长的头发散在脑后，因为个头太小，五官虽然有鼻子有眼的，但因为都凑在一起了，倒是看不清楚。
<br/>　　不过看这打扮，让芮冷玉省了疑心，喃喃道：“不会是道风吧……”
<br/>　　小人突然抬起头看她，道：“指头给我！”
<br/>　　是道风的声音，不过声音很小，十分虚弱。
<br/>　　“什么？”芮冷玉一下子愣住，本能地把指头伸到小人面前。
<br/>　　小人竟然张开了嘴，咬在她食指指肚上，这感觉就像是被小虫子咬了一口，接着芮冷玉便感觉到他真的像蚊子似的在吸自己的血。
<br/>　　不过只吸了几口（比蚊子吸得多不了多少），就松开了，透明的身体也变成了血色，然后他端坐在自己手掌心，张口吐出血气，身体又变得透明起来，那血气就环绕在他周身，忽明忽暗的，不断被他吸进去，接着吐出来。
<br/>　　芮冷玉虽然没见过这种情况，但猜他应该是在吐纳养息，怕自己活动手掌什么的惊动了他，于是把他轻轻放在石碑上沿，在旁边默默观察，眼看着他喷吐的血气越来越淡，周身的光泽却更加明亮饱满，心中也是高兴的。
<br/>　　“呼……”小人重重地吐出了一口气，缓了一会儿，说道：“总算又活过来了。”
<br/>　　芮冷玉听了惊道：“你之前真的死了？”
<br/>　　“元神俱灭，自然是死了。”
<br/>　　元神俱灭……芮冷玉不敢相信，喃喃道：“元神俱灭，那还怎么活过来？”
<br/>　　“不然，我要这龙晶做什么。不过就算有龙晶，我也只有五六层的把握。”
<br/>　　芮冷玉失声道：“你就不怕失败了？”
<br/>　　“总归要试一下，不然如何证道。”
<br/>　　芮冷玉正要细问，迎面走来一个人，抬头一看，是林三生。
<br/>　　林三生见到道风现在的样子，也大吃一惊，同时一颗悬着的心也放下了，询问他情况之后，问起在轩辕山究竟发生了什么。
<br/>　　道风一向不喜欢废话，于是长话短说，讲到上古邪神为自己而死，心中不免也是唏嘘。
<br/>　　芮冷玉二人听罢，更加震惊。
<br/>　　“照你这么说，你肉身、魂魄、元神全斩了，那你又是怎么复活的？”
<br/>　　“我没复活，只是借着龙晶重塑了元神。”
<br/>　　“元神怎么重塑？”芮冷玉惊问。
<br/>　　“其中玄妙，无法言说，我也是头一遭。”
<br/>　　林三生沉吟道：“得亏你复活了，不然真的不知道后果怎么样。不过……既然如此凶险，你为何要这么做？”
<br/>　　道风说道：“我连斩两尸，虽然艰难无比，靠的也是机缘，唯独这第三尸元神，却不知如何能斩。”
<br/>　　万物生灵，魂魄是基础，就像是最核心的分子，而元神是依附魂魄存在的神识的集合体，说白了就是一个人的思维所在。
<br/>　　肉身没了，魂魄还在，还可以做鬼，魂魄要是没了，元神也就没地方依附，立刻也就散去了。但是道风走的这条证道之路，斩去魂魄，靠的是“无中生有”的机缘和悟性，反倒更加强大，但是元神……一旦元神没了，生灵连最基本的思维都没了，一切灰飞烟灭，还谈什么证道？
<br/>　　这就是斩元神的悖论。
<br/>　　甚至道风一度也在怀疑，是不是真的有这种可能。不过青云子跟他说过，正因为斩元神之法看上去极其不可能，反而是真的，如果真那么容易，世上不知道有多少人走这条路斩尸证道了。
