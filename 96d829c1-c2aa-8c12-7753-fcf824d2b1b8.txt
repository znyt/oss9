凌晨五点多钟的药房空无一人，夏夕绾只打开了休息间的一盏壁灯，然后脱掉了身上的衣服。
<br/>
<br/>        少女白腻的娇肌上青紫不一，被咬破的伤口还沁着血迹，夏夕绾拿出了一瓶消毒水，然后用棉签开始处理自己的伤口。
<br/>
<br/>        嘶。
<br/>
<br/>        她白皙澄亮的眼眶都给疼红了。
<br/>
<br/>        这时休息间的门突然被推开了，一道俊俏淡漠的身影出现在了门边。
<br/>
<br/>        夏夕绾没有想到凌晨这个点会有人进来，她瞳仁一缩，几乎是瞬间就伸手扯过衣服遮住了自己。
<br/>
<br/>        “谁？”
<br/>
<br/>        夏夕绾扭头，门边是那个…喜欢趴在药房里睡觉的男人。
<br/>
<br/>        她竟然把这个神出鬼没的人给忘了。
<br/>
<br/>        门边的男人似乎也没有想到会是她，女孩戴了面纱，只见她一双黑漉灵动的水眸极其警惕的望向他，里面偏偏还因为刚才的疼痛而染上了星点的水光，羽捷轻颤着，那种过于精致的绝色美丽让人几乎移不开目光。
<br/>
<br/>        男人看了她一眼，然后关上了门，自觉的退了出去。
<br/>
<br/>        夏夕绾松了一口气，据她观察，这个喜欢睡觉的男人人品还是有的，应该不会偷窥，但是她还是不放心，就动作快速的处理了身上的伤口。
<br/>
<br/>        她手里没有药，这些伤口还需要抹药膏，夏夕绾想着待会儿等枢密上班了，她要买个好一点的药膏抹一下。
<br/>
<br/>        这样身上的伤口才能快点好。
<br/>
<br/>        夏夕绾打开了休息室的门，走了出去，但是很快她脚步一滞，因为她在门边的桌面上看到了一小瓶药膏。
<br/>
<br/>        白瓷玉瓶的小药膏，看着十分的珍贵。
<br/>
<br/>        这是哪里来的？
<br/>
<br/>        是刚才那个喜欢睡觉的男人送给她的？
<br/>
<br/>        夏夕绾现在很需要药膏，想了一下，她就收下了药膏，然后进了休息室将药膏抹在了身上。
<br/>
<br/>        刚才还火辣辣疼痛的伤口顿时感觉到了一股清凉，然后疼痛感迅速消失了。
<br/>
<br/>        这个药膏真的很神奇。
<br/>
<br/>        夏夕绾拿出了笔，她在一张字条上写了一行字，放在了那个喜欢睡觉的男人桌子上。
<br/>
<br/>        现在时间还很早，夏夕绾苍白的眼下已经有了疲倦的乌青，脸色看着很不好，所以她就合衣躺在休息室的小床上，很快就进入了梦乡。
<br/>
<br/>        这时，男人回到了自己的座位上，他很快就看到了那张字条，上面有一行娟秀的小字上一次的泡面加上这次的药膏，你已经帮我两次了，希望以后可以还给你。
<br/>
<br/>        男人将这张字条压在了一本书里，然后打开了一本医术无聊的看着…
<br/>
<br/>        ……
<br/>
<br/>        双双来的时候夏夕绾已经起床了，她人在枢密的藏书阁里。
<br/>
<br/>        “绾绾，这大清早的你怎么在这里研读医书啊，学霸都这么刻苦了，让我们学渣怎么活啊？”
<br/>
<br/>        夏夕绾翻阅了几本医书，现在陆寒霆的病情很棘手，她必须研制出药丸，但是有一味药她拿不准，需要找到一本医书。
<br/>
<br/>        “双双，我在找书。”
<br/>
<br/>        “什么书啊，我帮你一起找。”
