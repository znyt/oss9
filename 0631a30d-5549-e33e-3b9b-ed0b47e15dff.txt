        “我在感情上背叛了你？傅昼景，你不觉得你说这句话很可笑吗？”童筱颖嗤之以鼻道。
<br/>
<br/>        傅昼景心痛地扯了扯嘴角：“难道不是吗？不管是离婚前还是离婚后，我爱的女人，只有你。但是，你却爱上了我大哥。在感情上，难道不是你背叛了我吗？”
<br/>
<br/>        “我为什么要继续爱着你这个在外喜欢沾花惹草，不是跟这个女人上床，就是跟那个女人上床的男人？”童筱颖抬眸瞪着傅昼景，娇俏的脸蛋上没有任何表情。
<br/>
<br/>        “我大哥难道就不是这种男人吗？跟过他的女人，你难道不知道有多少个？”
<br/>
<br/>        “我不想和你废话，你放开我！”
<br/>
<br/>        “筱颍，我们复合，好吗？我知道，大哥他已经出国了。他身边不缺女人的，他不是真心爱你的。筱颍，他对你，只不过是感恩你救过他罢了。”
<br/>
<br/>        他的指腹，摩挲着她的脸颊，就像是在爱抚着这世间最珍贵的宝贝。
<br/>
<br/>        童筱颖听着他这话，觉得心很累，怪不得大哥不许她再见他，也是有原因的吧！
<br/>
<br/>        “你要我说多少遍？我们之间不可能了！你放过我，好不好？也放过你自己。”
<br/>
<br/>        她想要推开他，他却紧紧地捧着她的脑袋不放。
<br/>
<br/>        无奈之下，童筱颖不得不抬腿用膝盖往他脆弱的部位用力顶去。
<br/>
<br/>        傅昼景一阵吃痛地捂着身下，俊脸上表情扭曲。
<br/>
<br/>        看着他那痛苦的表情，就知道她这下脚的力度有多重了。
<br/>
<br/>        但是，她顾不上那么多，趁着他松懈之际，立即推开他后，松开了安全带，从副驾驶座上跳了下去。
<br/>
<br/>        他亲自来参加这种芝麻屁大的图纸答疑会，不为别的，只为了在见见他爱的筱颍。
<br/>
<br/>        可是……
<br/>
<br/>        傅昼景扶着车门，剑眉微蹙地凝望着童筱颖仓皇而逃的背影，心痛得无法呼吸。
<br/>
<br/>        如果，时光可以倒流……
<br/>
<br/>        如果，让他早一些发现，她就是孩提时期，烙在他心口上的那个“筱颍”。
<br/>
<br/>        他一定会好好地守护住她。
<br/>
<br/>        当初，他为什么要选择离婚啊？
<br/>
<br/>        他怎么就那么傻呢？
<br/>
<br/>        傻到高估了筱颍对他的感情，甚至认为无论他怎么做任何事，筱颍都会对他从一而终。
<br/>
<br/>        呵——
<br/>
<br/>        他真傻！
<br/>
<br/>        童筱颖担心傅昼景会追上来，一时回头，结果忘了查看前方的路况，不小心撞到了别人，就连同她自己也差点跌倒在地，是对方伸手拉了她一把。
<br/>
<br/>        “童筱颖，这么巧，你这是又回来了吗？”
<br/>
<br/>        闻声，童筱颖怔愣地抬眸看去。
<br/>
<br/>        居然是夏凌焕！
<br/>
<br/>        前前后后，确实有那么一段时间未见过这个男人了。
<br/>
<br/>        他是夏凌焰的堂弟，还是傅夜沉他干爹的儿子。
<br/>
<br/>        好像是这么一层关系吧！
<br/>
<br/>        童筱颖依稀记得这些关于他夏凌焕的信息。
<br/>
<br/>        “好巧。”童筱颖站稳了身子，微微一笑。
<br/>
<br/>        夏凌焕放了手，随之将另一只手上的袋子，举到了童筱颖的眼前，苦笑着恳求道：“中午，上我家去做顿饭，好不好？我妹今天来滨江城了，她又不愿意去下馆子。你手艺不错，她一定会喜欢吃你做的菜。”
<br/>
<br/>        “你妹妹？”童筱颖一愣，好像上次听夏凌焰提起过关于他夏凌焕家里的一些事情。
<br/>
<br/>        说是他夏凌焕的亲妹妹被十爷深爱的那个婴宁也就是夏玄烟给弄丢了，然后夏凌焕的父亲为了安抚他母亲，重新领养了一个一般大的女婴。
<br/>
<br/>        好像是这个情况吧！她记得也不大清楚了。
<br/>
<br/>        “拜托了！好不好？”夏凌焕双手合十，苦苦哀求起来，“看在我两曾是邻居，又是朋友一场的份上。”
<br/>
<br/>        童筱颖回过神来，忍俊不禁，接着莞尔一笑，微微点了点头。
<br/>
<br/>        跟着夏凌焕去了他的公寓里后，童筱颖便直接切入主题，系了围裙，在厨房里忙活了起来。
<br/>
<br/>        饭菜刚好上桌，夏凌焕领着一个女孩进了门，童筱颖一边解开身上的围裙，一边朝夏凌焕那边望去，看到跟在夏凌焕身边的女孩时，身心不禁一怔。
<br/>
<br/>        居然是她！

<br/>        早上，在傅氏集团大厦的一楼大厅里，叫住裴金澈的那个女孩。

<br/>        “这是我妹妹，夏玄灵。”夏凌焕微笑着介绍道。

<br/>        不等他介绍童筱颖，夏玄灵便俏皮地说道：“她一定是我的嫂子了，对不对？”

<br/>        原来他们彼此都认识。
