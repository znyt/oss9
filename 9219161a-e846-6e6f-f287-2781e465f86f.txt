　　“九霄琉璃焱的威力也增强许多，暂时先回天墉城吧。”
<br/>　　说着，神火自身躯上席卷而出，周围空间荡起一丝微不可查的涟漪，旋即身影就自上空消失的无影无踪。
<br/>　　眨眼间，数十里外。
<br/>　　此时，慕清霄已经化作人型姿态，立于高空之上，回头瞥了一眼花香镇，转身向天墉城掠去。
<br/>　　就在这时，星眸中的世界变得昏暗起来，时间也在此刻停止。
<br/>　　慕清霄的身影停滞在空中，浑身不得动弹，同时一道清脆悦耳的声音，在他的脑海中响起。
<br/>　　“奇怪，居然看不透你的命数，某种力量挡下本皇的探测，似曾相识的气息，远古？似乎更加遥远......”
<br/>　　身躯停滞在空中，脑海中清脆悦耳的女子声音，却是令得他汗毛炸立，但下一秒身躯就恢复控制权，天地也恢复正常。
<br/>　　哪怕身体恢复控制权，慕清霄依旧一脸懵逼的望着四周，难道自己的突破，引起了某尊大佬的注意？
<br/>　　听声音应该是女子，自称本皇，难道是......
<br/>　　想到此处，慕清霄微微摇头，心中更加警惕，暗道：“系统，刚才有人窥探我，可知道是谁？”
<br/>　　“叮，宿主需支付一千万掠夺积分。”
<br/>　　“你特么怎么不去抢！”
<br/>　　在心底暗骂一声，慕清霄就报着复杂的情绪，迅速离开此地，决定往后要低调。
<br/>　　不过，对方似乎没有敌意。
<br/>　　但这种窥探，令他心中非常的不舒服，仿佛自己的生命完全不再掌控中，而是掌控在别人手里。
<br/>　　“自己终究是太弱了。”
<br/>　　...
<br/>　　...
<br/>　　回到天墉城后，慕清霄率先回到自己的宫殿，从小兰口中得知，距离上次离开已经过去十天的时间。
<br/>　　期间，涵素真人派人来过一次。
<br/>　　得知掌门派人来过，慕清霄也不耽搁，立刻前往天墉城主殿。
<br/>　　片刻后，来到主殿，涵素真人正在与一名身穿蓝袍，风姿卓越的中年男子讨论着门下弟子的修炼情况。
<br/>　　“见过掌门。”
<br/>　　来到殿前，慕清霄微微行礼，打量了一眼蓝袍中年，完全看不透，甚至没有一丝的气息泄露，返璞归真。
<br/>　　仙风道骨，气势恢宏，具有长者的威严，但他却能从后者身上，感觉到一股恐怖的剑意。
<br/>　　若是猜的不错，面前的蓝袍中年，估计就是执剑长老，剑仙紫胤真人。
<br/>　　紫胤天资极高，修为超绝，德高望重又内敛自持，品行令人敬慕。
<br/>　　外冷内热，仙风道骨，看似冷峻肃穆，实则内敛而重情义，处事原则分明，对于天道循环等有较为透彻的感悟。
<br/>　　痴迷剑道，喜好收藏名剑，所持宝剑多为自己铸造，身边有两名古剑剑灵，对各类古剑也有过人的见识和认知。
<br/>　　现任昆仑山天墉城执剑长老，已经修炼成仙，剑术绝伦，道法高深。
<br/>　　三百年前，紫胤云游四海时，偶闻安陆虞家藏有千年古剑，或蕴剑灵，因心中喜悦，便前往安陆拜会虞家，因而认识红玉，帮助其救下被镜罔控制的虞晓莲。
<br/>　　随后，因与天墉城第六代掌门交好，受邀来到天墉城担任执剑长老，带来了自身擅长的铸剑秘术及御剑术。
<br/>　　正是因此，才使天墉城整体实力提高许多，成为天下第一大派。
<br/>　　慕清霄打量着紫胤真人的同时，后者也打量着他，眼中满是惊叹。
<br/>　　片刻后，紫胤微笑道：“没想到半个月过去，凝丹长老已经迈入练虚期了，真是令人望尘莫及的天赋。”
<br/>　　身旁的涵素真人也一脸惊叹，他虽然知道慕清霄资质妖孽，但也没有料到，短短半个月，后者就迈入了练虚期。
<br/>　　慕清霄不卑不亢，道：“执剑长老过誉了。”
<br/>　　紫胤真人眼中闪过一丝赞赏，道：“肺腑之言，凝丹长老的天赋，乃是紫胤有生以来，见过最佳之人。”
<br/>　　清霄报以微笑不再闲聊下去，看向涵素真人，道：“掌门，不知找在下何事？”
<br/>　　“半个月前，慕长老炼制的培元丹，令本座非常的满意，真是想不到，慕长老如此年轻，在丹道上的成就也如此高深呐。”
<br/>　　说着，涵素真人面露和煦笑容，心中却是非常惊叹。
<br/>　　慕清霄所炼制的五十炉培元丹，他亲自检验过，每一刻都是极品丹药，比还虚真人所炼制的，药效高上数倍。
<br/>　　很难想像，方年二十，不仅修为达到了练虚期，更是一位七品炼丹师，丹武双绝。
<br/>　　如此妖孽的天赋，令他与紫胤都感到汗颜，若非慕清霄现在是天墉城长老，涵素真人都想将他收为亲传弟子。
<br/>　　“最近门下有一批弟子，在外斩妖除魔，表现颇为优异，本座想请慕长老，帮忙炼制一批‘玄灵丹’给予门下表现优异的弟子作为奖励。”
<br/>　　玄灵丹，三品灵丹，有七成的几率令得筑基巅峰弟子，顺利迈入金丹期，对于门派来说，此类丹药非常的实用。
<br/>　　“这是在下分内之事。”
<br/>　　慕清霄微微点头，一口答应，炼制一批三品丹药，浪费不了他多长的时间。
<br/>　　闻言，涵素真人非常满意，越看越顺眼，将手伸进衣袖中，取出一枚储存玉镯。
<br/>　　“这枚储存玉镯你便收下，其中有一些凝丹堂都不曾有的丹方，以及多种高级灵药，炼制玄灵丹的材料也在其中。”
<br/>　　说着，涵素真人上前一步，将储存玉镯交给慕清霄。
<br/>　　接过玉镯，慕清霄查看了一下其中的东西，不动声色的收了起来，但心中却是非常的惊喜。
<br/>　　没想到，涵素真人居然给他送来这么一份大礼。
<br/>　　其中有许多的灵药，虽然品级不高，但胜在数量，而且还有许多凝丹堂都没有的丹方。
<br/>　　他当然清楚，涵素真人是在拉拢他，或者说，真心想将他留在天墉城。
<br/>　　“多谢掌门。”
<br/>　　见慕清霄将玉镯收下，涵素真人顿时松了口气，如此妖孽的天才，必须将之留在天墉城。
<br/>　　妖孽天才本就少之又少，若是往后让别的门派撬了墙角，他哭都没地方哭去。
<br/>　　要知道慕清霄不仅修炼上天赋妖孽，更重要的是，在丹道上的天赋亦是如此妖孽，如此一来，意义就完全不同了。
