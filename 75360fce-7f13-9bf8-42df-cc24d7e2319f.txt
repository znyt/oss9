　　此时，涂小安冷淡无波的望着不远处一条奄奄一息的蛇，原本通体鲜艳，却黯淡无光。
<br/>　　“你输了”
<br/>　　他已胜利者的姿态宣告这场大战可以落下帷幕了。
<br/>　　鸡冠蛇连抬一下蛇头的力气都没有了，倒在自己流的血的血泊中，它不相信自己怎么可能会输。
<br/>　　望着灵堂的方向，一个轮椅的老人让它咬牙切齿，它的仇还没报呢。
<br/>　　他还活着呢。
<br/>　　“身为同类，你为什么帮该死的人类”
<br/>　　鸡冠蛇不甘心，它用了整整五十年的光景才从一条无毒蛇，变成了变异家蛇，成精成妖。
<br/>　　转眼，一切成空。
<br/>　　“同类？”
<br/>　　涂小安惨然一笑，他现在虽是蛇身不假，可灵魂却还是人类的灵魂。
<br/>　　他倒也说不上帮，本来就是来拿蛇，总不能等着鸡冠蛇先将赵家灭门，他再出手吧。
<br/>　　这完全没有意义。
<br/>　　“偷偷告诉你的一个秘密”
<br/>　　只见金银蛇游曳的奄奄一息的鸡冠蛇面前，昂首而立，：“其实我并不是一条蛇，而是人，你信吗”
<br/>　　蓦然，鸡冠蛇像是回光返照一样的看着它，要不是身体脆弱，没有力气，它真想如弹簧一样的跳起来，破口大骂：“信尼玛的球”
<br/>　　可惜，不管这条蛇信不信，懂不懂，这都无关紧要，涂小安的蛇眸此时像是两汪平静无波的湖，然后微微的泛着涟漪：“一路走好吧”
<br/>　　接进来便是残忍的画面，只见这条要灭赵家满门的变异鸡冠王蛇，一点一点的被金银蛇吞噬到了肚子里去，没有半点挣扎。
<br/>　　赵家人顿时惊呼起来，先前还是两条蜿蜒大蛇撕杀，现在就转变成了蛇吃蛇。
<br/>　　同时他们是又庆幸，又害怕起来，那么厉害的家蛇都被另一条蛇干掉，吞掉。
<br/>　　那么接下来，这条蛇会不会想吃人。
<br/>　　赵老爷子看到鸡冠蛇被吞，一张消瘦而憔悴的老脸，痛哭流涕了起来。
<br/>　　没有人知道这位轮椅上的老人是激动的哭，还是痛苦的哭。
<br/>　　赵家墙头，李清照再次爬了上来，之前两条蜿蜒大蛇大战，宛如地震降临，让她只能找个地方躲起来避难。
<br/>　　等她好不容易再次爬到墙头，看到的却是蛇吃蛇的残忍画面。
<br/>　　那条鲜艳，能说话，能变色，能变大的家蛇，被另一条蛇活生生的吞进了肚子里去。
<br/>　　忽然，一股悲悯涌上心头，这是杀人蛇，它死有余辜，但李清照好心疼，心好痛。
<br/>　　仅仅一眼，她就喜欢上了这条独一无二的家蛇，鲜艳夺目，条纹五光十色的使人眼花缭乱，它是那么的美丽，那么的有灵性。
<br/>　　毫无疑问，这是一条有故事的蛇，李清照迫切的想知道它隐藏的故事是什么。
<br/>　　可这条蛇现在没了，没了。
<br/>　　涂小安在众目睽睽之下，将鸡冠蛇吞噬了进去，他的蛇躯发光发亮了起来，整条蛇吞进去，忽然，他脑海中出现了一股不属于他的记忆画面。
<br/>　　画面一道道的浮现，他就如一个坐在电影院看大屏幕的观众。
<br/>　　“这是鸡冠蛇的生前经历”
<br/>　　下一秒，涂小安了然于心，他开始认真的了解这条蛇的经历。
<br/>　　正常来说，一条蛇的记忆是有限的，它除了睡，就是在猎食，短短一生，毫无看点。
<br/>　　可鸡冠蛇不同，这种成为变异家蛇，活了不知道多少年。
<br/>　　涂小安也好奇为什么这条蛇非要灭赵家的门，非要让赵老爷子自杀。
<br/>　　一幕幕画面闪过..．
<br/>　　冷僻后院，一位年轻女子吹着笛箫，笛声悠扬飘荡，绵延回响，一条苍翠小蛇缭绕着她身边游曳，故作亲腻状，像是在为女子伴舞。
<br/>　　“咯咯咯...”
<br/>　　冷僻后院响起一阵银铃般的笑声。
<br/>　　下一个画面。
<br/>　　年轻女子坐在门沿上发呆，怀中抱着苍翠小蛇，它非常的乖巧，陪着自己的主人一起发呆，从清晨到黄昏，从夕阳西下，到黑幕降临。
<br/>　　一人一蛇，像是两小无猜，相处融洽。
<br/>　　在一个画面。
<br/>　　年轻女子挺着微微隆起的肚子，望着眼前乖巧的小蛇，含笑的说：“我的孩子要是出生了，你来帮我保护他好吗”
<br/>　　苍翠小蛇像是听懂了，小脑袋上下的点着，样子很是滑稽，却透着浓浓的灵性。
<br/>　　又是一个画面浮现。
<br/>　　年轻女子挺着大肚子，趟在病床上，她如玉的容颜苍白无血色，没有半点生机。
<br/>　　一条苍翠小蛇发疯似的用自己的小脑袋蹭着自己的主人，企图唤醒她。
<br/>　　可就算它如何的努力，她的主人都没在睁开眼望它一眼。
<br/>　　一道道低沉的嘶悯声在这个房间响起，那是小蛇绝望的呐喊，因为她的主人已经香消玉碎了。
<br/>　　一滴晶莹的眼泪从小蛇的眼眶中悄然落下。
<br/>　　小蛇哭了，她的主人死了，下一刻，它仰天咆哮，一双蛇眸变的嗜血可怕。
<br/>　　独独房内，一尸两命，唯有一条小蛇在哭，在悲痛欲绝，在愤怒。
<br/>　　“你可以成为我一个人的家蛇吗”
<br/>　　这段余音充斥在鸡冠蛇全部的记忆之中，无法磨灭。
<br/>　　一人一蛇之间的情感，纯粹而朴实，真实却透着梦幻。
<br/>　　看着看着，涂小安心中徒生一种悲凉，他终于知道鸡冠蛇的主人是谁了。
<br/>　　当他看到年轻女子的尸体被赵老爷子做成骨头挂在赵家房梁下的时候，在这片记忆中，充斥着鸡冠蛇无法仰止的杀气。
<br/>　　原来..．
<br/>　　原来这是一条宠物蛇为自己主人惨死而复仇的悲烈故事。
<br/>　　看完这段记忆，他好像做错了什么。
<br/>　　涂小安生前是捕蛇人，赵老爷子是他敬重的捕蛇大师，原来他年轻的时候做下此等不可原谅的事情。
<br/>　　他欺骗了一个善良天真的女老师感情，他没有尽到自己的责任，保护好她，更令人发指的是，他居然听了道士荒唐的话，将自己心爱的女人做成一块骨头，高高悬挂赵家房梁。
<br/>　　还企图换来赵家的一世安康。
<br/>　　该杀，该死。
<br/>　　如今的涂小安真正明白，赵老爷子为什么说该死的那个人是他。
<br/>　　确实，该死的那个人是他。
<br/>　　还有一段回忆。
<br/>　　“你好像并不喜欢他对吧，每次他一来，你就躲的远远的”
<br/>　　年轻女子笑着朝一个隐晦角落招了招手，苍翠小蛇游曳了出来。
<br/>　　“他是我未来孩子的父亲，他也是我唯一这辈子爱过的男子，我知道这种爱是错误的，但我也没的回头了”
<br/>　　“你答应我，不管他以后对我做了什么，你都不能伤害他，知道吗”
<br/>　　苍翠小蛇没有作答，年轻女子点了它一下小脑袋，笑道：“你不说话，我全当你答应了哦”
<br/>　　我答应主人不会杀你....原来如此。
<br/>　　鸡冠蛇想为主人报仇，只能费尽心思的逼着赵老爷子自杀。
<br/>　　“对不起，我不该阻止你，但你没完成的愿望，我会帮你实现”
<br/>　　涂小安在心底暗暗的留下了这样一句话，他吞了鸡冠蛇，总要为它做点什么，更要为那个可怜的女人做点什么。
<br/>　　她的死是那么的惨烈，几十年埋葬在岁月之中，无人知晓，无人提起，如空气般蒸发。
