老说出这个话后，心脏也开始不由自主的收缩了一下，这是给陈默上套呢，要是陈默他答应了地话，那帮子老兄弟就有些希望，要是拒绝，那么老兄弟还要继续遭罪啊。<br/>　　<br/>　　    “啊！我说我躺在这里，正舒服着呢，没有想到你过来这么说，看来是有预谋的啊咦？似乎纪老过来，是不是？”陈默半坐起来，盯着何老问道。<br/>　　<br/>　　    “呵呵！反应过来了？”<br/>　　<br/>　　    “不错！”陈默想到纪老无端出现，而且何老还那么说话，本来在给纪老看病的时候，心中就有些想法的，但是却非常的模糊，不过现在何老这么一说，自己也就明白过来了。这人啊，怎么这么多的心眼呢？<br/>　　<br/>　　    “也不是说对你耍心眼什么的，主要是一帮老兄弟之间关系还不错。看到我和老吴来到你这里后，这身体好了，吃的多了，脸色什么的也红润了，气色也不错，精神头更加的好，所以那些家伙们肯定会问啊。<br/>　　<br/>　　    而且老吴又回去检查了一下身体，发现恢复的不错，这就更加的了不得，大家都有些羡慕，所以就想都过来。”何老说道。<br/>　　<br/>　　    “这不行，要是人多了，地方都不够。”陈默想到这里的房间就那么几个，还真的不是现在就进人的时候。<br/>　　<br/>　　    “你这是有钱不赚啊！”何老也有些无语，只能这样说，而且他也知道，房间就那么几个，绝对的不足。<br/>　　<br/>　　    “没有办法，只能随缘了。等以后葫芦谷开发出来了再说喽！”陈默倒是没有太多的反应，反正赚的这些钱，也就是多少的问题。<br/>　　<br/>　　    “哈哈！这是不是你们年轻人说的佛系生活啊，人来人走都随缘，不强求也不奢望？”何老倒是听了解现在年轻的词语的，看来深受那个叫何兮的女孩影响。<br/>　　<br/>　　    “不错！我这就是宅男佛系生活，一切随缘吧！”陈默也不过多解释，笑着回答。<br/>　　<br/>　　    “说正经的，你出不出诊？”何老想到一些人，所以就问道。<br/>　　<br/>　　    “不出。”陈默回答。自己出诊干嘛，在这个看脸的社会，自己就是个年轻人，出去对人说是中医医师，有几个人能相信技术高超的。<br/>　　<br/>　　    如果给人看个病，还要解释半天，赚不了几个钱不说，还容易得罪人。还不如就在这里躺着舒服，能到这里来，能看的就看，看不了的走人就是了，多自在！<br/>　　<br/>　　    何老听着陈默的回答，倒是没有再说什么，既然不去，再劝也没有用，只能点点头，起身回到他们的院子当中。不过想到那老兄弟，只能在心里说抱歉了。对于陈默的拒绝，其实何老还是有准备的！<br/>　　<br/>　　    好吧，陈默还只是佛系宅男，还好没有想着做什么精致的猪猪宅男！<br/>　　<br/>　　    如果感觉不错，请推荐和收藏一下，谢谢大家哈！<br/>　　<br/>　　    7真人小姐姐在线服务，帮你找书陪你聊天，请微/信/搜/索热度网文或rdww444等你来撩~<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
