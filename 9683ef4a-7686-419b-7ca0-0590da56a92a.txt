　　苏莫心中咯噔一声，瞬间沉到了谷底，他怎么也没有想到，大祭司会亲自来此。
<br/>　　此地聚集的众多的势力，基本上都是武尊境的强者带队，并没有武圣境的超级强者前来。
<br/>　　在这个虚神境强者不出的时代，武圣已经是古灵星河的巅峰强者，谁会来浪费时间来此，均是各自势力的高阶武尊带队。
<br/>　　之前，苏莫根本没想到大祭司会亲临，现在则是彻底变了脸色。
<br/>　　“见过巫仑大祭司！”
<br/>　　“见过巫仑大祭司！
<br/>　　“巫仑大祭司亲临，在下有礼了！”
<br/>　　各大势力的强者之中，认识巫仑大祭司的人，纷纷向其见礼。
<br/>　　“嗯！”大祭司微微颌首，随即目光扫视四方，很快便看到了苏莫。
<br/>　　见苏莫端坐在土丘上，同样在望着自己，大祭司眸中杀机一闪，就想将苏莫抹杀。
<br/>　　不过，很快他便放弃了这个想法，既然他答应了火元圣师和杜府主，要在真圣界中解决此事，自然不会食言。
<br/>　　一切，都会在真圣界做个了断。
<br/>　　就算退一万步来说，苏莫侥幸在真圣界中活了下来，他再亲自出手也不迟。
<br/>　　只要他在这里，苏莫就已经是一个死人了。
<br/>　　况且，苏莫能在真圣界中活下来的可能性，基本可以忽略不计。
<br/>　　随即，大祭司袖袍一挥，无数人影闪烁，大量的巫族天才出现了，人数超过一万。
<br/>　　这一次，大祭司亲自出马，将巫族三大支脉和主脉的天才子弟，全部都一并带来了。
<br/>　　“苏莫！”
<br/>　　众巫族子弟之中，巫人王一眼就看到了苏莫，顿时大喝一声，咬牙切齿，全身气息都控制不住的涌动了起来。
<br/>　　“他就是苏莫？”
<br/>　　“才武帝境六重修为而已！”
<br/>　　“呵呵，这就是真圣榜第一，不过如此！”
<br/>　　“我翻掌间就能镇杀！”
<br/>　　一众巫族天才，顺着巫人王的目光，均是看到了苏莫，不少人纷纷开口，多是面带不屑之色。
<br/>　　天冥星支脉的巫族天才，对苏莫非常仇恨，但其它支脉以及主脉的人，则没有什么仇恨。
<br/>　　不过，他们来之前，巫仑大祭司已经有交代，让他们在真圣界斩杀苏莫这个仇人，他们自然也不会拒绝。
<br/>　　“诸位，苏莫就交给我了，你们无需多管！”巫人王深吸口气，压下心中的杀机，向其他人说道。
<br/>　　笑话，苏莫身上可是拥有掠夺血脉的秘密，他当然不能让其他人将苏莫杀了。
<br/>　　关于苏莫身上的这个秘密，巫人王没有告诉任何人，甚至连他自己身上血脉之力大减的事，都没有人知道。
<br/>　　这一次是他的机会，一举擒拿苏莫，获得这个秘密。
<br/>　　只要他能得到这个秘密，就能夺取别人的血脉，无论是天赋还是实力，都能暴涨。
<br/>　　只是想一想，巫人王都怦然心动。
<br/>　　“巫人王，我可是听说你曾经败给了苏莫，你能搞定吗？”一名巫族主脉的天才调笑道。
<br/>　　巫人王曾经败给巫尘的事情，也在其它几脉之中传开了，传闻中苏莫混进了天冥星，化身巫尘，打败了巫人王。
<br/>　　“哼！那是同阶之战，现在我能随意捏死他！”巫人王冷哼道。
<br/>　　“巫人王，那就交给你了，不要再让我出手！”大祭司的身边，一名身穿紫衣的巫族青年淡淡的说道。
<br/>　　此人脸庞狭长，一头长发飘舞，双臂交叉环抱在胸/前，给人一种玩世不恭的姿态。
<br/>　　“巫殇，无需你出手！”巫人王向对方说道。
<br/>　　这名玩世不恭的紫衣青年，正是巫族主脉武帝境修为的子弟中第一高手，同时也是第一天才，真圣榜排名第八的巫殇。
<br/>　　远处其它势力的人，听闻一众巫族子弟的交谈，暗暗惊讶，原来苏莫居然和巫族有仇，这些巫族天才都要杀苏莫。
<br/>　　这下有趣了，真圣榜第一的苏莫，如何能挡住一众巫族的天才？
<br/>　　而更让所有人感到有趣的是，若是有大仇恨的话，现在就可以解决，为何还要等待进入真圣界呢？
<br/>　　不过，虽然众人都不清楚其中的原委，但均是暗暗为苏莫感到可惜。
<br/>　　苏莫被排在真圣榜第一，虽然有些名不副实，但天赋超高毋庸置疑，在真圣碑留名的希望还是很大的，但现在有了巫族这些敌人，怕是要葬生在真圣界了。
<br/>　　一众神武学府的人，无论是近万弟子，还是十几名导师，除了尤天韩之外，个个都是有些懵逼。
<br/>　　这什么情况？
<br/>　　巫族的人都要在真圣界中杀苏莫？
<br/>　　这样的话，苏莫进入真圣界岂不是死路一条了？
<br/>　　领队的吴晋导师和唐婉导师对视一眼，两人脸色均是难看了下来，这种事情他们完全没有想到。
<br/>　　但是现在，巫族支脉的大祭司在此，他们也不敢有什么异动。
<br/>　　任云都和谢彬两人，他们和苏莫关系较好，现在不禁脸色一变。
<br/>　　两人转头看向苏莫，只见苏莫望着天空中的一众巫族之人，脸色亦是有些难看，仿佛是被吓到了。
<br/>　　其实，苏莫并不是被巫族的一众天才吓到了，而是被大祭司的突然到来，搞得心中没有了底气。
<br/>　　就算他能在真圣界中打败巫人王，就算他能打败所有巫族的天才，那出来之后呢？
<br/>　　他出来之后，大祭司绝对不会放过他，这一点苏莫能肯定。
<br/>　　这一次，他真的是危险了，似乎他前方只剩下了两条路，要么在真圣界中被巫族天才杀死，要么出来之后被大祭司杀死。
<br/>　　两条路，全部都是死路。
<br/>　　木璃导师站在苏莫前方，转头看向苏莫，又看向天穹上的众多巫族之人，柳眉微皱了起来。
<br/>　　她之所以来此，不是学府安排，而是自行前来，就是想要看看苏莫的表现，没想到居然会发生如此状况。
<br/>　　嗖！
<br/>　　就在此时，天穹之上的巫人王，身形滑落上空，如同一道闪电一般，瞬息间便来到了苏莫前方十数丈的半空中。
<br/>　　“苏莫，上一次在大荣城被你逃了，这一次你将逃无可逃！”巫人王居高临下，目光冷视着苏莫，森然的说道。
<br/>　　这一次他不再担心了，有大祭司在此，苏莫已经没有了逃跑的资格。
<br/>　　“你还没有让我逃的资格！”苏莫淡淡的说道，他不惧巫人王，也不惧任何巫族天才，他担心的只有大祭司一人。
<br/>　　此刻，他不禁心念急转，暗思脱身之策。
<br/>　　“是吗？”
<br/>　　巫人王不屑一笑，沉声道：“上次同阶一战我输给了你，这一次你会见识到我真正的实力！”
<br/>　　言罢，巫人王也没有和苏莫多言，便转身飞了回去。
<br/>　　尤天韩望着巫人王离去的背影，又看了看苏莫，嘴角勾起淡淡的笑容。
<br/>　　苏莫，必死无疑了，他已经不用出手了。
<br/>　　就算巫人王杀不死苏莫，其他巫族天才，也不会放过苏莫了。
