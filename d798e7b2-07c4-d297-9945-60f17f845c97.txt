夏夕绾背着包离开了教室。
<br/>
<br/>        大家将礼物都塞到了她的桌肚子里，塞了满满的，然后追出去，跟在了夏夕绾的身后。
<br/>
<br/>        这简直成了A大的奇观，一道最靓丽的风景线，大家簇拥着她，将她簇拥在中间，一行人浩浩荡荡的。
<br/>
<br/>        范恬笑道，“绾绾，你没有吓到吧，我们A大很久没有来美女了，他们都是翻墙去T大看厉嫣然校花的，现在你来了，他们就都看你了，你就是他们争着抢着的新玩具。”
<br/>
<br/>        夏夕绾勾唇表示理解。
<br/>
<br/>        这时很多人拿出手机来拍照，将夏夕绾的照片放在了A大的论坛上，标题就是热烈欢迎A大新校花！
<br/>
<br/>        此时，一辆劳斯莱斯幻影豪车缓缓停在了T大的学校门口，陆寒霆来了，来接厉嫣然放学了。
<br/>
<br/>        他并没有下车，而是将驾驶车窗缓缓滑了下来，让新鲜的空气镀进来，安静的在等。
<br/>
<br/>        这时前方传来了一阵骚动，是隔壁A大门口出现了浩浩荡荡的一群人，正值学生放学的高峰期。
<br/>
<br/>        他深邃的狭眸往那里扫了一眼，然后淡淡的移开，可是两秒后，他又将视线落了过去。
<br/>
<br/>        陆寒霆在茫茫人群里一眼就看到了被簇拥在中间的夏夕绾。
<br/>
<br/>        女孩纤柔的身影一下子就冲击到了他的眸底，她穿着学校的校服，跟周围那些女学生们穿的并没有什么不同，但是就那样不同。
<br/>
<br/>        相同的衣服她身段最好，发育娇好的少女身体如弱柳拂之，纤莹的曲线勾勒到极致，她肤白如雪，那种牛奶白让人生出一种想摧毁的禁忌感，后面挂着一个小小而精致的黑色背包，绝丽雅致，纤尘不染。
<br/>
<br/>        陆寒霆瞳仁狠狠的一缩，他不知道心里有什么东西被刺痛了，他只是盯着那道纤柔的身影一下子眼眶又烫又热。
<br/>
<br/>        很疼，很疼。
<br/>
<br/>        厉嫣然开心跑出来的时候就看到了这一幕，陆寒霆英挺的坐在劳斯莱斯的驾驶座上，单手握着方向盘，他在看一个女孩儿，盯着一个女孩儿看，因为太专注连眼皮都没有眨一下。
<br/>
<br/>        厉嫣然从来没有见过他用这种眼神去看一个女孩儿。
<br/>
<br/>        他始终都是淡然疏冷的，跟她在一起的时候也淡淡的，哪怕她亲他时，他也没有什么反应，她以为他对女孩子不热衷的。
<br/>
<br/>        可是现在，他用一种炙热到发狠的目光去看一个女孩儿。
<br/>
<br/>        厉嫣然也察觉到了今天的不同，平时她放学，T大的门口人潮涌动，尤其是A大那些不学无术的富二代们都会涌过来偷看她，虽然她看不上那些人，但这已经是一种习惯了。
<br/>
<br/>        今天，那些人没有来。
<br/>
<br/>        一个都没有来。
<br/>
<br/>        厉嫣然抬头看向A大的校门口，现在那里人潮涌动，平时看她的那些人都围在另一个女孩身边。
<br/>
<br/>        厉嫣然看向了夏夕绾。
<br/>
<br/>        这时一辆炫红色的跑车停在了A大的门口，孙进带着几个公子哥来了，口气嚣张，“夏家那个小丑八怪呢，快点让她出来！”
