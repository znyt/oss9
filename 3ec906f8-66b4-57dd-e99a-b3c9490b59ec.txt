　　第567章
<br/>　　夜已经暗下来。
<br/>　　书院到现在还未对程大雷形成合围，他的动作过分快了，身子一点一跃，便在夜幕中消失不见。万一谁落单被程大雷遇见，结果也就是一剑被杀。
<br/>　　今夜的书院万卷山，山门前在杀人，山顶在杀人，到处都在杀人。
<br/>　　山门前，燕不归陷入剑阵包围中，他一身蛮力，有泼天本事，但面对书院的剑阵，却依旧冲杀不出去。勇则勇矣，却是负伤的雄狮，杀威犹在，但血却在一点点流尽。
<br/>　　程大雷的情况同样谈不上有多好，仗着身上那股机灵劲，他没有陷入燕不归那种不堪的境地。但他也没能力冲出去，而且，身上还扛着一个人，力气一点点流逝，程大雷的动作已不复刚开始的灵便。而且，扑上来的高手越来越多。
<br/>　　有些头疼呐。
<br/>　　“大当家……”刘发财虚弱的声音道。
<br/>　　程大雷心中感慨万千，是自己害得刘发财落入这般境地，但他却一句抱怨也无，什么是亲兄弟，这就是亲兄弟了。
<br/>　　“放心，我不会丢下你的。”
<br/>　　“嗯呐，我说的就是这个意思，如果不是你，我怎么会被搞成这个样子。如果你敢丢下我，我做鬼也不会放过你。”
<br/>　　“……”程大雷。
<br/>　　……
<br/>　　……
<br/>　　随着时间流逝，程大雷遇到的高手越来越多，有些他也没能力干净利落的收拾，而一旦开始缠斗，自己必然会陷入他们的包围。
<br/>　　现在的状况，程大雷也没有太大把握，无非是全力以赴，在绝境中拼出一线生机罢了。
<br/>　　“大当家，咱们现在去哪里？”
<br/>　　“找我的牛。”程大雷可不舍得将墨玉乌月兽丢给书院：“你知道书院的牛棚在哪里么？”
<br/>　　“这个……我不清楚。”
<br/>　　程大雷当初上山时，还是被书院奉为上宾的，他的座骑也有专人照顾，程大雷倒是没有过分在意。如今想找到自己的黑牛离开，倒是不知道书院把牛棚定在哪里。
<br/>　　一时半刻，想要找到也不容易。今天的事情对程大雷来说，有些憋屈了。他一腔郁闷无法发泄，忽地昂起头，发出一阵长啸。
<br/>　　啸声从山顶散开，声彻云霄。
<br/>　　“找到了，他在那里。”
<br/>　　“快过来，一定要杀了。”
<br/>　　“恶贼在那里，我要替师弟报仇。”
<br/>　　“你疯了。”刘发财道：“这是将他们都引过来，你想死，可也别带上我，我还想活呢。”
<br/>　　“有没有人说过，你的嘴很碎。”
<br/>　　程大雷颇有些无奈道，不过这也证明刘发财身上的伤势没伤到根本。一个将死之人，话不可能这么多。
<br/>　　静下心来，侧耳倾听。周围密密麻麻的脚步声，这是书院弟子正向自己合围而来。
<br/>　　昂。
<br/>　　一声牛吼从半山腰传来，程大雷眼睛一亮：找到了。
<br/>　　他背起刘发财，循着声音找去。声音越来越急，一声连着一声，那黑牛似乎也知道主人陷入危险中，声音躁狂不安。
<br/>　　刘发财轻咦一声，道：“大当家，您的座骑很神呐。”
<br/>　　“当然啦，我的座骑盔甲，件件都有来历，你以为随便找头牛就可以冒充我。”
<br/>　　“吹嘘的事情，过了这桩事咱们慢慢聊，如今，是不是脱身要紧。”
<br/>　　现在，程大雷的黑牛也陷入了合围中。它在牛棚里不安生，上蹿下跳，虽然背上没长翅膀，但总给人一种随时起飞的感觉。
<br/>　　看守牛棚的人，自然要按住它，大家虽然都对程大雷恨之入骨，实在也没有走火入魔到要对一个畜生动手。大家只是想要控制住他……然后，他们发现控制不住。
<br/>　　这黑牛蛮力惊人，又性格暴戾，当初秦蛮几人都没能制住它，况几个养牛的普通人。
<br/>　　粗重的打着响鼻，发出暴躁的昂声，就连牛棚都被它撞了半边墙。养牛的杂役想圈住它，又怎么做得到，有两人被它撞断了肋骨，倒在地上动弹不得。
<br/>　　这时候，程大雷从山下冲过来，口中发出一声呼啸。
<br/>　　那黑牛顿时一激灵，发出一声愉悦的长啸，四蹄腾空，竟直接从众人头顶跃起，冲向程大雷。
<br/>　　一伙杂役都呆立在当场。
<br/>　　程当家将刘发财搁在黑牛背上，满意的拍拍它的丑脑袋。
<br/>　　“我们走。”
<br/>　　程大雷心中再无挂碍，此地是险地，早早脱身要紧。
<br/>　　也是他没携带自己趁手的鬼面斧，不然，纵然有千军万马拦路，他也是赶闯一闯的。
<br/>　　“程当家，留步。”
<br/>　　程大雷正准备离开，一道声音忽然响起，有一人拦在程大雷面前。纵有千军万马，他也赶闯，可面前这人，却能令程大雷不得不停下脚步。
<br/>　　书院夫子，亲自出马。
<br/>　　对方不过一个老朽，白发苍苍，靠手中的竹杖才能站稳，但却让程大雷不得不谨慎。
<br/>　　对方也是绝世等级的人物，何谓绝世，就是全天下干这行的，满打满算也就你们几个顶尖人物。倒也不是说程大雷面对夫子必败，毕竟，传说级的人物不出世，绝世高手间的过招，胜负谁也说不好。
<br/>　　老人小孩女人出家人，不在江湖上出现则以，一旦行走江湖，必然有一两招保命的本事。
<br/>　　“夫子能否让条路？”
<br/>　　“程先生能否留步？”
<br/>　　夫子不可能让路，程大雷没办法留步。说来说去，终究是要打一场。是老前辈弄死愣头青，还是新冒头的茬子，摆平老江湖。
<br/>　　程大雷晃了晃肩膀，将匹夫剑出鞘，在夫子面前，他可没自信摆弄拔剑术。
<br/>　　一剑袭来，直来直往。
<br/>　　“老家伙，与我让路。”
<br/>　　好快的一剑，便是夫子也不敢硬接。他蹂身后退，手中的竹杖啪地一声爆开，从中露出一柄狭长的细剑。
<br/>　　过了多少年，夫子这柄竹中剑再次现世。
<br/>　　退步避开程大雷这一剑，等着程大雷力气使劲，新力未生之时，夫子同样刺出一剑。
<br/>　　同样好快，剑光压过星光，璀璨一剑，令旁观着心生寒意。
<br/>　　夫子在帝国名气不小，但那是因为他的道德，他的学识，很少有人知道，夫子同样是一位用剑高手，绝世高手。
<br/>　　二人各自刺了一剑，平分秋色，互不相让。
<br/>(本章完)
