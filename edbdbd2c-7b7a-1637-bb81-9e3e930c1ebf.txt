　　对于这次会武的规则，苏莫虽然感觉有些无奈，但也没有什么在意，反正他也不准备上场了，时间长些就长些吧！
<br/>　　“既然如此，本长老也不废话，会武现在正式开始！”阴泽长老环视全场，袖袍一挥，朗声宣布道。
<br/>　　嗖！
<br/>　　阴泽长老话音一落，一声身影便如同闪电一般，飞到了中央巨大无比的战台之上。
<br/>　　这庞大无比的战台是由阵法之力构建而成，武者落在战台上，显得非常的渺小。
<br/>　　这率先上场之人，不是别人，正是道宗的吕罡。
<br/>　　这种比赛的规则，明显是主动上场挑战的人，更占便宜，因为可以选择实力弱的对手。
<br/>　　而吕刚，心理早已憋着一口气，准备在这次会武之中，为自己正名，也为道宗正名，所以第一个便上场了。
<br/>　　“姓名！”
<br/>　　这时，北方看台上，在阴泽道圣的下方位置，一名太阴族的褐袍老者朗声问道。
<br/>　　“吕罡！”吕罡大声道，他知道这是负责记录战绩的人。
<br/>　　褐袍老者闻言，便点了点头，将吕罡之名记录了下来。
<br/>　　随后，吕罡的目光在南方七大势力之人所在的看台上扫视，最终落在了黑炎族所在的阵营之中。
<br/>　　目光望着炎火，吕罡眸中战意沸腾，不过，他虽然心中憋气，但心中还是放弃了第一战便挑战炎火。
<br/>　　炎火的实力，他也心中有数，不比他差多少，第一战就对上对方，极为不明智。
<br/>　　这是积分赛事，获得积分才是要务，至于挑战炎火，等会有的是时间。
<br/>　　而且，这第一战他要赢得漂亮，给所有人看看他的实力，所以，他心中放弃了挑战炎火。
<br/>　　“你，下来一战！”吕罡目光落在了黑炎族一位不知名青年的身上，就算不挑战炎火，他也依旧选择了黑炎族之人，实在是黑炎族的人太让他恼火了。
<br/>　　七大势力的人之中，除了修为达到武尊，或者半圣境修为的长老之外，所有的弟子均是参加会武之人，个个都是武帝境九重的修为，所以不需要询问，便能直接挑战。
<br/>　　嗖！
<br/>　　那名黑炎族青年闻言，身形一闪，便飞落在了战台上。
<br/>　　“姓名？”负责记录的褐袍老者，再次朗声问道。
<br/>　　“炎清！”黑炎族青年报上自己的名号，他眼眸望着吕罡，眸中充满了凝重之色，显然知道吕罡的强大。
<br/>　　“小子，两日前在浮屠塔外，你不是很嚣张吗，现在我就让你知道，你是多么的不堪！”
<br/>　　吕罡望着眼前的黑炎族青年，面上带着一丝冷笑，上次在浮屠塔外，此人可是叫嚣的厉害，狂妄的姿态不比炎火差。
<br/>　　“哼！谁输谁赢还不一定呢！”黑炎族青年冷哼道，言罢，他身上陡然燃烧起了熊熊的火焰。
<br/>　　这火焰成乌黑之色，仿若汹涌的魔火，散发着恐怖的高温，瞬间就将他周身的空间焚化。
<br/>　　“给我败！”
<br/>　　吕罡霸气无比，暴喝一声，猛然一拳向对方轰击而去。
<br/>　　这一拳极为霸道，拳劲如同洪荒猛兽，又如同一道汹涌的长河，以碾压一切的威势，瞬间跨越千里距离。
<br/>　　“给我破！”眼见拳劲临身，黑炎族青年双掌向前一推，一团巨大的黑色火焰，便飞速的迎向拳劲，
<br/>　　瞬息之间，黑色火焰和拳劲碰撞在了一起。
<br/>　　轰！
<br/>　　一声爆响，黑色的火焰瞬间炸裂，四散而开，化为漫天火雨。
<br/>　　而拳劲轰碎了黑色火焰之后，虽然威力大减，但依旧向黑炎族青年身上袭杀而去。
<br/>　　“不好！”黑炎族青年见此，顿时面色一变，但此时他再想躲避，已是来不及了。
<br/>　　电光火石之间，他只能极力防御，只见其身上的黑色火焰猛然大涨，形成了一片火海，将其完全包裹在了其内。
<br/>　　下一息，拳劲狠狠的轰击在了火海之中。
<br/>　　啊！
<br/>　　一声惨叫声响起，只见那名黑炎族的青年，身形直接被轰飞了出去。
<br/>　　倒飞近万里，直接砸在了战台的边缘，口中碰触一大口鲜血。
<br/>　　“吕罡胜！”
<br/>　　这时，负责记录的褐袍老者高声喊道，他不仅负责记录，也负责主持整个比赛过程。
<br/>　　所有观战之人，看到这一幕，都不禁有些惊讶，这吕罡的实力，貌似比想象中还要强啊！
<br/>　　一招！
<br/>　　只是随意的一招，便打败了对手，这种战力不可谓不强，说不定真有前十的实力呢！
<br/>　　一招打败了对手的吕罡，嘴角一勾，面上带着强大的自信。
<br/>　　他目光环视四方，见到很多人惊讶的表情，心中暗喜，他要的就是这个效果，一招击败对手，让别人看看，他们道宗弟子不输于任何人。
<br/>　　“不过如此！”炎火望着得意的吕罡，却是不屑一笑，根本没有放在心上，因为那名青年，只是他们黑炎族一个很普通的族人而已。
<br/>　　旋即，吕罡大步的返回了看台，并且得的了真枯半圣的赞赏。
<br/>　　毕竟，会武第一战，吕罡秒败对手，算是给道宗争光了！
<br/>　　吕罡前脚刚返回，便又有人踏上了战台，挑战继续，很快便再次开始了战斗。
<br/>　　“冯长老，这会武能举行多长时间？”苏莫毫不关心会武，向冯光问道。
<br/>　　“大概会持续十日左右吧，或许会更久！”冯光道，根据以往会武的情况，一般都会持续十日以上。
<br/>　　苏莫微微点了点头，心中颇为无奈，之前盼着能看到青璇，现在对方来了他却没有机会接近。
<br/>　　遥望着对面的青璇，苏莫率一沉吟，觉得有必要试验一番，看看对方还是否认识他！
<br/>　　虽然对方被抹除了记忆，但说不定会对他感到熟悉，或者有一点印象。
<br/>　　随即，苏莫目光灼灼，如同一道无形的电光，紧紧的盯着宏青璇，他相信对方肯定能注意到他的目光。
<br/>　　北方看台上，云悠悠端坐在阴泽长老的身边，静静的观看着战台上的比斗。
<br/>　　对于七大势力的会武，她亦是颇感兴趣，可以观看一番七大势力中顶尖天才的实力。
<br/>　　云悠悠的另一边，端坐着一名面容清丽的绿衣女子，此女琼鼻凤眼，容貌不俗，但纤薄的嘴唇给人一种刻薄之感。
<br/>　　“悠悠，你看那边，有个家伙一直在盯着你！”
<br/>　　这时，绿衣女子率先感受到了苏莫的目光，对云悠悠道：“呵呵！看这家伙如此灼热的目光，估计又是痴迷你的人！”
