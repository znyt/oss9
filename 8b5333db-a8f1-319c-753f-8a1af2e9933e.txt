陆寒霆是真的打算跟厉嫣然订婚结婚的，他想要将夏夕绾给彻底的忘记，但是奶奶一个电话打了过来，他想，他终究还是不能放开她的手，不能让任何人欺辱她。
<br/>
<br/>        刚才撞车的时候他就在想，如果救不了她，那就一起死吧。
<br/>
<br/>        很多路人跑了过来，“天哪，这里发生交通意外了，快点打电话叫救护车！”
<br/>
<br/>        现场聚集了很多人，这时有一个英俊的男人出现在了不远处，一身的黑色风衣，颀长俊美，是一直没有露面的…苏希。
<br/>
<br/>        苏希出现了。
<br/>
<br/>        一个黑衣手下恭敬的给苏希撑着一把黑伞，“宫主，既然陆寒霆将事情解决了，那我们也该走了，这一次夏夕绾失踪已经闹得满城风雨，好几批人手都在往这里赶，帝都四大家厉陆徐夜都有参与，我们不宜露面。”
<br/>
<br/>        苏希看着倒在地上的两个人，有人试图将陆寒霆和夏夕绾分开，但是昏迷不醒的陆寒霆紧紧的抱着女孩，他修长凌厉的五指扣在夏夕绾身上，任谁怎么扒都扒不开。
<br/>
<br/>        夏夕绾也在昏迷，除了额头受的伤，她看起来还好，纤柔玲珑的身体被严实的裹在陆寒霆的黑色大衣里，她闭着眼睛躺在陆寒霆精硕的胸膛里，被他护在羽翼里，风雨不侵，巴掌大的绝色小脸看着柔软而安心。
<br/>
<br/>        苏希淡淡的收回了目光，“走吧。”
<br/>
<br/>        “是，宫主。”
<br/>
<br/>        “对了，将那个范恬的个人资料送到夏邦的手上，让他转呈夏夕绾，经此一役，整个帝都城都要热闹起来了。”苏希喃喃的嗓音随着他颀长而神秘的身影一起消失了，好像他从来就没有出现过。
<br/>
<br/>        很快，几辆豪车疾驰而来，耳畔响起了尖锐的刹车声，夏邦和蓝烟拉开车门就跑了过来，“绾绾！”
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾做了一个很长的梦，她梦到妈咪了。
<br/>
<br/>        在梦里，她已经躺在床上睡着了，妈咪就坐在她的身边，不停的抚摸她的小脸，带着怜爱和心疼，“绾绾，妈咪只能陪你到这里了，以后的路要靠你自己走了，妈咪相信你可以走的很远，直到…走到妈咪的身边…”
<br/>
<br/>        她在梦里惊醒了，梦里黑漆漆的，伸手不见五指，她四处寻找妈咪，恐慌不安，“妈咪，你在哪里…妈咪，你快出来，绾绾害怕…”
<br/>
<br/>        很快，她摔了一跤，直接跌坐在了地上。
<br/>
<br/>        疼。
<br/>
<br/>        她也说不上哪里疼，就好像浑身都在疼，眼里有大颗大颗的泪珠砸落了下来，她哭了。
<br/>
<br/>        突然，有一个人走了过来，高大英挺的他缓缓单膝蹲地，伸手将她给搂在了怀里，他的薄唇落在她通红的眼睛上，一遍遍的亲吻，还勾去了她眼角的泪，“绾绾，别怕，我来了。”
<br/>
<br/>        是…陆寒霆。
<br/>
<br/>        他抱着她，在她耳畔低声缱绻绾绾，别怕，我来了。
<br/>
<br/>        夏夕绾纤长的羽捷颤了颤，倏然睁开了眼，她醒了。
<br/>
<br/>        她现在在医院的病房里，入眼都是白色，还有一股刺鼻的消毒水味，夏邦和蓝烟一直陪着，见她醒来，两个人迅速跳了起来，“绾绾，你终于醒了！”
