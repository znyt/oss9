　　这只恶鬼是一具较为凝实的虚影，高达两丈有余，青面獠牙，全身漆黑，两双鬼爪尖锐至极，如同两柄锋利的钢叉。
<br/>　　嗷！
<br/>　　恶鬼嚎叫一声，便急速向苏莫扑了过来，锋利的鬼爪向前一探，就欲将苏莫撕成碎片。
<br/>　　“死！”
<br/>　　苏莫立刻取出帝皇剑，急速向前一斩，剑光所过，天地一分，恶鬼的身体顿时被撕裂，而后随风消散。

<br/>　　这鬼影高达百丈，眸中爆射出赤红色的光芒，血盆大口中獠牙长达数丈，浓烈的鬼气森然无比。

<br/>　　鬼影乍一出现，便向苏莫急速飞扑而去，大口一张，猛然一吸。
<br/>　　尸罗殿殿主出手了，其他三大殿主也没有袖手旁观，尽数出手了。

<br/>　　“阴冥魔爪！”
<br/>　　阴罗殿殿主大手一抓，立刻幻化出一只黑色的魔爪，撕开空间，向苏莫当头罩去。
<br/>　　魔爪长达数十丈，仿佛是真实的鬼爪一般，其上的掌纹都清晰可见，威势同样惊天动地。

<br/>　　这一刀快如闪电，后发先至，率先临近了苏莫。
<br/>　　至于黄泉魔宗大长老阎泰，则是没有出手，他负责压阵，若是苏莫想逃跑，他会第一时间出手阻拦。
<br/>　　四道攻击从四个方向，向苏莫展开了绝杀。
<br/>　　鬼影大口一吸，顿时产生了一股诡异的吸力，居然让苏莫体内的武魂都颤动了起来，想要飞出体外。
<br/>　　“居然会吞噬！”
<br/>　　苏莫心中一惊，他还是第一次遇到能吞噬武魂的攻击。
<br/>　　一直都是他吞噬别人的武魂，今日倒好，别人居然想吞噬他的武魂！
<br/>　　不过，面对四人的攻击，苏莫并没有选择逃窜。
<br/>　　心神一动，他立刻便催动了吞噬武魂抵挡鬼影的吞噬之力，而后他手中帝皇剑光芒璀璨，剑光挥洒，斩向阎罗殿主袭来的刀芒。
<br/>　　于此同时，苏莫一心几用，九柄本命灵剑飞射而出，化为了三组，每三柄本命灵剑为一组，分别迎向了鬼影、拳茫、和魔爪。
<br/>　　轰！
<br/>　　一声爆响，帝皇剑挥出的剑气，率先和阎罗殿主的刀气碰撞在了一起，顿时发生了惊天爆炸。
<br/>　　轰隆隆！！
<br/>　　剑气和刀气同时崩碎，化为漫天细密的气劲，****方圆数千里，将一方空间都射成了筛子。
<br/>　　而另外三道袭来的攻击，也分别于三柄灵剑展开了碰撞。
<br/>　　那鬼影仿佛有吞噬武魂的能力，但是苏莫的吞噬武魂一催动，吞噬之力比鬼影的吞噬力量强大十倍。
<br/>　　那鬼影，遭到苏莫吞噬之力的笼罩，顿时身形巨颤，仿佛是害怕了一般。

<br/>　　不过，两组灵剑虽然不敌，却也成功阻挡了鬼爪和拳茫的势头，给苏莫腾出了时间。
<br/>　　“给我灭！”
<br/>　　苏莫一声暴喝，体内五枚神丹极力催动，浑厚的玄力如大河奔流，在他全身经脉中流淌。
<br/>　　玄力注入帝皇剑，让得帝皇剑剑芒暴涨千丈，化为了一柄巨大无比的三色神剑。
<br/>　　唰！唰！
<br/>　　苏莫瞬间连斩两剑，两道浩大无比的剑气贯穿天地，狠狠的斩向拳茫和鬼爪。
<br/>　　瞬息之间，两道剑气便狠狠的斩中了拳茫和利爪。
<br/>　　轰隆隆！！
<br/>　　巨响震天，浩荡八荒，毁灭般的爆炸席卷九霄，拳茫炸开，鬼爪粉碎，剑气崩灭。
