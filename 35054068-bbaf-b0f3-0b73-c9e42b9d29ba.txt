　　第655章
<br/>　　万卷山上，到处漂白，从山脚到山顶，山道两侧挂着白绫。
<br/>　　李行哉拾阶而上，步步缓行，这样的季节，登山观海还是有些景色可以看的。
<br/>　　山道两侧，除了被风扬起的白绫，还有跪得整整齐齐得书院弟子。
<br/>　　如果是以前，李行哉是轮不到这种待遇的，原先的书院夫子肯降阶相迎已经是他天大的面子。然而，今时不同往日，夫子已经死了，书院青黄不接，已经远非兴盛时可比。
<br/>　　今天，书院的面子被李行哉狠狠踩在脚下，也或许，以后再没有站起来的机会。
<br/>　　他们如此做，当然换不来李行哉的尊重，而是心中更加不屑。怪不得曾经和程大雷相处时，程大雷对天下读书人极尽冷嘲热讽之能事。这些人口口声声宣扬的东西，原来他们自己也不相信呐。
<br/>　　最后，李行哉来到书院万卷楼前，或者说，是万卷楼的废墟前。被一把火烧毁的藏书楼，废墟已经被清理掉，但多多少少还能有些残留的痕迹。
<br/>　　“这座楼是程大雷烧掉的？”李行哉问。
<br/>　　书院新任的院长张明理回答道：“是呐，那程贼十分可恶，多少珍贵的典籍付之一炬，实在是伤天害理，罪大恶极。”
<br/>　　一旁的楚云生掩着口咳嗽了几声，也不知道心里想着什么。
<br/>　　张明理说得咬牙切齿，李行哉倒是不觉得如何，程大雷会做出这种事他一点也不觉得奇怪。
<br/>　　忽然间，似乎某件奇怪的事发生，李行哉忽然觉得程大雷烧毁万卷楼，有些太过分了。前辈先贤留下的珍贵书籍，如今毁于一旦，而已经毁了，便再也看不到了。
<br/>　　“那程大雷……当真有些可恶。”
<br/>　　就连李行哉都忍不住道了一声，他身后众人早已议论纷纷。不仅是山上在议论，山下琅琊城中的居民也同样在议论。
<br/>　　他们这次被刘顶天祸害得不轻，多少亲人都死在这次战乱中。提起那刘顶天便破口大骂，而追本溯源，那刘顶天从何处来的？自然是从林少羽来的，因为他打的是正义教的旗号。而林少羽又是从何处来的……
<br/>　　当然是程大雷一手调教出来的。
<br/>　　如今刘顶天死了，林少羽也死了，大家需要一个要憎恨的存在，自然是程大雷。
<br/>　　这件事本来和程大雷扯不上太大的关系，但东扯西扯还是落在程大雷身上。
<br/>　　不仅仅是琅琊城一地，帝国大部多多少少都受过程大雷的祸害。本来这些陈年遗恨大家都差不多忘了干净，可几乎在同一时间想起。
<br/>　　江南李星想着当初自己怎么被程大雷赶得走投无路，落草为寇的百里胜想着程大雷如何与宋伯康联手针对自己……尤其是长安城，在尉迟离的葬礼上，无端端出现一副挽联，内容倒是无关紧要，落款赫然就是程大雷的恶名。
<br/>　　这是赤果果的示威呐，从来死者为大，一死百了，怎么可以公然在他人的葬礼上进行如此卑鄙无耻的行径。
<br/>　　将军府恨得咬牙切齿，如果摸得着程大雷，大家恐怕会立刻将其千刀万剐，方才能消心头之恨。
<br/>　　而相府方面的人也是心中戚戚，是呐，尉迟离死了，也不代表帝国便是他们的囊中之物，还有各路诸侯，还有一个程大雷。
<br/>　　话说，这山贼折腾大家已经够久了，是不是该腾出手来将他清理掉。
<br/>　　在同一时间，整个帝国都在讨论程大雷，说他是如何作恶多端，是如何残暴……李乐天也将程大雷赏金抬到十万，显然是也想起自己被程大雷折腾的经历。
<br/>　　而对这一切，程大雷确实不知情，他将系统奖励的广播技能使用后，便回屋睡觉去了。
<br/>　　而在一觉睡醒之后，程大雷发现自己的恐惧值正在以极其恐怖的速度向上飙升。
<br/>　　程大雷一身冷汗，呆呆的坐在床榻上，看着自己的恐惧值飙升，但却完全不知发生了怎样的事。
<br/>　　很快他拥有的恐惧值便已经达到一个以前想都不敢想的数值。
<br/>　　程大雷打开系统商店，以他现在的拥有的恐惧值，已经不在乎那点入场费。现在系统商店可购买的货物有：火药、皮甲、黑鬃马、青苓散……
<br/>　　另外程大雷可以通过点将台兑换绝世、顶级、优秀、普通三种人物。
<br/>　　程大雷虽然拥有极大量的恐惧值，但却没必须兑换的东西。朔方城虽然缺少人口，但完全可以通过招募流民扩充，也不需要浪费恐惧值。
<br/>　　程大雷逛了逛，随手关掉系统界面，翻个身继续补觉。
<br/>　　睡着前，他心中还有一个困惑：究竟是发生了什么事情。
<br/>　　日子一天一天过去，朔方城在按部就班的发展，由和珅接手管理，倒没有什么需要程大雷插手的事情。这段日子里，不断有流民迁入蛤蟆城，但还远远达不到任务要求的十万人。
<br/>　　这段时间内，程大雷身边并没有什么大事发生，他无非是在朔方城和蛤蟆城两边跑，平时关注一些阿喜送过来的情报。
<br/>　　程大雷身边无大事可叙，但最近帝国可是颇为热闹。在经历一段时间的招兵买马后，各路诸侯都变得不安分起来。互相之间的摩擦越来越多，已经有小规模的战斗发生。
<br/>　　而最引人注目的还是长安，尉迟离的葬礼过后，相府开始清除异己，想要将帝国朝廷变成一家天下。李乐天这帝国天子有被架空的危险，他开始和崔相掰手腕，但明显双方不是一个量级的对手，李乐天的处境很艰难。
<br/>　　京州的权力斗争，也吸引了宋伯康的兵力，他在京州边界屯兵，给崔相助威。
<br/>　　如此，程大雷的压力小了很多，有了安心发展的空间。
<br/>　　帝国已快成一锅乱粥，凉州倒是风平浪静，一日一过去，程大雷没找到恐惧值飙升的原因，但却等到了刘发财。
<br/>　　他与黑衣老尼结伴去东海，跨越上万里，如今算是平安归来。难得可贵的是，他们顺利找到华氏一族，且将他们带到了蛤蟆城。
<br/>　　程大雷得到消息后，第一时间便回到蛤蟆城。
<br/>(本章完)
