        &lt;h3&gt;第三卷第三百二十一章  撞&lt;/h3&gt;
<br/>
<br/>        陈天天陪陈元良去了，留下冯睿一个人在家。
<br/>
<br/>        冯睿心里特别的难受，亲生女儿没呆上几天，她费尽了心思都哄不回来的那种，只要见了陈家人，心就飞走了。
<br/>
<br/>        儿子和儿媳妇也不回来，偶尔就打个电话，现在的她真的是孤苦伶仃，年轻的时候并不觉得一个人有多孤单，等到上了年纪，才发现一个人的日子真的是难熬。
<br/>
<br/>        思来想去，自己的这俩个亲生的孩子还不如夏颜，夏颜隔三差五的给她打打电话，俩个人偶尔会在一起吃顿饭，相互还有走动，尤其是夏颜还帮着陈伟冯将工作调动了，虽然陈伟冯不怎么来看她，但毕竟是她的儿子，只要儿子一切都好，她也就心安了。
<br/>
<br/>        这几天夏颜也没有出差，不如下班了去夏颜家里给夏颜做饭，俩个人在一起吃饭好过她一个人孤零零的在家里。
<br/>
<br/>        冯睿下班直接去了夏颜家里，打开门，听见夏颜的卧室有动静，冯睿心里一慌，莫不是进贼了？
<br/>
<br/>        她蹑手蹑脚的进到了厨房，拿起菜刀就朝夏颜的卧室走去。
<br/>
<br/>        推门，一副活生生的游戏春色图展现在冯睿的面前。
<br/>
<br/>        冯睿手中的菜刀“啪”掉到了地上。
<br/>
<br/>        她来不及捡起菜刀，捂上眼睛，她好像看见了老马。
<br/>
<br/>        冯睿这么多年和陈子建一直恩恩爱爱，见不得那些乌七八糟的事情，当她看到夏颜和老马的时候，她震惊的无以复加，况且她知道老马很早就和唐红莲的关系很好，当初俩个人都准备结婚了，只是因为唐红莲带着夏颜，马家人不同意。
<br/>
<br/>        老马心里窝着火，怒声对夏颜说：“家里怎么会随随便便的来人？”
<br/>
<br/>        夏颜也不知道冯睿会突然到她家里来，平时还打个电话，今天怎么连电话也不打。
<br/>
<br/>        “她有我家的钥匙，平时来之前会给我打电话的。今天她没有说要来。”夏颜也很恼火。
<br/>
<br/>        “你出去看看她走了没有？”老马也是认识冯睿，他们算是一个学校的校友，冯睿和唐红莲比他高一级，之前和唐红莲的关系冯睿也是清楚的。
<br/>
<br/>        “好。”夏颜穿戴整齐出来没有看到冯睿，但是看到了她的钥匙放在了客厅的茶几上，夏颜心里一惊。
<br/>
<br/>        夏颜打开门，朝楼梯道看了一眼，没有看见冯睿的影子。
<br/>
<br/>        她冲着老马说：“人不见了，我去找找。”
<br/>
<br/>        冯睿觉得心里不能接受夏颜和老马的事情，她发誓以后不到夏颜的家里了，便将夏颜的钥匙还给了夏颜，放在夏颜的茶几上，步行回家。
<br/>
<br/>        她在寻思要不要将这件事情告诉唐红莲，唐红莲也真是可悲，好强了一辈子，临老了遭受双重背叛，真是可怜可悲，也不知道前世造了什么孽。
<br/>
<br/>        当冯睿想到“造孽”两个字的时候，脑子里像是劈过一道雷，将她炸了个激灵，前世唐红莲造孽不造孽她不知道，这一世却是将陈子昂的前途活生生的耽误了。
<br/>
<br/>        报应来得真快，陈子昂什么都没有作，唐红莲的女儿已经开始背叛她了。
<br/>
<br/>        冯睿突然就想到了自己，她家已经是散了，丈夫离开她，儿子不理她，女儿就更不用说了，难道这不是报应吗？
<br/>
<br/>        往事一幕幕的出现在冯睿的眼前，冯睿看到了面目可憎的自己，她喃喃自语：“我怎么会成为这样的人？我怎么就成了这样的人？子建，对不起，我爱你，天天，对不起，妈妈没有好好养育你，伟冯，对不起，我只是想着让你有个好前程，还有，还有子昂，这么多年，我一直拿你做幌子，各种利用你。”
<br/>
<br/>        冯睿胡思乱想的漫无目的的走着，她都不知道该往哪里走，悔恨充斥着她的心，她拿起手机给陈子建发了个短信：“子建，祝你幸福，你的资料在电脑的f盘里，加密的密码是你的生日，改天你回去拷贝走。”
<br/>
<br/>        这一生都做了些什么事情啊，虽然当时帮夏颜的时候不知道是陈子昂，可是要是换了另外一个人是不是就毁了人家的一生。
<br/>
<br/>        “冯妈，冯妈。”夏颜开着车追上来了。
<br/>
<br/>        冯睿看着从车里探头出来的夏颜，叹了口气：“颜颜，我回呀，今天本来想过来给你做饭的，我没想到......你......你们在家里”
<br/>
<br/>        “我送你回去吧。”夏颜看了一眼冯睿。
<br/>
<br/>        “不用了，颜颜，我自己走走，你回去吧，那个马总还在家里吧？”冯睿都感觉脸上火辣辣的。
<br/>
<br/>        “我送你吧，你回家还有一段路。”夏颜坚持要送冯睿。
<br/>
<br/>        冯睿心里有些烦躁，她实在是不想坐夏颜的车。
<br/>
<br/>        她站在夏颜的面前艰难的对夏颜开口：“老马和你妈关系很好，之前他们......他们都准备结婚了的。你们.......你们这样你妈不知道吧？”
<br/>
<br/>        夏颜的眼神像刀子一样剜在冯睿的身上：“她怎么会知道呢？你不说我不说他不说。”
<br/>
<br/>        “颜颜，你，你.......算了，你回去吧，我走一走，散散心。”冯睿实在是想不通为什么夏颜会和老马在一起。
<br/>
<br/>        冯睿转身慢慢的朝前走去，夏颜看着冯睿一点一点的走远。
<br/>
<br/>        她靠在座椅上，点了一支烟，烦躁的抽了几口，将烟狠狠的掐灭了。
<br/>
<br/>        “颜颜，你在哪里？”老马的电话打过来了。
<br/>
<br/>        “我在找冯睿，她问我和你的事情我妈知道不知道，她还说你之前都准备和我妈结婚了，你们的事情她倒是很了解啊？”
<br/>
<br/>        老马在电话那头沉默了一会：“我们的事情不能让你妈知道！”
<br/>
<br/>        “怎么了你怕了？”夏颜冷冷一笑。
<br/>
<br/>        “我怕你妈伤心，颜颜，你看我们要不，要不就过去了吧？”老马坐在夏颜的沙发上，女人还真是个麻烦事。
<br/>
<br/>        “我不伤心？”夏颜口气不好，冲着电话喊。
<br/>
<br/>        “那怎么办？这事情要是冯睿给你妈说了，我们还能在一起吗？”
<br/>
<br/>        “她不会说出去的。”夏颜说。
<br/>
<br/>        “你怎么知道她不会说的？她告诉你了？你能保证她以后不说？”老马在电话里三连问。
<br/>
<br/>        夏颜将电话将副驾驶上一扔，一脚油门踩下去。
<br/>
<br/>        冯睿慢慢悠悠的朝前走着，听到了后面的汽车声，转头看了一下，来不及侧身，只觉得脚下一疼，便被远远的摔了出去。
<br/>
<br/>        夏颜回到家的时候，老马斜躺在沙发上，看了一眼夏颜：“怎么才回来？”
<br/>
<br/>        “这个家里只来过俩个人，一个是你，一个是冯睿，她再也不会来了。你呢？”夏颜死死的盯着马总。
<br/>
<br/>        “我？你让我来我就来，不让我来我就不来啊。”
