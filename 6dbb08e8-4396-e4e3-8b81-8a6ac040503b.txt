　　器灵老者一边开口，这云羽神殿的大门便是突然开启，这古朴的铜门，已经是七万年没有开启。
<br/>　　开启之后，其中却是坐着五尊骸骨。
<br/>　　这五尊骸骨，虽然身上以无血肉，但是看起来却有着神韵，似乎比起活人，还要更像活人一般。
<br/>　　七万年的岁月，这五尊骸骨，居然依然不朽，坐在这云羽神殿之中。
<br/>　　“这五尊骸骨，生前到底是有着什么样的实力，才可以数万年依然不朽？”
<br/>　　楚风眠惊叹一声。
<br/>　　在强大的武者，陨落之后，也只剩下一具白骨，也许足以百年不朽，但是千年，万年，必定会风化，化为一团粉末。
<br/>　　可这五尊骸骨，却是足足七万年不朽，如今一看，都没有任何变化。
<br/>　　甚至是让楚风眠有些感觉到，这五尊骸骨似乎都还没有死一样，而是如同活人一般，随时可能复生过来。
<br/>　　“这是我们云羽天宫的五位太上长老。”
<br/>　　器灵老者在一旁，平静的开口道。
<br/>　　“拜见五位前辈。”
<br/>　　楚风眠跟寒月漓，都是低声行礼道。
<br/>　　这五尊太上长老，绝对是位列九域之巅的人物，就算以死，化为白骨，也值得楚风眠尊敬。
<br/>　　“我云羽天宫，七万年后，才终究开启一次，没想到第一次居然就碰到了你们，只能说是我云羽天宫之幸事了。”
<br/>　　器灵老者看着楚风眠，不由的感叹了一声道。
<br/>　　“小家伙，你愿不愿意拜入我云羽天宫，担任我们云羽天宫的天帝？”
<br/>　　“天帝？”
<br/>　　听到这个称呼，楚风眠不由惊呼一声道。
<br/>　　这云羽天宫的历代掌门，居然号称天帝？
<br/>　　“不错，我云羽天宫的掌门，都为天帝，任何一人，都足以称霸九域，号令四方，为何不成称为天帝？”
<br/>　　器灵老者傲然道。
<br/>　　七万年前，云羽天宫，便是当代无敌的存在，号令四方，无人敢不从。
<br/>　　“当然，你若不是不敢接下这个名号，你现在也可以随意离开，我云羽天宫历代天帝，绝不会有任何一个废物。”
<br/>　　“称霸九域，号令四方，不错，这个天帝，我当了。”
<br/>　　楚风眠听到器灵老者的话，大笑连连道。
<br/>　　他要踏平七大宗门，位于九域之巅，这天帝的名号，也的确是正适合他。
<br/>　　他又有什么可以拒绝的？
<br/>　　“好。”
<br/>　　器灵老者听到楚风眠回答的干脆，利落，眼神中也是露出了几分喜色。
<br/>　　只有这样，发自心底的自信，张狂，才能真正成为一代强者。
<br/>　　若是就连称霸九域的心都没有，那如何还能称王称霸。
<br/>　　在楚风眠的身上，器灵老者看到了一位强者之心。
<br/>　　拥有这样一颗强者之心，才能真正有机会，踏上九域之巅，成为九域之帝。
<br/>　　“从今日起，这云羽天宫的主人，便是属于你，其中的种种财富，都归你所用，不过只有一点，你，不能辱了天帝之名，不然其他人不杀你，老夫也要杀你！”
<br/>　　器灵老者，突然向着楚风眠眉心一点，从楚风眠的额头之上，便是突然出现了两道羽翼般的印记，逐渐的，便是消失。
<br/>　　在楚风眠的脑海之中，突然无数记忆，涌了进来，这记忆之中。
<br/>　　七万年前，云羽天宫的种种，全部都被楚风眠看在眼中。
<br/>　　而除了这些记忆之外，还有一道法决，随着记忆，来到了楚风眠的脑海之中。
<br/>　　“云羽化天剑！”
<br/>　　这是一道剑决。
<br/>　　曾经云羽天宫之中，最为强大的剑术。
<br/>　　这器灵老者，也是看出了楚风眠修炼的功法，以剑为主，所以便将这云羽天宫之中最为强大的剑决，交给了楚风眠。
<br/>　　这云羽化天剑，是以速度为极致的剑术，出剑速度，比起风还要快，甚至是剑灭人，剑风未来。
<br/>　　虽然剑走偏锋，但是这云羽化天剑，绝对是楚风眠所见到过的最强剑术之一。
<br/>　　“厉害！”
<br/>　　楚风眠都不由的感叹一声，能够将剑道，快这一道，走到极致，创造出这剑术的人，绝对是一尊剑道老祖。
<br/>　　若是没有剑走偏锋，此人很有可能都会领悟剑之真谛，成就一代剑帝。
<br/>　　“云羽化天剑！融入我身！”
<br/>　　楚风眠长喝一声，这云羽化天剑种，有着无数值得借鉴的地方，其中的种种奥秘，全部融入到了楚风眠的九域剑术之中。
<br/>　　他要开创的，是世间前所未有的剑术，这世间的种种剑术，他都要融入其中，开创完美。
<br/>　　“好！融入了这云羽化天剑，在速度之上，我的九域剑术，要比起之前，快上无数倍！”
<br/>　　楚风眠已经感觉到，他九域剑术的蜕变，这等剑术，对楚风眠而言，都有着无尽的好处。
<br/>　　只是，只有一道，这还太少了。
<br/>　　“云老，不知这云羽天宫中，还有没有其他剑术，可以前来一观。”
<br/>　　楚风眠向着器灵老者开口道。
<br/>　　这器灵老者，既然是云羽天宫的器灵，那楚风眠便是称他为云老。
<br/>　　“怎么？剑道一路，向来专精，学的多也没有用。”
<br/>　　云老听到楚风眠的话，皱了皱眉道。
<br/>　　“这云羽化天剑，已经是我云羽天宫中最好的剑术你，想要参悟，可并不轻松。”
<br/>　　这云老的话外之音，便是告诉楚风眠，贪多嚼不烂。
<br/>　　听到云老的话，楚风眠脸色倒是没有任何变化，这个道理，楚风眠自然清楚，云老提醒，也是为了他好。
<br/>　　不过楚风眠要自创剑术，必定是要集百家之长再行，对他来说，剑术自然是越多越好。
<br/>　　“云老，看剑！”
<br/>　　突然之间，楚风眠轻喝一声，从他手中，祖龙至尊剑突然出窍，猛然一剑之间，便是向着云老刺了过去。
<br/>　　这一剑的速度，快到极点，就连剑光都没有看到，这剑锋，已经是逼近云老。
<br/>　　“这是？”
<br/>　　云老的目光，都是微微露出几分诧异，突然一指，点了过去，正好点在这剑锋中心，这才令这一剑，停了下来。
<br/>　　“自创剑术？你居然已经开创了自己的剑术？”
<br/>　　短短交手瞬间，云老便是看得出来，楚风眠的剑术，不属于世间的一切剑术，而是另外开辟了一条道路，创造出了一门前所未有的剑术。
