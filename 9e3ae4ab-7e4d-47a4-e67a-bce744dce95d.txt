　　城主府内部，尉迟宝林大步流星的走进主厅，对四周满眼的异族风情建筑视而不见。
<br/>　　莎车城不大，真的不大，加上又属于边境地区，建筑虽然满满的异族味儿，但实际上并不显得如何高大上，对于在大唐见惯了世面的尉迟来说，吸引力确实差了些。
<br/>　　“你就是城主？”时间不大，尉迟已经站到一个胖胖中年人面前。
<br/>　　胖城主抹着头上不断滴下来的冷汗，瞪了身边哭哭啼啼的几个女眷一眼，然后才畏畏缩缩的说道：“是，是的。尊敬的大唐将军，小人就是城主，冒犯天颜还请恕罪！”
<br/>　　“弄条红绳子给他系到脖子上，粗一点的，别弄断了。”尉迟宝林面无表情的看着眼前的胖子，等到绳子被栓到他的脖子上，这才阴森森的说道：“从现在开始，只要你敢把这绳子摘下来，老子就屠了你全家，明不明白。”
<br/>　　“明，明白。”感受着脖子上如同狗链子一样的绳索，胖子的肥脸苦的像是死了老子娘。
<br/>　　那绳子明显就是一根浸了人血的麻绳，挂在脖子上就像是死人从地狱中伸出来的索命的鬼手，让胖子的冷汗顺着后背飕飕的淌。
<br/>　　“明白就好，现在告诉我，你是什么人？”尉迟才不管胖子是什么感觉，性子憨直的他目的性很强，只要决定了某些事情，就很难再被改变。
<br/>　　“大，大唐人！”胖子回答的很痛快。
<br/>　　刚刚外面闹腾了那么长时间，他已经听的清清楚楚，自然不会给自己找不痛快。
<br/>　　“行，知道自己是大唐人就好，一会儿等我们撤了，你就派人出去挨家挨户的给老子查，谁家门上要是没有挂红布、红绳一类的东西，该怎么办，你知道吧？”尉迟的大手往胖子的肩膀上一拍，笑着说道，只不过那笑容在胖子眼中显得分外狰狞。
<br/>　　“知道，那些不挂的都是有二心之人，全家杀光！”为了自己的小命，胖子选择了妥协。
<br/>　　在古代人命不值钱，对于生活条件艰苦、物资短缺的西域来说更是如此。在这里没有人会在乎一些平民百姓的生命，大家都挣扎在死亡线上，只要能保住自己的命，杀一些人并不算什么。
<br/>　　而且城主本身也不是什么好东西，否则也不可能外面在打仗，而他还依旧躲在城主府里面享乐。
<br/>　　由此可见这胖子也不是什么好东西，治下百姓的生命，在能保住自己命的前提下，死上一些还是值得的。
<br/>　　“不错，你很有悟性，孺子可教也！”得到让自己满意的答案，尉迟宝林夸奖了胖子一句，只是用词却是显得有些不伦不类。
<br/>　　没办法，宝林没啥文化，孺子可教还是以前孔颖达等一帮老货夸奖李承乾时候，他在一边听来的。
<br/>　　至于尉迟宝林自己……，他得到的评语一般都是朽木不可雕也！
<br/>　　“谢将军夸奖，小人能为大唐天兵略尽绵薄之力，该当的，该当的！”不过很明显，胖子也不是什么文化人，对于尉迟宝林似是而非的夸奖也没听出什么来，只是点头哈腰的跟在他身边。
<br/>　　“行了，别跟着老子，快点办你的事去。”尉迟宝林被一个胖子跟在身边觉得十分不爽。
<br/>　　“是是是，小人马上就去。将军，要不要到后宅休息一下？”胖子胡乱答应着，眼睛却一个劲的往那些被押在一旁的女眷身上看，其中意思不言而预。
<br/>　　“滚一边去，当老子是什么人，用你玩过的贿赂老子，你老小子的脖子难道比刀子还要硬？”长安城出来的纨绔对这种事情如何能不明白，看到胖子的眼神，尉迟立刻就知道他是什么意思。
<br/>　　不过，想想这些女子都被那个肥肥的胖子搞过，尉迟立刻就没了什么兴趣。
<br/>　　而且他现在是统军将领，这种违反军纪的事情，说什么也不会带头去干，身为武勋贵族，必要的‘原则’尉迟还是有一些的。
<br/>　　盏茶时间之后，尉迟摆脱了那个胖子，让他安排手下去执行任务，自己则是带着人离开了城主府，将城主一家全部放了。
<br/>　　得到解脱的胖子直到看不见尉迟的影子之后，长长的呼了一口气，抬起袖子用力的抹了一把头上的冷汗。
<br/>　　“老，老爷！”胖城主正在纠结应该怎么办，是不是要一心归顺大唐的时候，老管家拿着一块红布走了过来。
<br/>　　“老爷，小人给你把这，这个包起来吧，您的脖子都快要破皮了。”在胖子狐疑的目光中，老管家指指他脖子上的绳子说道。
<br/>　　“嘶……，快，快给我包上。”胖子听到老管家的话这才觉得脖子上火辣辣的疼。
<br/>　　刚刚实在是太过紧张，根本就顾不上脖子疼不疼，现在精神一放松，粗糙麻绳对他脖子造成的伤害立刻让他觉得后脖子的位置疼的锥心。
<br/>　　于是，老管家、侍女、老婆一群人围着城主大人忙碌起来，片刻之后，胖子脖子上的绳子已经消失不见，取而代之的是看上去十分美观的扁平布片。
<br/>　　“阿西，你说，那尉迟将军看不到绳子，什不会？”摩挲着脖子上被包起来的绳子，胖子有些犹豫。
<br/>　　“老爷，怎么会呢，您可以说这样是为了更好的保护那条绳子，所以才包起来的。”老管家谄媚的说道。
<br/>　　“是么？”胖子犹豫了一会儿，又纠结的说道：“那你说，这东西应该叫什么？总不能绳子绳子的叫吧？”
<br/>　　“这……”老管家被胖子给问住了，一时不知如何作答。
<br/>　　胖子的某个老婆，见所有人都在愁眉不展，突然脑中灵光一闪，挤到他身边说道：“老爷，您看这个东西扁扁的像一条带子，又是系在脖领之上，依妾身看，不如就叫‘领带’吧？”
<br/>　　“领带？”胖子嘀咕了一句，眼前一亮，狠狠一拍大腿：“不错，领带，以后这东西就叫领带了。”
<br/>　　远在长安的李承乾丝毫不知道，因为尉迟宝林的一次恶搞，在敌人的脖子上系了条绳子，结果竟然提前上千年搞出了领带。
<br/>　　如果知道的话，真不知道是应该哭还是应该笑！
