<br/>　　悄然间，鸡冠蛇脑海中画面浮现，那年轻女子含笑的望着它：“你可以成为我一个人的家蛇吗”

<br/>　　余音袅袅，五十年未尽，它的主人早已尸骨无存。

<br/>　　没错，是涂小安阻止的，当了路人甲那么久，也应该做点什么了。

<br/>　　涂小安并没有口吐人言，而是用意识力传输过去。
<br/>　　“那我就吞了你，吞了你，我会变的更强”
<br/>　　鸡冠蛇凶煞乖张，血盆大口一张，骤然朝着金银色射去，顷刻间就到。
<br/>　　“来的好”
<br/>　　实力暴涨之后，涂小安都不知道自己成长到何种的地步，从八级到十五级，被巨蟒基因改造，被系统强化，他甚至有一步登天的快感。
<br/>　　“铿锵...铿锵...铿锵...！”
<br/>　　旋即，宅院中就算出金属板的声音，火星四溅，这是蛇与蛇，肉身之间的碰撞，蛇口之下的撕咬。
<br/>　　对于无手无脚的蛇来说，它们的打斗是简单粗暴的。
<br/>　　要嘛气势上就压倒对方，要嘛咬死跟毒死对方。
<br/>　　两条蛇的进化让肉身坚硬如铁，刀枪不入，一条蛇尾拍出，便蕴含着无与伦比的力量，顷刻间飞沙走石，地面一处处如蜘蛛网一样的裂痕蔓延。
<br/>　　鸡冠蛇变色，它的气息也越发的可怕，这种是它的战斗形态。
<br/>　　当初的变异吸血蛇让白镇人民闻风丧胆，但是跟这条蛇一比，还真有点小巫见大巫的意思。
<br/>　　好在这条蛇只针对赵家，否则要是换成吸血蛇那种凶残，见人就咬的性格，还真是一场灾难。

<br/>　　宅院仿佛在被一场狂风肆虐，灵堂外的众人精神处于半痴半呆的状态之中。
<br/>　　这尼玛的两条是蛇？？？
<br/>　　涂小安这次并没有将电蜂释放出来攻击，因为他深深的明白，面对这二阶生物，那些只能算小手段。
<br/>　　虽是雷电，但是电的威力却远远不够对这条变异鸡冠蛇照成实质性的伤害。

<br/>　　“吼...！”
<br/>　　此时，两蛇对撞之后分外，鸡冠蛇惊天咆哮起来，顿时风云搅动，一股强风朝着金银色席卷而去，宅院随着这一吼，不知有多少隐藏的小型生物莫名的被震死。

<br/>　　这条不知来历的蛇既然也如此的可怕，自打两条对战，赵老爷子就被管家扶起来，重新做回轮椅上。
