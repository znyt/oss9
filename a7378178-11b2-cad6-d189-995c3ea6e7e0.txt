        “不知道。对方称呼您为‘四少’，而且，他还说，说什么四嫂去药店买了什么事后药，未出店门，就已经把那药吃了。”落以柔慢条斯理地复述。
<br/>
<br/>        傅夜沉倏地蹙起了剑眉，脸色铁青，黑眸深处隐隐地跳跃着两簇火苗。
<br/>
<br/>        落以柔望向傅夜沉的俊脸，见他脸色不好，立马识趣地闭上了嘴。
<br/>
<br/>        想必，那男人口中的“四嫂”指的就是童筱颖吧！
<br/>
<br/>        童筱颖吃了事后药，不就是摆明了不想怀宝宝吗？
<br/>
<br/>        情节再严重点，那就是童筱颖不想怀上他傅夜沉的孩子。
<br/>
<br/>        落以柔忽然间暗自窃喜起来。
<br/>
<br/>        “今天所有的行程取消，另行安排。”傅夜沉忽然撂下这话后，拿起手机直接离开了总裁办。
<br/>
<br/>        落以柔连忙点头颔首，应了声：“是。”
<br/>
<br/>        本来，整个设计部的集中办公室里，键盘和鼠标的声音，“啪嗒啪嗒”、“滴答滴答”地响个不停。
<br/>
<br/>        但是，在傅夜沉走进设计部大门，站台小姐连忙从前台办公桌后起身，都来不及去通知部长。
<br/>
<br/>        当他只身往那集中办公室的过道上一站，站台小姐一声“傅总好”，让室内所有的声音戛然而止。
<br/>
<br/>        傅夜沉立在过道上，将集中办公室扫了一遍后，目光最终落在了那边靠窗的座位上。
<br/>
<br/>        因为嫌办公室里的键盘声和鼠标声太吵，童筱颖早上过来开始工作的时候就摘掉了两耳上的助听器。
<br/>
<br/>        她坐在电脑前聚精会神地审核结构图，并认真地记录着每一条的审核记录，导致她压根就不知道傅夜沉此刻就站在这儿的过道上，目光灼灼地盯着格子间里埋头工作的她。
<br/>
<br/>        “傅总，您今天是来视察我们设计部的吗？”孙莉莉接到站台小姐的告知后，连忙从办公室里出来，笑盈盈地走到了傅夜沉的身旁，毕恭毕敬地询问。
<br/>
<br/>        傅夜沉的黑眸里透着冷冽的精光，浑身散发的寒气，让周遭好像被笼罩在他的冷酷气场中。
<br/>
<br/>        他一声不吭地望着那边的童筱颖，大家也瞬间不敢轻举妄动。
<br/>
<br/>        孙莉莉脸上的笑容渐渐收敛，顺着傅夜沉的目光看了过去，得知这问题的来源点在童筱颖身上，于是主动往童筱颖那边走了过去。
<br/>
<br/>        直到自己的肩膀被人拍了拍，童筱颖才抬起头来，望向身边这个惊扰自己的人。
<br/>
<br/>        “傅总有事找你。”孙莉莉接着说道。
<br/>
<br/>        童筱颖一直看着孙莉莉的嘴唇，虽然听不清她说了什么，但是通过她嘴唇的动作，她读出了“傅总有事找你”这句话。
<br/>
<br/>        自从听力几近丧失后，她每次与人面对面谈话时，目光的焦点总会先落在对方的嘴唇上。
<br/>
<br/>        那段时间的唇语培训，于她而言还是起到了效果，便于她在没有佩戴助听器的情况下，也能正常与人交流。
<br/>
<br/>        童筱颖见孙莉莉对自己使眼色，她本能地站起身来，往傅夜沉那边看去。
<br/>
<br/>        见傅夜沉的脸色有些难看，心里莫名其妙地感到有些不安，好像是自己做了什么坏事，被“家长”发现了一样心虚得不得了。
<br/>
<br/>        重新戴上助听器后，她刚走到他的跟前，他却二话不说地转身离去。
<br/>
<br/>        童筱颖下意识地看向孙莉莉，孙莉莉再次对她使眼色，示意她跟过去。
<br/>
<br/>        无奈之下，她聋拉着脑袋，默默地跟在了傅夜沉的身后。
<br/>
<br/>        大哥，好像很生气的样子……
<br/>
<br/>        他是在生她的气吗？
<br/>
<br/>        为什么要生气？
<br/>
<br/>        童筱颖觉得自己好像没有哪个地方惹恼他了啊！
<br/>
<br/>        就在她为他感到疑惑不解，突然一头撞上了一堵肉墙。
<br/>
<br/>        童筱颖抚着额头，抬起头来时，只见傅夜沉已经转过身来，气势凌人地瞪着她。
<br/>
<br/>        “大哥，你是不是不开心？”童筱颖觉得自己在明知故问，顿了顿后，又补充了一句，“你为什么不开心？”
<br/>
<br/>        傅夜沉未吭声，突然向她伸过手去，一把抓住了她的手腕，拽着她继续往前走。
<br/>
<br/>        这一路从设计部外坐他的VIP专属电梯直达他的总裁办，他全程一句话也不说，但她能感觉到他愤怒到了极致。
<br/>
<br/>        傅夜沉将童筱颖拽进办公室里的时候，落以柔还在整理资料，结果他把所有的火气往落以柔身上撒，冷冷地呵斥道：“这里不需要你了，滚！”
<br/>
<br/>        落以柔被傅夜沉这声怒斥给吓了一跳，下意识地看了童筱颖一眼，瘪了瘪嘴，连忙放下手中的资料，颔首欠身，迅速转身离去。
<br/>
<br/>        童筱颖刚想说什么的时候，傅夜沉再次拽着她往楼上走，然后毫不客气地将她直接甩到了那张单人床上。
