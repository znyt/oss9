　　玉虚宗，坐落在时空界域的边缘地带，属于永恒神宫的下属势力。
<br/>　　这是一个不大的宗门，弟子十余万，山门坐落在一座数千丈的雄伟高峰之上，建筑连绵起伏，倒也显得颇为气派。
<br/>　　宗主玉虚子，拥有九品低级的修为，在时空界域，也算是有名的强者，威震一方地域。
<br/>　　今日，在玉虚宗的宗门大殿之中，玉虚子今日迎来了一位不速之客。
<br/>　　“阁下是何人？不知有何指教？”
<br/>　　一身道袍，面容儒雅，头发半白，显得仙风道骨的玉虚子，坐在主座之上，望着下方座位之上，面色古板的黑衣青年。
<br/>　　根据弟子禀报，有一位强者求见，得知对方也是九品的修为，他便接见了。
<br/>　　不过，玉虚子打量了一番这名黑衣青年，从未见过，不知是何方人物。
<br/>　　“玉虚子前辈，晚辈只是一个无名小卒，名讳不足挂齿！”
<br/>　　黑衣青年摇了摇头，淡淡的问道：“今日，在下来此，是有事和前辈相商！”
<br/>　　“什么事？”玉虚子皱眉闻言，眼前之人，藏头漏尾，连名讳都不敢暴露，一看就不会有好事。
<br/>　　不过，对方修为强大，不弱于他，他也不好直接赶人。
<br/>　　“不知前辈和终极神王殿的绫裳宫主，是否还有来往？”黑衣青年沉声问道。
<br/>　　这黑衣青年，不是别人，正是紫霄的密门弟子楚天墟。
<br/>　　“绫裳？”
<br/>　　玉虚子闻言一怔，随即摇了摇头，道：“绫裳离开我玉虚宗，已经数十万年了，哪里还有来往？”
<br/>　　玉虚子面容平静，心中却是心念电转，他虽然不是什么大人物，但对于神域的一些大事，却也是有所耳闻的。
<br/>　　最近这些年，秦陨的事，闹的风波极大，而那个秦陨，就是绫裳麾下的弟子。
<br/>　　而目前，好像是秦陨和绫裳，一起消失不见了。
<br/>　　玉虚子心中无奈，看来是有人要找秦陨，然后连他和绫裳的关系，都调查清楚了。
<br/>　　“没有来往了？”
<br/>　　楚天墟闻言，古板的面容之上，现出一丝惊疑，继续问道：“前辈，据我所知，你当年对绫裳有大恩，而绫裳是一个重情重义之人，不会和你断了联系吧？”
<br/>　　“怎么？你认为老夫在骗你？”玉虚子脸色一沉，面现不悦之色。
<br/>　　“前辈你误会了，晚辈不是这个意思！”
<br/>　　楚天墟摇了摇头，沉声道：“在下需要前辈帮一个忙，只要前辈愿意，十亿极品源石相送！”
<br/>　　“什么忙？”玉虚子沉声问道，十亿极品源石虽然惊人，但他也不会为了源石，而去做有违道义之事。
<br/>　　“很简单，将绫裳引出来！”楚天墟一字一顿的说道。
<br/>　　“这不可能！”
<br/>　　玉虚子闻言，立刻出言拒绝，道：“老夫与绫裳之间，早已没有任何关系，如何能引她出来？”
<br/>　　对于眼前黑衣青年的目的，玉虚子心中依然明了，此人是想靠他找到绫裳，从而再找到那个秦陨。
<br/>　　他绝对不会帮这个忙，不说自己帮不了，就算能帮，也不会去帮。
<br/>　　“前辈，百亿极品源石，你真的要放弃吗？”
<br/>　　楚天墟眼眸微眯了下来，蛊惑道：“这么多的源石，足以让你们玉虚宗，更上一个层次！”
<br/>　　“小子，老夫修武一百五十万载，区区百亿极品源石，你就想收买老夫吗？”
<br/>　　玉虚子阴沉着脸，随即长身而起，冷漠的道：“阁下请离开吧！玉虚宗恕不招待了！”
<br/>　　“前辈莫要急躁！”楚天墟摆了摆手，大马金刀的坐在座椅上，完全没有起身的意思。
<br/>　　“前辈，听说您唯一的儿子早年不幸陨落，只留下了一女；您这孙女生的冰雪聪明，天赋极为不凡，深受您的疼爱，准备过几年将她送进永恒神宫？”楚天墟声音淡淡，仿佛是要与玉虚子叙家长里短。
<br/>　　“你什么意思？”玉虚子脸色一变，大声质问道。
<br/>　　“没什么意思！”
<br/>　　楚天墟古板的脸庞之上，罕见的露出了一丝笑容，道：“听说前段时间，您的孙女离开了玉虚宗，恐怖是回不来了！”
<br/>　　“放肆！”
<br/>　　玉虚子闻听此言，立刻怒发冲冠，一掌将身下的座椅拍成了齑粉，大声怒喝道：“你把池儿怎么样了？”
<br/>　　“前辈息怒！”
<br/>　　楚天墟摆了摆手，冷笑一声道：“您那可爱的孙女，被晚辈请去一个地方做客了，只要您老愿意帮晚辈这个忙，你的孙女自然会回来，但倘若你不愿意帮忙，那就不好说了；如此冰雪聪明、可爱伶俐的丫头，若是遭遇不测，那真是可惜了！”
<br/>　　“你找死！”
<br/>　　玉虚子闻听此言，顿时暴怒，他的孙女就是他的禁忌，谁若是敢加害，那就是要他的命。
<br/>　　言罢，玉虚子大手一张，浑厚的源力涌动，庞大的气势冲天而起，瞬间就将宫殿掀飞了出去。
<br/>　　“你若是敢动手，您孙女会不会健全，晚辈可不敢保证！”楚天墟面色平淡，看也不看玉虚子，淡淡的说道。
<br/>　　他丝毫无惧，不说他手上有筹码，就算没有筹码，一个小小的玉虚子，也不是他的对手。
<br/>　　虽然他和玉虚子一样，都是九品低级的修为，但是，他的实力，已经堪比九品高级，直追神王级别的强者了。
<br/>　　“你……！”玉虚子闻言，顿时面色一滞，准备击出的一招，硬生生的止住了。
<br/>　　嗖嗖嗖！！
<br/>　　大殿中巨大的动静，已经惊动了整个玉虚宗，无数的弟子和长老，纷纷前来。
<br/>　　“所有人都退下！”
<br/>　　玉虚宗大手一挥，命令所有人退避，随即，对楚天墟道：“随我到密室一谈！”
<br/>　　“可以！”楚天墟点了点头，他不担心对方有什么轨迹，只要对方见不到孙女，就不敢轻举妄动。
<br/>　　随后，玉虚子便带着楚天墟，进入了玉虚子一处隐秘的密室之中。
<br/>　　“阁下，你应该知晓，我玉虚宗乃是永恒神宫的下属势力，你如此做法，无异于在叫板永恒神宫。”
<br/>　　密室之中，玉虚子冷视着楚天墟，声音冰冷刺骨，他在强压着心中的怒火。
<br/>　　“自然知晓！”
<br/>　　楚天墟微微点了点头，又道：“不过，这都不重要，重要的是，一个不慎，您的孙女将再也不会出现！”
<br/>　　对于永恒神宫，楚天墟自然万分忌惮，不然的话，他也不用使出这种手段了，不过，他有自己的计划，能保证万无一失。
<br/>　　“你……！”玉虚子心中怒不可竭，但是，因为心有忌惮，他又不好发作。
<br/>　　少倾，玉虚子压下心中的愤怒，沉声问道：“我若是帮了你，你能否保证池儿的安全？”
<br/>　　“我与你往日无仇，近日无怨，只要您帮我引出绫裳，我自然会让您的孙女平安归来！”楚天墟郑重的说道。
<br/>　　“好吧，你想让老夫怎么做？”沉默了片刻，玉虚子声音低沉的问道，此事不能鲁莽，只能先应了对方，再见机行事了。
<br/>　　“前辈，你这样做……！”
<br/>　　随后，楚天墟告诉了玉虚子他的计划，怎么样引绫裳现身。
<br/>　　两人在密室之中，商量了半刻钟，商议好了详细的计划，楚天墟才告辞离去。
<br/>　　玉虚子，亲自将楚天墟，送出了山门。
<br/>　　送走楚天墟之后，玉虚子也没有闲着，立刻召集几名玉虚宗的高层，商议对策。
<br/>　　与此同时，立刻安排人手，调查楚天墟的身份。
<br/>　　对于玉虚子而言，他的孙女池儿，一定要安全的救回来，但同时，他也不能让绫裳陷入万劫不复之境。
<br/>　　毕竟，当年绫裳在玉虚宗，是他最为看重的弟子，虽然后来绫裳走了，但是，对方也从未忘记他的恩情。
