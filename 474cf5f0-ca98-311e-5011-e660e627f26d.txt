　　苏莫强势击伤何传岩，待何传岩攻击之时，就说自己是为了保命，而无奈反击，然后将何传岩杀死。
<br/>　　此种做法，在所有人看来，实在是无耻没有节操！
<br/>　　这种话也能说出来，真是没谁了！
<br/>　　不过，苏莫的果决狠辣，也是让众人心中微凛，这是个杀伐果决的主啊！
<br/>　　天空中。
<br/>　　苏莫叹了口气，何传岩不接受他的挑战，他只能这么做了！
<br/>　　所谓的无奈反击，也就是一个借口而已，学府若真的想罚他，有没有借口都是一样。
<br/>　　少倾，苏莫转头看了一眼小岛上的聂乾和韩盈两人，向两人点了点头，立刻便向战界的出口飞去。
<br/>　　他现在连杀两人，已经闹出了很大的动静，还是暂时不要露面的好！
<br/>　　聂乾和韩盈两人，面色有些呆滞，依旧有些没有回过神来。
<br/>　　因为，他们两人比其他人更了解苏莫，他们知道苏莫还有一种吞噬武魂。
<br/>　　太阳武魂、吞噬武魂！
<br/>　　双生武魂的天才！
<br/>　　还有大日神体！
<br/>　　聂乾和韩盈，越想越觉得震撼，越想越觉得不可思议。
<br/>　　这种天赋，已经超越了学府中那些圣子了！
<br/>　　十几座小岛之上，不少人望着苏莫离去的背影，眼神闪烁，暗暗沉思。
<br/>　　神武学府派系繁杂，现在突然出现了一个苏莫这种天才，估计又有一番风云了。
<br/>　　只是不知道，苏莫有没有加入某个派系。
<br/>　　太史轩眼眸微眯，嘴角勾起了一丝笑意，不知其心中所向。
<br/>　　木璃目光复杂，心绪久久不能平息！
<br/>　　……
<br/>　　“哈哈！苏莫，汝这才像妖孽天才的作风！”
<br/>　　回到阁楼之后，苏莫的脑海中，便响起了阎魔鬼圣的笑声。
<br/>　　苏莫能战胜冷江，出乎他的意料，但也在情理之中。
<br/>　　因为通过这些时日的相处，他知道苏莫是非常谨慎的人，没有一丝把握的话，根本不可能应战。
<br/>　　“狗屁吧！”苏莫闻言叹了口气，今日他暴露的可是不少，尤其是大日神体这种厉害的天赋，都彻底暴露了。
<br/>　　好在，他动用三种战魂和其它剑武魂的事情，应该没有人发现吧！
<br/>　　不过，话说回来，三种战魂的威力着实强大，虽然不是器武魂，但与剑气一起攻击出去，效果依然巨大。
<br/>　　看来，他就算武魂晋升到战魂以后，不需要再升级了，也要多多吞噬他人战魂，这可是大杀器啊！
<br/>　　“苏莫，汝修炼的是什么功法？”阎魔鬼圣没有在乎苏莫的语气，沉声问道。
<br/>　　对于苏莫拥有大日神体，阎魔鬼圣没有多少意外，毕竟他早已知道苏莫能吞噬血脉，拥有特殊体质，也在情理之中。
<br/>　　他毕竟好奇的是，苏莫的玄力，为何如此的浑厚？威力如此的强大？
<br/>　　就算是苏莫修炼尊级功法，乃至圣级功法，玄力也不可能如此的厉害吧！
<br/>　　“我修炼了五种功法！”苏莫沉声说道，他也不准备隐瞒对方，毕竟还需要对方的指导。
<br/>　　若是对方不了解他的修炼情况，自然也无法指导。
<br/>　　“五种功法？同时修炼？”阎魔鬼圣闻言愕然，心中有些不解。
<br/>　　因为在武道一途，无论是何种族，是何修为境界，都只能修炼一种功法。
<br/>　　若是换了一种功法，以前的功法就要舍弃，根本不能同时修炼。
<br/>　　“嗯！五行属性的功法，同时修炼！”苏莫点头道。
<br/>　　阎魔鬼圣闻言，不免心中一震，难怪苏莫的玄力威力如此的巨大，远超修为境界，原来是集合了五行之力。
<br/>　　但是，五种功法，是如何修炼的？
<br/>　　“汝为何五种功法，能同时修炼？”阎魔鬼圣好奇的问道。
<br/>　　“因为五种功法，已经融合成了一种功法！”
<br/>　　苏莫微笑着说道，言罢，他继续道：“好了，鬼圣前辈，你也不要再问了，日后自然会知晓。”
<br/>　　阎魔鬼圣闻言，便没了声息，但他心中越发震惊。
<br/>　　对于苏莫了解的越深，他越是感觉到苏莫天赋的恐怖，这简直是万古罕见。
<br/>　　他有一种预感，苏莫若是成长起来，绝对能震动整个天荒星域。
<br/>　　呼！
<br/>　　苏莫深吸口气，随即在阁楼内盘膝坐了下来。
<br/>　　冷江一事解决了，他无事一身轻，可以静心修炼，突破到武帝之境了。
<br/>　　沉吟了片刻，苏莫手臂轻轻一挥，一具透明的棺椁，便出现在了他的面前。
<br/>　　棺椁正是玄天灵柩。
<br/>　　玄力灵柩之中，芊寻月安静的躺在其内，仿佛正在熟睡一般。
<br/>　　绝世的容颜，倾世的风姿，没有丝毫的改变，仿似画中的仙子。
<br/>　　“很快我就能救活你了！”望着玄天灵柩中那凄美的身影，苏莫的面色平静了下来，低声自语。
<br/>　　只要他突破到武帝之境，就能救活芊寻月了。
<br/>　　救活了对方之后，他便也不欠对方了！
<br/>　　芊寻月之事，就像是一根刺，深扎在苏莫的心间，不救活对方，他一辈子都不会心安。
<br/>　　宏青璇也是如此，不救出对方，他同样难以心安。
<br/>　　只不过，他根本无力解救宏青璇，苏莫的心中已经决定了，待修为再强一些，他就要去巫族所在的星球，去救青璇。
<br/>　　无论对方是死是活，无论对方身处何地，他都一定要去寻找。
<br/>　　宏青璇之事，他目前无能为力，但救活芊寻月，已经近在眼前了。
<br/>　　“汝救不活他！”
<br/>　　就在此时，一声幽幽的声音，在苏莫的脑海中响起，正是阎魔鬼圣的声音。
<br/>　　苏莫闻言一怔，随即眉头一皱，道：“你说什么？”
<br/>　　“吾说你救不活她，就算是吾在巅峰时期，都救不活她！”阎魔鬼圣说道。
<br/>　　“你胡说八道，怎么可能救不活？”苏莫闻言，仿佛受了刺激一般，顿时炸毛了，放声大吼了起来。
<br/>　　救不活？
<br/>　　怎么可能？
<br/>　　当初天涯海阁的胡副堂主可是说了，由武帝境的强者出手，便能救活芊寻月。
<br/>　　现在，阎魔鬼圣居然告诉他，救不活！
<br/>　　这让苏莫无法接受！
<br/>　　“哼！你打开棺椁，让吾查探一番！”阎魔鬼圣冷哼一声道，仿佛苏莫的姿态，让他也动了怒气。
<br/>　　苏莫闻言有些迟疑，这老魔头不会是想夺舍芊寻月吧？
<br/>　　不过转念一想，又不太可能。
<br/>　　对方在他的吞噬武魂之中，根本耍不了什么花招！
<br/>　　随即，苏莫点了点头，便起身打开了玄天灵柩。
