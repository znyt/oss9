　　众人闻言，皆是下意识的看去，戒备着进行查看。
<br/>　　那壁虎怪物的身体，已经彻底失去了所有生命特征，但和先前血肉饱满的样子不同，现在这具尸体，就如同一个被风干了一般，几乎可以用皮包骨来形容了。
<br/>　　体积也是缩水了不少，非常明显。
<br/>　　之前没有发现，是因为刘哔和保安小哥在戒备周围，并且楼道的灯光并不明亮，所以才忽略过去了。
<br/>　　直到，白小飞出声提醒，他们才发现了不对劲的地方。
<br/>　　让保安小哥和白小飞感觉不对劲的，是这个尸体突然之间的变化，还在半小时前，这还是一个有血有肉的怪物啊！
<br/>　　怪不得这么久了，都没有引来什么东西，要知道血腥味，是很容易引来尸兄和变异生物的。
<br/>　　这栋楼里不可能只有这么一只变异动物。
<br/>　　而让刘哔感觉不对劲的地方，和两人并不相同，毕竟这种状况是他一手造成的，只是这样，他根本不会吃惊。
<br/>　　但……壁虎干尸的位置被移动了！
<br/>　　虽然很细微，但刘哔还是眼尖的察觉出了这一点，这具尸体，好像被什么东西翻动过！
<br/>　　是尸兄么……或者其他变异动物？
<br/>　　翻动尸体而不啃咬，也没有任何声音，更没袭击他们。
<br/>　　这个家伙，有智慧的么？
<br/>　　会不会……是先前在大头尸兄身上感应到的那个东西！
<br/>　　诸多想法，如同电光火石，只是一瞬间就在刘哔脑海中形成，紧接着，他的精神高度集中了起来。
<br/>　　“嘶~”
<br/>　　像是毒蛇吐信的声音响起，刘哔眉头一挑，身体向旁边动了一下，躲过了一根细小的舌头。
<br/>　　真是阴魂不散，这栋大楼里，为什么会有这么多奇葩的东西！
<br/>　　刘哔转了下身，看到身后突然出现的一个奇形怪状的尸兄。
<br/>　　那是一个全身蓝色的小矮人，长相丑陋，皮包骨头，下半身却和一只硕大的变异青蛙连在一起，体型比正常人类还要高出许多。
<br/>　　“呀~被躲过去了。”
<br/>　　蓝色小人裂开嘴巴，露出一口交错的锯齿，在阴沉沉的楼道中，显的格外渗人。
<br/>　　“咕噜……”
<br/>　　变异青蛙抽回舌头，开口吐出了一枚绿色液体，刘哔身影一动，擦着绿色液体的边缘躲了过去。
<br/>　　瞬间，爆裂的声音响起，绿色液体擦过刘哔，径直射在了走廊尽头的墙壁上，将混凝土墙壁，硬生生的砸开了一个大洞。
<br/>　　“该死，这是什么东西？”
<br/>　　此时，白小飞和保安小哥也反应了过来，纷纷上前帮忙，只是变异青蛙的速度太快，即使是白小飞也有些跟不上它的动作。
<br/>　　更不要说保安小哥了，为了避免误伤，他甚至都不敢使用手中的枪支。
<br/>　　“不行，速度太快，根本无法瞄准。”
<br/>　　咬了咬牙，保安小哥拼命跑动着，丝毫不敢停下，那样很容易受到变异青蛙的攻击。
<br/>　　没被强化过的体质，是根本不可能硬抗变异兽攻击的。
<br/>　　“你先撤出去，保证自己安全！”
<br/>　　趁攻击之余，白小飞抽空吼了一句，但他本身的攻击，大部分也落不到青蛙身上。
<br/>　　即使打中，也不过是一些无关紧要的地方，根本造成不了太大的伤害。
<br/>　　“咦，这怪物，防御不怎么样嘛，那也就只是速度快一些了？”
<br/>　　刘哔手中的黑色棍棒再次燃烧起来，只是随手一挥，便将变异青蛙的一根前肢砸的稀烂，这种强度，对他来说简直和豆腐没什么区别。
<br/>　　包括变异兽那引以为傲的速度，在他眼中也只是从正常人的龟速，变成了蚁爬而已。
<br/>　　“啊！你伤了我的‘呱呱’！我要吃了你！”
<br/>　　蓝色小人握起拳头，声音带着恼意，随后左臂一指，数十发绿色液体冲向刘哔，封死了他所有的退路。
<br/>　　没法躲避么……
<br/>　　眼中闪过一丝不屑，刘哔左手虚抓，在身前幻化成了十只能量手掌，将绿色液体一一捏爆。
<br/>　　他又怎么能被这种层次的攻击伤到……至少，也要驱使这只青蛙的主人前来吧。
<br/>　　这只青蛙绝对不是偶然出现在这里的，从一开始，这就是一场有计划有预谋的伏击。
<br/>　　先是变异蚊群，紧接着是不知名的怪物截断电梯，逼迫他们出来，然后双头壁虎封路，最后是这个变异青蛙。
<br/>　　这绝对不是巧合，这些家伙一个个上来，就是为了拖延时间，似乎……像是在等待着什么东西！
<br/>　　是那个大头尸兄的本体么？
<br/>　　倒要见识见识，究竟是一个什么样的东西……
<br/>　　站在原地未动，面前密集的绿色液体猛然爆开，随后，一阵惊天动轰鸣声响起，顿时尘土飞扬，整个走廊从中间被炸穿，白小飞和变异青蛙一起，伴随着落石掉落进了下一层。
<br/>　　“无知小儿，你伤不了我。”
<br/>　　从尘雾中走出，刘哔身上没带一点伤势，甚至衣服都没有破碎，些许尘土落在他的肩头，也没有显出一丝狼狈。
<br/>　　并不是每次攻击都能躲开。
<br/>　　躲不掉的，
<br/>　　硬接下来就好了。
<br/>　　当你的力量，强大到足以碾压对手的时候，任何细节与技巧，都不需要了。
<br/>　　见到自己的攻击未奏效，蓝色小人脸上露出不敢相信的表情，刚才的攻击，耗费了它太多能量，短时间内，它根本无法使用第二次。
<br/>　　可是，在往常的情况下，这个攻击都能秒杀掉敌人啊，今天为什么却失去效果了呢。
<br/>　　这让它本来就不算高的智商，更是难以理解。
<br/>　　“小鬼，你爸妈没教过你，不要乱吐口水吗？”
<br/>　　趁着变异青蛙进行攻击的间隙，白小飞摸到它的近前，一刀劈向了变异青蛙的头颅。
<br/>　　这个怪物最大的依仗就是速度，只要能将它黏住，打败它也只是时间问题。
<br/>　　恰好，怪物发动向刚才那样攻击的时候，是不可以进行移动的。
<br/>　　如果是单对单，刘哔也无法趁机进行攻击，这点小小的间隙，自然可以忽略。
<br/>　　但，现场还有白小飞在，这就成了一个很好的机会！
<br/>　　真是石乐志啊……
<br/>　　白小飞嘴角露出一抹笑容，眼看他手中的长刀就要劈在变异青蛙头顶，情况却突然发生变化。
