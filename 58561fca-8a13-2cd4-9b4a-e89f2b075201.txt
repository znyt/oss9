　　圣龙秘境之中，最为逆天的两位准帝，就是小叶帝，小绝帝，两位接受了力量传承的准帝了。
<br/>　　可纵然是他们二人到来，面对这独魔皇，也要拼尽全力斩杀，至于肉身去抗毁灭之光，更是在自己找死。
<br/>　　可楚风眠抗下毁灭之光，却是无比轻松，难道说楚风眠已经是超越了小叶帝，小绝帝？
<br/>　　“短短一个月不到的时间，此人怎么会逆天到如此地步？”
<br/>　　一名五灵宗圣子不可置信的说道。
<br/>　　他曾经见到过楚风眠，那个时候楚风眠虽然实力也无比恐怖，但是却跟现在远远无法相比。
<br/>　　一个月不见，楚风眠的实力已经是发生了翻天覆地一般的变化。
<br/>　　“一个小小准帝？”
<br/>　　那独魔皇也看清楚了楚风眠的身影，尤其是看见楚风眠身上境界的时候，他也是无比的不可思议。
<br/>　　能够硬抗他毁灭之光的人，居然是一名准帝？连帝尊都不是。
<br/>　　“楚风眠？你就是楚风眠？”
<br/>　　那独魔皇，听到这个名字，突然间的看向楚风眠，脸上的愤怒之色突然一扫而空，取而代之的是无比炙热的眼神。
<br/>　　“好，圣龙秘境弟子，不错，看来你就是少帝要找的那个楚风眠了！”
<br/>　　“没想到来一趟九域，还真是大有收获！小子你招惹了少帝，还敢出现在我们魔族面前？正好今日就抓住你，去少帝哪里邀功去！”
<br/>　　轰隆！
<br/>　　这独魔皇在看见楚风眠的一刻，便是调转方向，猛然之间冲这楚风眠冲了过来。
<br/>　　头顶之上的独角，再次颤动，化为一道红色光芒，直指楚风眠轰击而来。
<br/>　　“不好！”
<br/>　　“这独魔皇，可不好对付，你们快去出手，相助楚风眠！”
<br/>　　木麟看着这一幕，急忙的喊道，同时压抑着伤势，一步上前，想要帮忙。
<br/>　　“几个八劫古帝的魔皇罢了，算不得什么。”
<br/>　　楚风眠嘴角露出几分不屑之色道。
<br/>　　“炎撼世，你去帮助他们，对付其他的魔族，这四个魔皇，我一个人解决了。”
<br/>　　“一个人解决？”
<br/>　　听到楚风眠的话，那一群五灵宗圣子都愣住了。
<br/>　　他们虽然刚刚见到了楚风眠的实力，随意一道剑气，都足以斩杀一名七劫古帝实力的魔族。
<br/>　　这种实力，的确是站在了八劫古帝的巅峰了。
<br/>　　可眼前这四名魔皇，可都是八劫古帝，这四位魔皇联起手来，除非是九劫古帝，不然谁敢说一个人足以解决了他们？
<br/>　　就算是得到了力量传承的小叶帝，小绝帝等人，也不敢说出这种话。
<br/>　　“好大的口气！”
<br/>　　听到楚风眠的话，那夜叉皇，鬼魅皇，阎魔皇，这三尊魔皇，看向楚风眠的眼神都是无比的冰冷。
<br/>　　“给我死！”
<br/>　　夜叉皇脾气最为暴躁，眼神之中杀机涌现，不跟楚风眠废话，直接出手了。
<br/>　　夜叉皇手心猛然一抓，周围的魔意便是向着他的手心涌了过去，当即化为了一根长矛，对着楚风眠一矛猛然刺了过来。
<br/>　　“蝼蚁而已！”
<br/>　　楚风眠看着逼近过来的夜叉皇，脸色的不屑之色丝毫未减，在他这夜叉皇的长矛接近过来的一刻。
<br/>　　楚风眠背后的天涯剑匣，当即打开，两道剑气飞了出来，嗖的一下子，打在了长矛之上。
<br/>　　咔嚓！
<br/>　　这魔意所凝聚的长矛，跟两道剑气碰撞的瞬间，被一下子打碎。
<br/>　　“什么？”
<br/>　　夜叉皇都没有反应过来，楚风眠的剑气居然是强大到了这个地步。
<br/>　　他刚刚想要退后，却是为时已晚，这两道剑气对着夜叉皇的胸口当即斩杀而去，轰然之间，胸口被贯穿，鲜血狂飙，已经是奄奄一息。
<br/>　　“一招击败？”
<br/>　　“这？怎么会如此逆天的人物，这楚风眠才是一名准帝啊，准帝一招击败八劫古帝的魔皇？”
<br/>　　盯着那奄奄一息的夜叉皇，五灵宗圣子都是愣住了。
<br/>　　就连那见识过楚风眠的实力的炎撼世，都愣住了。
<br/>　　他之前见到楚风眠击败那阿修罗皇的时候，也是花费了一些手段。
<br/>　　可现在，楚风眠甚至是本体未动，单单是背后剑匣打出的两道剑气，就足以轰杀了夜叉皇。
<br/>　　这种实力已经是强横到了一种不可思议的地步。
<br/>　　纵然是大罗天星域各个势力之中的天才，都没有一人达到了如此逆天的地步。
<br/>　　另一边，那三名魔皇，也是被镇住了。
<br/>　　“这小子的实力怎么这么恐怖？”
<br/>　　“怪不得少帝都想要杀了此子！这种实力，简直逆天！”
<br/>　　“一起出手，诛杀了此子，此子不除后患无穷！”
<br/>　　阎魔皇，鬼魅皇，独魔皇这三尊魔皇，互相交谈一声，眼神之中都是起了必杀之意。
<br/>　　楚风眠现在才只是准帝，实力就达到了这种地步，一旦是让楚风眠踏入帝尊境，那还得了？
<br/>　　更何况，楚风眠还是少帝点名要杀之人。
<br/>　　无数魔意凝聚，在空中足足化为了十几道魔爪，同时向着楚风眠轰击而来。
<br/>　　这每一道魔爪的力量，都丝毫不亚于八劫古帝的全力一击，几乎是相当于十几名八劫古帝，在一起对着楚风眠出手。
<br/>　　恐怖的力量之下，周围的空间都在崩塌，这周围空间，都无法承受如此恐怖的力量。
<br/>　　“在挣扎，也无济于事！”
<br/>　　楚风眠目光冰冷无比。
<br/>　　在这十几道魔爪即将降临的一刻，楚风眠右手一挥，祖龙至尊剑，突然出鞘，从天涯剑匣之中，飞了出来。
<br/>　　祖龙至尊剑，散发出令人胆寒的剑锋。
<br/>　　在天涯剑匣的温养之中，如今祖龙至尊剑，也带有着几分天涯剑匣之力，在不断的蜕变。
<br/>　　终究有一日，这祖龙至尊剑，也有蜕变成极道仙兵的一天。
<br/>　　手持灵剑，楚风眠身上的气息，便是截然不同，剑意入芒，冲天而起。
<br/>　　楚风眠是一名剑修，唯有手持灵剑的一刻，才是他最强的状态。
<br/>　　“正好就以你们，来试剑！”
<br/>　　“三阳烈炎剑！”
<br/>　　嗡！
<br/>　　在剑锋之上，炙热的烈焰，熊熊燃烧而起，冲天剑出。
<br/>　　轰隆！
<br/>　　一剑出，烈焰焚天，漫天魔爪，在这熊熊火焰之下，全部毁灭在天空之上。
<br/>　　同时这烈焰，更是不止如此，向着那一群魔族直接燃烧而去。
<br/>　　整片天际，都成为了火焰的海洋。
