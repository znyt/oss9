　　房间中，苏莫正在努力的冲击境界瓶颈，希望在千选战到来之前，成功突破修为。
<br/>　　咚！咚！咚！
<br/>　　然而，他才修炼了两个时辰的功夫，门外响起了急促的敲门声。
<br/>　　“进来！”苏莫睁开了眼眸，无奈的喊道。
<br/>　　咯呀！
<br/>　　房门被推开，龙腾和司空炎、池儿三人，大步的走了进来。
<br/>　　“师尊，有个好消息！”池儿率先开口，迫不及待的说道。
<br/>　　“什么好消息？”苏莫疑惑的问道。
<br/>　　“苏……！”司空炎正欲开口，结果又被池儿抢了先，只见池儿面色振奋，大声说道：“师尊，有一种宝物，肯定能助你突破修为。”
<br/>　　司空炎张了张口，话语卡在了嗓子里，无语的瞥了池儿一眼，心中几乎吐血。
<br/>　　随即，他也不再多言，就让池儿讲述了。
<br/>　　“什么宝物？”苏莫好奇了起来，若是真有这种宝物，时间允许的范围之内，他一定要得到。
<br/>　　“这宝物叫混沌玉髓，混沌古山之上便有！”池儿急忙说道。
<br/>　　“混沌玉髓？”苏莫闻言，顿时眼眸一亮，只是听名字，他就感觉到这宝物对他肯定有用。
<br/>　　“这混沌玉髓为何物？“苏莫再次问道。
<br/>　　这一次，池儿不做声了，转头看了看龙腾和司空炎，笑着道：“两位师叔，你们说吧！”
<br/>　　龙腾无奈的笑了笑，随即面上笑容收敛，郑重的望着苏莫，道：“这混沌玉髓，乃是混沌神朝特有之物，诞生于混沌古山之上，无比的珍贵，对于混沌族而言，此物是修炼的圣物，但是，此物非常稀少，目前混沌古山之上有没有还不能确定；而且，就算是有，你也不容易得到。”
<br/>　　“不错，此物太过稀少，所以，我们首先要确定混沌古山上有没有，若是有的话，再想办法得到。”司空炎沉声道，这基本上是苏莫唯一的机会了。
<br/>　　只有得到混沌玉髓，才能顺利的踏入五品修为。
<br/>　　苏莫闻言，沉默不语，心中在思索办法。
<br/>　　“苏莫，之前我和司空炎商量了一下，不如让大长老前去，看看混沌族的强者，愿不愿意让出混沌玉髓？”龙腾沉声说道。
<br/>　　为了得到玉髓，他们龙族就是花费一些代价，也无不可。
<br/>　　“龙族和混沌神朝的关系怎么样？”苏莫问道，这是一个关键，若是关系很好，或许有机会。
<br/>　　“关系一般！”龙腾道，天龙神朝和混沌神朝，虽然一直有来往，但是并不密切，只能说是一些表面上的关系。
<br/>　　主要是，混沌神朝的混沌族，比较低调，有超然物外的形态，不会过多和各大神朝深交。
<br/>　　“还是我自己去吧！“苏莫叹了口气说道，他也是混沌族之人，说不定亲自去更加有机会。
<br/>　　而且，上次在龙都城，有一位面容儒雅的混沌神朝的神王，言语貌似有招揽之意。
<br/>　　“你自己去也行！”龙腾点了点头。
<br/>　　“龙腾，你还记得上次去龙都城的那位混沌神朝的神王吗？此人是何名讳？”苏莫沉声问道。
<br/>　　“那位是未天神王，混沌族！”龙腾道。
<br/>　　“好，我就去找此人！”苏莫深深的吸了口气，随即便大步的离开了房间，走出了宫殿。
<br/>　　宫殿之外，有着两名混沌古山的弟子，修为不高，均是四品的修为。
<br/>　　这两人，是在宫殿之外，听候吩咐的弟子。
<br/>　　“这位兄弟，在下求见未天神王，还请带路！”苏莫看向其中一名弟子，向对方抱了抱拳道。
<br/>　　“未天神王？”那名弟子闻言皱眉，若是龙族的长老求见，他自然立刻带路，但是，不是龙族的长老，他怎么可能随便带人前去求见。
<br/>　　“在下苏莫！”苏莫见对方的表情，立刻自爆名讳。
<br/>　　“什么？你就是苏莫？”这名弟子一听，顿时双眼一瞪，面带不可思议之色。
<br/>　　苏莫之名，他自然知道，而且是如雷贯耳。
<br/>　　此刻，他立即恍然，苏莫一直在天龙神朝，看来这次和龙族之人同来了。
<br/>　　“请跟我来！”随即，这名弟子立刻点了点头，为苏莫带路，向顺着一条小道，向古山上方走去。
<br/>　　混沌古山很大，这名弟子带着苏莫，在山上走了片刻，来到了古山的上方区域，走到了一出山崖之前。
<br/>　　山崖之上，坐落着一座宫殿，气势恢弘，通体金黄，仿佛是用纯金打造而成。
<br/>　　“未天长老，苏莫求见！”
<br/>　　那名弟子来到宫殿之前，对着紧闭的大门，恭敬的喊道。
<br/>　　不过，宫殿之中，并未回音，弟子连续喊了三声，依旧如此。
<br/>　　“苏莫，未天长老不在，应该在处理事务！”弟子看向苏莫，摇了摇头说道，最近，各大神朝的人陆续来临，古山上事务繁多，未天长老不在住处很正常。
<br/>　　“我在此地等他！多谢你了！”苏莫向对方道谢。
<br/>　　“这……好吧！”那名弟子略一迟疑，还是点了点头，苏莫也算是有头有脸的人物，不至于暗中做什么事情。
<br/>　　随即，这名弟子不再多言，便踏步离去。
<br/>　　苏莫来到宫殿之前，在大门口盘膝而坐，一边修炼一边等待了起来。
<br/>　　这一等，就是两日的时间。
<br/>　　两日后的中午时分，人影一闪，一道挺拔的身影，出现在了苏莫的身前。
<br/>　　此人是位中年男子，身穿白衣，面容儒雅，如同是一位文士，气质极为独特。
<br/>　　“苏莫！”中年男子看到宫殿门前盘坐的苏莫，面色微微讶然，显得很是意外。
<br/>　　“晚辈见过前辈！”苏莫急忙结束了修炼，起身向中年男子行礼，此人正是未天神王。
<br/>　　“进来吧！”未天神王点了点头，一挥手，宫殿的大门自行打开了，他大步的走进殿内。
<br/>　　他心中清楚，苏莫无事不登三宝殿，会在这里等他，一定有要事。
<br/>　　“嗯！”苏莫点头，跟着未天神王走进了宫殿中。
<br/>　　宫殿内，有议事大厅，两人来到大厅中，分宾主落座。
<br/>　　“苏莫，你有何事？”未天神王坐在主座之上，望着下方的苏莫，开门见山的问道。
<br/>　　“前辈，我想要混沌玉髓！”苏莫目光凝视着对方，同样开门见山的说出了来意，对方直接，他也不会拐弯抹角。
