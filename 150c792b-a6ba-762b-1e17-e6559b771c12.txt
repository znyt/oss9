　　钦陵在李绩被调回长安之后，暂时成了整个东北地区在军事方面的主要负责人。
<br/>　　因为东北地区一面靠海，两在靠近内陆，此时已经没有了外在的威胁，靺鞨人因为生活条件的改善变的好控制了许多。
<br/>　　再加上钦陵娶了一个黑水部的女人为妻，在靺鞨人的眼中，成了靺鞨人的女婿，所以他说的话在东北也比较好使，已经隐隐有了东北王的趋势。
<br/>　　但尽管如果，钦陵对长安传来的命令还是不敢有丝毫的大意。
<br/>　　东北的其他人或许不知道李承乾的厉害，但钦陵却不是一般人，他很清楚如果长安的那位新皇帝如果没有一定的把握，绝对不会放任他在东北称王称霸。
<br/>　　而且从一直游弋在大海上的北海舰队的装备来看，东北靺鞨人根本不足以与内地一较长短，那种装填金属子弹的步枪可以完胜他们的火铳，那种可以超过距离发射炮弹的贞观炮也不是他们那种只能打出一千五百步的贞观可以比拟的。
<br/>　　另外还有一点就是火药武器极度依赖后勤，他们的武器装备如果没有内地的火药支持，与烧火的棍子几乎没有任何的区别。
<br/>　　在钦陵看来，东北其实就是一个工业基地，负责向内陆输出钢铁、洧水、煤炭等等物资，就算是有一定的军事实力，也是为了防御西边的铁勒人，还有北边的那些个全身长毛的野人。
<br/>　　“相公，在想什么呢？”钦陵的夫人，也就是当初黑水部族长之女玉儿，端着一碗热腾腾的饺子走进他的书房，见钦陵在发呆便小声的问道。
<br/>　　“陛下的手谕，让我领兵出征倭岛。”钦陵放下手中的军令以及半块虎符，对着身边的女子淡淡一笑：“为夫要出征了。”
<br/>　　“又要出征么？！”玉儿愣了一下，笑容渐敛。
<br/>　　“大唐以武立国，若是想要功名自然要在战场上搏来。”钦陵拉过玉儿的手，微笑依旧：“再说我们的孩子也快要出世了，我这个当爹的怎么也要给他搏个未来吧，总不能让我们的孩子受了委屈。”
<br/>　　“偏是你有道理！”听钦陵提到孩子，玉儿的小脸微微一红，伸手打掉那只按在她小腹上的手，把一大盘饺子往老公面前一推：“快点把饺子吃了吧，这是你最爱吃的，我，我去给你准备出征的东西。”
<br/>　　想到丈夫就要奉皇命出征，玉儿的神情又有些黯然，转过身时眼眶已经微微泛红。
<br/>　　不过她也知道，这不是她一个女人可以阻止的，不说皇命难违，就说她自己的相公本人，也一直在期待着这一战。
<br/>　　想当初倭岛之战，松赞干布随军出征，虽然没有独自领兵但也算是打的有声有色，自视甚高的钦陵作为松赞干布的同族，看到这一切自然心中会有些不舒服，生出一较长短的心思也很正常。
<br/>　　可是倭岛之战打了一半便停了，李承乾出于某种目的从岛上撤回了军力，这让本想请战出击的钦陵未免有些遗憾。
<br/>　　现在，机会终于又来了，倭人自己作死先是打探大唐武器装备的机密，而后又想要暗算太上皇，如此举动彻底激怒了长安那位新皇，打算给倭人一个狠狠的教训。
<br/>　　所以钦陵此时的心情是十分激动的，迫不及待的想要展示一下自己的实力，不将倭岛杀个天翻地覆绝不收兵。
<br/>　　……
<br/>　　一直游弋在海参崴和登州之间的李震也同时收到了来自长安的命令，协助钦陵登岛，而后倭人片帆不得下海。
<br/>　　对于这样的命令李震自然是乐见其成，天天在大海上漂着也是极无聊的一件事情，可偏偏此前李承乾命令收兵，不得与倭人交战，这却让他这个前‘海狼’的头子觉得有些郁闷。
<br/>　　而现在有了这份命令，李震这个北海舰队的大都督终于可以得展所长，好好利用一下这些新到手的钢铁战舰了。
<br/>　　……
<br/>　　秦吉胜等人并不知道自己的阴谋已经败露，他们还在做着生擒太上皇李二的美梦，想着利用李二人换取倭国独立，想着利用李二换来更先进的武器装备，当然，这个不是要成品，而是要制造工艺。
<br/>　　釜山港因为倭人的到来已经封港，毕竟他们是在大唐犯了错的，需要在军队的押送下离开。
<br/>　　秦吉胜站在码头之上，回望着十余年前还属于被他们控制的土地，眼中射出不甘的光芒：我会回来，一定会回来，当我再次回来的时候，要成为这片大陆的主人！
<br/>　　心中发着毒誓，秦吉胜踏上了归国的战船，在大唐战舰的押送下缓缓离港。
<br/>　　一天，两天，距离大唐的国界越来越远，可让倭人奇怪的是那些大唐的战舰似乎并不想离开，就好像要把他们一直押送到家一样。
<br/>　　如此情况却是秦吉胜措手不及的，他还想着半路派人南下，去寻找李世民的踪迹，如果大唐军队一直跟着他们，这计划还如何实施。
<br/>　　于是秦吉胜便想着与大唐军队商量一下，询问一下唐军的目的。
<br/>　　但他很快就发现，唐军对他相要沟通的想法根本就是置之不理，不管他如何打出旗语想要联系，对方都没有任何的反应。
<br/>　　隐约间，秦吉胜心中有了一种不好的预感，开始变的坐力不安。
<br/>　　他的儿子见到老爹这个样子，也有些着急，在一个夜晚敲开了他的房门，有些担心的问道：“父亲，是不是出什么事情了？为什么你这几天显得如此紧张？”
<br/>　　“唐军的行为太诡异了，他们不应该跟着我们出海这么远，我总是觉得他们似乎有什么阴谋。”秦吉胜有些担心的看着窗外，夜色下，大唐战舰巨大的船身就好像一只巨兽一样笼罩着他。
<br/>　　“我觉得应该不会吧？如果他们真的想要干什么，完全可以把我们留在大唐，或者在刚刚出海的时候就把我们全部杀死，以大唐舰队的能力完全可以做到这一点啊。”
<br/>　　秦吉胜看着儿子一脸懵懂的一样，叹了口气说道：“你不懂，我是担心我们的计划已经被唐人知道了，如果那样的话，我们就要大祸临头了！”
