夏夕绾想了一下，当年这场婚约是她妈咪跟陆寒霆的妈咪订下的，那整个陆家，陆老夫人，陆司爵，柳招娣，都应该认识她妈咪。
<br/>
<br/>        只不过，他们都不知道她是林水瑶的女儿。
<br/>
<br/>        夏夕绾心头一动，感觉自己又找到了一条妈咪曾经走过的路，这条路上不知道有没有留下妈咪的脚印，她必须去。
<br/>
<br/>        “好，子羡，我跟你一起去。”夏夕绾点头答应。
<br/>
<br/>        这时蓝烟迅速出声道，“陆二少，你将绾绾带回陆家是可以的，但是你一定要保护好我家绾绾，你妈那个人…”
<br/>
<br/>        “蓝妈！”夏夕绾迅速打断了蓝烟。
<br/>
<br/>        蓝烟的话虽然没有说完，但是陆子羡已经察觉到了什么，他还是很了解自己的妈的，即使不问也能猜到大概是他离开帝都这段时间，他妈是找过夏夕绾什么麻烦了。
<br/>
<br/>        陆子羡俊美的眉眼里覆上一层清冷的寒气，他没有多说什么，只是低声道，“绾绾，相信我，我妈以后不会再找你麻烦的，这些年我一直住在外面，除了今天回一趟陆家，我们以后都不会见到他们的。”
<br/>
<br/>        蓝烟满意的点了点头，这婆媳关系是很要命的问题，陆子羡把话丢在这里，就是不会让夏夕绾再受委屈的意思。
<br/>
<br/>        夏夕绾不需要陆子羡给自己出头，上一次柳招娣也没有讨到好，她并不想将这件事告诉陆子羡，让他夹在中间为难。
<br/>
<br/>        不过现在她是陆子羡的未婚妻了，那她跟柳招娣的关系也变得不一样了，陆子羡这么一说，倒省去很多麻烦。
<br/>
<br/>        夏夕绾点了点头，“恩，好！”
<br/>
<br/>        ……
<br/>
<br/>        陆家。
<br/>
<br/>        因为陆子羡打电话回来过了，说是今晚会带着自己找回的未婚妻回家，所以柳招娣很早就亲自进了厨房，开始准备丰盛的晚餐。
<br/>
<br/>        柳招娣很开心，脸上都挂着笑，她看向客厅里的陆司爵，“司爵，你说林水瑶的女儿长大了是什么样了？”
<br/>
<br/>        陆司爵低眸看着手里的一份财经报，嗓音淡漠的答了一句，“应该不差。”
<br/>
<br/>        “一定是的，想当年林水瑶是何等风采，她的女儿肯定不差，而且林水瑶身份无比尊贵，凌驾众生之上，乃是来自于九州的…”
<br/>
<br/>        柳招娣的话戛然而止，她自己吓出了一身的汗，刚才她差点就将林水瑶的身份给说漏嘴了。
<br/>
<br/>        柳招娣看了一眼陆寒霆，陆寒霆已经来了，现在正慵懒的抵靠在沙发上，手里拿着手机，很明显并没有在意到她的话。
<br/>
<br/>        柳招娣这才放下心。
<br/>
<br/>        陆寒霆的确没听到柳招娣在说些什么，现在他拿着手机盘弄，心烦意乱，他想的都是夏家的寿宴。
<br/>
<br/>        有没有名门公子出席，夏夕绾有没有相亲，她是不是又有了新欢…这些问题盘旋在他的脑海里，挥之不去。
<br/>
<br/>        陆寒霆一点都不想去关注夏家的寿宴，但是他的心还是克制不住的飘到了夏夕绾的身上。
<br/>
<br/>        这时“叮铃”一声，陆宅的门铃响了。
<br/>
<br/>        柳招娣迅速跑了过去，“一定是子羡带着他的未婚妻回来了！”
<br/>
<br/>        柳招娣打开了大门。
