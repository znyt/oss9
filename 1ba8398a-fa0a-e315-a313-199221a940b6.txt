　　六道魔门，可是极为古老的门派。
<br/>　　比起九魔城，还要更加古老，不过在记载中，六道魔门早已经是被毁灭了。
<br/>　　没想到今日，居然还能碰到六道魔门的人。
<br/>　　“怎么，楚公子对洛夕姑娘，似乎有些兴趣？”
<br/>　　看着楚风眠的眼神，李长老脸色变的凝重了许多，认真的开口道。
<br/>　　“这洛夕姑娘，可不是一般人，楚公子若是对她有什么想法的话，还是打消主意吧。
<br/>　　楚风眠看着李长老，轻笑着开口道。
<br/>　　“怎么？这洛夕姑娘，似乎身份并不一般？”
<br/>　　“洛夕姑娘的身份，在通天阁中都是地位极高，所以楚公子还是万万不要去打她的主意了。”
<br/>　　李长老出言提醒道。
<br/>　　“不然的话，惹来的只能是灾祸。”
<br/>　　“灾祸？楚某纵横一生，还从来没有怕过什么灾祸。”
<br/>　　楚风眠肆意大笑几声，开口道。
<br/>　　“李长老无须在意，我对洛夕姑娘只是有些好奇罢了，至于其他的，并没什么。”
<br/>　　“但愿如此吧。”
<br/>　　李长老苦笑一声。
<br/>　　这些年轻人，尤其是还有着几分实力的年轻人，在看到洛夕之后，还从来没有可以无动于衷的。
<br/>　　若非李长老已经是年是以高，只怕也会被洛夕迷倒。
<br/>　　就在这交谈之际，看台上，已经是开始在拍卖第二件宝物。
<br/>　　这第二件宝物，是一把巨斧灵器，乃是一件地级中品灵器，品级虽然不高，但是毕竟古老，其中刻画的无数纹路，都要超越如今这个时代。
<br/>　　威力比起如今的地级上品灵器，还要强大许多，也算是难得一见的宝物了。
<br/>　　只是这种东西，对楚风眠来说，可没有任何意义。
<br/>　　他有着祖龙至尊剑，丰饶碑，人皇罐，寻常的天级灵器，他都不一定看得上。
<br/>　　更何况这区区地级灵器。
<br/>　　白萱也是如此。
<br/>　　白萱在这颜古龙帝洞府中，机缘巧合得到了一把地级极品的灵剑，白萱本就算是一尊剑修。
<br/>　　如今白萱的实力，已经是远超还没有来到虚空海之前，现在的她回到北邙学院，晋升核心弟子，几乎是板上钉钉的一件事。
<br/>　　第三件，第四件，第五件宝物……
<br/>　　一件件宝物，纷纷被拿出来拍卖，除了灵器，还有着不少诸天奇物，一些特殊材料，甚至是远古的功法。
<br/>　　这些东西，对楚风眠都没有多少用处，所以楚风眠便是一直没有出手购买。
<br/>　　楚风眠清楚，现在拍卖的东西，都是一些寻常之物。
<br/>　　真正有价值的宝物，应该都会放在最后才会拍卖。
<br/>　　不过就算是一些寻常之物，在外面，也都是无价之宝，根本无法买到的东西。
<br/>　　“第十二件宝物，通天剑气！”
<br/>　　灵尊真人开口一声，便是一尊侍女，手中拿着一枚透明晶体，走了出来。
<br/>　　在这透明晶体之中，隐隐约约可以看见在其中的三道剑气，这三道剑气，凌厉至极。
<br/>　　透过晶体，楚风眠都能跟感觉到，其中剑气的强大。
<br/>　　“这至少是一尊超越了生死境的大能，留下的剑气。”
<br/>　　楚风眠看到这通天剑气，低声自语道。
<br/>　　“通天剑气？通天阁拍卖会上，居然连这个东西都拿了出来。”
<br/>　　坐在一帮的李长老，都是似乎吓了一跳，失声说道。
<br/>　　“这是曾经通天阁的第一强者，通天剑尊，遗留下来的剑气，据说其中蕴含着通天剑尊的剑道真解，谁若是可以得到，有可能参悟通天剑尊的剑道。”
<br/>　　楚风眠心中暗自想到。
<br/>　　“参悟通天剑尊的剑道？这倒是远远没有那么容易，就凭这剑气，若是可以参悟的出剑道的话，至少也要是一尊剑道巅峰的强者，才能做到。”
<br/>　　“这种强者的实力根本不会在通天剑尊之下，根本不可能会浪费如此大的代价去参悟其他人的剑道。”
<br/>　　不过虽然无法参悟通天剑尊的剑道，但是若是一尊剑修得到，融入到自己的剑道之中，也是相当于受到了前辈的指点，有着极大的用处。
<br/>　　这通天剑气，对楚风眠无用，楚风眠的剑道，还要超越通天剑尊，这通天剑气对他没有任何作用。
<br/>　　但是对神力境，甚至是生死境的剑修来说，这通天剑气，都是极大的宝物。
<br/>　　通天剑气！
<br/>　　坐在楚风眠旁边的白萱，看到这通天剑气出现的一刻，眼神都是瞪得豆大，充满着羡慕，眼神中尽是渴望。
<br/>　　白萱所修的剑术，跟楚风眠不同，乃是一种万剑诀，收集天下万剑，成就大成的剑术。
<br/>　　通天剑气这种无上强者的剑气，正是对她最好的补品。
<br/>　　“白萱师姐，想要这通天剑气？”
<br/>　　楚风眠看着白萱的眼神，笑着开口道。
<br/>　　“白萱师姐若是想要的话，我就给你拍下如何？”
<br/>　　“这，你刚刚已经花费了那么多的财富，买下了建木树枝，在拍下这通天剑气的话，损失太大了吧。”
<br/>　　白萱听到楚风眠的话，先是一喜，随后眼神中变的暗淡了许多。
<br/>　　这通天剑气，可绝不是什么便宜的东西，足以引得在场无数剑修的争夺，价值极高。
<br/>　　白萱虽然想要买下这通天剑气来，但是以她的财富，只怕是远远做不到这一点。
<br/>　　“无妨，建木树枝的那点财富，不过是九牛一毛罢了，白萱师姐需要，我就拍下，也算是还白萱师姐那一万功勋值的人情了。”
<br/>　　楚风眠看着白萱，笑着说道，买下这通天剑气的财富，对楚风眠来说，也远远不算什么。
<br/>　　“第十二件拍品，通天剑气，起拍价，一万灵石！”
<br/>　　灵尊真人在看台上看，长喝一声。
<br/>　　“两万灵石！”
<br/>　　“三万灵石！”
<br/>　　“五万灵石！”
<br/>　　“十万灵石！”
<br/>　　下方无数剑修，纷纷出价，价格一路飙升。
<br/>　　这通天剑气对剑修的作用，比起无数诸天奇物都要有用的多，纷纷出言争抢。
<br/>　　“二十万灵石！”
<br/>　　在这无数加价的时刻，一道身影，却是猛然穿透出来，令在场突然一片寂静。
<br/>　　这声音，正是来自十号包厢，荒古门弟子，那古林的声音。
<br/>　　“可恶，又是荒古门的人！”
<br/>　　“此人争抢了无数宝物了，这通天剑气，居然也来争抢。”
<br/>　　“可惜我们无法跟此人对抗，荒古门，庞然大物，七大宗门，跟他们对抗，简直是在找死！”
<br/>　　下方的无数剑修，都在愤愤不平，却没任何一人敢再次出价。
