        童筱颖觉得自己这是要高兴好，还是要不高兴好？
<br/>
<br/>        瞧，她的丈夫是多么的信任她，又或者说，他的丈夫认为她是真的疯了。
<br/>
<br/>        又是两个小时过后，刚好凌晨，傅昼景一身疲惫地赶了回来。
<br/>
<br/>        童筱颖哭肿了双眼，就坐在客厅的沙发上等着他回来。
<br/>
<br/>        傅昼景一步步沉重地走到童筱颖跟前，然后蹲下身去，拉住童筱颖那只放在膝盖上的手，仰着脸，凝视着童筱颖那双呆滞的眼睛，深沉地说道：“筱颍，我会跟季思妍打官司，把小景夺过来，然后，我们一起抚养小景，好不好？”
<br/>
<br/>        所以，这就是他的打算？
<br/>
<br/>        他压根就没想过要跟她解释什么，反倒是想着如何解决这件事情。
<br/>
<br/>        没错，与其去计较过去，倒不如设法决定现在。
<br/>
<br/>        童筱颖的声音已经沙哑，双手仍旧在不停地颤抖，眼泪又不争气地夺眶而出。
<br/>
<br/>        她一张嘴，上下牙床也忍不住地打起了颤儿：“我要一个真相。”
<br/>
<br/>        “这三年来，我从未做过对不起你的事情。”傅昼景斩钉截铁地回答。
<br/>
<br/>        “那三年前呢？”童筱颖开始咄咄逼人。
<br/>
<br/>        傅昼景沉默不语了一会儿，不正面回答问题，反倒质问起她来：“那你呢？筱颍，三年前，你又是否做过对不起我的事？”
<br/>
<br/>        “你在说什么啊？”童筱颖身心一怔，有些难以置信地瞪着傅昼景。
<br/>
<br/>        傅昼景剑眉微蹙，索性破罐子破摔地接着说：“你在‘夜惑’里的那些事。”
<br/>
<br/>        “我在‘夜惑’里驻唱，就是对不起你了吗？”童筱颖有气无力地反问。
<br/>
<br/>        以前她为了这事藏着掖着，只是不希望他的自尊心受打击，现在，于她而言，他的自尊心似乎已经没有那么重要了。
<br/>
<br/>        “除了卖唱，你还卖过身吧！筱颍，如果不是你背着我卖身，我又岂会……”
<br/>
<br/>        “啪——”一道清脆的巴掌声，在空气里响了个通透。
<br/>
<br/>        童筱颖的一巴掌，掴在了傅昼景的脸颊上，阻止了他欲要说下去的话。
<br/>
<br/>        这一瞬间，傅昼景瞳孔紧缩。
<br/>
<br/>        “我、童筱颖、不是那种会为了钱而出卖自己身体的女人！”童筱颖顿字顿句地维护自己的清誉。
<br/>
<br/>        傅昼景紧接着不以为然地呵斥：“那你证明给我看！”
<br/>
<br/>        “证明？什么证明？”童筱颖精致的黛眉微微蹙起。
<br/>
<br/>        傅昼景忽然站起身来，抓住了童筱颖的双肩，直接将她压倒在了沙发上。
<br/>
<br/>        他的俊脸满是复杂的神情，微微眯起狭长的眸子，透着令人捉摸不透的深邃。
<br/>
<br/>        童筱颖一阵心悸，忽然间，忘记了要挣扎、要反抗，与他的目光灼灼相对。
<br/>
<br/>        “我……可以证明，去医院做检查。”童筱颖淡淡地开口道。
<br/>
<br/>        她宁可去医院，也不肯让他碰吗？
<br/>
<br/>        傅昼景俊雅的脸上，表情有些轻微地扭曲，他那双黑幽深邃的瞳孔慢慢收缩，目光渐渐地涣散，找不着焦点。
<br/>
<br/>        两人彼此沉默了许久，傅昼景才缓缓地开口说道：“筱颍，我们……不要闹了好不好？三年前，我是做过对不起你的事情。可是，在澳洲的这三年，除了你，我没碰过第二个女人。我承认，季思妍带着小景找上门来，我最近这段时间，白天有偷偷地去陪小景，但是，我的心，一直在你这儿。”
<br/>
<br/>        白天陪儿子，晚上陪老婆，如果这事不捅开，他就打算一直这么干下去吗？
<br/>
<br/>        童筱颖垂着眸子，默不吭声。
<br/>
<br/>        傅昼景俯下身来，一把环住童筱颖的身子，紧紧地搂着，在她耳畔低语哀求：“筱颍，原谅我，好不好？我去跟季思妍打官司，把小景的抚养权拿过来。我们俩一起抚养小景，我相信筱颍，将来会是一个好妈妈。”
<br/>
<br/>        “小景在楼上，已经睡着了。”童筱颖任由傅昼景抱着，一动也不动地低沉道。
<br/>
<br/>        她的心，到底有多大？
<br/>
<br/>        竟然能包容丈夫的出轨，连带着包容了那个私生子。
<br/>
<br/>        傅昼景为什么那么在乎小景，别人不懂，但她童筱颖此刻却懂了。
<br/>
<br/>        因为，同命相怜。
<br/>
<br/>        傅昼景能体会，那种作为私生子的痛苦。
<br/>
<br/>        毕竟是自己的亲生骨肉，所以他同情那孩子，怜悯那孩子。
<br/>
<br/>        而殊不知，这是两人彼此折磨彼此的开始。
<br/>
<br/>        后来，傅昼景为了争夺季小景的抚养权，真的去跟季思妍打了官司。
<br/>
<br/>        因为有她童筱颖的配合，所以傅昼景赢了这场官司，季小景归他傅昼景和童筱颖一起抚养，改名冷尚轩。
<br/>
<br/>        但是，这场原配与小三的战争，作为原配的童筱颖来说，她赢得并不开心。
