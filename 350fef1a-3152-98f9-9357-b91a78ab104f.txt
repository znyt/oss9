顾夫人十分讨厌叶翎，所以一进来劈头盖脸的就骂叶翎是小野种小贱人！
<br/>
<br/>        顾夜瑾抿了一下薄唇，然后不动声色的看了那个黑衣保镖一眼，黑衣保镖恭敬道，“夫人，少爷，我先下去了。”
<br/>
<br/>        黑衣保镖走了出去。
<br/>
<br/>        顾夜瑾单手抄裤兜里，俊美的五官上溢出了几分森寒不悦，“妈，保持你顾夫人的修养，不要开口就骂人，你知道的我不喜欢你这样。”
<br/>
<br/>        顾夫人穿着高贵华丽，保养的也很年轻，盘着一丝不苟的发，可以看得出来年轻时也是一个美人。
<br/>
<br/>        不过她的眉眼里藏着幽怨恨毒，一看就是那种常年得不到丈夫关怀和疼爱的女人。
<br/>
<br/>        见自家儿子袒护着叶翎，顾夫人迅速拽紧了轮椅的扶手，“夜瑾，我哪句话说错了，叶翎的母亲孙怡是你父亲的初恋情人，虽然当年她嫁入了高门叶家，她还勾搭你父亲，两个人在婚内双双出轨。”
<br/>
<br/>        “那一天孙怡和你父亲在酒店开房的时候被我和叶青缇当场捉奸，后来叶青缇开车带着孙怡回家的时候两个人就发生了车祸，全部丧生，而我也是在回家的途中出了车祸，失去了这双腿，成为了一个废人，这个轮椅我一坐就是十五年！”
<br/>
<br/>        “你父亲对孙怡那个贱人用情至深，当年不顾我的反对将叶翎那个小野种带回了家，你父亲这些年对你严厉苛责，却对叶翎那个小野种百般呵护，如果不是做过DNA亲子鉴定，我都怀疑叶翎是你父亲的种！”
<br/>
<br/>        “夜瑾，你别告诉我你没看出来你父亲那龌蹉阴暗的心思，叶翎如今长大了，美貌比她母亲孙怡还胜了一筹，你父亲早就盯上叶翎那个小贱人了…”
<br/>
<br/>        “够了！”这时顾夜瑾出声打断了顾夫人，他俊美的脸庞镀在昏暗里，看着暗黑而森冷，“妈，如果没有别的话，我让人送你回去。”
<br/>
<br/>        “夜瑾，你这是怎么了，当年孙怡和你父亲出轨，毁了两个家庭，叶翎就是你仇人的女儿，你看着她不觉得膈应么？”
<br/>
<br/>        “还是说叶翎给你灌了什么迷魂汤，不光你父亲盯着她，你也看上她了，你们父子还想要抢一个女人？”
<br/>
<br/>        顾夜瑾将薄唇抿成了一道森冷泛白的弧线，“妈，我说过了，我不爱她，我爱的是别人。”
<br/>
<br/>        顾夫人对自己未来的儿媳是有严格要求的，必须是门当户对，家世清白，受过高等教育的世家千金，还有这个人绝对不能是叶翎，或者是跟叶翎牵扯上任何关系的人。
<br/>
<br/>        叶翎，是顾夫人卡在心头的刺，碰一碰就觉得疼痛刺骨。
