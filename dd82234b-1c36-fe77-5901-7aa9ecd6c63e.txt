　　训练国王时所用的一周路程，楚涵边走边玩用了两天就到了王城，天运国并不大，国土富饶而已。
<br/>　　同时在路过的各个城镇中，天运国的人也都知道了即将打仗，人心有些恐慌。
<br/>　　但当楚涵到达的王城的时候，边境的消息就已经在远处一座座城池的传递中，国王亲自帅兵上战场的消息，很久就会人尽皆知。
<br/>　　对此毫不知情的王城，则是在被围困的水深火热之中。
<br/>　　王城居民出不来，被困在屋中等待着弹尽粮绝，国师还在夜夜笙歌，大将军跟摆设一样连对方打到城门了都没办法解决。
<br/>　　一干大臣急的跳脚，但却毫无办法，只能眼睁睁的看着城门一点点失守。
<br/>　　城门处的战争持续的如火如荼，楚涵到达的时候，一地的尸体，明成国过来攻打主城的人可不多，竟然打成这个德行！
<br/>　　无语的摇摇头，楚涵大摇大摆的走城门口走进去。
<br/>　　这显眼的样子，当场引发了两拨人马的目光注视。
<br/>　　战争也不是时时刻刻都在打，两方似乎说好了，每天早晚休战。
<br/>　　楚涵走入城门的时候，正是休战时，双方人马都在两边休整呢，愣是将偌大的城门空了出来。
<br/>　　于是楚涵就这么穿门而过，一个正眼都没给两方人。
<br/>　　两边的人都呆住了，不明白这忽然冒出来的人，是哪一方的。
<br/>　　这时候出乎意料的，先一步出言阻拦楚涵的，是明成国一方。
<br/>　　“站住！”这边的一个将军将长刀横在楚涵面前，满脸凶色：“再敢进一步，斩你首级！”
<br/>　　楚涵看了他一眼，而后伸出脚，绕过了这柄长刀，完全无视了他的话，继续往前走！
<br/>　　这一举动相当于挑衅，明成国的士兵当即围攻而来，那名将军更是一刀斩向楚涵身后。
<br/>　　楚涵修罗战斧拎在手上不禁紧了紧，他望了眼旁边的天运国士兵，一个个目光呆滞跟死了没什么两样。
<br/>　　这种军队，留着还不如埋了！
<br/>　　于是不分敌我，楚涵一斧斩出，浩大的月影斩凭空出现，刷刷刷的在地上带起极其锋利的声音，划出一道极长的黑色轨迹，呼啸着向着城门削了过去。
<br/>　　月影斩带起的风暴，不仅让明成国的人死伤惨重，也让天运国的这帮士兵血流成河！
<br/>　　无差别的伤亡，终于引起了两方将领的重视，大将军从远处赶来，大喝着让天运国的人集体攻击楚涵。
<br/>　　明成国的人同样如此，两军竟然默契的开始融为一体，务必要将楚涵干掉！
<br/>　　这就让楚涵更不爽了，给天运国的大将军及这一干士兵，直接套上了叛军的名头。
<br/>　　于是月影斩和时空月影斩交错出现，外加修罗战斧的不断挥动，楚涵一下子就大开杀戒，当着听闻风声赶来的天运国大批居民的面，一口气将这群人杀的尸横遍野。
<br/>　　很快，王城轰动，王国震荡。
<br/>　　一下子这里就围满了人，各个大臣全部赶来，只是等到他们来的时候，战斗已经进入了尾声。
<br/>　　一片的尸体，鲜血流淌将整个护城河染红，一片血色刺激着无数人的眼球。
<br/>　　明成国的人楚涵一个没放过，天运国的人，则是留下一干放弃挣扎的士兵，大将军则是被楚涵用一根绳子绑了起来，一手牵着绳子的一头，将其拖着走。
<br/>　　拖在地上，拖出了一地的血迹！
<br/>　　“城门开口了，收拾一下，王城恢复自由进出。”楚涵留下这句话，就拖起大将军往城内走。
<br/>　　楚涵的话让很多人都一震，大概所有人都以为他是来屠城的，却没想到他是来解决这场战争？
<br/>　　城门失守的几日战火，竟然被他一个人解决了？
<br/>　　很多人都不可置信，但亲眼所见不得不信！
<br/>　　强大的战力，在这一瞬间体现的淋漓尽致！
<br/>　　楚涵拖着嗷嗷大叫的大将军走着，路过的人群统统让开了一条道，就像是巡视一般，当着全体王城居民的面，把大将军当狗一样的溜。
<br/>　　大将军无法挣脱，楚涵用碾压式的战力将他彻底压制，他只能不断的谩骂，口中不停的吐出各种污秽之词。
<br/>　　关于这个大将军到底是不是内奸，楚涵根本不在意，反正不管是不是，这种毫无建树一点能力都没有的人，根本就不配待在这个位置。
<br/>　　而造成王城沦陷这么多天，此人也罪孽深重，楚涵杀起来毫无心理波动。
<br/>　　更何况……
<br/>　　他还急着回地球呐！
<br/>　　一路拖着大将军到达国师府，此时的国师府也听闻了消息，不再歌声阵阵，而是彻底的一片死寂。
<br/>　　国师坐在正堂上，一脸阴沉五指紧握的望着一步步走向自己的楚涵。
<br/>　　他看到了楚涵身后用一根绳子拖着的大将军，也看到了一路延伸至这里的血迹，更听说了城门口仅仅几分钟前发生的事。
<br/>　　又是这个男人！
<br/>　　这个在国王房间里出现的男人！
<br/>　　这个男人到底是谁，他为什么这么强！
<br/>　　“你……”国师率先开口，声音有些沙哑：“其实我们可以合作。”
<br/>　　说着，国师似乎有些激动，决定放弃一切以保性命：“你想要什么我都可以给你，天运国国王怎么样？你自己当国王，现任国王随你玩弄！”
<br/>　　楚涵望着他，像在看一个白·痴。
<br/>　　国师继续道：“还不够？我可以给你钱，以及明成国的……”
<br/>　　话音还没落下，忽的一阵钟声响起，当当当响了很久。
<br/>　　国师彻底愣住：“怎么可能？”
<br/>　　楚涵不懂这钟声是什么意思，但他感受到了当钟声消失后，整个王城的轰然爆发声，像是在欢呼喝彩。
<br/>　　紧接着，国师府前的道路上，一群人咆哮着跑过，声音亢奋无比激动。
<br/>　　“国王亲自领兵杀阵！打退了边境的明成国军队！”
<br/>　　“国王亲自领兵进攻！占领了明成国两座城池！”
<br/>　　两则消息，同时到达。
<br/>　　当楚涵破开城门的一刹那，就已经有人将准备好的消息传递到了王城，快马加鞭，用驰骋的速度，向全天运国的人分享这两则令人振奋的消息。
<br/>　　国师面如死灰，呆坐在位置上，他知道彻底败了，二十年的谋划，一瞬间就败了！
