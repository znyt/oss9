　　炼化雷灵珠中的能量，一举突破大乘期，其中菩提子起到了不可或缺的作用。
<br/>　　如果没有悟道菩提子，在关键时刻令慕清霄有所感悟，卡在合体九重巅峰的可能性会非常的大。
<br/>　　“法则，真是玄妙。”
<br/>　　领悟法则后，修士挥手间就能引动自然界的恐怖能量，劈山断岳，使江水逆流不在话下。
<br/>　　回过神来，慕清霄自储存空间中取出两颗透明的圆珠，正是土灵珠与雷灵珠，但其中的能量已经被炼化一空。
<br/>　　将其中的能量吸收炼化后，慕清霄并没有将之丢弃。
<br/>　　灵珠不知是如何形成，似乎能够吸收天地间精纯的自然能量，并且储存在其中。
<br/>　　如果将土灵珠安置在地心当中，过个三年五载在将之取出，不知会凝聚出何等庞大的能量。
<br/>　　若是用得好，简直是刷经验必备神器啊！
<br/>　　嘴角牵起一丝笑意，慕清霄直接将两颗毫无能量的灵珠装进储存空间中。
<br/>　　如果有机会，定要寻找一个能量充沛的地方，将灵珠安置，等过个几年在回来取，绝对能节省许多修炼时间。
<br/>　　等到时候，自己肯定已经用不上，但他的女人却能够派上用场，绝对能替她们节省一大把的时间。
<br/>　　回过神来，慕清霄视线在四周环顾一圈，此地荒山野林，没有村庄更没有城市，山脚下皆是绿葱葱一片。
<br/>　　既然雷灵珠已经炼化，那就没有回雷州的必要了。
<br/>　　系统发布的任务，他已经完成四分之一，接下来还有三颗灵珠，若是不加快速度，也不知要等到猴年马月。
<br/>　　况且，可不仅仅只有他在搜集灵珠，景天等人恐怕也在为灵珠四处奔波。
<br/>　　想到此处，慕清霄不再怠慢，取出地图仔细的看了一阵，他所在的位置是距离雷州百里外的一处无名山谷，荒无人烟。
<br/>　　接下来的目的地，自然是火鬼王所在的极乐世界。
<br/>　　火灵珠在火鬼王手中，水灵珠在紫萱手中，风灵珠在神界夕瑶手中。
<br/>　　三个女子，分别是鬼族，妖族，神族，若说谁最好对付，无疑是极乐世界中的火鬼王。
<br/>　　火鬼王的实力估计也就与古藤林中的古藤老怪相当，而且对付火鬼王，慕清霄比较有优势。
<br/>　　夕瑶是神族，暂且不说本身实力，神界强者可不少，更何况还有一个地仙境二重的天帝守着。
<br/>　　至于紫萱，女娲族后裔，是一个被宿命负累的可怜女人，饱受****煎熬三生三世之苦。
<br/>　　信息中说她活了三生三世，但具体活了多久，谁也不清楚，最重要的是，她将水灵珠修炼成了自己的内丹。
<br/>　　这种情况，慕清霄想要取水灵珠会变得非常麻烦，不仅要用武力压制，取出水灵珠更等于废了她。
<br/>　　思来想去，慕清霄微微摇头，后者又不是他的女人，自己貌似没必要想这么多。
<br/>　　到时候软的不行来硬的，水灵珠他是志在必得。
<br/>　　至于招惹蜀山的徐长卿，慕清霄根本就没有担忧过。
<br/>　　哪怕徐长卿资质不错，但一个连自己感情都不敢坦然面对的人，根本不足为惧。
<br/>　　在慕清霄看来，徐长卿就是一具傀儡，一个走在清微安排道路上的傀儡，成就注定有限。
<br/>　　神界夕瑶，女娲后裔紫萱排除，那么目标就只能是极乐世界中的火鬼王了。
<br/>　　做下决定后，慕清霄扫了一眼地图，地图上一处黑色圆点上，清晰的标记着酆都两字，正是极乐世界所在。
<br/>　　传闻，酆都是一座鬼城，联想到火鬼王，情况也就解释的通了。
<br/>　　慕容紫英曾经说过，黑色圆点代表禁区，是万分凶险之地，一般踏入其中的修士，都有去无回。
<br/>　　能够自由进出酆都，并且平安无事的，绝对都是拥有恐怖修为的修士。
<br/>　　要么是奉命行事，要么是去处理事情，但有一点，他们都不敢在鬼族的地盘上乱来。
<br/>　　在各大仙门强者不出的情况下，火鬼王就是巅峰强者，犹如悬在修士们头顶的利剑，随时都可能要了他们的命。
<br/>　　而且，仙门也不敢大规模的进攻鬼界，这里面牵扯太多，一方动八方齐动，至今没有哪一方打破平衡。
<br/>　　一旦有一方打破平衡，大战便会蔓延至六界。
<br/>　　除了某些野心极大的家伙，大部分都希望这样的平衡维持下去，最起码他们的地位能保住。
<br/>　　将复杂的情绪抛之脑后，扫了眼地图，确定酆都的位置后，慕清霄手一挥，空间犹如水面荡起一丝涟漪。
<br/>　　旋即，两只手同时伸出，往相反的方向一扯，一抹令人余悸的黑色就出现在视线中。
<br/>　　空间被双手渐渐拨开，一扇不知通往何处的门扉出现在面前，虚无的气息扑面而来，慕清霄却面色如常。
<br/>　　先前在雷州走的急，没有仔细打量，望着其中漆黑一片，慕清霄眼底闪过一抹好奇。
<br/>　　空间门开启的虚空隧道中，除了古朴沧桑的气息，他并没有察觉到一丝的危险，更没有传说中的虚空乱流。
<br/>　　不再多想，凝望着虚空，慕清霄迈步跨入其中。
<br/>　　待他整个身躯都莫入其中后，门扉渐渐合拢，最终化作黑点消失不见。
<br/>　　下一秒，百里外的上空轻颤，一道黑色裂缝出现在虚空，一只手伸出，慕清霄整个人自其中垮了出来。
<br/>　　仅仅瞬间便跨越百里，若是阴人绝对是阴死人不偿命。
<br/>　　站在上空，扫了眼四周，以他现在的修为，空间门能跨越的距离，似乎只有百里。
<br/>　　对于大乘期间修士而言，跨越百里的距离似乎并不远，但运用在战斗中，根本无解。
<br/>　　因为空间门一旦施展，几乎无迹可寻，若是不敌，直接施展空间门遁走，绝对是保命的最佳手段。
<br/>　　除非对方能够保证在虚空中不迷失方向，否则根本不敢跟进。
<br/>　　在慕清霄的印象中，拥有在虚空中移动的种族，似乎就只有斗破苍穹中的太虚古龙。
<br/>　　酆都的位置比较偏僻，扫了眼四周，脚下光秃秃一片，似乎是一座小型沙漠，慕清霄拨开虚空，再次钻入其中。
